# --description--

**目标：** 实现以下用户需求并通过所有测试以完成实验。

**用户需求：**

1. 您应该有一个名为 `pyramid` 的函数，该函数接受三个参数。
2. 第一个参数应该是一个字符串，表示要在金字塔中重复的图案字符。
3. 第二个参数应该是一个整数，表示金字塔中的行数。
4. 第三个参数应该是一个布尔值。
5. 当第三个参数为 `false` 时，`pyramid` 函数应该返回一个字符串，其中图案字符被重复并排列成一个顶点向上的金字塔。
6. 当第三个参数为 `true` 时，金字塔应该有一个顶点向下的形状。
7. 顶点行应该有一个单独的图案字符，其他每一行应该比前一行多两个图案字符。
8. 每一行都应该以足够多的空格开始，以使每行的中心字符位于同一列，并且每行末尾不应该有任何空格。
9. 金字塔应该以换行符开始和结束。

例如，调用 `pyramid("o", 4, false)` 应该给出这个输出：

```js

   o
  ooo
 ooooo
ooooooo

```

function pyramid(char, count, isInverted) {
    const rows = []
    for (let i = 1; i <= count; i++) {
        if (isInverted) {
            rows.unshift(" ".repeat(count - i) + char.repeat(2 * i - 1))
        } else {
            rows.push(" ".repeat(count - i) + char.repeat(2 * i - 1))
        }        
    }
    return "\n" + rows.join("\n") + "\n";
}

console.log(pyramid("#", 10, false))
