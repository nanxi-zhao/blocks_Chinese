# --description--

在这个实验中，你将创建一个将字符串转换为标题大小写的函数。标题大小写意味着每个单词的第一个字母都是大写的，而单词的其余部分都是小写的。

"Web Development Is Awesome" 就是一个标题大小写字符串的例子。

**目标**：完成下面的用户需求并通过所有测试以完成实验。

**用户需求：**

1. 你应该有一个名为 `titleCase` 的函数，它接受一个字符串作为参数。
1. `titleCase` 函数应该返回一个字符串，其中每个单词的第一个字母都是大写的，而单词的其余部分都是小写的。
1. `titleCase("I like to code")` 应该返回 `"I Like To Code"`。
1. `titleCase("javaScript is fun")` 应该返回 `"Javascript Is Fun"`。

# --solutions--

```js
function titleCase(str) {
  return str
    .split(' ')
    .map(word => word.charAt(0).toUpperCase() + word.substring(1).toLowerCase())
    .join(' ');
}
```
