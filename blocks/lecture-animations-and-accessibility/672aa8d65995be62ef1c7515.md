# --interactive--

CSS动画允许你在网页上创建动态、视觉上引人入胜的效果，而无需使用JavaScript或复杂的编程。它们提供了一种方式，可以在指定的持续时间内平滑地在不同样式之间转换元素。

核心来说，CSS动画由两个主要组件组成：`@keyframes`规则和animation属性。

`@keyframes`规则定义了动画的阶段和样式。它指定了元素在动画过程中各个时间点应该具有的样式。

这是一个示例：

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}
```

这个名为`slide-in`的`@keyframes`规则定义了一个将元素从左到右移动的动画。百分比代表动画的进度，`0%`是开始，`100%`是结束。

在你的`@keyframes`动画中的`translateX`函数控制着元素在动画进入视图时的水平位置。

要将此动画应用到元素上，你可以使用`animation`属性。这个示例还无限重复动画，这样你就可以看到它的效果：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sliding-element">你好，我滑进来了！</div>
```

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}

.sliding-element {
  animation: slide-in 2s ease-out infinite;
}
```

:::

这将`slide-in`动画应用到元素上，持续时间为2秒，缓动函数为`ease-out`。

`animation`属性实际上是几个单独属性的简写：

`animation-name`指定要使用的`@keyframes`规则。

`animation-duration`设置动画完成所需的时间。

`animation-timing-function`定义动画如何随时间进展 - 例如`ease`、`linear`、`ease-in-out`。

`animation-delay`指定动画开始前的延迟时间。

`animation-iteration-count`设置动画应该重复的次数。

`animation-direction`确定动画是应该向前播放、向后播放还是交替播放。

`animation-fill-mode`指定动画前后元素的样式。

`animation-play-state`允许你暂停和恢复动画。

你可以单独使用这些属性以获得更精确的控制：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="complex-animation">看我的颜色变化！</div>
```

```css
.complex-animation {
  animation-name: color-change;
  animation-duration: 3s;
  animation-timing-function: linear;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes color-change {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: green;
  }
}
```

:::

这创建了一个动画，使元素的背景色在红色、蓝色和绿色之间连续过渡。

CSS动画可以通过各种事件触发，比如悬停在元素上：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">悬停在我上面！</button>
```

```css
.button {
  background-color: blue;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: red;
}
```

:::

虽然这个示例使用了`transition`属性，这对于基本效果来说更简单，但它展示了CSS如何创建交互式、动画化的元素。

需要注意的是，虽然CSS动画很强大，但应该适度使用。过度使用动画可能导致性能不佳，并且可能分散注意力或对某些有无障碍需求的用户造成问题。始终考虑为喜欢较少运动的用户提供减少或禁用动画的选项。

CSS动画提供了一种无需依赖JavaScript就能创建引人入胜、交互式网页体验的方式。通过理解`@keyframes`和`animation`属性的原理，你可以以高性能和无障碍的方式让你的网页设计变得生动起来。
