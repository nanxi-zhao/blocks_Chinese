# --interactive--

与所有文本元素一样，你需要确保应用于文本输入框的样式是可访问的。这意味着字体需要适当大小，颜色需要与背景有足够的对比度。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form class="accessible-form">
  <label for="username">用户名</label>
  <input type="text" id="username" name="username" placeholder="输入你的用户名">
  <button type="submit">提交</button>
</form>

```

```css
body {
  background-color: #f9fafb;
  color: #222;
  padding: 2rem;
}

.accessible-form {
  max-width: 320px;
  margin: 0 auto;
}

label {
  display: block;
  font-weight: 600;
  margin-bottom: 0.5rem;
}

input[type="text"] {
  width: 100%;
  padding: 0.6rem 0.8rem;
  font-size: 1rem;
  border: 2px solid #555;
  border-radius: 4px;
  background-color: #fff;
  color: #111;
}

input[type="text"]:focus {
  outline: 3px solid #1e90ff;
  border-color: #1e90ff;
}

button {
  margin-top: 1rem;
  padding: 0.6rem 1rem;
  font-size: 1rem;
  background-color: #1e90ff;
  color: #fff;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

button:hover,
button:focus {
  background-color: #187bcd;
}

```

:::

另一件需要记住的事情是，你应该允许用户修改输入。例如，如果是文本区域，你不应该移除调整大小的能力。当用户缩放页面时，输入框也应该相应地缩放。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form class="accessible-form">
  <label for="message">你的消息</label>
  <textarea id="message" name="message" placeholder="在这里输入你的消息..."></textarea>
  <button type="submit">发送</button>
</form>

```

```css
body {
  background-color: #f9fafb;
  color: #222;

}

.accessible-form {
  max-width: 480px;
  margin: 0 auto;
}

label {
  display: block;
  font-weight: 600;
  margin-bottom: 0.5rem;
}

textarea {
  width: 100%;
  min-height: 120px;
  padding: 0.8rem;
  font-size: 1rem;
  border: 2px solid #555;
  border-radius: 4px;
  background-color: #fff;
  color: #111;
  resize: both; 
  box-sizing: border-box; 
}

textarea::placeholder {
  color: #555;
  opacity: 1;
}

textarea:focus {
  outline: 3px solid #1e90ff;
  border-color: #1e90ff;
```

输入元素也是可聚焦的。当你编辑样式时，应该注意在元素聚焦时保持明显的指示器，比如粗边框。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form class="accessible-form">
  <label for="name">全名</label>
  <input type="text" id="name" name="name" placeholder="输入你的全名">

  <label for="email">邮箱地址</label>
  <input type="email" id="email" name="email" placeholder="you@example.com">

  <button type="submit">提交</button>
</form>

```

此外还要考虑错误状态。当用户的文本未通过输入验证时，会有视觉指示器显示存在问题。

要使错误消息动态更新，你需要使用JavaScript，这将在未来的课程中学习。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form class="accessible-form">
  <label for="email">邮箱地址</label>
  <input 
    type="email" 
    id="email" 
    name="email" 
    placeholder="you@example.com" 
    aria-describedby="email-error"
  >
  <p id="email-error" class="error-message">
    请输入有效的邮箱地址。
  </p>

  <button type="submit">提交</button>
</form>

<script src="index.js"></script>
```

当你为元素添加个人风格时，要确保错误状态仍然明显，并且看起来与聚焦指示器不相似。考虑到所有这些，你现在可以创建引人入胜的用户表单了。
