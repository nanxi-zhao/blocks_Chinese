# --description--

在 Python 中，类（class）和对象（object）是紧密配合来组织与管理数据的：你用类来定义共享的行为与结构，然后通过类创建具体的对象来使用这些行为。

通俗地说，类就像一张蓝图或模板，用来生成具有相同结构与行为的一组对象。

下面我们看一下如何定义类并用类创建对象。

要创建类，使用 `class` 关键字后接类名和冒号，然后在类内部可以定义初始化器（initializer，也就是 `__init__`）、属性和方法等。

属性类似于类内部的变量，用来保存数据；方法是定义在类中的函数，用来让对象执行某些操作或改变自身状态。
