# --description--

在之前的课程里你已经了解过属性是属于对象的变量，用来表示对象的状态或行为。

例如，`Car` 类通常会包含 `brand` 与 `model` 作为属性：

```python
class Car: 
    def __init__(self, brand, model): 
        self.brand = brand 
        self.model = model 

my_car = Car('Lamborghini', 'Gallardo') 
print(my_car.brand) # Lamborghini 
print(my_car.model) # Gallardo 
```

但有时你在运行时才知道需要哪些属性：比如脚本从用户输入或配置文件接收属性名，这些属性无法提前写死在代码中。

这就需要动态处理属性 —— 你可以使用属性名的变量来访问、修改、检查甚至删除属性，从而让程序能灵活应对运行时的数据或输入。

Python 提供了四个便捷的内置函数来动态操作对象属性：`getattr()`、`setattr()`、`hasattr()` 和 `delattr()`。

它们分别用于读取、创建/更新、检测与删除属性。下面分别示例说明它们的用法：

`getattr()` 允许在运行时读取一个对象的属性（当你不知道属性名时）。如果属性不存在，会抛出 `AttributeError`，或者如果提供了默认值则返回该默认值。

用法格式为 `getattr(object, attribute_name, default_value)`（示例代码保持不变以便直接运行）。

当属性名来自变量（例如用户输入或文件）时，`getattr()` 特别有用，因为这时无法使用 `object.attribute_name` 的静态语法。

此外，若要查看对象拥有的所有属性，可以使用 `dir()` 返回属性名列表，并结合过滤忽略以双下划线开头的方法与可调用的方法来列出实际属性（示例代码保持不变）。

`setattr(object, attribute_name, value)` 可以动态创建或更新属性，常用于根据运行时读取到的配置数据设置对象属性（示例代码保持不变）。

`hasattr(object, attribute_name)` 用于在操作或删除属性前先检查其是否存在，返回布尔值，通常是良好的防御性编程习惯。

`delattr(object, attribute_name)` 则用于动态删除属性，适合在保存或清理对象前移除敏感或临时属性（示例代码保持不变）。

结合这些工具，你可以在运行时灵活管理对象的属性，使代码更具适应性和鲁棒性。
