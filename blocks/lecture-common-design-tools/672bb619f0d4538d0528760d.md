# --description--

设计是每个企业级 Web 应用的基础。因此，设计师和开发者需要紧密合作，才能打造既以用户为中心又美观实用的界面。

开发者熟悉常见的设计工具可以更好地利用设计师的成果。大多数工具在矢量设计和原型制作方面表现突出。

矢量设计是使用数学公式来定义线条、形状与颜色的数字绘图方式；而原型制作则是创建产品或界面交互模型的过程。

下面介绍几款开发者应该了解的常用设计工具。

Figma 是一款非常常见且重要的设计工具，专注于用户界面与用户体验（UI/UX）设计。它基于云端，方便设计与开发团队随时协作，内置功能包括：

- 矢量化设计
- 自动布局
- 评论与反馈系统
- 历史版本记录
- 实时协作
- 设计系统等功能

你可以通过网页版开始使用 Figma，也可以下载桌面应用。Figma 的免费套餐功能已相当丰富，很多场景无需付费即可完成工作。

Sketch 是另一款开发者应当熟悉的设计工具，与 Figma 一样以矢量设计为主，主要用于 UI/UX 设计。

Sketch 因界面直观且操作简洁而受欢迎，便于快速制作原型，也常被设计师用于创建界面、图标与网页布局。

不过 Sketch 的限制在于缺乏原生云端界面，并且仅在 macOS 上可用。

Adobe XD 也是一款用于 UI/UX 的矢量设计与原型工具，擅长与 Photoshop、Illustrator、After Effects 等 Adobe 系列应用无缝集成，方便实现交互原型与动画工作流。

Adobe XD 同时支持 Windows 与 macOS，并提供云端协作功能；通常使用其桌面应用能获得更佳体验。

另一个值得一提的工具是 Canva，它适合用于制作海报、封面、演示文稿、短视频等多种视觉内容。Canva 的上手门槛低、模板丰富，适合入门与快速产出专业视觉素材。

Canva 还支持网页界面设计并具备与团队协作的能力，可通过网页版或 Android 应用访问。

其他常见且值得了解的设计工具还包括 Framer、InVision、Adobe Photoshop、Adobe Illustrator 与 Miro。
