# --interactive--

通配选择器（`*`）是一种特殊的 CSS 选择器，它会匹配文档中的所有元素。

它常用于对页面上所有元素应用统一样式，例如在重置或规范化样式时很有用，以减少不同浏览器间的差异。

通配选择器既可以在特定上下文中选择所有元素，也可以全局应用于整个文档。

下面演示如何使用通配选择器将整个 HTML 文档的 `margin` 和 `padding` 设为 0：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::


在这个示例中，`*` 将所有元素的 margin 和 padding 重置为 0，这是常见的 CSS 重置技巧。

通配选择器的 specificity 值最低，对四部分数值的贡献为 `(0, 0, 0, 0)`。

因此，任何其他选择器（类型、类、ID、内联样式等）都能覆盖由 `*` 设置的样式。

让我们看看下面的 HTML 和 CSS 示例：

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::


示例中，`*` 将文本颜色设为 `blue`，但随后 `p` 类型选择器将 `p` 元素改为 `red`，类选择器 `.highlight` 将文本改为 `green`，而 ID 选择器 `#unique`（优先级最高）最终把颜色设为 `purple`。
