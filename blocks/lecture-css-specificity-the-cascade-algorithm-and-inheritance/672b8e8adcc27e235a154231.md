# --interactive--

类型选择器（又称元素选择器）根据标签名来选择元素。

它们是 CSS 的基础，用来对某类 HTML 元素的所有实例应用样式。

类型选择器用法简单，直接写元素的标签名即可。

下面示例演示如何针对页面上所有段落元素使用类型选择器：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::


在这个示例中，所有 `p` 元素的文本颜色会变为 `blue`。

与其它选择器相比，类型选择器的 specificity 较低，其值为 `(0, 0, 0, 1)`。

这表示类型选择器会被类选择器、ID 选择器和内联样式覆盖，但在没有更高优先级规则时仍然生效。


下面看一个示例：类选择器会覆盖类型选择器的样式。

示例包含两个带类名的段落：

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::


两个段落元素都拥有类名 `para`。

在 CSS 中，类型选择器作用于所有段落，而类选择器只作用于带有 `para` 类的元素。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::


具有 `para` 类的所有段落会变为红色（`red`），而不是蓝色（`blue`），因为类选择器的优先级高于类型选择器。
