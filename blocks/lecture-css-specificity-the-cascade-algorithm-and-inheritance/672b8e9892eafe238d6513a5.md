# --interactive--

类选择器是 CSS 的重要组成部分，允许开发者用相同的类名为多个元素应用统一样式。

它们非常灵活，适合在整个网站中高效地复用样式。

类选择器以一个点号 (`.`) 加类名来表示，可用于任何 HTML 元素。

下面是一个使用类选择器的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::


在该示例中，所有带有 `highlight` 类的元素会把文本颜色设为 `green`。

类选择器的优先级高于类型选择器，但低于 ID 选择器和内联样式。

类选择器的 specificity 为 `(0, 0, 1, 0)`，这意味着它可以覆盖类型选择器，但会被 ID 或内联样式覆盖。


类选择器可以与其他选择器组合，从而形成更具体的规则。

下面示例把段落类型选择器与类选择器结合：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::


此规则只会作用于同时是 `p` 且带有 `bold-text` 类的元素，使其文本加粗。
