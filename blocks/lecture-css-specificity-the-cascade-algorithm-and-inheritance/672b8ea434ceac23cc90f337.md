# --interactive--

ID 选择器是 CSS 中最强力的选择器之一，用于为具有唯一标识符的特定元素设置样式。

它们非常适合针对需要单独样式的元素进行精确定位。

ID 选择器以 `#` 开头，后接 ID 名。一个 HTML 文档中通常不应有重复的 ID。

下面是一个使用 ID 选择器的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::


在示例中，ID 为 `unique` 的元素会把文本颜色设为 `purple`。

ID 选择器的 specificity 很高，超过类型选择器和类选择器，但低于内联样式。ID 的 specificity 为 `(0, 1, 0, 0)`。

因此，ID 选择器可以覆盖类和类型选择器，但会被内联样式覆盖。
