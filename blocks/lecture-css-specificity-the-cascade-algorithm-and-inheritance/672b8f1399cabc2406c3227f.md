# --interactive--

CSS 中的 `!important` 关键字用于提升某条样式规则的优先级，使其能够覆盖同一属性的其它声明。

当使用 `!important` 时，浏览器会优先应用该样式，而不考虑其它选择器的 specificity。

例如，假设你有一个带内联样式的段落元素：

:::interactive_editor

```html
<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

:::


在这个示例中，段落元素有 `lightblue` 的背景色和 `black` 的文本色。

在 CSS 文件中，我们对该段落应用了如下样式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black;
  color: white;
}
```

:::


因为内联样式的优先级高于类、ID 和类型选择器，因此文件中设置的黑色背景和白色文字不会生效。

要覆盖这些内联样式，可以使用 `!important`，例如：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black !important;
  color: white !important;
}
```

:::


`!important` 写在属性值后、分号前。

使用后，该规则会生效并覆盖其它冲突规则（即便那些规则的 specificity 更高或是内联样式）。

`!important` 并不会改变选择器本身的 specificity，它只是强制让带 `!important` 的规则被应用。

一个常见的合理场景是：当你无法修改第三方库或框架的 CSS 时，用 `!important` 临时覆盖其样式。

但过度使用 `!important` 会破坏 CSS 的自然层叠逻辑，增加维护和调试难度，因此应尽量少用。
