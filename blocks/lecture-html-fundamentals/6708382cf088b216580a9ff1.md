# --interactive--

`id` 属性用于为 HTML 元素添加唯一标识符。

下面是一个带有 `id` 为 `title` 的 `h1` 元素示例。

在 `h1` 元素下面添加一个 `id` 为 `"subtitle"` 的 `h2` 元素，随意填写文本并在预览窗口查看变化。

:::interactive_editor

```html
<h1 id="title">电影评论页面</h1>

```

:::

你可以在 JavaScript 或 CSS 中通过 `id` 名称（如 `title`）来引用该元素。下面是一个 CSS 示例，引用 `id` 为 `title` 来把文本颜色设置为红色：

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>示例页面</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <h1 id="title">电影评论页面</h1>
  </body>
</html>
```

```css
#title {
  color: red;
}
```

:::

前面的井号（`#`）表示你在选择一个 `id`。`id` 名称在文档中应保持唯一，且不能包含空格。下面是错误示例：

```html
<h1 id="main heading">Main heading</h1>
```

浏览器会把空格也当作 `id` 名的一部分，这会给样式和脚本选择带来问题。`id` 值只能包含字母、数字、下划线或连字符。

与 `id` 不同，`class` 的值可以重复且允许包含空格，用于为元素添加多个类名。

下面是为 `div` 应用类名 `box` 的示例：

```html
<div class="box"></div>
```

如果想给元素添加多个类名，可用空格分隔，例如：

```html
<div class="box red-box"></div>
```

再配合样式表示例：

```css
.box {
  width: 100px;
  height: 100px;
}

.red-box {
  background-color: red;
}

.blue-box {
  background-color: blue;
}
```

总结：什么时候用 `id` 而不是 `class`？如果你想针对单个、特定的元素，使用 `id`；如果想把样式应用到多个元素，使用 `class`。

## --answers--

当你想要一个只应用于一个元素的唯一标识符时，使用 `class`。

### --feedback--

记住，`id` 名称是唯一的，每个 HTML 文档只能使用一次。
