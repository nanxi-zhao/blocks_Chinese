# --interactive--

HTML 实体（也叫字符引用）是一段用来表示 HTML 中保留字符的字符序列。

举个例子，如果你想在页面上显示文本 `This is an <img/> element`，直接把 `<img />` 放进去会被解析为真正的图片元素（即使添加了 `src` 和 `alt`），这并不是我们想要的显示结果。

:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

当 HTML 解析器看到小于号（`<`）后跟标签名时，会把它当作元素处理，所以不会原样显示 `<img/>`。

为了解决这个问题，可以使用 HTML 实体。下面示例用小于号（`<`）和大于号（`>`）的实体表示法，这样页面会显示 `This is an <img/> element`。

试着在段落元素下面添加 `&lt;p&gt;learning is fun&lt;/p&gt;`，你应该会看到页面上显示 `<p>learning is fun</p>`。

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

这类引用称为“命名字符引用”（named character references），以 `&` 开始，以 `;` 结束。使用命名引用可以避免解析器把它当作实际的 HTML 元素。

另一种类型是十进制数字引用（decimal numeric reference）：以 `#` 和十进制数字跟随并以 `;` 结束。

下面是用十进制数字引用表示小于号的示例。

在编辑器中修改代码可以看到不同的符号。比如可以使用 `&#169;` 表示版权符号，使用 `&#174;` 表示商标符号。

:::interactive_editor

```html
&#60;
```

:::

最后一种是十六进制数字引用（hexadecimal numeric reference）：以 `#x` 开头，后面跟十六进制数字并以 `;` 结束。

下面是使用十六进制数字引用表示小于号的示例。

在编辑器中修改代码可以看到不同符号，例如 `&#x20AC;` 表示欧元符号，`&#x03A9;` 表示希腊大写欧米伽符号。

:::interactive_editor

```html
&#x3C;
```

:::
