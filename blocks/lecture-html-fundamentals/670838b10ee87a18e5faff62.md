# --interactive--

`script` 元素用于在 HTML 中嵌入或引用可执行代码，通常用于为页面添加交互性。常见应用有交互式游戏、图片轮播、以及对用户输入进行实时验证等。

下面是一个在 HTML 文档中使用 `script` 元素的示例。将 `alert("Welcome to MasterPuti");` 前面的 `//` 删除并刷新页面，你应该会看到弹出提示。

:::interactive_editor

```html
<body>
  <script>
    // alert("Welcome to MasterPuti");
  </script>
</body>
```

:::

虽然技术上可以把所有 JavaScript 代码直接写在 `script` 标签内，但最佳实践是把 JavaScript 放在外部文件，再通过 `src` 属性引入，这样可以实现关注点分离（separation of concerns），让 HTML 与 JavaScript 各自负责自己的任务，便于维护。示例：

```html
<script src="path-to-javascript-file.js"></script>
```

`src` 属性用于指定外部 JavaScript 文件的位置。
