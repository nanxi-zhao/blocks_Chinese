# --description--

Open Graph（OG）协议让你能控制网站内容在社交媒体（如 Facebook、LinkedIn 等）上的展示效果。通过设置这些 OG 属性，你可以提升内容在用户动态中的吸引力，从而提高点击率。通常在 HTML 的 `head` 区通过一组 `meta` 元素来设置这些属性。

第一个重要的 OG 属性是 `title`。示例如下，为 MasterPuti 首页设置 OG 标题：

```html
<meta content="MasterPuti.org" property="og:title" />
```

`property` 属性应指定为 `og:title`，`content` 属性则写上希望在社交媒体上显示的标题。

下一个重要的属性是 `type`：

```html
<meta property="og:type" content="website" />
```

`type` 用于表明被分享内容的类型，例如文章（article）、网站（website）、视频（video）或音乐（music）等。

第三个重要属性是 `image`：

```html
<meta
  content="https://cdn.MasterPuti.org/platform/universal/fcc_meta_1920X1080-indigo.png"
  property="og:image"
/>
```

示例中使用的图片指向 MasterPuti 的 logo。OG 图片应保证高质量并符合平台推荐的尺寸与比例。多数社交平台会提供图片尺寸规范，例如 Facebook 建议：

"为了在高分辨率设备上获得最佳展示，使用至少 1200×630 像素的图片；最低推荐使用 600×315 像素以保证较大的图片显示效果。"

第四个重要属性是 `url`：

```html
<meta property="og:url" content="https://www.MasterPuti.org" />
```

除此之外还有许多可选 OG 属性，例如 `description`、`audio`、`video` 与 `locale` 等，但 `url`、`image`、`type` 与 `title` 是最常见且最重要的。

这些 OG 属性如何影响 SEO 呢？当内容在社交媒体上被分享时，精心设置的 OG 元数据能提升在用户动态中的展示效果，从而提高点击率。更高的点击率可能向搜索引擎传递出内容相关且受欢迎的信号，有助于间接提升搜索表现。
