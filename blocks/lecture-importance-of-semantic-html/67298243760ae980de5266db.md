# --description--


“语义”是指词语或短语的含义。把 HTML 看成一种语言也很合适：HTML 元素同样携带语义信息。就像一篇文章会有标题、图片、强调文本等，HTML 文档也有相应的结构。

当我们说一个元素有“语义”时，意思是它传递了特定的含义。例如，`p` 元素语义上表示一段文本：

```html
<p>
  Let me tell you about my fantastic holiday in Paris.
  I saw the impressive Eiffel Tower up close!
</p>
```

大多数 HTML 元素都是有语义的；`div` 是少数不具语义的容器之一。那为什么语义重要呢？

首先，语义化的 HTML 能让使用屏幕阅读器等辅助技术的用户有更好的体验；其次，它有助于搜索引擎正确理解页面内容，从而提升搜索排名（也就是 SEO）。

最后，语义化还能让开发体验更顺畅：当你看到 `nav`、`header`、`main` 等元素名称时，就能快速定位页面结构，而不用在一大堆 `div` 里苦寻导航区域。

在本节中，我们会继续了解这些概念、如何使用语义元素，以及为什么语义化 HTML 如此重要。你会发现，写语义化的 HTML 会让你的页面对用户和开发者都更友好！
