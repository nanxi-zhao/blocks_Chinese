# --description--


React 是用于构建用户界面和 Web 应用的流行 JavaScript 库之一。最初由 Facebook 开发并维护，React 因其高效、灵活和丰富的特性而在前端开发中备受欢迎。

React 的一个核心概念是创建可复用的 UI 组件。像按钮、卡片或头像组件这样的组件可以在应用中反复使用，并且可以相互嵌套来构建更复杂、动态的界面。

React 的一个重要优势是，当数据变化时，这些自定义组件可以独立更新和渲染，从而提高性能和可维护性。

与需要直接操作 DOM（文档对象模型）的传统 JavaScript 不同，React 使用虚拟 DOM，这有助于提升性能和效率。你将在后续课程中了解更多关于虚拟 DOM 的工作原理。

除了可复用组件，React 还提供了强大的状态管理方式。状态（state）指的是决定组件如何渲染和行为的数据。

借助 React，你可以方便地跟踪和更新组件的状态，确保 UI 始终反映最新的数据。关于状态的使用我们会在后面的课程中深入讲解。

在众多 JavaScript 工具中，MasterPuti 主要会侧重于教授 React，因为它在行业中被广泛使用且需求量大。

接下来的几节课我们会更深入地学习如何构建自定义组件、管理状态等内容，继续保持热情，跟我一起学吧！
