# --description--


与小型的纯 HTML/CSS/JavaScript 项目相比，启动一个新的 React 项目需要额外做一些配置以确保一切能够正确运行。与其自己手动逐项配置，不如使用现有工具来帮你快速搭建。

其中一个很受欢迎的项目搭建工具是 Vite。Vite 在法语里有“快速”的意思，它是为现代 Web 项目提供更快开发体验的构建工具。Vite 可以与 React 一起使用，也支持 Vue、Svelte 等框架，甚至可以用于纯 JavaScript 项目。

要用 Vite 创建新项目，需要在命令行中运行相应命令。如果你使用的是 Windows，可以用命令提示符或 PowerShell；如果是 Mac，则可使用终端（Terminal）。

打开命令行后，你可以运行以下命令：

```bash
npm create vite@latest my-react-app -- --template react
```

该命令会使用 Vite 的 React 模板创建一个名为 `my-react-app` 的新 React 项目。

随后你可以打开该项目，查看 Vite 为你生成的 React 项目样板代码。

Vite 的优点之一是它只会提供启动项目所需的最基本文件，让你能更快上手。

要运行项目，先安装依赖，命令如下：

```bash
cd my-react-app
npm install
```

`cd my-react-app` 用于切换到项目目录。

`npm install` 会安装项目运行所需的依赖，比如 React、ReactDOM 以及其他第三方包。

运行 `npm install` 时，npm 会读取项目目录下的 `package.json` 并安装其中列出的所有依赖，从而确保项目所需的组件完整。

`package.json` 是项目的重要配置文件，包含项目名称、版本、依赖项以及脚本等元数据，有助于管理项目。

安装依赖后，你会在项目中看到一个 `node_modules` 文件夹。

`node_modules` 存放了项目所需的所有包及其依赖的实际代码。

要运行开发服务器，请执行 `npm run dev`，然后在浏览器中打开 `http://localhost:5173/`。

你会看到 Vite 提供的起始模板。

如果需要停止本地服务器，请在命令行按 `CTRL + C`。

要查看模板代码，可以进入项目的 `src` 目录，打开 `App.jsx` 文件。

从那里开始修改文件，保存后即可在浏览器中看到实时更新。

现在，你已经准备好开始构建自己的 React 应用啦，继续动手实践吧！
