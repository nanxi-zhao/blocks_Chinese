# --description--


在之前的课程中，你已经学习了 JavaScript 中的导入（import）和导出（export）。本节我们来看看在 React 中如何导入和导出组件。

在示例中，我们有一个名为 `Cat` 的组件，放在 `Cat.jsx` 文件中。之所以用 `.jsx` 后缀，是因为该文件主要包含 JSX 代码。

这个 `Cat` 组件会返回一段 JSX 标记，展示一只名为 Mr. Whiskers 的猫的标题和图片：

```jsx
function Cat() {
  return (
    <div className="card">
      <h2>Mr. Whiskers</h2>
      <img
        src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/running-cats.jpg"
        alt="Tuxedo cats running on dirt ground."
      />
    </div>
  );
}
```

If we want to use our `Cat` component in another file, we need to first export it like this:

```jsx
function Cat() {
  return (
    <div className="card">
      <h2>Mr. Whiskers</h2>
      <img
        src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/running-cats.jpg"
        alt="Tuxedo cats running on dirt ground."
      />
    </div>
  );
}

export default Cat;
```


这里我们使用了 `default` 关键字，因为这是模块的默认导出。你也可以在组件定义的同一行直接导出组件，像下面这样：

```jsx
export default function Cat() {
  return (
    <div className="card">
      <h2>Mr. Whiskers</h2>
      <img
        src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/running-cats.jpg"
        alt="Tuxedo cats running on dirt ground."
      />
    </div>
  );
}
```


你可以在父组件文件中导入子组件，或在根组件文件中导入。本例中我们将在根组件中导入 `Cat` 组件。

每个 React 项目通常都有一个顶层组件，通常称为 `App.jsx`：

```jsx
export default function App() {
  return (
    // return component here
  );
}
```


该文件通常位于项目的 `src` 目录中。关于常见的项目结构我们会在后续课程中进一步讲解。

要在根组件 `App` 内使用 `Cat` 组件，可以这样导入：

```jsx
import Cat from "./Cat";

export default function App() {
  return (
    // return component here
  );
}
```

Now, you can return the `Cat` component inside the `App` component like this:

```jsx
import Cat from "./Cat";

export default function App() {
  return (
    <Cat />
  );
}
```


随着你不断构建自己的 React 项目，你会越来越熟练地组织组件、在需要的地方导入它们，并通过组合这些模块化组件来创建更复杂的 UI。继续实践，你会做得越来越好！
