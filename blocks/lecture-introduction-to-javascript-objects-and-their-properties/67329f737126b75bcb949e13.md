# --interactive--

在 JavaScript 中，对象是一个基础的数据结构，用来存放和组织相关的数据与功能。

可以把对象想象成一个装资料的抽屉柜，里面有不同的文件夹和文件，每一项信息都称为属性（property），由名称（或键）和值组成。

```js
const exampleObject = {
  propertyName: value;
}
```

对象非常灵活，构成了 JavaScript 的核心。事实上，JavaScript 中几乎所有东西都可以被视为对象或以对象的方式使用，这包括数组、函数，甚至在某些情形下的原始类型（比如字符串和数字）。

这种以对象为中心的设计使 JavaScript 非常强大且富有表现力。下面看一个创建对象的示例：

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};
```

在这个示例中，我们创建了一个名为 `person` 的对象，它有三个属性：`name`、`age` 和 `city`。每个属性通过冒号分隔名称与值。

接下来我们看看如何访问这些属性。访问对象属性主要有两种方式：点（dot）表示法和括号（bracket）表示法。

点表示法是最常见也最直观的方式，语法如下：

```js
objectName.propertyName
```

用点表示法访问 `person` 对象的示例：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person.age);   // 30
```

:::

当你知道属性名且该名称是有效的 JavaScript 标识符（不以数字开头且不包含特殊字符或空格）时，点表示法简洁可读，是优选方式。

而括号表示法允许你在方括号中使用字符串来访问属性，示例如下：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person["name"]); // Alice
console.log(person["age"]); //  30
```

:::

括号表示法比点表示法更灵活，因为它可以处理那些不是有效标识符的属性名，例如带空格或以数字开头的属性：

:::interactive_editor

```js
const oddObject = {
  "1stProperty": "Hello",
  "property with spaces": "World"
};

console.log(oddObject["1stProperty"]);  // Hello
console.log(oddObject["property with spaces"]);  // World
```

:::

另一个优点是，括号表示法可以使用变量动态访问属性：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "Wonderland"
};

let propertyName = "city";
console.log(person[propertyName]); // Wonderland
```

:::

当你在编写代码时不知道确切的属性名，或者属性名来自用户输入或其他动态来源时，括号表示法特别有用。

值得注意的是，对象可以包含不仅仅是字符串或数字，还可以包含数组或其它对象。

掌握对象及其用法对 JavaScript 学习非常重要，因为它们在语言本身以及众多库与框架中被广泛使用。
