# --interactive--

有好几种方法可以从对象中移除属性，其中最直接且常用的是使用 `delete` 操作符。

使用 `delete` 会从对象中删除指定的属性。下面是 `delete` 的使用示例：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

在这个示例中，`person` 对象最初包含 `name`、`age` 和 `job` 三个属性。使用 `delete` 后，`job` 属性被移除，之后访问 `person.job` 会得到 `undefined`。

另一种“移除”属性的方式是使用解构赋值配合剩余参数（rest）。这种方式并不会在原对象上删除属性，而是创建一个不包含某些属性的新对象：

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

上例中通过解构提取出 `job` 和 `city`，并将剩余属性收集到 `remainingProperties` 新对象中，从而得到一个不包含 `job` 与 `city` 的对象。

学会如何移除对象属性是 JavaScript 中很实用的技能，能帮助你动态地操作对象并清理不需要的数据。继续练习，你会越来越熟练！
