# --interactive--

在 JavaScript 中，你经常会遇到嵌套对象或对象中包含数组的复杂数据结构。这些结构可以表示层级丰富的数据，但也需要明确的方式来访问和操作内部数据。下面我们来看看如何有效地遍历这些嵌套结构。

访问嵌套对象的属性与访问普通对象类似，仍然可以使用点表示法或括号表示法，只不过需要把访问器串联起来以逐层向下索引。

例如，下面是一个包含联系信息的嵌套对象：

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

要访问 Alice 的工作电话号码，可以这样链式访问属性：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

同样可以使用括号表示法，当属性名包含空格或特殊字符，或当你需要使用变量动态访问属性时，括号表示法非常有用：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

:::

现在看一看对象属性的值为数组的情况。下面的 `person` 对象包含一个地址数组：

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

下面示例展示如何访问 Alice 的工作地址城市：

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

:::

在上述示例中，`person.addresses` 是地址数组。访问数组中第二项时使用索引 `1`，然后再使用点表示法访问该地址对象的 `city` 属性。

掌握如何访问嵌套对象和数组中的属性对于处理复杂数据结构至关重要。继续练习，你会越来越得心应手！
