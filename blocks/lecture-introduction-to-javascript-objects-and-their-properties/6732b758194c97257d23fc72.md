# --interactive--

在 JavaScript 中，构造函数是一种用于创建和初始化对象的特殊函数。它通常用 `new` 关键字调用，并可以在新创建的对象上初始化属性和方法。

本课将介绍如何使用 `Object()` 构造函数。`Object()` 会创建一个新的空对象。示例：

```js
new Object()
```

调用 `new Object()` 时，它会返回一个可用于存储值的新对象。

`Object()` 构造函数可以带或不带 `new` 关键字使用。当以函数方式（不带 `new`）调用时，其行为取决于传入值的类型。下面是不使用 `new` 的示例：

:::interactive_editor

```js
const num = 42;
const numObj = Object(num); // Creates an object wrapper for the number

console.log(numObj);
console.log(typeof numObj); // "object"
```

:::

如你在第二个 `console.log` 中看到的，`numObj` 是一个对象。这是因为我们使用 `Object()` 构造函数将数字包装为对象。

如果向 `Object()` 传入 `null` 或 `undefined` 会发生什么？

:::interactive_editor

```js
const newObj = new Object(undefined);
console.log(newObj); // {}
```

:::

结果会是一个空对象。另一种使用场景是当你处理未知类型的值并需要确保它是对象时。来看下面的例子：

:::interactive_editor

```js
function toObject(value) {
  if (value === null || value === undefined) {
    return {};
  }

  if (typeof value === "object") {
    return value;
  }

  return Object(value);
}

console.log(toObject(null));

console.log(toObject(true));

console.log(toObject([1, 2, 3]));
```

:::

在这个示例中，我们有一个名为 `toObject` 的函数。第二个判断会检查值是否为对象类型，如果为 `true` 就直接返回该值。该判断同时也会识别数组，因为数组是对象的特殊形式。

如果两个条件都不成立，函数会返回 `Object(value)`，把输入转换为对象。这适用于数字、字符串和布尔值等原始类型。

大多数情况下，你不会用 `Object()` 构造函数来创建新对象，通常会使用对象字面量语法（例如 `const objectLiteral = { name: "Beau" }`）。不过，理解 `Object` 构造函数的基础用法仍然很有帮助，掌握这些细节会让你在编程中更自信，继续加油！
