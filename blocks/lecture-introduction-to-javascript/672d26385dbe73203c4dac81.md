# --interactive--

JavaScript是一种强大的编程语言，它为网站带来交互性和动态行为。

虽然HTML和CSS是用于构建页面内容结构和样式元素的标记语言，但JavaScript超越了这些，通过实现更复杂的功能，如处理用户输入、制作动画元素，甚至构建完整的Web应用程序。

例如，当你点击按钮、提交表单或悬停在菜单上时，JavaScript决定了页面的行为。

以下是这三者如何协同工作的示例：

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

在这个示例中，HTML用于定义内容：标题（`h1`元素）和按钮（`button`元素）。

CSS用于为标题应用样式，比如使文本变为绿色。JavaScript用于在点击按钮时显示警告消息。

总而言之，HTML提供结构，CSS添加样式，JavaScript启用行为，从而创建完整且交互式的Web体验。
