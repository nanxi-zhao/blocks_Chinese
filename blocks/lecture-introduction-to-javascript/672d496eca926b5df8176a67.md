# --description--

在 JavaScript 中，数据类型指的是变量所保存值的种类。

变量是带名称的容器，用于存储特定数据类型的值，便于在代码中引用和操作。数据类型帮助程序判断正在处理的数据是数字、文本还是其他类型。

JavaScript 有几种常见的基本数据类型，我们会在后续课程中逐一详细讲解。现在先简要介绍这些数据类型。

第一个要了解的是 `Number`（数字）类型。

`Number` 类型既表示整数也表示浮点数。例如整数有 `7`、`19`、`90`；浮点数有 `3.14`、`5.2` 等，浮点数就是带小数点的数字。

下一个数据类型是 `String`（字符串）。

`String` 是被引号包裹的一串字符或文本。这里有两个示例：

```md
"Hello, world"
```

```md
'JavaScript'
```

另一个常用的数据类型是 `Boolean`（布尔）。

`Boolean` 只有两个可能的值：`true` 或 `false`。你可以用布尔值来表示页面是否正在加载，或者用户是否已登录。

接下来是 `undefined` 和 `null` 两种类型。

`undefined` 表示变量已声明但尚未赋值；`null` 表示变量被刻意设置为“无值”。我们会在后续课程中更详细地讲解它们的区别。

下一个要看的是 `Object`（对象）类型。

`Object` 是更复杂的数据类型，可以保存键值对的集合。键（也称为属性名）类似于数据的标签，而值就是你要存储的实际数据。下面是一个示例：

```js
let book = {
  title: "The Great Gatsby",
  author: "F. Scott Fitzgerald",
  year: 1925
};
```

在这个对象中，`title`、`author` 和 `year` 是键（或属性名），而 `The Great Gatsby`、`F. Scott Fitzgerald` 和 `1925` 是对应的值。

对象中的每个键值对称为一个属性。上面的 book 对象有三个属性。这只是对象和属性的基础介绍，后续课程会深入到更高级的概念。

最后两种要提到的是 `Symbol` 和 `BigInt`。

`Symbol` 是一种特殊的值类型，总是唯一且不可变，常用于为属性创建唯一标识符：

```js
Symbol('mySymbol');
```

`BigInt` 用于表示超过 `Number` 类型范围的超大整数：

```js
1234567890123456789012345678901234567890n;
```

在这个示例中，我们通过在超大数字末尾添加 `n` 来创建 `BigInt`。

`Symbol` 和 `BigInt` 在日常开发中使用较少，但了解它们很有帮助。

理解这些数据类型可以帮助你在程序中正确处理不同种类的数据，因为每种类型都有自己的特性和行为。
