# --description--

在 JavaScript 中，字符串是一系列用于表示文本数据的字符。字符串是语言的原始数据类型之一，与数字（numbers）、布尔值（booleans）、`null` 和 `undefined` 同属基本类型。

在 JavaScript 中创建字符串可以使用单引号（`'`）或双引号（`"`）。

下面是创建两个保存字符串值的变量的示例：

```js
let singleQuotes = 'This is a string';
let doubleQuotes = "This is also a string";
```

尽管可以使用单引号或双引号来创建字符串，但保持风格一致很重要。如果字符串以单引号开始，也必须以单引号结束。

双引号同理。下面的例子会抛出错误，因为它以双引号开始却以单引号结束：

```js
const improperStr = "Do not do this';
```

另外需要注意的是，字符串是不可变的（immutable）。在编程中，不可变意味着一旦创建就不能被更改。因此，当你创建一个字符串后，不能直接修改它的字符；如果想要改变字符串，需要创建一个新的字符串。

下面是将新字符串赋给 `developer` 变量的示例：

```js
let developer = "Jessica";
developer = "Quincy";
```

由于字符串不可变，我们不能直接修改最初的字符串，这就是为什么要把新字符串赋给 `developer` 变量。

字符串是编程中非常重要的部分，后续的课程你会学到更多高级的字符串处理技巧，帮助你构建更动态、更具交互性的应用程序。
