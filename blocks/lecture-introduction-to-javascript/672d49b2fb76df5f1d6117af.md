# --description--

在 JavaScript 中处理文本是编程的重要组成部分，通常需要将不同的文本片段连接起来，这个过程称为字符串拼接（concatenation）。

本讲重点介绍字符串拼接的常见方法，尤其是使用 `+` 运算符、`+=` 运算符，以及 `concat()` 方法。

`+` 运算符是最简单也最常用的拼接方法之一。它可以将多个字符串连接起来，或将字符串与保存文本的变量组合。

这是一个示例：

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName; // John Doe
```

在这个例子中，我们使用 `+` 将 `firstName` 和 `lastName` 与一个空格（`" "`）连接成全名。

使用 `+` 拼接时的一个缺点是如果不注意空格，可能会出现连在一起的情况：

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + lastName; // "JohnDoe"
```

因此使用 `+` 拼接时要注意空格问题。

如果需要在已有字符串后追加内容，可以使用 `+=` 运算符。这在需要逐步构建字符串时非常方便。

示例：

```js
let greeting = 'Hello';
greeting += ', John!';

// greeting 现在是字符串 Hello, John!
```

需要记住的是字符串是不可变的（immutable），这意味着一旦创建字符串就不能就地修改它。

在上面的例子中，原始的 `Hello` 并没有被修改，而是创建了一个新的字符串 `Hello, John!`，并把 `greeting` 指向了这个新字符串。

另一种拼接字符串的方式是使用 `concat()` 方法。

在学习 `concat()` 之前，先简单区分一下函数（function）和方法（method）：函数是可复用的代码块，用来执行特定任务；方法是属于对象的函数，用来操作该对象中的数据。

后续课程会更深入讲解函数、对象和方法。现在只需知道 JavaScript 提供了很多内置方法，例如 `concat()`。

示例：使用 `concat()` 将两个字符串连接起来：

```js
let str1 = 'Hello';
let str2 = 'World';

let result = str1.concat(' ', str2); // Hello World
```

在这个示例中，我们用 `concat()` 将 `str1`、一个空格（`' '`）和 `str2` 连接成一个字符串。

总结：对于简单拼接，`+` 运算符最直观；当你需要逐步构建字符串时，`+=` 非常方便；当需要连接多个字符串时，`concat()` 也很有用。
