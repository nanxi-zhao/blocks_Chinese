# --description--

在上一节课中，你已经了解了 Python 是什么以及它能做些什么。现在，我们来看看如何在本地机器上设置和使用 Python。

在 Windows 和 macOS 上安装 Python 的推荐方式是从 Python 官方网站下载安装程序。本节也会简要介绍在 Linux 上运行 Python 的方法。

访问 `https://www.python.org/`，将鼠标移到 “Downloads” 上，会弹出一个窗口，显示适用于你操作系统的最新 Python 版本。

先介绍在 macOS 上安装 Python 的步骤：

- 点击前述弹窗中显示的版本按钮，安装包（`.pkg`）会自动开始下载。
- 下载完成后，打开 `.pkg` 安装程序，然后在弹出的窗口中点击“Continue”。
- 按提示继续点击“Continue”，直到出现“Installation Type”一栏，然后点击“Install”。
- 如有需要，输入你的密码以授权安装，然后开始安装。
- 安装完成后，你会看到一条提示，说明 Python 已成功安装。
- 点击“Close”按钮，安装就完成了！

你可以通过在终端运行 `python --version` 或 `python3 --version` 来验证安装是否成功。

也可以在终端中运行 `python` 或 `python3` 来进入 Python 解释器。

<dfn>终端（terminal）</dfn> 是一个基于文本的界面，允许你通过输入命令与计算机交互。每个操作系统都有默认的终端应用：在 macOS 上可以使用 Terminal，Windows 上可以使用 Command Prompt 或 PowerShell，Linux 各桌面环境也有各自的默认终端（例如 GNOME Terminal、Konsole 等）。

需要注意的是，在一些较旧的 macOS 或 Linux 系统上，`python` 可能对应的是 Python 2，而 `python3` 才是 Python 3。如果你运行 `python --version` 看到了类似 `Python 2.7.18` 的版本号，说明系统可能依赖于旧版 Python 的某些软件。此时建议使用 `python3` 来运行你的 Python 代码。

在 Windows 上安装 Python 的步骤如下：

- 打开 `https://www.python.org/`，将鼠标移到 “Downloads” 上，会看到一个写着 “Download for Windows” 的弹窗和当前版本的下载按钮。
- 点击版本号，Windows 安装程序（`.exe`）会自动开始下载。
- 下载完成后，双击运行安装程序，按照提示操作。
- 当你看到 `Add python.exe to Path` 选项时，请勾选该选项，然后点击 `Install Now`，这会让以后使用 Python 更方便。

你可以在 PowerShell 等命令行中运行 `python --version` 来验证安装，也可以直接运行 `python` 进入解释器。

在 Linux 上，大多数主流的发行版（例如 Ubuntu、Debian、Fedora）通常会自带 Python。

只需打开终端并运行 `python --version` 或 `python3 --version`：

如果两个命令都没有显示 Python 版本，你可以到 `https://www.python.org` 上查找适用于你发行版的安装包，或在线搜索你所用发行版推荐的安装方式。
