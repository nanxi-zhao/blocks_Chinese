# --description--

每种编程语言都有向终端输出数据的方式，可能是内置方法、函数、属性或关键字。在 Python 中，可以使用 `print` 函数将数据输出到终端。我们来仔细看一下 `print`，让你能自信地使用它。

学习任何编程语言时，第一件事通常就是写一个简单的 `Hello world!` 程序。在 Python 中，用 `print` 函数就能轻松实现。

要打印 `Hello world!`，只需把字符串放在 `print` 的括号里：

```python
print('Hello world!') # Hello world!
```

后续课程会详细介绍字符串和函数；现在可以把字符串理解为被单引号（`'`）或双引号（`"`）包裹的一串字符。

在 `print('Hello world!')` 示例中，`'Hello world!'` 是传递给 `print` 的一个**参数（argument）**。你也可以通过逗号传入多个参数同时打印，例如：

```python
print('My favorite colors are', 'blue', 'green', 'red')

# Output: My favorite colors are blue green red
```

当使用逗号分隔时，Python 会在每个项之间自动插入一个空格，这在你想把多条信息放在一起打印时非常方便。
