# --description--

Python 是一门动态类型语言，像 JavaScript 一样，你无需为变量显式声明类型。语言会根据你赋给变量的值来推断其数据类型。

例如：

```python
name = 'John Doe' # Python 会推断这是字符串
age = 25 # Python 会推断这是整数
```

这与一些静态类型语言（如 C#、Java、C++）不同，那些语言需要在声明时指定类型，例如：

```csharp
string name = 'John Doe'
int age = 25
```

动态类型让 Python 的编码更快速、更灵活，但也可能带来运行时才会暴露的类型错误，因此在大型项目中需要更谨慎地测试和类型检查。

下面是你在 Python 中常用的数据类型：

- 整数（Integer）：不带小数点的整数，例如 `10` 或 `-5`。

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- 浮点数（Float）：带小数点的数值，例如 `4.41` 或 `-0.4`。

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.50
```

- 复数（Complex）：包含实部和虚部，例如 `6 + 7j`。

```python
my_complex_var = 3 + 4j
print('Complex:', my_complex_var) # Complex: (3+4j)
```

- 字符串（String）：用单引号或双引号包裹的一串字符，例如 `'Hello world!'`。

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- 布尔值（Boolean）：表示真或假的类型，写作 `True` 或 `False`。

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- 集合（Set）：无序且元素唯一的集合，例如 `{4, 2, 0}`。

```python
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- 字典（Dictionary）：以键值对形式存储的集合，使用花括号表示，例如 `{'name': 'John Doe', 'age': 28}`。

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- 元组（Tuple）：不可变的有序集合，使用圆括号表示，例如 `(7, 8, 4)`。

```python
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- Range：表示一系列数字，常用于循环，例如 `range(5)`。

```python
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- 列表（List）：有序集合，元素可以是不同类型。

```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None：表示值缺失或无值的特殊类型。

```python
my_none_var = None
print('None:', my_none_var) # None: None
```

许多语言会将类型分为原语类型（primitive）和引用类型（reference）。原语类型通常是简单且不可变的，而引用类型可以包含多个值，并可能是可变的或不可变的。但在 Python 中，所有数据都被视为对象，有些对象是不可变的，有些是可变的。

不可变类型一旦创建便无法修改。你可以把变量重新指向其他对象（重新赋值），但无法在原地修改对象的内容。Python 中的不可变类型示例包括字符串、整数、浮点数、布尔值、元组和 range。

下面示例显示虽然可以给变量重新赋值，但不能直接修改字符串（因为字符串是不可变的）：

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello

greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

相反，可变类型可以在原位修改，不需要重新命名。列表和字典就是常见的可变类型示例。

例如更新列表中的元素：

```python
nums = [1, 2, 3]
nums[0] = 4

print(nums) # [4, 2, 3]
```

要获取变量的数据类型，可以使用 `type()` 函数：

```python
my_var_1 = 'Hello world'
my_var_2 = 21

print(type(my_var_1)) # <class 'str'>
print(type (my_var_2)) # <class 'int'>
```

下面是本课中涉及的数据类型及其在终端中的表示：

```python
my_integer_var = 10
print('Integer:', my_integer_var, '| Type:', type(my_integer_var))  # Integer: 10 | Type: <class 'int'>

my_float_var = 4.50
print('Float:', my_float_var, '| Type:', type(my_float_var))  # Float: 4.50 | Type: <class 'float'>

my_complex_var = 3 + 4j
print('Complex:', my_complex_var, '| Type:', type(my_complex_var))  # Complex: (3+4j) | Type: <class 'complex'>

my_string_var = 'hello'
print('String:', my_string_var, '| Type:', type(my_string_var))  # String: hello | Type: <class 'str'>

my_boolean_var = True
print('Boolean:', my_boolean_var, '| Type:', type(my_boolean_var))  # Boolean: True | Type: <class 'bool'>

my_set_var = {7, 5, 8}
print('Set:', my_set_var, '| Type:', type(my_set_var))  # Set: {7, 5, 8} | Type: <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var, '| Type:', type(my_dictionary_var))  # Dictionary: {'name': 'Alice', 'age': 25} | Type: <class 'dict'>

my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var, '| Type:', type(my_tuple_var))  # Tuple: (7, 5, 8) | Type: <class 'tuple'>

my_range_var = range(5)
print('Range:', list(my_range_var), '| Type:', type(my_range_var))  # Range: [0, 1, 2, 3, 4] | Type: <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print('List:', list(my_list), '| Type:', type(my_list)) # List: [22, 'Hello world', 3.14, True] | Type: <class 'list'>

my_none_var = None
print('None:', my_none_var, '| Type:', type(my_none_var))  # None: None | Type: <class 'NoneType'>
```

另一种检查变量类型的方法是使用内置的 `isinstance()` 函数，它用于判断一个变量是否属于某个类型。

`isinstance()` 接受一个对象和你要检查的类型，返回布尔值。示例如下：

```python
isinstance('Hello world', str) # True
isinstance(True, bool) # True
isinstance(42, int) # True
isinstance('John Doe', int) # False
```

虽然 Python 是动态类型语言，但仍然可以添加类型提示（type hints）。类型提示是可选的，主要用于给其他开发者或编辑器提供静态分析和文档化信息。变量类型的简单示例：

```python
user_name: str = 'John Doe'
user_age: int = 24
```

下面是对函数参数和返回值进行类型提示的示例：

```python
def greet(name: str, age: int) -> str:
    return f'Hello, {name}, age {age}.'
```

以及同时使用变量和函数类型提示的示例：

```python
def greet(name: str, age: int) -> str:
    return f'Hello, {name}, age {age}.'

user_name: str = 'John Doe'
user_age: int = 24

print(greet(user_name, user_age)) # Hello, John Doe, age 24.
```

注意，与在编译时强制类型的 TypeScript 不同，Python 的类型提示主要用于静态分析、文档和编辑器支持，并不会在运行时强制类型检查。但这些提示能帮助开发者及早发现问题并提升代码可读性，尤其在大型项目中非常有价值。
