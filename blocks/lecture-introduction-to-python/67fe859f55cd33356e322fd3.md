# --description--

函数是可重用的代码片段，只有在被调用时才会执行。许多编程语言都提供内置函数以便快速上手，Python 也不例外；我们在之前的课程已经介绍过像 `print()` 这样的内置函数。

另一个常用的内置函数是 `input()`，它可以向用户提示并读取输入：

```python
name = input('What is your name?') # 用户输入 "Kolade" 并按回车
print('Hello', name) # 输出: Hello Kolade
```

另外，`int()` 可以将数字、布尔值或数字字符串转换为整数：

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

你也可以定义自定义函数。使用 `def` 关键字，后面接函数名、一对圆括号和冒号，然后在新的一行中书写函数体。

下面是一个名为 `hello` 的示例函数，它会在终端打印 `Hello World`：

```python
def hello():
    print('Hello World')
```

要运行该函数，调用它的名字并加上一对括号：

```python
hello() # Hello World
```

注意 `print('Hello World')` 前的缩进。缩进决定了 Python 中的代码块，属于同一缩进层级的语句被视为一个代码块。

与使用大括号标识代码块的语言不同，Python 使用缩进来表示代码块。虽然可以使用两个或四个空格来缩进，但官方风格指南建议使用四个空格。

在循环和条件语句中也会出现代码块，这些内容会在后续课程中讲到。

下面是一个简单函数示例，打印两个数的和：

```python
def calculate_sum(a, b):
    print(a + b)
```

可以看到 `calculate_sum` 的括号中有 `a` 和 `b`，它们被称为参数（parameters）。参数是占位符，用来在调用函数时接收传入的值（arguments）。

参数只有在调用函数时传入实参（arguments）才会被使用。例如：

```python
calculate_sum(3, 1) # 4
```

如果调用函数时未提供正确数量的参数，会抛出 `TypeError`：

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

函数还可以使用 `return` 关键字返回一个值。如果没有显式使用 `return`，Python 默认返回 `None`。

例如：

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

上例中，`calculate_sum` 打印了结果但没有返回值，因此赋值给 `my_sum` 的结果是 `None`。使用 `return` 可以把值返回：

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

现在 `calculate_sum` 会返回 `a` 与 `b` 的和，并将其存入 `my_sum`。
