# --description--

Python 提供了许多内置方法，让字符串处理变得轻松。下面列出一些常用的方法：

- `upper()`：返回一个将所有字符转换为大写的新字符串。

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`：返回一个将所有字符转换为小写的新字符串。

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`：返回去除指定前后字符的新字符串；若不传参数，则去除首尾空白字符。

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`：返回一个将所有匹配 `old` 的子串替换为 `new` 的新字符串。

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`：以指定分隔符分割字符串，返回字符串列表；若不指定分隔符，则按空白字符分割。

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`：使用指定分隔符将可迭代对象的元素拼接为字符串。

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`：判断字符串是否以指定前缀开头，返回布尔值。

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`：判断字符串是否以指定后缀结尾，返回布尔值。

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`：返回子串首次出现的索引，若未找到返回 `-1`。

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`：返回子串在字符串中出现的次数。

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`：返回一个首字母大写、其它字母小写的新字符串。

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`：如果字符串中所有字母均为大写则返回 `True`，否则返回 `False`。

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`：如果字符串中所有字母均为小写则返回 `True`，否则返回 `False`。

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`：返回每个单词首字母大写的新字符串。

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
