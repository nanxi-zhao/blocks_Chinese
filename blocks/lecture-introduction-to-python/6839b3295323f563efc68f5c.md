# --description--

增强赋值（augmented assignment）将二元运算与赋值合并为一步操作：对变量执行运算后，将结果存回同一个变量。

如果你熟悉 JavaScript，可能听说过 `+=`（加赋值）或 `-=`（减赋值）等操作符，Python 中也有这些，只是通常称作增强赋值。

增强赋值的基本语法为：

```python
variable <operator>= value
```

这等价于更冗长的写法：

```python
variable = variable <operator> value
```

例如，用增强赋值给现有变量加 5：

```python
my_var = 10
my_var += 5

print(my_var) # 15
```

不使用增强赋值时的等价写法：

```python
my_var = 10
my_var = my_var + 5

print(my_var) # 15
```

增强赋值的优点是更简洁可读，避免重复书写变量名，从而降低因打字错误导致的潜在问题。

几乎所有运算符都支持增强赋值。下面列出一些常见的示例：

- 减法赋值（`-=`）：将右侧运算数从左侧变量中减去并将结果存回左侧变量：

```python
count = 14
count -= 3

print(count) # 11
```

- 乘法赋值（`*=`）：将左侧变量乘以右侧运算数并将结果存回左侧变量：

```python
product = 65
product *= 7

print(product) # 455
```

- 除法赋值（`/=`）：将左侧变量除以右侧并将结果存回左侧变量：

```python
price = 100
price /= 4

print(price) # 25.0
```

- 整除赋值（`//=`）：对左侧变量执行整除并将结果存回左侧变量：

```python
total_pages = 23
total_pages //= 5

print(total_pages) # 4
```

- 取模赋值（`%=`）：计算左侧变量除以右侧的余数并存回左侧：

```python
bits = 35
bits %= 2

print(bits) # 1
```

- 幂赋值（`**=`）：将左侧变量提升到右侧的幂并把结果存回左侧：

```python
power = 2
power **= 3

print(power) # 8
```

还有用于位运算的增强赋值，例如 `&=`、`^=`、`>>=`、`<<=` 等。

部分增强赋值也可以用于字符串。例如，加赋值非常适合字符串拼接：

```python
greet = 'Hello'
greet += ' World'

print(greet) # Hello World
```

乘法赋值可用于重复字符串：

```python
greet = 'Hello'
greet *= 3

print(greet) # HelloHelloHello
```

其他某些增强赋值在字符串上会抛出 `TypeError`：

```python
greet = 'Hello'
greet -= ' World'

print(greet) # TypeError: unsupported operand type(s) for -=: 'str' and 'str'


greet = 'Hello'
greet /= 'World'

print(greet) # TypeError: unsupported operand type(s) for /=: 'str' and 'str'
```

如果你想知道递增（`++`）和递减（`--`）操作在 Python 中是否可用，答案是否定的。Python 有意不采用 C 风格的自增自减操作，以保持语言的清晰与明确。

用 `x += 1` 替代 `x++`，更直观地表明你将变量 `x` 的值加 1。

在 Python 中写 `++x` 只是对一元正号的重复应用，并不会做增量操作：

```python
my_var = 5

print(+my_var)   # 5
print(++my_var)  # 5
print(+++my_var) # 5

my_var += 1

print(my_var) # 6
```
