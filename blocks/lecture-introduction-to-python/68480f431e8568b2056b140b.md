# --description--

在上一课中，你已经学会如何使用比较运算符和条件语句来控制程序流程。

虽然这些工具非常强大，但在实际编程中你常常需要同时比较多个条件，这可能导致嵌套的条件判断，例如：

```python
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

上面的例子会先判断 `is_citizen` 是否为 `True`，如果是，就继续进入嵌套的 `if` 判断 `age` 是否大于等于 `18`。因为 `age` 满足条件，终端会打印 `You are eligible to vote`。如果 `is_citizen` 为 `False`，则会打印 `You are not eligible to vote`。

当条件变得更复杂时，可以使用 Python 的 `and`、`or` 和 `not` 运算符来简化表达。

在深入这些运算符之前，我们先了解什么是 truthy 和 falsy 值。

在 Python 中，每个值在逻辑上下文中都有其固有的布尔意义：有些值被认为是 **truthy**（在逻辑判断中相当于 `True`），有些被认为是 **falsy**（在逻辑判断中相当于 `False`）。

一些常见的 falsy 值包括：

- `None`
- `False`
- 整数 `0`
- 浮点数 `0.0`
- 空字符串 `""`

其他像非零数字或非空字符串等则被视为 truthy。

如果你想检查某个值是 truthy 还是 falsy，可以使用内置的 `bool()` 函数，它会把值显式转换为布尔值：truthy 转为 `True`，falsy 转为 `False`。例如：

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

了解了 truthy 和 falsy 后，我们来看布尔运算符（也称逻辑运算符），它们可以把多个表达式组合起来，构建更复杂的条件逻辑。

Python 中有三个布尔运算符：`and`、`or` 和 `not`。

先看 `and` 运算符。

`and` 接受两个操作数：如果第一个操作数为 falsy，则返回第一个操作数；否则返回第二个操作数。要使 `and` 表达式为 truthy，两个操作数都必须为 truthy。

例如：

```python
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

在上例中会打印数字 25，因为当第一个操作数为 `True` 时，`and` 会继续计算并返回第二个操作数。`and` 属于短路运算（short-circuit）：Python 从左到右检查值，遇到能确定结果的值就停止计算。

你常在 `if` 语句中使用 `and` 来判断多个条件是否同时成立。下面是把之前嵌套 `if` 改写为 `and` 的例子：

```python
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

在该例中，`is_citizen` 与 `age >= 18` 都为真，所以整个表达式为真，执行 `if` 分支。

接下来是 `or` 运算符。`or` 会返回第一个为 truthy 的操作数；如果第一个为 falsy，则返回第二个。只要有一个操作数为 truthy，`or` 表达式就为 truthy；`or` 也是短路运算符。例子如下：

```python
age = 19
is_employed = False

print(age or is_employed) # 19
```

上例打印 19，因为第一个操作数 `age` 为 truthy。

在条件判断中，如果需要判断“至少有一个条件为真”，可以使用 `or`：

```python
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

在此例中，`age < 18` 为假，但 `is_student` 为真，因此 `or` 表达式为真，会打印优惠信息。

最后看 `not` 运算符：它只接受一个操作数并取反其布尔值，把 truthy 转为 `False`，falsy 转为 `True`。不同于前两个运算符，`not` 总是返回 `True` 或 `False`。

例如：

```python
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

在条件判断中常用 `not` 来检查某个条件不为真，例如：

```python
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

因为 `is_admin` 是 `False`，所以 `not is_admin` 为真，会打印 “Access denied for non-administrators.”。

现在你已经了解了 truthy/falsy 值、`and`、`or`、`not` 以及短路行为，就能写出更灵活、可读性更强的条件逻辑啦，继续加油，你做得很棒！
