# --interactive--

在 JavaScript 中，字符串是一连串用于表示文本的数据字符。字符串是语言中的原始数据类型之一，与数字、布尔值、`null` 和 `undefined` 并列。

在 JavaScript 中创建字符串可以使用单引号（`'`）或双引号（`"`）。

下面是创建两个保存字符串值的变量的示例：

:::interactive_editor

```js
let singleQuotes = 'This is a string';
console.log(singleQuotes);
let doubleQuotes = "This is also a string";
console.log(doubleQuotes);
```

:::

虽然可以用单引号或双引号来创建字符串，但保持一致性很重要。如果字符串以单引号开始，就必须以单引号结束。

同样地，双引号也必须配对。下面的例子会抛出错误，因为它以双引号开始却以单引号结束：

```js
const improperStr = "Do not do this';
```

另一点要注意的是：字符串是不可变的（immutable）。在编程中，不可变意味着一旦创建就不能被直接修改。因此，当你创建一个字符串时，无法直接改变其内部的字符；如果想修改内容，需要创建一个新的字符串。

下面是将新字符串赋值给 `developer` 变量的例子：

:::interactive_editor

```js
let developer = "Jessica";
console.log(developer);
developer = "Quincy";
console.log(developer);
```

:::

由于字符串不可变，我们不能直接修改第一个字符串的内容，所以才需要将新的字符串赋给 `developer` 变量。

字符串在编程中非常重要，后续课程你会学到更多实用的处理技巧，用它们来创建更动态、更交互的应用。
