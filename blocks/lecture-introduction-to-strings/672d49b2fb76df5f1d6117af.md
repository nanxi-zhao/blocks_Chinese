# --interactive--

在 JavaScript 中，处理文本是编程中非常常见且重要的部分，通常需要将若干段文本合并在一起，这个过程称为字符串连接（string concatenation）。

本课将聚焦常见的字符串连接方法，特别是使用 `+` 运算符、`+=` 运算符，以及 `concat()` 方法的用法。

`+` 运算符是最简单也最常用的字符串连接方式之一，它可以将多个字符串或包含文本的变量拼接在一起。

下面是一个示例：

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName; 
console.log(fullName); // John Doe
```

:::

在这个例子中，我们使用 `+` 运算符把 `firstName` 和 `lastName` 变量与一个空格（`" "`）拼接成完整的姓名。

使用 `+` 运算符时一个常见的缺点是：如果不注意空格的位置，可能会出现拼接后缺少空格的问题。

下面是一个缺少空格的示例：

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";

let fullName = firstName + lastName; 
console.log(fullName); // "JohnDoe"
```

:::

因此在使用 `+` 连接字符串时，要留意空格以确保输出格式正确。

如果需要在现有字符串上追加内容，可以使用 `+=` 运算符。当你想逐步构建字符串或向已有字符串添加更多文本时，这个操作很方便。

下面示例展示如何使用 `+=` 将字符串追加到另一个字符串后面：

:::interactive_editor

```js
let greeting = 'Hello';
greeting += ', John!';

console.log(greeting); // "Hello, John!"
```

:::

记住字符串是不可变的（immutable），也就是说一旦创建字符串，你不能直接修改它的内部字符。

在上面的例子中，原始的 `Hello` 并没有被原地修改，而是生成了一个新字符串 `Hello, John!`，并由 `greeting` 引用。

另一种连接字符串的方法是使用 `concat()` 方法。

在介绍 `concat()` 之前，先简要区分一下“函数”和“方法”的概念：

函数是可重用的一段代码，用来执行特定任务并可以传入不同参数调用；方法是与对象关联的函数，作用于该对象的数据。

后续课程中我们会更深入讲解函数、对象和方法的工作原理。现在先知道 JavaScript 有很多内置方法可以使用，例如 `concat()`。

下面是使用 `concat()` 将两个字符串拼接在一起的示例：

:::interactive_editor

```js
let str1 = 'Hello';
let str2 = 'World';

let result = str1.concat(' ', str2); 
console.log(result); // Hello World
```

:::

在这个例子中，我们使用 `concat()` 把 `str1`、一个空格（`' '`）和 `str2` 合并成一个字符串。

总结一下：当需要简单拼接少量字符串或变量时，`+` 是最直接的选择；当需要逐步构建字符串或追加新内容时，`+=` 很方便；如果要合并多个字符串，`concat()` 也是一种可选方式。
