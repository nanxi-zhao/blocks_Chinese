# --interactive--

前几课向你介绍了 `console.log()`，本课将更深入地讲解它的用途和使用方法。

在 JavaScript 中，`console.log()` 是一个简单但非常有用的工具，用来在浏览器的控制台中输出信息。开发者常用它来调试和检查代码，以便在开发过程中观察程序运行情况。

你可以用 `console.log()` 输出文本或变量到控制台，确认代码是否按预期执行。

使用 `console.log()` 时，把想要输出的值或消息放在圆括号内调用即可。下面是一些示例：

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

第一个示例会在浏览器控制台打印 `Hello, world!`，第二个示例会打印数值 `5`。

下面是另一个使用 `console.log()` 的例子：

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

你也可以将多个值用逗号分隔传入 `console.log()`，例如：

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

这种方式有助于一次性记录多条信息。

`console.log()` 方法可以帮助你在程序运行时监控代码，更容易发现错误并理解程序的运行行为。
