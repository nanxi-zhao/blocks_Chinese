不断对剩余区间折半，直到找到目标或确认目标不存在（这时返回 `-1`）。

下面是二分查找的示例代码：

```python
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1

    while low <= high:
        mid = (low + high) // 2  

        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return -1
```


实现二分查找时，先确定 `low` 和 `high` 索引，代表当前查找范围。

当 `low` 大于 `high` 时，表示已无可查范围，目标不存在，应停止查找并返回 `-1`。

否则计算中间索引 `mid`，若中间元素等于目标值，则返回 `mid`；若中间元素小于目标值，则把 `low` 更新为 `mid + 1`（在右半区继续查找）；否则把 `high` 更新为 `mid - 1`（在左半区继续查找）。

重复该过程直到找到目标或确认目标不在列表中。

二分查找的时间复杂度是 `O(log n)`，因为查找范围每次对半缩小；空间复杂度是 `O(1)`。

线性查找和二分查找在不同场景下都有用，理解它们的差异和适用条件会帮助你在算法题或实际工程中选对工具。
