# --interactive--

外边距（margin）和行高（line-height）在为列表项设置间距时非常重要，它们能提升可读性和视觉效果。

我们先从使用外边距来调整列表项间距开始吧！

通过给 `li` 元素设置外边距，你可以在列表项之间创造空间。这个方法可以控制每个列表项外部的间距，方便你增大或减小项间的空隙。

下面看一个包含三个列表项的无序列表示例。

:::interactive_editor

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

默认情况下，HTML 不会在列表项之间添加太多间距。

要在每个列表项底部添加间距，你可以使用 `margin-bottom` 属性，示例如下：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  margin-bottom: 40px;
}
```

:::

在这个示例中，每个无序列表项底部会应用 `40px` 的外边距。

另一种调整列表项间距的方式是使用 `line-height`（行高）。

`line-height` 用来调整单个列表项内部文本行之间的垂直间距。

While it primarily affects the spacing between lines of text within each item, it can also indirectly influence the overall spacing between list items if the items contain only a single line of text.

If list items have multiple lines of text, the `line-height` will affect the spacing between those lines, but it does not directly adjust the spacing between separate list items themselves.

To control the spacing between individual list items, you would use `margin` or `padding` properties instead.

在这个示例中，每个无序列表项底部会应用 `40px` 的外边距。

另一种调整列表项间距的方式是使用 `line-height`（行高）。

`line-height` 用来调整单个列表项内部文本行之间的垂直间距。

当列表项只包含一行文本时，行高也会间接影响列表项之间的整体间距；但如果列表项有多行文本，`line-height` 只会影响同一项内部行与行之间的间距，而不会直接调整不同列表项之间的外部间距。

若要控制各个列表项之间的间距，建议使用 `margin` 或 `padding`。

下面用前面的无序列表示例展示如何给列表项应用 `line-height`：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
li {
  line-height: 2; 
}
```

:::

在这个示例中，`line-height: 2;` 会把行高设置为字体大小的两倍，从而在每个列表项内部产生更多的垂直空间。
