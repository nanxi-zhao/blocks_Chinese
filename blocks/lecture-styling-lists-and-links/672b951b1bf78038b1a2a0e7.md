# --interactive--

在CSS中，`list-style` 属性用于控制网页上列表的外观。

无论您是在处理有序列表（`ol`）还是无序列表（`ul`），`list-style` 属性都能让您自定义列表项的显示方式。

`list-style` 属性实际上是以下三个其他属性的简写：

- `list-style-type`
- `list-style-position`
- `list-style-image`

每一个都在定义列表外观方面发挥着不同的作用。

`list-style-type` 属性允许您定义列表中使用的项目符号或数字的类型。

对于无序列表，您可以从几种项目符号样式中选择，比如圆点、圆圈或方块。

对于有序列表，您可以使用不同的编号系统，如十进制数字、罗马数字，甚至是字母字符。

下面是使用 `list-style-type` 的示例：

:::interactive_editor

```html
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

在这个示例中，无序列表的项目符号被更改为方形。

`list-style-type` 属性是三者中最常用的，因为它直接影响列表中项目符号或编号样式的外观。

`list-style-position` 属性控制项目符号或数字相对于列表项内容的位置。您可以使用两个值：`inside` 和 `outside`。

当您使用 `outside` 值时，项目符号或数字出现在内容外部，这是默认行为。

而当您使用 `inside` 值时，项目符号或数字出现在内容内部，这可能导致文本换行并与项目符号或数字对齐。

下面是使用 `list-style-position` 的示例：

:::interactive_editor

```html
<ul style="list-style-position: inside;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<ul style="list-style-position: outside;">
  <li>Item 4</li>
  <li>Item 5</li>
  <li>Item 6</li>
</ul>
```

:::

在这个示例中，为两个不同的无序列表分别提供了 `inside` 和 `outside` 值。

当您想要控制列表内容的对齐方式时，特别是在单个列表项中有多个文本行的情况下，`list-style-position` 属性非常有用。

`list-style-image` 属性允许您使用图像作为列表项的项目符号。这对于为列表添加独特的视觉风格很有用。

下面是使用 `list-style-image` 的示例：

:::interactive_editor

```html
<head>
  <style>
    ul {
      list-style-image: url('https://cdn.MasterPuti.org/platform/universal/MasterPuti-org-gravatar.jpeg');
      list-style-position: inside;
    }
  </style>
</head>
<body>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</body>
```

:::

在这个示例中，项目符号被替换为自定义的 MasterPuti 徽标，为列表增添了个性化风格。

使用 `list-style-image` 时，请确保您选择的图像较小且适合您网页的设计。如果图像太大或太复杂，会使列表难以阅读。

您可以将这三个属性 —— `list-style-type`、`list-style-position` 和 `list-style-image` —— 合并为一个简写属性 `list-style`。

在简写属性中，值的顺序并不重要，但可以同时指定这三项。

下面是使用简写属性的示例：

:::interactive_editor

```html
<ul style="list-style: square inside url('https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg');">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

:::

在这个示例中，列表项使用方形项目符号，位置为内容内部，并使用自定义图片作为项目符号。

但是，如果图像不可用或未渲染，方形项目符号将作为后备选项。
