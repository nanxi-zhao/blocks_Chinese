# --description--

Fetch API允许Web应用程序发出网络请求，通常用于从服务器检索或发送数据。这个API提供了一个`fetch()`方法，你可以用来发出这些请求。让我们看一个如何使用`fetch`的基本示例：

```js
fetch('https://api.example.com/data')
```

在这个例子中，我们向`https://api.example.com/data`发出一个`GET`请求。这将返回一些数据，我们需要将其转换为JSON格式，然后可以在任何地方使用。

默认情况下，Fetch API使用`GET`方法来检索数据。这将在下一课中介绍，连同其他常见的HTTP请求方法。

现在，让我们讨论一些从网络获取的常见资源类型。

在我们的Web应用程序中，我们需要一些常见数据，如天气数据、职业列表数据、国家名称列表、国家代码或国家标志图标/图片。使用这些数据我们可以让我们的应用程序更加信息丰富和交互性强。多亏了Fetch API，我们可以从网络获取这些资源。

图片是一些经常获取的资源。你可能使用fetch来静态或动态加载图片（基于用户操作），并在你的Web应用程序上显示它们。

文本文件是另一种经常获取的资源类型。这可能包括配置文件、日志文件，甚至是你想在网页上显示的整个文档。

在某些情况下，你可能获取音频或视频文件。Fetch API也可以处理这些类型的资源，允许你处理各种各样的数据类型。
