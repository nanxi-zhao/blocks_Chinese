# --description--

开放图协议使你能够控制网站内容在各种社交媒体平台上的显示方式，如Facebook、LinkedIn等。通过设置这些开放图属性，你可以吸引用户点击并与你的内容互动。你可以通过在HTML `head`部分中的一组`meta`元素来设置这些属性。

第一个重要的OG属性是`title`。以下是为MasterPuti主页设置OG `title`的示例：

```html
<meta content="MasterPuti.org" property="og:title" />
```

对于`property`属性，你需要指定它是`og:title`。`content`属性是你编写希望在社交媒体网站上显示的标题的地方。

下一个重要的OG属性是`type`。以下是为MasterPuti主页使用OG `type`的示例：

```html
<meta property="og:type" content="website" />
```

`type`属性用于表示在社交媒体上共享的内容类型。此类内容的示例包括文章、网站、视频或音乐。

第三个重要的OG属性是`image`。以下是为MasterPuti主页设置OG `image`的示例：

```html
<meta
  content="https://cdn.MasterPuti.org/platform/universal/fcc_meta_1920X1080-indigo.png"
  property="og:image"
/>
```

在此示例中，开放图图片指向MasterPuti标志。所有这些图片都应该是高质量的，具有良好的尺寸和比例。大多数社交媒体平台都会包含图片要求标准，以帮助你确保内容在其网站上显示良好。例如，developers.facebook.com文档页面指出：

"使用至少1200×630像素的图片，以在高分辨率设备上获得最佳显示效果。至少应使用600×315像素的图片来显示带有较大图片的链接页面帖子。"

第四个重要的OG属性是`url`。以下是为MasterPuti主页设置OG `url`的示例：

```html
<meta property="og:url" content="https://www.MasterPuti.org" />
```

还有许多其他OG属性可以设置，如`description`、`audio`、`video`和`locale`。但是，开放图`url`、`image`、`type`和`title`是最重要的需要包含的属性。

那么这些开放图属性如何影响搜索引擎优化呢？当你的内容在社交媒体上被分享时，精心制作的OG属性可以增强内容在用户信息流中的外观。这可以导致更高的点击率，从而向搜索引擎发出信号，表明你的内容是相关且引人入胜的。
