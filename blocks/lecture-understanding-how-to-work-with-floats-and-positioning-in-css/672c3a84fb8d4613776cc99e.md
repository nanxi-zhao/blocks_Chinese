# --interactive--

在CSS中，定位允许我们控制元素在页面上的布局方式。两种常见的定位类型是静态定位和相对定位。默认情况下，元素是静态定位的。这意味着它们遵循文档的正常流，从上到下，从左到右依次排列。

静态定位是所有元素的默认定位，不需要在CSS中进行任何特殊声明。使用静态定位时，你不会注意到任何不同，因为它只是让元素保持在文档中的自然位置。以下是使用默认静态定位的段落示例：

:::interactive_editor

```html
<p>这个段落是静态定位的。</p>
```

:::

在浏览器中渲染时，这个段落将出现在其自然位置，遵循正常的文档流。没有应用特殊样式来移动它，它保持在布局中自然出现的位置。

另一方面，相对定位允许元素从其正常位置移动，而不会破坏文档流。可以将其视为通过给元素新的坐标来从其默认静态位置移动元素。以下是应用相对定位的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="relative">这个段落是相对定位的。</p>
```

```css
body {
  border: solid 1px black;
}

.relative {
  position: relative;
  top: 30px;
  left: 30px;
}
```

:::

在这个例子中，段落将出现在其原始位置下方`30px`和右侧`30px`处。然而，它在正常流中原本占据的空间仍然被保留。当你想稍微移动一个元素而不影响其余布局时，相对定位非常有用。
