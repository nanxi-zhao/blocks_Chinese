# --interactive--

绝对定位允许你将元素从正常的文档流中取出，使其独立于其他元素。当一个元素被绝对定位时，它被放置在自己的图层中，完全独立于布局中的其他所有内容。这对于创建浮动UI功能（如模态框、工具提示或下拉菜单）非常有用，这些功能可以覆盖页面上的其他元素。

默认情况下，绝对定位的元素相对于最近的已定位祖先元素放置。如果没有找到已定位的祖先元素，该元素将相对于初始包含块放置，初始包含块通常是浏览器的视口。你可以使用[top](file:///e:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/eslint-plugin-react/lib/util/pragma.js#L42-L42)、[bottom](file:///e:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/eslint-plugin-react/lib/util/pragma.js#L42-L42)、[left](file:///e:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/eslint-plugin-react/lib/util/pragma.js#L42-L42)和[right](file:///e:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/eslint-plugin-react/lib/util/pragma.js#L42-L42)属性来移动绝对定位的元素，指定它应该距离包含元素边缘多远。

例如，设置[top: 30px](file:///e:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/eslint-plugin-react/lib/util/pragma.js#L42-L42)和[left: 30px](file:///e:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/eslint-plugin-react/lib/util/pragma.js#L42-L42)将使元素距离包含块的顶部和左侧边缘`30px`。以下是如何在CSS中对元素应用绝对定位的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="positioned">绝对定位</div>
```

```css
body {
  background-color: #eeeeee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

当应用此代码时，元素将从正常的文档流中移除，并放置在距离其包含块顶部和左侧`30px`的位置。

绝对定位对于创建需要浮动在其余内容之上的元素特别有用，如覆盖层或模态窗口。然而，由于元素从文档流中移除，如果没有正确处理，它也可能创建间隙或导致其他元素坍塌在一起。
