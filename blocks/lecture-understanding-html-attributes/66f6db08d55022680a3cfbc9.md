﻿# --interactive--

HTML，全称为超文本标记语言（Hypertext Markup Language），是一种用于创建网页的标记语言。当你访问一个网站并看到诸如段落、标题、链接、图像和视频等内容时，那就是 HTML。

这是一个使用 HTML 元素的小例子。尝试在编辑器中编辑一些文本，并在预览窗口中查看更改。

:::interactive_editor

```html
<h1>Main heading element</h1>

<p>I am a paragraph element.</p>
```

:::

HTML 通过使用元素来表示网页的内容和结构。大多数元素都有一个开始标签和一个结束标签。有时这些标签被称为开始标签和结束标签。在这两个标签之间，你将拥有内容。这个内容可以是文本或其他 HTML 元素。

这是另一个段落元素的例子。将编辑器中的文本更改为 `I love coding!` 并在预览窗口中查看结果。 

:::interactive_editor

```html
<p>I am a paragraph element.</p>
```

:::

开始标签和结束标签都以左尖括号（`<`）开始，以右尖括号（`>`）结束，标签名称放在这些尖括号之间。虽然 HTML 标签名称不区分大小写，但广泛接受的约定和最佳实践是用小写书写它们。

这里是开始和结束段落标签的特写：

```html
<p>
```

```html
</p>
```

区分开始标签和结束标签的是结束标签中左尖括号后面紧跟的正斜杠（`/`）。一些 HTML 元素没有结束标签。这些被称为 void 元素。

这里是一个 void 元素的图像元素示例：

```html
<img>
```

请注意，这个图像元素没有结束标签，也没有任何内容。Void 元素不能有任何内容，只有开始标签。

有时你会看到在 `>` 之前使用 `/` 的 void 元素，如下所示：

```html
<img />
```

虽然许多代码格式化工具如 Prettier 会选择在 void 元素中包含 `/`，但 HTML 规范指出 `/` 的存在"不会将开始标签标记为自闭合，而是不必要的，没有任何效果"。

在实际开发中，你会看到两种形式，所以熟悉两者都很重要。

如果你想显示一张图片，你需要在图像元素中包含几个属性。属性是用于调整 HTML 元素行为的特殊值。

这是一个带有 `src` 属性的图像元素示例。将 `src` 属性的值更新为 `"https://cdn.MasterPuti.org/curriculum/cat-photo-app/cats.jpg"`，你会看到图像变成两只猫在平静地睡觉。

:::interactive_editor

```html
<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/running-cats.jpg" />
```

:::

`src` 属性用于指定该图像的位置。对于图像元素，最好包含另一个称为 `alt` 属性的属性。`alt` 属性用于为图像提供简短的描述性文本。

这是带有 `src` 和 `alt` 属性的图像元素示例。尝试通过将 `src` 值更新为 `"https://.MasterPuti.org/curriculum/cat-photo-app/cats.jpg"` 来破坏图像。你会看到图像消失，只显示 `alt` 文本。  

:::interactive_editor

```html
<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch." />
```

:::

那么，你可能想知道仅靠 HTML 是否足以构建一个网站。答案是：看情况。如果你正在构建一个只显示文本和图像的小型练习项目，仅 HTML 可能就足够了。但是，如果你要创建一个现代的专业网站，你需要有 HTML、CSS 和 JavaScript。

HTML 用于内容和结构。CSS 用于样式设计。JavaScript 用于为你的网页添加交互性。一个很好的类比是将 HTML、CSS 和 JavaScript 与一栋完整的建筑进行比较。

HTML 代表构成墙壁的砖块、混凝土和钢筋。它是使建筑坚固的基础。CSS 代表使房屋看起来美丽的内部和外部设计。JavaScript 代表确保水电不间断供应的电气和水系统。
