# --description--

让我们来学习一些可以用来提高工作效率的命令行快捷键。

终端和命令行快捷键可以帮助你简化工作流程并最大化工作效率。对于Linux和macOS，它们都可以追溯到Unix，许多这些快捷键将是相同的。但对于Windows，会有一些差异。

第一个快捷键是向上箭头键。按下这个键将允许你逐个向后浏览命令历史记录。向下箭头键则可以用来向前浏览命令历史记录。这两个键允许你快速浏览之前运行的命令，找到需要再次运行的命令。

许多shell还提供自动完成功能，根据你开始输入的内容提出命令建议。tab键可以用来填充剩余的建议，快速用完整语法填充你的命令行。这些建议的生成方式因shell而异。例如，在zsh中这些建议基于你最近的命令历史。但在PowerShell中，它们基于可用的命令、变量和参数（较少考虑最近使用的时间）。

有时命令可能会在终端上产生大量输出。这些额外的干扰可能会妨碍你的工作，或破坏你的工作流程。在Linux和macOS设备上的*nix基础终端中，你可以按Control + L清除整个屏幕并开始一个新的干净提示符。在PowerShell中，你需要运行[cls]命令（你可以将其绑定到Control + L等组合键）。

如果你需要中断命令以停止其执行，可以使用Control + C来终止进程。这将终止命令并创建一个新的提示符，让你可以继续处理其他需要处理的事情。对于PowerShell用户，Control + C也用于复制文本 - 只有在上下文不明确时（例如没有选择要复制的文本时）才会用于终止进程。

接下来的几个快捷键只在*nix基础终端中可用，PowerShell没有替代方案。

有时你可能需要多任务处理，允许进程或命令在后台运行，而你处理其他事情。按Control + Z将当前进程置于后台任务中并返回到命令行，你可以在那里继续工作。当你需要将注意力转回后台任务时，可以使用[fg]来恢复它。

最后，你可能需要连续运行两次命令。你可以按向上箭头键向后浏览到之前运行的命令，但你也可以在命令行中输入两个感叹号（[!!]）并按回车 - 这将自动运行最后执行的命令。

许多终端提供更多的快捷键，我鼓励你阅读特定设置的文档，找到最适合你的快捷键。但这些基本的通用快捷键应该是一个很好的起点，帮助你最大化工作效率并成为终端向导。
