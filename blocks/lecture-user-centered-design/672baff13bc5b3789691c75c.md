# --interactive--

**注意**：一些交互式示例可能使用了你尚未学习的CSS。不用担心理解所有代码。示例的目标是向你展示这些设计概念的预览，以便你更好地理解工作原理。

在网页设计方面，你可以使用许多类型的导航辅助工具。示例包括顶部导航栏、侧边栏和页脚。但如果你的网站比较复杂，导航层级较深，你可能需要考虑使用面包屑导航。

面包屑导航是一种导航辅助工具，向用户显示他们在网站层级结构中的位置。以下是电子产品网站模拟示例中的面包屑导航外观：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  首页 / 电子产品 / 手机 / 智能手机XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

在大多数网站中，面包屑导航显示在页面顶部，向用户显示他们到达当前页面的路径。从[首页]开始，用户导航到[电子产品]类别，然后到[手机]类别，最后到[智能手机XYZ]产品。你在搜索产品或特定信息时可能已经与网站上的面包屑导航进行过交互。

使用面包屑导航很有帮助，因为它可以帮助用户了解他们在网站层级结构中的位置以及如何导航回之前的页面。当用户来自搜索结果或外部链接并需要了解他们所在页面的上下文时，这尤其有用。

在设计面包屑导航时，有几个需要考虑的因素。第一个是决定分隔符是什么。分隔符是分隔不同层级的字符。常见的分隔符包括大于号（[>]）、右角引号（[»]）和正斜杠（[/]）。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  首页 &gt; 电子产品 &gt; 手机 &gt; 智能手机XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

第二个考虑因素是面包屑导航的位置。面包屑导航通常放置在页面顶部，要么在主导航栏上方，要么在其下方。用户不应该费力寻找面包屑导航，所以确保它们可见且易于找到。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<nav class="main-nav">
  <a href="#">首页</a>
  <a href="#">电子产品</a>
  <a href="#">手机</a>
</nav>
<div class="breadcrumbs">
  首页 &gt; 电子产品 &gt; 手机 &gt; 智能手机XYZ
</div>
```

```css
.main-nav {
  background-color: #333;
  padding: 1rem;
}

.main-nav a {
  color: white;
  margin-right: 1rem;
  text-decoration: none;
}

.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

另一个考虑因素是面包屑导航的大小。你需要确保面包屑导航足够大以便轻松阅读，但不要太大以至于占用页面上太多空间。记住，面包屑导航不是作为主要导航工具，而是作为次要导航辅助工具。在页面上信息量很大的网站中，用户可以轻松看到他们在层级结构中的位置，并使用面包屑导航导航回之前的页面。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  首页 &gt; 电子产品 &gt; 手机 &gt; 智能手机XYZ
</div>
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</p>
```

```css
.breadcrumbs {
  font-size: 1rem;
  color: #555;
  margin: 1rem 0;
}   

p {
  font-size: 1.2rem;
  line-height: 1.6;
}
```

:::

那么你应该在什么时候在网站上使用面包屑导航呢？如果你的网站很简单且层级结构较浅，面包屑导航可能没有必要。浅层结构意味着网站只有几个导航层级，所以面包屑导航可能不会增加太多价值。通常，你会在电子商务网站、新闻网站和其他具有深层结构的网站（如技术文档网站）上看到面包屑导航。
