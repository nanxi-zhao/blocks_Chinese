# --interactive--

**注意**: 一些交互式示例可能使用了你尚未学习的 CSS。不用担心理解所有代码。示例的目的是向你展示这些设计概念的预览，以便你更好地理解它们的工作原理。

视觉层次是指你布局和显示页面内容的方式，以引导观看者的注意力。

强大的层次结构可以为眼睛提供清晰的路径，确保你传达的信息能按照你预期的顺序被接收。

让我们考虑一个基本的页面布局，其中页面的 HTML 在语义上是正确的，但应用的样式并未创建出强烈的视觉层次。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    color: #333;
    background-color: #fff;
  }

  header, nav, main, section, footer {
    padding: 10px;
    margin-bottom: 10px;
    border: 1px solid #ccc;
  }

  h1, h2, h3 {
    font-weight: normal;
    font-size: 16px;
    margin: 5px 0;
  }

  nav a {
    text-decoration: none;
    color: #333;
    margin-right: 10px;
    font-size: 14px;
  }

  p {
    font-size: 14px;
    margin: 5px 0;
  }
</style>

<header>
  <h1>我的网站</h1>
</header>

<nav>
  <a href="#">首页</a>
  <a href="#">关于</a>
  <a href="#">服务</a>
  <a href="#">联系</a>
</nav>

<main>
  <section>
    <h2>欢迎</h2>
    <p>这是主页的欢迎部分。</p>
  </section>

  <section>
    <h2>我们的服务</h2>
    <p>在这里我们描述我们提供的服务。</p>
  </section>

  <section>
    <h2>联系我们</h2>
    <p>联系我们获取更多信息。</p>
  </section>
</main>

<footer>
  <p>&copy; 2025 我的网站</p>
</footer>
```

:::

如果字体大小没有区别，尽管内容通过标题分隔，但文档流没有明显的视觉指示。

要创建视觉层次，你应该对标题层级应用不同的字体大小。你也可以使用像"标注框"这样的元素来突出显示特定部分。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    line-height: 1.6;
    padding: 20px;
    background: #f9f9f9;
    color: #333;
  }

  h1 {
    font-size: 32px;
    margin-bottom: 10px;
  }

  h2 {
    font-size: 24px;
    margin-top: 20px;
    margin-bottom: 8px;
  }

  h3 {
    font-size: 18px;
    margin-top: 15px;
    margin-bottom: 6px;
  }

  p {
    font-size: 16px;
    margin-bottom: 12px;
  }

  .callout {
    background-color: #fff3cd;
    border-left: 5px solid #ffc107;
    padding: 15px;
    margin: 20px 0;
  }
</style>

<h1>理解视觉层次</h1>
<p>视觉层次通过引导用户的注意力来帮助用户浏览和理解内容。</p>

<h2>标题层级</h2>
<p>使用不同的字体大小创建结构，使内容更易于扫描。</p>

<h3>三级标题</h3>
<p>这个较小的标题进一步细分章节，而不会压倒主要内容。</p>

<div class="callout">
  <strong>提示：</strong>使用这样的标注框来突出显示重要笔记或关键要点。
</div>
```

:::

视觉层次还可以帮助提高用户转化率。例如，你可以利用标注框进一步吸引用户注意行动号召（CTA）按钮。
