# --interactive--

**注意**: 一些交互式示例可能使用了你尚未学习的 CSS。不用担心理解所有代码。示例的目的是向你展示这些设计概念的预览，以便你更好地理解它们的工作原理。

"比例"是指某物的大小。

在网页设计中查看比例时，你关注的是不同元素之间的大小关系，以及这些元素如何适应不同的屏幕尺寸。

在元素中使用正确的比例在视觉层次中起着重要作用。较大的元素会吸引更多的注意力，这可以按照你想要的方式引导用户浏览内容。

例如，标题和段落之间的视觉分离会吸引读者的注意力，但比例应该适当，以获得引人注目的文本，将读者吸引到该部分。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 20px;
    background-color: #fdfdfd;
    color: #333;
  }

  .section {
    margin-bottom: 40px;
  }

  .big-heading {
    font-size: 40px;
    font-weight: bold;
    margin-bottom: 12px;
  }

  .paragraph {
    font-size: 16px;
    max-width: 600px;
    line-height: 1.6;
  }
</style>

<div class="section">
  <div class="big-heading">发现智能设计的力量</div>
  <p class="paragraph">
    优秀的设计不仅仅关乎颜色或字体——它也关乎比例。
    像这样的大标题会立即吸引你的注意力，而下面的段落则提供上下文和细节。
    正确的比例创建了清晰的视觉路径，引导读者的眼球跟随。
  </p>
</div>
```

:::

不过，比例不仅适用于文本。对于图像也很重要。虽然横幅图像的比例在桌面布局中可能有意义，但在移动布局中可能太大。

通过将图像缩小到更合适的比例，你可以保持视觉冲击力，同时确保网站上的信息可见。

:::interactive_editor

```html
<style>
  body {
    font-family: sans-serif;
    padding: 20px;
    background-color: #fefefe;
    color: #333;
  }

  .banner {
    max-width: 100%;
    height: auto;
    display: block;
    margin: 0 auto 20px auto;
    border-radius: 8px;
  }

  .content {
    max-width: 600px;
    margin: 0 auto;
    font-size: 16px;
    line-height: 1.6;
  }

  @media (max-width: 600px) {
    .banner {
      max-width: 90%;
    }

    .content {
      font-size: 15px;
    }
  }
</style>

<img 
  src="https://placehold.co/1200x400/png?text=Large+Banner+Image"
  alt="横幅" 
  class="banner"
>

<div class="content">
  <p>
    这个横幅图像在大屏幕上看起来很棒，但在较小的设备上会自动缩小。
    这样，它仍然能产生强烈的视觉印象，而不会将实际内容推到屏幕之外。
    正确缩放图像有助于在不同布局之间保持平衡和可访问性。
  </p>
</div>
```

:::

比例对于交互性和实际使用你的网站也很重要。如果导航栏中的文本比例不合适，手机用户将很难点击链接。

如果你适当地调整比例，你最终会得到不仅更容易阅读，而且对移动用户来说更容易点击的链接。

:::interactive_editor

```html
<style>
  body {

    margin: 0;
    background-color: #fafafa;
  }

  .navbar {

    padding: 15px 20px;
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
  }

  .nav-link {
    color: white;
    text-decoration: none;
    font-size: 18px;
    padding: 10px 15px;
    display: inline-block;
    border-radius: 4px;
  }

  .nav-link:hover {
    background-color: #0059b3;
  }

  @media (max-width: 600px) {
    .nav-link {
      font-size: 20px;
      padding: 14px 18px;
    }
  }
</style>

<nav class="navbar">
  <a href="#" class="nav-link">首页</a>
  <a href="#" class="nav-link">关于</a>
  <a href="#" class="nav-link">服务</a>
  <a href="#" class="nav-link">联系</a>
</nav>
```

:::

比例在你的设计中有许多重要作用。我们已经涵盖了基础知识，所以你现在应该对它的重要性有了基本的了解。
