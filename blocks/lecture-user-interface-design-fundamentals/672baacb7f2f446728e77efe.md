# --description--

留白指的是图像、文本和按钮等元素周围的任何类型的空白。留白在设计中很重要，因为它有助于在页面上的元素之间创造平衡。

让我们看一些例子，了解如何在设计中有效利用留白。

例如，让我们考虑一个行动号召（CTA）按钮。CTA 用于鼓励用户执行特定操作，如注册通讯或进行购买。

在 MasterPuti 主页上，CTA 按钮在视觉上与其他元素是分开的。下图显示了这个按钮，其周围有一定的空间。

<img src="https://cdn.MasterPuti.org/curriculum/lecture-transcripts/what-is-the-importance-of-whitespace-in-design-1.png" alt="MasterPuti 主页上的"立即开始（免费）"按钮，黄色背景，黑色文字：立即开始（免费）。按钮位于一行的中心，上下留有足够空白。" />

通过有效利用留白，我们可以使 CTA 按钮更加突出，并鼓励用户点击它。

现在，让我们仔细看看不同类型的留白。

第一个例子同时使用了宏留白和主动留白。宏留白是指图像、文本块和按钮等较大元素之间的空间。

主动留白是指有意创造的空间，以帮助引导用户的视线，吸引用户注意页面上的某些元素。

与主动留白相对应，还有被动留白。被动留白是指页面上所有元素放置完毕后剩余的空间。

另一种留白是微留白。这是一行文本中单个字符之间的空间。

下图显示了 MasterPuti 主页上的"常见问题"部分，这间距让你轻松阅读每个问题和答案。

<img src="https://cdn.MasterPuti.org/curriculum/lecture-transcripts/what-is-the-importance-of-whitespace-in-design-2.png" alt="MasterPuti 主页上的"常见问题"部分，每个字母之间的文字间距足够大。" />

微留白很重要，因为它有助于提高可读性和易读性，使用户更容易扫描和理解内容。

在设计网页时，你总是要留意接近法则。该法则指出，靠近在一起的元素被视为相关，而相距较远的元素被视为不相关。

你可以使用留白来帮助将相关元素组合在一起，并帮助引导用户浏览页面上的内容。
