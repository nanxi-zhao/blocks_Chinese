# --interactive--

CSS，全称层叠样式表（Cascading Style Sheets），是现代网页开发的重要组成部分。它是一种标记语言，用于为HTML应用样式。简单来说，如果HTML是网页的结构，那么CSS就是让它看起来美观的部分。

下面是一个导航栏样式的例子。

**注意**：现在还不用担心理解CSS代码。在未来的课程和研讨会中，你会学到这些是如何工作的。这只是让你了解一下用一点点CSS能做些什么。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">首页</a></li>
        <li><a href="#">关于我们</a></li>
        <li><a href="#">服务</a></li>
        <li><a href="#">联系我们</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

CSS的主要作用是将网页的内容与其视觉呈现分离开来。这种分离使网页开发者能够创建更灵活、更易维护的网站。

通过CSS，你可以控制网页的布局、颜色、字体和整体视觉外观，而无需更改HTML结构。

让我们考虑一个简单的类比。如果你把网站看作一栋房子，HTML就是地基和框架，而CSS则是油漆、壁纸和装饰品，让房子在视觉上吸引人且独特。

CSS通过选择HTML元素并对其应用样式来工作。

下面是一个展示未样式化[按钮]元素和样式化按钮的例子。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">未样式化按钮</button>
<button class="round-btn">圆角按钮</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

这些样式可以包括颜色、字体大小等属性。通过改变这些属性，你可以在不更改内容的情况下大幅改变网页的外观。

CSS最强大的方面之一是其创建响应式设计的能力。

这意味着通过CSS，你可以让你的网站在任何设备上看起来都很棒，无论是台式电脑、平板电脑还是智能手机。

CSS允许你根据查看网站的设备屏幕尺寸调整布局、字体大小和其他视觉元素。

尝试调整预览窗口的大小，看看布局如何适应屏幕尺寸。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>欢迎来到我的响应式网站</h1>
    <p>这个布局会适应你的屏幕尺寸！</p>
</header>

<main class="content">
    <section class="box">盒子1</section>
    <section class="box">盒子2</section>
    <section class="box">盒子3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* 响应式：适用于平板电脑及以下设备 (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* 响应式：适用于手机 (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

CSS的另一个重要特性是它的层叠性，这也是其名称中"层叠"的由来。

这意味着样式可以被继承和覆盖，从而实现样式的层次结构。在未来的课程中，你会学到更多关于它是如何工作的。

CSS还支持使用外部样式表。

这意味着你可以将所有的样式规则保存在一个单独的文件中，然后将其链接到多个HTML页面。

这一特性大大增强了网站的可维护性，特别是对于较大的网站。你不必在每个页面上逐一更改样式，只需在一个CSS文件中进行更改，就能影响所有链接的页面。

在网页开发领域，CSS在创建视觉吸引力强、响应迅速且用户友好的网站方面起着至关重要的作用。

它使开发者能够将简单的HTML文档转变为引人入胜的网络体验，吸引用户的注意力并增强他们与网页内容的互动。
