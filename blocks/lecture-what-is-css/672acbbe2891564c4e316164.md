# --interactive--

CSS负责网页的样式。所有这些样式都由各种CSS规则组成。

CSS规则由两个主要部分组成：**选择器**和**声明**块。

让我们来看看基本语法：

```css
selector {
  property: value;
}
```

选择器是CSS中用于识别和定位特定HTML元素以进行样式设置的模式。

选择器的例子包括类型选择器、类选择器和ID选择器。

基本语法中提供的花括号被称为声明块。声明块为给定的选择器应用一组样式。

在声明块内，你会有一系列声明。每个声明由一个属性和一个值组成。

属性是CSS标识符，用于指定正在设置样式的特征。属性的一个例子是[background-color]属性。

值是应用于该属性的特定设置。例如，如果属性是[background-color]，值可以是[purple]，将背景颜色设置为紫色。

在每个属性名称后，你需要放置一个冒号，而在每个值后，你应该有一个分号。

现在你已经了解了CSS规则的语法，让我们来看一个例子。点击交互式编辑器中的[styles.css]选项卡查看CSS代码。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>示例段落元素</p>
<p>另一个示例段落元素</p>
```

```css
p {
  color: blue;
}
```

:::

在这个CSS规则中，类型选择器定位页面上的所有段落元素。

在声明块内，有一个声明。

该声明由[color]属性组成，值设置为[blue]。这将把所有段落元素的文本颜色更改为[blue]。

如果你想将相同的样式集应用到多个选择器，你可以创建一个选择器列表，每个选择器用逗号分隔。

下面是样式化多个选择器的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">示例标题</h1>
<h2 class="subheading">示例子标题</h2>
<p>这个段落不受选择器影响。</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

在这个选择器列表中，有一个[id]选择器定位[id]值为[title]的HTML元素。所有[id]选择器必须以井号[#]符号开头。

然后是一个逗号，后面跟着一个[class]选择器，定位[class]值为[subheading]的所有HTML元素。所有类选择器必须以点[.]开头。

整个选择器列表将接收相同的样式，文本颜色设置为[navy]。
