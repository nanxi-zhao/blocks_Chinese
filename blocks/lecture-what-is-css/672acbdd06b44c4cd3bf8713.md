# --interactive--

当你开始使用HTML和CSS时，你会注意到即使在编写任何CSS之前，一些样式就已经应用到你的网页上了。这些样式被称为"浏览器默认样式"或"用户代理样式"。

这些浏览器默认样式提供基本格式，以确保HTML元素在所有浏览器中以可读的方式显示。

然而，这些样式在不同浏览器之间可能略有差异，这就是为什么在为网站设计一致外观时理解它们很重要。

让我们来看看应用到各种HTML元素的一些常见浏览器默认样式。

标题是使用最广泛的HTML元素之一，默认样式具有不同的大小和粗细。

例如，[h1]（最高级标题）通常比[h2]、[h3]等低级标题更粗更大。

这些标题标签有助于定义内容的层次结构。

以下示例显示了从[h1]到[h6]的所有六个标题元素。

:::interactive_editor

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

:::

另一个具有默认样式的是[hr]元素，它创建一条水平线，通常用于在视觉上分隔内容部分。浏览器通常对此元素应用简单的线条样式。

为了更好地理解，让我们看这个代码示例：

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<hr>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

水平线显示为一条细线，文本上下有间距，以区分内容的不同部分。

接下来，我们将看看[blockquote]元素。

块引用用于表示从其他来源引用的文本部分。浏览器通常添加缩进，有时会使文本倾斜。

缩进有助于将块引用与其余文本区分开来，明确表示此内容是从其他来源引用的。

让我们看看这个代码示例：

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<blockquote>I think, therefore I am. (Rene Descartes)</blockquote>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

在上面的示例中，[blockquote]元素将对文本产生缩进，这是由浏览器默认样式应用的。

另一个具有默认样式的是锚元素（[<a>]），它默认以蓝色和下划线样式显示，使其可识别为链接。

让我们看看以下HTML示例：

:::interactive_editor

```html
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<a href="https://masterputi.com/">访问MasterPuti学习页面</a>
<p>Paragraph 3</p>
<p>Paragraph 4</p>
```

:::

上面的代码有四个段落元素，中间有一个锚元素。锚元素以蓝色和下划线样式显示，引导用户前往MasterPuti学习页面。

最后，我们将看看无序列表和有序列表元素。

浏览器为列表添加基本格式，包括缩进和项目符号或数字，具体取决于你使用的是无序列表（[ul]）还是有序列表（[ol]）。

让我们看一个示例：

:::interactive_editor

```html
<p>Sample Paragraph</p>
<ol>
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ol>
<ul>
  <li>item</li>
  <li>another item</li>
  <li>yet another item</li>
</ul>
<p>Ending Paragraph</p>
```

:::

在上面的代码示例中，我们使用了无序列表和有序列表。这两个列表默认都会稍微向右缩进。

对于所有这些示例，你看到了浏览器如何为这些HTML元素应用默认间距、不同的字体大小和粗细。

在后续课程中，你将学习如何使用CSS重置来重置这些默认样式。
