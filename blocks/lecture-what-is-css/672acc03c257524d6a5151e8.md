# --interactive--

在CSS中，[width]和[height]属性用于控制网页上元素的尺寸。

这些属性可以用不同的单位定义，如像素（[px]）、百分比（[%]）、视口单位（[vw]、[vh]）等。

[width]属性指定元素的宽度。如果不指定宽度，则默认设置为[auto]。这让浏览器根据元素的内容、父元素和显示类型来确定元素的宽度。对于[div]元素，[width: auto]使其扩展以填充其父容器的完整宽度。

[height]属性指定元素的高度。类似地，高度默认为[auto]，这意味着它会根据内部内容进行调整。

下面是使用[width]和[height]属性的示例：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 100px;
      height: 100px;
      background-color: skyblue
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

在这个示例中，我们有一个名为[box]的[div]元素。这个元素将在高度和宽度上占据[100px]，背景颜色将是[skyblue]。

像素是CSS中的固定尺寸测量单位，提供对尺寸的精确控制。

[min-width]属性指定元素的最小宽度。即使内部内容更小，元素也不会缩小到此值以下。

[min-height]指定元素的最小高度。它确保元素不会变得比设定值更短。

下面的示例演示了[min-width]和[min-height]是如何工作的：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 50px;
      min-width: 100px;
      height: 50px;
      min-height: 100px;
      background-color: lightcoral;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

上面的示例演示了[min-width]和[min-height]是如何工作的。

尽管盒子的[width]和[height]设置为50px，但它实际上将是[100px]乘[100px]。这是因为[min-width]和[min-height]设置为[100px]，比指定的[width]和[height]更大。

记住，如果[min-width]或[min-height]大于[width]或[height]，它们将覆盖较小的值。这确保元素不会变得太小，这对于保持一致且可用的设计很重要。

[max-width]指定元素可以增长到的最大宽度，即使有足够空间让它更宽。

[max-height]指定元素可以增长到的最大高度，无论内容大小如何。

下面的示例演示了[max-width]和[max-height]如何覆盖[width]和[height]：

:::interactive_editor

```html
<head>
  <style>
    .box {
      width: 200px;
      max-width: 150px;
      height: 200px;
      max-height: 150px;
      background-color: lightgreen;
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
```

:::

上面的示例演示了[max-width]和[max-height]如何覆盖[width]和[height]。尽管盒子设置为[200px]乘[200px]，但它实际上将是[150px]乘[150px]。这是因为[max-width]和[max-height]设置为[150px]，这更小。

记住，当[max-width]或[max-height]小于[width]或[height]时，它们优先。这对于控制布局中元素的最大尺寸很重要。

CSS优先考虑[min-width]和[min-height]而不是[width]和[height]。[max-width]和[max-height]在值超过其限制时限制尺寸。

例如，如果将[width]设置为[600px]，将[max-width]设置为[500px]，元素将被限制为[500px]宽。[max-width]覆盖更宽的设置，使元素保持在指定的最大尺寸内。

这确保元素保持在所需的尺寸范围内，无论标准的宽度和高度值如何。
