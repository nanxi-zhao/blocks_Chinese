# --interactive--

在HTML和CSS中，元素被分类为**内联元素**或**块级元素**，这种分类决定了它们在文档布局中的行为方式。

理解这种区别对于控制内容在网页上的显示方式至关重要。

块级元素是默认占据可用全宽的元素，横跨其容器的宽度。

这些元素总是在新行开始，并将其他内容推到下一行，创建一个内容"块"。

块级元素默认应用了CSS属性[display: block;]。此属性确保元素拉伸以填充容器的宽度并在新行上出现。

一些常见的块级元素包括[div]元素、段落、标题、有序列表、无序列表和section元素。

下面是块级元素的代码示例：

:::interactive_editor

```html
<p style="border: 2px solid red;">
  First paragraph
</p>
<p>Second paragraph</p>
```

:::

在此示例中，我们有两个段落元素，其中第一个元素周围有红色边框。

这两个段落元素不在同一行上，因为它们默认是块级元素。

因此，两个段落元素都将占据其容器的全宽，在这种情况下是[body]元素。

当你希望内容垂直堆叠时，块级元素是理想的选择，例如段落、章节或较大的内容块。它们通常用于定义网页的布局和结构。

内联元素与块级元素不同，只占据所需的宽度，不在新行开始。这些元素在内容内流动，允许文本和其他内联元素与它们一起出现。

内联元素默认应用了CSS属性[display: inline;]。此属性确保元素保持在内容流中，不会换到新行。

常见的内联元素包括[span]、[anchor]和[img]元素。

下面是一个更好地理解内联元素的示例：

:::interactive_editor

```html
<p>This is a
  <span style="color: red; border: 2px solid green;">red</span>
  word inside a paragraph.
</p>
```

:::

在此示例中，我们有一个嵌套在段落元素内的[span]元素。[span]元素具有[red]文本颜色和[green]边框，以便你能更好地看到高亮的单词。

如你所见，[span]元素只占据"red"这个词的空间，不会将后续内容推到新行。

内联元素最适合用于为行内较小部分的文本或内容设置样式，例如强调一个单词、创建超链接或为段落的部分内容应用特定样式。
