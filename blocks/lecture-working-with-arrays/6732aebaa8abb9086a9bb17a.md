# --interactive--

在上一节中，你已经学习了如何使用数组以及如何访问数组中的不同元素。下面是如何访问数组第二个元素的快速回顾：

:::interactive_editor

```js
const fruits = ["apple", "banana", "cherry"];

console.log(fruits[1]); // "banana"
```

:::

由于数组使用从 0 开始的索引，第一个元素的索引是 `0`，第二个元素的索引是 `1`，以此类推。需要注意的是，如果你尝试访问数组中不存在的索引，JavaScript 会返回 `undefined`。

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[3]); // undefined
```

:::

在上面的示例中，`fruits` 数组没有索引为 `3` 的元素，因此日志会显示 `undefined`。现在我们来看看如何更新数组中的元素。你可以通过给特定索引赋新值来更新元素。

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[1] = "blueberry";
console.log(fruits); // ["apple", "blueberry", "cherry"]
```

:::

在这个示例中，我们把索引为 `1` 的 `banana` 替换为 `blueberry`。只要你知道元素的索引，就可以用这种方法更改数组中的任意元素。你也可以通过给尚不存在的索引赋值来向数组添加新元素：

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[3] = "date";
console.log(fruits); // ["apple", "blueberry", "cherry", "date"]
```

:::

不过在这样做时要小心。如果你将值赋给远大于当前数组长度的索引，会在中间创建许多 `undefined` 元素，这可能导致意想不到的行为。随着你继续学习 JavaScript，你会发现这些访问与更新数组元素的方法在许多编程任务中都是基础且非常有用的，无论是实现简单的代办列表，还是处理复杂的数据结构。
