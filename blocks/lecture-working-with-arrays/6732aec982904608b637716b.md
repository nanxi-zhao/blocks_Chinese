# --interactive--

JavaScript 中的数组是动态的，这意味着你可以方便地向数组添加或从数组中移除元素。对数组起始和末尾进行添加或移除操作的四个主要方法是：`push()`、`pop()`、`shift()` 和 `unshift()`。下面我们来详细看看这些方法。

`push()` 方法用于向数组末尾添加一个或多个元素。`push()` 的返回值是数组的新长度。下面示例将一个新水果添加到 `fruits` 数组中：

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");

## --text--

```js

arr.push(4);
arr.unshift(0);
console.log(arr);
```
