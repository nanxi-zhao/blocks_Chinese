# --interactive--

`audio` 和 `video` 元素让你在 HTML 文档中添加音频和视频内容。`audio` 元素支持流式的音频格式，如 mp3、wav 和 ogg。`video` 元素支持 mp4、ogg 和 webm 格式。

要将音频内容包含在你的网页中，你可以使用带有 `src` 属性的 `audio` 元素，该属性指向你的音频文件的位置。

正如你在预览窗口中看到的，页面上没有显示任何内容。

:::interactive_editor

```html
<audio src="https://cdn.MasterPuti.org/curriculum/js-music-player/cruising-for-a-musing.mp3"></audio>
```

:::

如果你想在页面上看到音频播放器，那么你可以添加带有 `controls` 属性的 `audio` 元素。

在预览窗口中按下播放键来听 Quincy Larson 的一首歌曲。要听不同的歌曲，请将 `src` 值更改为 `"https://cdn.MasterPuti.org/curriculum/js-music-player/never-not-favored.mp3"`。

:::interactive_editor

```html
<audio src="https://cdn.MasterPuti.org/curriculum/js-music-player/cruising-for-a-musing.mp3" controls></audio>
```

:::

`controls` 属性使用户能够管理音频播放，包括调节音量、暂停或恢复播放。`controls` 属性是一个布尔属性，添加到元素中以启用内置播放控制。如果省略，则不会显示任何控件。

除了 `src` 和 `controls` 属性外，还有其他几个属性可以增强 `audio` 元素的功能。`loop` 属性是一个布尔属性，用于使音频连续重放。

下面是一个使用 `loop` 属性播放 Quincy Larson 的歌曲 "Can't stay down" 的示例。要看到循环效果，请将播放头移动到歌曲接近结尾处，一旦播放完成，它将重新开始。

:::interactive_editor

```html
<audio
  src="https://cdn.MasterPuti.org/curriculum/js-music-player/can't-stay-down.mp3"
  loop
  controls
></audio>
```

:::

你可以使用的另一个属性是 `muted` 属性。当出现在 `audio` 元素中时，布尔属性将默认以静音状态启动音频。下面是一个使用 `muted` 属性的示例。

要听到音乐，请点击音频播放器中的音量图标。

:::interactive_editor

```html
<audio
  src="https://cdn.MasterPuti.org/curriculum/js-music-player/can't-stay-down.mp3"
  loop
  controls
  muted
></audio>
```

:::

谈到音频文件类型，不同的浏览器支持不同的类型。为此，你可以在 `audio` 元素内使用 `source` 元素，浏览器将选择它能理解的第一个源。下面是为 `audio` 元素使用多个 `source` 元素的示例：

```html
<audio controls>
  <source src="audio.ogg" type="audio/ogg" />
  <source src="audio.wav" type="audio/wav" />
  <source src="audio.mp3" type="audio/mpeg" />
</audio>
```

浏览器会首先尝试 ogg 类型，如果无法播放音频，就会下移到列表中的下一个类型。

`video` 元素也支持我们目前所学到的所有属性。下面是一个使用带有 `loop`、`controls` 和 `muted` 属性的 `video` 元素的示例。

在开头的 `video` 标签中添加 `autoplay` 属性，使视频自动播放。

**注意**: 这里使用 `width` 属性是为了让视频更小，更好地适应预览窗口。你将在未来的课程中了解更多关于 `width` 属性的知识。

:::interactive_editor

```html
<video
  src="https://archive.org/download/BigBuckBunny_124/Content/big_buck_bunny_720p_surround.mp4"
  loop
  controls
  muted
  width="400"
></video>
```

:::

对于 `src`，或源属性，我们使用的是 archive.org 上名为 "Big Buck Bunny" 的视频。如果你想在加载视频时显示图像，可以使用 `poster` 属性。此属性不适用于 `audio` 元素，是 `video` 元素独有的属性。下面是一个使用 `poster` 属性的示例，内容由 peach.blender.org 提供。

```html
<video
  src="https://archive.org/download/BigBuckBunny_124/Content/big_buck_bunny_720p_surround.mp4"
  loop
  controls
  muted
  poster="https://peach.blender.org/wp-content/uploads/title_anouncement.jpg?x11217"
  width="400"
></video>
```
