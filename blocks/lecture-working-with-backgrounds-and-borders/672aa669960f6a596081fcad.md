# --interactive--

在 CSS 中处理背景图像时，你可以使用一些属性来控制图像的显示方式，让页面看起来既美观又有趣。

本节我们将重点介绍 `background-size`、`background-repeat`、`background-position` 和 `background-attachment` 这几个属性，掌握它们会让你的布局更灵活、更有表现力。

先从 `background-image` 属性开始了解吧：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::


上面的 CSS 为 `body` 元素设置了一张猫咪背景图。

如果你想设置背景图的大小，可以使用 `background-size` 属性。

使用 `contain` 时，图像会在不被裁切或拉伸的前提下尽可能放大，从而完整显示图片内容。

下面是一个使用 `background-size: contain` 的示例：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::


我们将 `min-height` 设置为 `100px`，这样背景图在内容很少时也能可见，整体布局看起来更平衡、更讨喜。

如果把 `background-size` 改为 `cover`，背景图会按比例缩放并覆盖整个 `body` 元素，可能会裁切图片的部分区域以适应画面。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::


上面的示例中，你可能注意到背景图会不断重复。

默认情况下，背景图会在水平方向和垂直方向上重复以填充整个元素，但你可以通过属性来控制这种行为。

使用 `background-repeat` 并设置为 `no-repeat` 可以禁止重复显示。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::


当把 `background-size` 设为 `contain` 且 `background-repeat` 设为 `no-repeat` 时，图片将不再重复显示。

如果你希望背景图只在水平方向重复，可以把 `background-repeat` 设为 `repeat-x`。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::


如果希望在垂直方向重复背景图，则使用 `repeat-y`。

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::


要在页面上定位背景图，可以使用 `background-position` 属性。

`background-position` 用来设置图像在元素中的显示位置，既可以使用关键字（如 `top`、`bottom`、`left`、`right`、`center`），也可以使用像素或百分比值进行精确控制。

下面示例将背景定位在水平居中、垂直靠上：

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::


上面的 CSS 会把背景图水平居中、垂直靠上显示。

最后，`background-attachment` 决定背景图是在页面滚动时随内容移动，还是固定在屏幕上不动。

常见取值有 `scroll`（默认，随内容滚动）和 `fixed`（固定在视口，不随滚动移动）。

Here is an example using the `fixed` value for the `background-attachment` property:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::


上面的 CSS 会让背景图即使在页面滚动时也保持固定位置，产生漂亮的视觉效果。

你也可以使用 `background` 简写属性把多个设置合并到同一行，写法更简洁。

下面是一个示例：

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::


上面的例子把 `background-image`、`background-position` 和 `background-attachment` 合并到了同一行。

掌握这些属性后，你就能轻松控制背景图的显示效果，让网页既美观又实用。继续加油，你已经在变得越来越棒了！
