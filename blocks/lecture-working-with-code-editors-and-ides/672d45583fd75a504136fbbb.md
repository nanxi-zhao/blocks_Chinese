# --description--

要安装 VS Code，你需要访问他们的网站以获取适合你操作系统的安装程序。确保你访问的是 VS Code 网站，而不是 Visual Studio 网站。

对于 Windows，你可以下载一个 EXE 安装程序。下载完成后，运行该文件，安装向导将引导你完成安装过程。

对于 Mac，你可以下载一个包含 `.app` 的 zip 存档，你可以运行它。你也可以通过 homebrew 安装 VS Code。

在 Mac 上打开终端应用程序并运行以下命令：

```sh
brew install --cask visual-studio-code
```

在 Linux 上，你可以直接下载 `.deb` 或 `.rpm` 文件。安装完成后，你可以使用适当的可执行文件或 CLI 命令（通常是 `code`）来启动应用程序。现在你已经准备好开始编码了。
