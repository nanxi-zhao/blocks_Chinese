# --interactive--

在使用 CSS 设计网页时，定义颜色的最常见方法之一是使用十六进制颜色值。十六进制颜色值，或十六进制代码，是在 RGB 色彩模型中表示颜色的简洁方式。在本课中，我们将探讨什么是十六进制代码，它们在 CSS 中是如何工作的，以及何时使用它们。

十六进制代码（十六进制代码的简称）是一个六字符的字符串，用于表示 RGB 色彩模型中的颜色。"十六进制"指的是十六进制数系统，使用数字 `0` 到 `9` 和字母 `A` 到 `F`。

在颜色的上下文中，十六进制代码指定组成特定颜色的红、绿、蓝（RGB）的数量。每个十六进制代码以井号（`#`）开头，后跟六个字符，定义颜色中红、绿、蓝的强度。

例如，黑色的十六进制代码是 `#000000`，白色的是 `#FFFFFF`。前两个字符代表红色值，接下来的两个代表绿色，最后两个代表蓝色。

在 CSS 中，你可以通过使用十六进制代码来为元素应用颜色。以下是基本语法：

```css
element {
  color: #RRGGBB;
}
```

在这里，`#RRGGBB` 是实际十六进制代码的占位符。`RR`、`GG` 和 `BB` 对分别代表红、绿、蓝的强度。

这些对可以从 `00`（最低强度）到 `FF`（最高强度）不等。数字越高，最终混合中的该颜色就越多。例如，让我们为一些元素应用十六进制颜色：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>这是一个标题。</h1>
<p>这是一个段落。</p>
```

```css
h1 {
  color: #FF5733; /* 一种红橙色 */
}

p {
  background-color: #4CAF50; /* 一种绿色 */
}
```

:::

在第一种情况下，`#FF5733` 是一种红橙色。分解如下：

- `FF` 是红色的最高值（完全饱和）。
- `57` 是适量的绿色。
- `33` 是少量的蓝色。

在第二种情况下，`#4CAF50` 是一种绿色：

- `4C` 是红色的低值。
- `AF` 是绿色的强值。
- `50` 是蓝色的中等值。

十六进制代码很受欢迎，因为它们精确且易于使用。它们允许你微调颜色，非常适合需要一致性的设计。

大多数设计软件，如 Adobe Photoshop 或 Figma，为所选颜色提供十六进制代码，这使得直接将这些值复制粘贴到 CSS 文件中变得容易。

在 Web 开发中，十六进制代码通常用于定义文本颜色、背景、边框和其他视觉元素。例如：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>示例</button>
```

```css
body {
  background-color: #F0F0F0; /* 浅灰色背景 */
}

button {
  background-color: #007BFF; /* 一种蓝色 */
  color: #FFFFFF; /* 白色文本 */
}
```

:::

这确保了你的颜色在整个网站或应用程序中保持一致。

在某些情况下，你可能会注意到用三个字符而不是六个字符编写的十六进制代码的简写格式。当每种颜色对中的两个字符相同时，这是可能的。例如，`#FF5733` 不能缩短，但 `#FFFFFF`（白色）可以写成 `#FFF`。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>这是一个标题。</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* 白色的简写 */
}
```

:::

简写 `#FFF` 等同于 `#FFFFFF`。同样，`#000` 代表黑色，完整形式是 `#000000`。

十六进制代码因其精确性和兼容性而受欢迎，允许在所有浏览器中使用超过 1600 万种颜色。当你需要特定的品牌颜色、想要控制命名颜色或 HSL 不易定义的确切色调，或者当你从默认提供十六进制代码的设计软件复制颜色时，它们是理想的选择。
