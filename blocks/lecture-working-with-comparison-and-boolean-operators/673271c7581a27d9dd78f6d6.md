# --interactive--

在前面的课程中，您首次接触了布尔值的概念，但在本课中，我们将深入探讨布尔值的工作原理以及相等和不相等运算符的工作方式。

布尔值是一种只有[true](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)和[false](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)值的数据类型。它们很有用，因为您可以根据某些条件来做一些事情。当您想要评估某事是否应该发生时，布尔值是必不可少的，比如决定某人是否可以访问您应用程序中的某个功能。这是一个将值[true](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)设置给名为[isOldEnoughToDrive](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)的变量的示例：

:::interactive_editor

```js
let isOldEnoughToDrive = true;

console.log(isOldEnoughToDrive); // true
```

:::

您可以像这样在条件语句中使用这个变量：

:::interactive_editor

```js
let isOldEnoughToDrive = true;

if (isOldEnoughToDrive) {
 console.log("You're old enough to drive"); // You're old enough to drive
} else {
 console.log("Sorry, you are not old enough to drive");
}
```

:::

条件语句帮助您根据条件在代码中做出决策。这个示例使用了所谓的[if/else](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L31-L35)语句。

如果[isOldEnoughToDrive](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)是[true](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)，那么句子"You're old enough to drive"将被记录到控制台。否则，如果[isOldEnoughToDrive](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)是[false](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)，那么句子"Sorry, you are not old enough to drive"将被记录到控制台。由于[isOldEnoughToDrive](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)变量被设置为[true](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)，第一个句子将被记录到控制台。您将在未来的课程中了解更多关于[if/else](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L31-L35)语句的内容。

要比较两个值，您可以使用相等或严格相等运算符。比较的结果将是[true](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)或[false](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)的布尔值。这是一个使用相等运算符比较字符串和数字的示例。相等运算符由双等号（[==](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L49-L49)）表示。

:::interactive_editor

```js
console.log(5 == "5"); // true
```

:::

在此示例中，JavaScript将字符串["5"](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L49-L49)转换为数字[5](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L49-L49)，然后检查它们是否相等。由于两个值现在相同，结果为true。相等运算符在检查每个值是否相等之前使用类型强制转换。

这与严格相等运算符不同，严格相等运算符不执行类型强制转换。严格相等运算符将检查类型是否相同以及值是否相同。这是一个使用严格相等运算符比较数字和字符串的示例。该运算符由三等号（[===](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L61-L61)）表示。

:::interactive_editor

```js
console.log(5 === '5'); // false
```

:::

以下比较将是[false](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)，因为字符串数据类型与数字数据类型不相同。如果您需要检查某物是否不等于另一个值，则可以使用不相等或严格不相等运算符。这是一个使用不相等运算符（[!=](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L71-L71)）比较数字与字符串的示例。

:::interactive_editor

```js
console.log(5 != "5"); // false
```

:::

在此示例中，结果将是[false](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)，因为不相等运算符首先将字符串值转换为数字，然后比较值。由于值会相同，它将返回[false](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)。如果您尝试使用严格不相等运算符，则会得到不同的结果。严格不相等运算符由感叹号后跟两个等号（[!==](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L81-L81)）表示。

:::interactive_editor

```js
console.log(5 !== "5"); // true
```

:::

结果将是[true](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)，因为严格不相等运算符不执行任何类型强制转换。由于数字[5](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L49-L49)不等于字符串["5"](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L49-L49)，结果为[true](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271c7581a27d9dd78f6d6.md#L17-L17)。

尽可能使用严格不相等和相等运算符被认为是最佳实践，因为它们不执行类型强制转换。在专业项目中，大多数时候您会看到代码库通常更喜欢这两个运算符而不是不相等和相等运算符。
