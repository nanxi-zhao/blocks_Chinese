# --description--

在我们深入之前，您应该了解CSS中的at-rule是什么。At-rules是为浏览器提供指令的语句。您可以使用它们来定义样式表的各个方面，如媒体查询、关键帧、字体等。您将在未来的课程中了解更多关于媒体查询和关键帧等概念。这次，您将学习[@font-face](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L15-L17) at-rule。

使用[@font-face](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L15-L17)，您可以通过指定字体文件、格式和其他重要属性（如粗细和样式）来定义自定义字体。这是基本语法。您可以看到[@font-face](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L15-L17)后跟一个由花括号包围的块：

```css
@font-face {
  /* Descriptors */
}
```

在花括号内，您需要包含描述符来自定义字体。让我们看看一些最常用的描述符。[font-family](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L24-L24)描述符指定您将在整个样式表中用来引用该字体的名称。例如，假设您定义了这个[@font-face rule](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L23-L25)。它定义了[font-family](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L24-L24)描述符，其值为[MyCustomFont](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L24-L24)：

```css
@font-face {
  font-family: "MyCustomFont";
}
```

在这种情况下，您需要在样式表中使用该名称，无论您想将该自定义字体族分配给与CSS选择器匹配的所有元素。例如，您将使用它作为[font-family](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L24-L24)的值：

```css
body { 
  font-family: "MyCustomFont"; 
}
```

但是为了使[@font-face](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L15-L17) at-rule有效，您还需要指定[src](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L42-L42)。这包含对字体资源的引用。它基本上是一个由逗号分隔的外部引用或本地安装的字体名称列表。它还可以包含关于字体资源格式和技术的提示。

在下面的代码中，定义了[src](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L42-L42)。值是一个由逗号分隔的URL列表，放在不同的行上以提高可读性：

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
    url("path/to/font.woff"),
    url("path/to/font.otf");
}
```

您可以调用[url()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L43-L43)函数在样式表中包含文件。在这种情况下，我们正在包含字体文件。您需要在括号和引号内写入文件路径，包括文件扩展名。为了提高可读性，您可以将每个资源写在不同的行上，但最后一个应该以分号结尾。

对于每个字体资源，您还可以指定格式。这是可选的。这是浏览器关于字体格式的提示。如果省略格式，资源将被下载，并在下载后检测格式。如果格式无效，资源将不会被下载。

可能的字体格式包括[collection](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L53-L53)、[embedded-opentype](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L53-L53)、[opentype](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L53-L53)、[svg](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L53-L53)、[truetype](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L53-L53)、[woff](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L53-L53)和[woff2](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L53-L53)。

这是带有字体格式的示例。注意我们如何在括号和引号内写入特定格式：

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
    url("path/to/font.otf") format("opentype"),
    url("path/to/font.woff") format("woff");
}
```

在此示例中，您可以看到我们指定了WOFF2格式、OpenType格式和WOFF格式。

woff代表"Web Open Font Format"。WOFF和WOFF2之间的区别是用于压缩数据的算法。OpenType是由Microsoft和Adobe开发的可缩放计算机字体格式，允许用户访问字体中的附加功能。它在主要操作系统中广泛使用。

除了指定格式外，您还可以指定字体资源的技术。这也是可选的。这是指定第二个字体资源技术的示例。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
    url("path/to/font.otf") format("opentype") tech(color-COLRv1),
    url("path/to/font.woff") format("woff");
}
```

这些是[@font-face](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd834cedccefd5939a913.md#L15-L17)规则的基础知识。使用这个at-rule，您可以指定字体文件来为您的独特设计定义自定义字体。
