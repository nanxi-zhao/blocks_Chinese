# --interactive--

在上一课中，您接触了CSS网格，它可以用来在网页中创建复杂且流畅的布局。在本课中，我们将探讨如何使用[fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226732b19aa1cacd0a75c.md#L46-L46)单位创建灵活的网格布局。

让我们从这个HTML标记开始，它将代表我们的网格容器：

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

在CSS中，我们将容器的显示属性设置为网格。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

[grid-template-columns](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226732b19aa1cacd0a75c.md#L46-L46)用于设置每列的大小。在这种情况下，每列大小将是容器的[25%](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226732b19aa1cacd0a75c.md#L46-L46)。然后[gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226732b19aa1cacd0a75c.md#L47-L47)属性用于在每列周围创建空间。

到目前为止，我们一直在使用百分比来设置列大小，但我们也可以使用[fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226732b19aa1cacd0a75c.md#L46-L46)单位。

[fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226732b19aa1cacd0a75c.md#L46-L46)单位是一个分数单位，代表网格容器空间的一部分。

这是重构后使用[fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226732b19aa1cacd0a75c.md#L46-L46)单位而不是百分比的代码样子。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

每列将占据可用空间的一个分数。由于有四列，每列将在网格容器的可用空间中占有相等的份额。

当您开始构建网格布局时，您会发现自己更经常想要使用[fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226732b19aa1cacd0a75c.md#L46-L46)单位，因为它们提供了一种灵活的比例方式来分配空间，允许您创建响应式布局，这些布局可以适应不同的屏幕尺寸，而无需手动调整像素值。
