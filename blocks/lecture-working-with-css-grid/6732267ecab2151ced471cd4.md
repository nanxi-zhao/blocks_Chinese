# --interactive--

在之前的课程中，我们稍微讨论了如何在网格项目之间创建空间。但在本课中，我们将深入探讨如何在网格布局中使用[row-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L77-L77)、[column-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L48-L48)和[gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L116-L116)属性。

但首先我们需要回顾一下CSS网格中的轨道是什么。轨道是两个相邻网格线之间的空间。当您使用CSS Grid时，这些线会自动创建。在这种情况下，轨道通常指构成网格布局的行和列。

要在CSS Grid中创建列之间的间隙，可以使用[column-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L48-L48)属性。此属性的可接受值包括像素、[em](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L14-L14)单位、百分比或[normal](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L16-L16)关键字。

如果您对[column-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L48-L48)属性使用[normal](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L16-L16)值，那么结果将是网格布局的[0](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L16-L16)和多列布局的[1em](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L16-L16)。

这是一个四列网格布局的标记示例：

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

对于CSS，我们将显示设置为网格，并将[column-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L48-L48)属性设置为10像素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

如果我们想更改示例以创建两行蓝色框并在行之间创建更多空间，可以使用[row-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L77-L77)属性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

在这个修改后的示例中，我们将[row-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L77-L77)属性设置为30像素，并更改[grid-template-columns](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L75-L75)以使用两个[1fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L75-L75)单位而不是四个来创建两行框。

就像[column-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L48-L48)属性一样，[row-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L77-L77)属性的可接受值可以包括百分比、[em](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L14-L14)和像素。

如果您想使用创建行和列之间间隙的简写方式，可以使用[gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L116-L116)属性。这是基本语法：

```css
gap: row-value optional-column-value;
```

如果您为[gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L116-L116)属性指定一个值，那么该值将应用于行和列。如果您指定两个值，那么第一个值将应用于行，第二个值将应用于列：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

[gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L116-L116)简写属性的可接受值包括百分比、像素、[em](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L14-L14)甚至[calc()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L127-L127)值。但您不能在这里使用[fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L14-L14)单位。

[row-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L77-L77)、[column-gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L48-L48)和[gap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732267ecab2151ced471cd4.md#L116-L116)属性提供了灵活的方式来控制CSS网格布局中项目之间的间距。通过使用这些属性，您可以轻松创建具有行和列之间一致且可调整间隙的视觉吸引力网格。
