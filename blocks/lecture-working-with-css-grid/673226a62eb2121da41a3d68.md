# --interactive--

[minmax()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L51-L51)函数定义网格轨道的大小范围，指定行或列可以占据多少空间。

请记住，您可以使用[px](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L12-L12)（像素）、[rem](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L12-L12)甚至[em](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L12-L12)等单位设置轨道大小，以及使用分数单位（[fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L12-L12)）。

[minmax()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L51-L51)函数通过允许您为网格轨道设置最小大小和最大大小，将事情更进一步。

这是[minmax()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L51-L51)函数的语法：

```css
minmax(min, max)
```

[min](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L19-L19)是网格轨道的最小大小，可以使用像素、百分比或[auto](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L22-L22)设置。而[max](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L19-L19)是网格轨道的最大大小，您可以使用相同的单位设置。

这两个值以这种方式协同工作：

- [min](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L19-L19)值确保网格轨道永远不会缩小到设定大小以下。

- [max](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L19-L19)值限制网格轨道可以增长的大小。

网格轨道大小根据内容和容器大小在[min](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L19-L19)和[max](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L19-L19)值之间动态调整。

让我们看一个实际的例子：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div>
    <h2>项目 1</h2>
  </div>
  <div>
    <h2>项目 2</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: minmax(150px, 300px) 1fr;
  gap: 20px;
}

.grid-container > div {
  background: crimson;
  padding: 20px;
  text-align: center;
}
```

:::

这里发生了什么？

第一个列，[minmax(150px, 300px)](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L51-L51)，将始终至少为[150px](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L51-L51)，最多为[300px](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L51-L51)，具体取决于可用空间。

另一方面，第二列，[1fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L51-L51)，将占据网格容器中任何可用的剩余空间，因为没有其他列来共享空间。

[minmax()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L51-L51)函数相对于固定大小甚至[fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226a62eb2121da41a3d68.md#L12-L12)单位的优势在于它更灵活，使其成为适应性和响应性的理想选择。
