# --interactive--

[grid-column](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L365-L365)和[grid-row](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L365-L365)属性允许您指定网格布局中网格项目的水平和垂直位置。

换句话说，它们都允许您通过引用网格线来控制网格项目开始和结束的位置。这些网格线是分隔您已经使用[grid-template-rows](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L15-L15)和[grid-template-columns](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L15-L15)属性定义的行和列的边界。

让我们来看看[grid-row](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L365-L365)和[grid-column](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L365-L365)的语法，以及一个实际的例子。

这是[grid-row](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L365-L365)属性的语法：

```css
grid-row: <start-line> / <end-line>;
```

以及[grid-column](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L365-L365)的语法：

```css
grid-column: <start-line> / <end-line>;
```

[<start-line>](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L19-L19)是项目开始的网格线，[<end-line>](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L19-L19)是项目结束的网格线。两者都是从1开始索引的，也就是说，您从1开始计数，而不是从0开始。

请记住，行的网格线是基于[grid-template-rows](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L15-L15)属性中指定的行数生成的。列也是如此，使用[grid-template-columns](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L15-L15)属性。

对于实际的例子，让我们从您同时使用[grid-template-rows](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L15-L15)和[grid-template-columns](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L15-L15)属性时浏览器生成的线开始。

这是一个有4列3行的网格示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4个相等的列 */
  grid-template-rows: repeat(3, 100px); /* 3个相等的行 */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

检查网格容器（[grid](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L53-L53)类）显示每行和每列都由两条线界定——行或列开始处的起始线，以及行或列结束处的终止线。

您可以针对这些线开始使用[grid-row](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L365-L365)和[grid-column](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L365-L365)属性来确定应该将项目放置在哪里。

以下是如何让第一个网格项目占据前两列的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4个相等的列 */
  grid-template-rows: repeat(3, 100px); /* 3个相等的行 */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

通过这个设置，您是在说第一个网格项目应该从第1列开始，在第3列之前结束。

第一个项目现在将占据两列，第四个项目被推到第二行。太棒了！

如果您还想让第一个项目占据两行，您可以指定[grid-row](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L365-L365)为[1 / 3](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226afcd33991dd751937a.md#L157-L157)：

```html
  <div class="item9">9</div>
  <div class="item10">10</div>

  <div class="item12">12</div>
</div>
```

```css

  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4个相等的列 */
  grid-template-rows: repeat(3, 100px); /* 3个相等的行 */
  gap: 10px;
}

  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::
