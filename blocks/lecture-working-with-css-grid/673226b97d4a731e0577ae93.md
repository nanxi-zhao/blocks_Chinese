# --interactive--

[grid-template-areas](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性允许您使用命名标签来设计可视化的网格布局。

然后，您使用[grid-area](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性将标签分配给特定的网格项目。换句话说，这些命名标签也称为"网格区域名称"。

以下是[grid-template-areas](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性的基本语法：

```css
grid-template-areas:
 'header header header'
 'left-sidebar main right-sidebar'
 'footer footer footer';
```

从基本语法中需要记住的要点：

- 像[header](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L53-L53)和[main](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L53-L53)这样的值是网格区域的名称。

- 字符串中的每个空格分隔值对应于一个列。

- 每个字符串代表网格中的一行。

因此，在语法中，我们有一个3乘3的网格容器。

定义模板后，您必须使用[grid-area](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性和命名标签（或区域）作为值，告诉CSS特定元素属于网格的哪个区域。

[grid-area](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性将网格项目连接到您在[grid-template-areas](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)中定义的命名区域。

展示[grid-template-areas](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性功能的一种流行方法是创建经典的圣杯布局。

如果您不知道这是什么，圣杯布局是一种网页设计模式，包含页眉、页脚、两个侧边栏和一个主要内容区域。它确保主要内容具有优先级，而侧边栏和其他部分在布局内响应式调整。

存在许多解决方案来实现圣杯布局，但使用[grid-template-areas](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)和[grid-area](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性是创建它的最直接方式。

以下是圣杯布局的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="header">
    <h2>页眉</h2>
  </div>
  <div class="sidebar-left">
    <h2>左侧边栏</h2>
  </div>
  <div class="main"><h2>主要内容</h2></div>
  <div class="sidebar-right">
    <h2>右侧边栏</h2>
  </div>
  <div class="footer">
    <h2>页脚</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-areas:
    'header header header'
    'sidebar-left main sidebar-right'
    'footer footer footer';
  grid-gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

.header {
  grid-area: header;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-left {
  grid-area: sidebar-left;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.main {
  grid-area: main;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-right {
  grid-area: sidebar-right;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.footer {
  grid-area: footer;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}
```

:::

请注意，[grid-template-areas](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)和[grid-area](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性可以独立使用。

[grid-template-areas](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性专门用于通过在网格容器内映射命名的网格区域来定义可视化布局。

另一方面，[grid-area](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性用于定位单个网格项目，可以通过指定它们的行和列位置，或通过引用使用[grid-template-areas](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/673226b97d4a731e0577ae93.md#L189-L189)属性定义的命名区域。
