# --interactive--

CSS `transform` 属性是一个强大的工具，允许你在不影响其他元素布局的情况下修改网页上元素的视觉呈现。它使你能够对元素应用各种变换，如旋转、缩放、倾斜或平移（移动）它们在2D或3D空间中的位置。

`transform` 属性通过将数学变换应用于元素的坐标系统来工作。这意味着你可以操作元素的形状和位置，同时保持其原始位置和文档流不变。

让我们探索一些常见的变换函数。这里是一个盒子元素的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
}
```

:::

我们已将 `body` 设置为具有实心黑色边框，这样你就可以看到嵌套在 `body` 元素中的 `.box` 元素。

`translate` 函数将元素从当前位置移动。这里是一个使用 `translate` 函数的更新示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 100px);
}
```

:::

这个CSS规则将把具有 `box` 类的元素从原始位置向右移动50像素，向下移动100像素。

`rotate` 函数围绕一个固定点旋转元素，这是使用 `rotate` 函数的示例，用于前面的 `.box` 元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: rotate(45deg);
}
```

:::

这将使元素顺时针旋转四十五度。

`scale` 函数允许你更改元素的大小。这是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: scale(1.5, 2);
}
```

:::

这将使元素的宽度变为原始大小的一点五倍，高度变为原始大小的两倍。

你可以在单个声明中组合多个变换：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}
```

:::

这将使元素向右和向下移动50像素，旋转45度，并缩放为原始大小的一点五倍。

虽然 `transform` 属性在创建视觉上吸引人的设计方面很强大，但在使用它时考虑可访问性很重要。以下是一些需要注意的重要可访问性问题。

屏幕阅读器可能无法准确传达变换后的内容。例如，如果你使用 `transform` 重新排列元素的视觉顺序，屏幕阅读器仍将按照原始DOM顺序读取内容。这可能会导致依赖屏幕阅读器的用户感到困惑。

当使用 `scale` 调整文本大小时，要小心不要使其过小或过大。极小的文本难以阅读，而过大的文本可能会溢出其容器并变得无法阅读。通常最好使用适当的字体样式技术来调整文本大小。

如果你使用 `transform` 进行动画效果，要注意对运动敏感的用户。过度或快速的动画可能会让一些人感到不适，甚至引发癫痫发作。考虑为用户提供减少或关闭动画的方法。使用3D变换时，请记住并非所有用户都以相同的方式感知深度。确保通过3D效果传达的任何关键信息也以2D格式或通过文本提供。

如果你使用 `transform` 来隐藏或显示内容，请确保内容对屏幕阅读器和键盘导航仍然可访问。隐藏的内容应该是真正隐藏的，例如使用 `display: none;` 或 `visibility: hidden;`，而不是仅仅在视觉上移出屏幕。

当对按钮或链接等交互元素应用 `transform` 时，确保可点击区域保持直观且易于定位。剧烈变换的按钮在视觉上可能令人困惑或难以点击，特别是对于有运动障碍的用户。

总之，虽然CSS `transform` 属性在创建视觉动态的网页设计方面很强大，但必须在考虑可访问性的情况下负责任地使用它。始终使用各种辅助技术测试你的变换元素，并考虑为可能受变换影响的信息或功能提供替代访问方式。
