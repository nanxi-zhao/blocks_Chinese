# --interactive--

CSS重置是一个样式表，用于移除Web浏览器应用于HTML元素的所有或部分默认格式。例如，你可能已经注意到某些元素，如段落和标题，即使你没有在自定义样式表中明确定义它们，也默认具有外边距。

你还会在各种HTML元素的广泛属性中看到这种情况。不同的浏览器也可能对HTML元素应用不同的默认样式。Google Chrome中的默认样式可能与Mozilla Firefox或Microsoft Edge中的不完全相同。这可能导致跨浏览器的样式不一致，你应该尽可能避免这种情况。

为了解决这个问题，CSS重置会移除默认样式。通过移除所有或部分默认样式，你可以为设计获得一致的基线，并最大限度地减少跨浏览器和设备的潜在不一致性。移除默认样式还可以使样式化过程更容易，因为你只会看到在自定义样式表中明确应用的样式。CSS重置有两种主要方法：你可以定义自定义CSS重置或使用第三方CSS重置。

自定义CSS重置是你从头开始创建的样式表，以适应项目的需求。这样你可以控制将被重置的特定样式，并具有很大的灵活性。然而，你还需要投入时间来开发和维护样式表。这是一个非常常见的CSS规则示例，用于重置所有HTML元素的外边距和内边距：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>示例标题</h1>
<p>这是一个段落。</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

它通常写在CSS样式表的顶部。星号选择器是一个通配符选择器，匹配所有HTML元素，因此它们在所有四边都有默认的外边距为零和默认的内边距为零。这将为你提供一个起点，然后你可以在样式表的下方使用更具体的CSS选择器来自定义它们。

你可以使用这种方法选择任何HTML元素并重置其默认样式，只需创建一个自定义样式表并使用适当的CSS选择器来匹配元素并设置样式。

但如果你想节省时间，这是一个耗时的过程，你也可以使用第三方CSS重置。这些样式表已经预先构建，所以你可以直接下载并将它们添加到你的项目中。一个很好的第三方CSS重置示例是`Normalize.css`。这个样式表规范化了各种HTML元素的样式，同时仍然保持一些有用的默认样式，特别是那些对可访问性很重要的样式。它还纠正了错误和样式不一致性。

另一个选择是`sanitize.css`。这是一个CSS库，你可以用来确保默认样式在所有主要现代浏览器中保持一致。这个库与`Normalize.css`一起开发，所以它们一起发展。它还有 individual stylesheets，你可以为特定目的下载，如规范化表单和排版。

有很多选项可供选择，但你应该选择最适合项目需求的选项。你也可以结合两种方法，使用第三方CSS重置和自定义重置。

在使用CSS重置时，考虑可访问性也很重要。你的Web应用程序应该对每个人都可访问。因此，你不应该重置那些可能对屏幕阅读器或其他辅助技术有帮助的样式。你还应该考虑这些额外样式表对应用程序性能的影响，因为它们必须在应用自定义样式之前下载。

通过移除默认样式，CSS重置为你提供了一个空白的起点来实现你的设计。这导致了跨浏览器和跨设备更统一和一致的用户体验。
