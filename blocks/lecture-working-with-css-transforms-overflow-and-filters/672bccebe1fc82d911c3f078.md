# --interactive--

CSS `filter` 属性是一个强大的工具，允许你对网页上的元素应用图形效果。它特别适用于调整图像、背景甚至文本的视觉呈现，而无需修改原始资源。

`filter` 属性可用于创建各种效果，如模糊、颜色转换和对比度调整。让我们讨论filter属性的工作原理并探讨一些常见示例。`filter` 属性的基本语法很简单：

```css
selector {
  filter: function(amount);
}
```

在这里，`function` 代表你想要应用的特定滤镜效果，`amount` 通常是决定效果强度的值。现在让我们看看一些常见的滤镜函数及其用途。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="一只可爱的橙色猫咪仰躺着。">
```

```css
img {
  filter: blur(2px);
}
```

:::

`blur` 函数对元素应用高斯模糊，数值以像素为单位指定，表示模糊的半径。这个CSS规则将对页面上的所有图像应用2像素的模糊效果。`blur` 效果可用于在设计中创建深度感或遮蔽图像的某些部分。

`brightness` 函数调整元素的亮度。`0%` 的值会使元素完全变黑，而超过 `100%` 的值会增加亮度。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="一只可爱的橙色猫咪仰躺着。">
```

```css
img {
  filter: brightness(150%);
}
```

:::

这个CSS规则将图像元素的亮度增加 `50%`。亮度调整可用于让图像更加突出或创建褪色效果。

`grayscale` 函数将元素转换为灰度。数值以百分比定义，其中 `100%` 是完全灰度，`0%` 保持图像不变。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="一只可爱的橙色猫咪仰躺着。">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

这个规则将图像元素转换为完全灰度。`grayscale` 可用于创建复古外观或弱化页面上的某些元素。

`sepia` 函数对元素应用棕褐色调。与灰度一样，它使用百分比值：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="一只可爱的橙色猫咪仰躺着。">
```

```css
img {
  filter: sepia(80%);
}
```

:::

这个规则对图像元素应用 `80%` 的棕褐色效果。棕褐色效果非常适合创建复古或怀旧的外观。

`hue-rotate` 函数对元素应用色相旋转。数值以度为单位定义，表示围绕色相环的旋转。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="一只可爱的橙色猫咪仰躺着。">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

这个规则将图像元素的色相旋转 `90` 度。色相旋转可用于创建迷幻效果或调整图像的整体配色方案。

`filter` 属性最强大的方面之一是能够组合多种效果。你可以通过用空格分隔来对同一元素应用多个滤镜：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="一只可爱的橙色猫咪仰躺着。">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

这个规则对图像元素应用增加的对比度、略微增加的亮度和微妙的棕褐色效果。

通过组合滤镜，你可以创建复杂而独特的视觉效果，以满足你的设计需求。CSS filter 属性是一个多功能工具，允许对Web元素进行创造性的视觉操作。

虽然我们已经涵盖了一些最常见的滤镜函数，但还有其他可用的函数，如 `contrast`、`invert` 和 `saturate`。与任何强大的功能一样，重要的是要小心使用滤镜来增强你的设计，而不要压倒用户或损害可访问性。
