# --description--

JavaScript 中的日期和时间并不总是容易处理。在专业代码库中，您可能会使用一个已经为您解决了很多这些问题的库。

但有时您需要使用 JavaScript 内置的 `Date` 对象。

在本课中，我们将介绍 `Date` 对象的基础知识以及在 JavaScript 中处理日期和时间时最常用的一些方法。

`Date` 对象是 JavaScript 中的内置对象，允许您处理日期和时间。以下是创建新 `Date` 对象的基本语法：

```js
const now = new Date();
```

`new` 关键字用于创建 `Date` 对象的新实例，然后将 `Date` 对象分配给变量 `now`。如果您将 `now` 的值记录到控制台，您会看到基于运行代码的计算机系统时钟的当前日期和时间。

如果您将 `now` 的值记录到控制台，这里是一个您会看到的示例响应：

```js
const now = new Date();
console.log(now);
// Mon Mar 15 2024 14:30:00 GMT-0700 (Pacific Daylight Time)
```

对于时间，它使用的是军事时间格式，所以 `14:30` 是 `下午2:30`。`GMT-0700` 是时区偏移量，`Pacific Daylight Time` 是时区名称。

您还可以通过提供年、月、日、时、分、秒和毫秒值作为参数，向 `Date` 对象传递特定的日期和时间。

以下是使用特定日期和时间创建新 `Date` 对象的示例：

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
// Wed Jul 04 1776 12:00:00 GMT-0700 (Pacific Daylight Time)
```

当您需要处理特定日期和时间而不是当前日期和时间时，这很有用。输入必须始终是字符串，并且格式应该能被 `Date` 对象识别。

要获取当前日期和时间，您可以使用 `Date.now()` 方法，该方法返回自 `1970年1月1日00:00:00 UTC` 以来的毫秒数。这被称为 Unix 纪元时间。

Unix 纪元时间是计算机系统中表示日期和时间的常见方式，因为它是一个可以轻松存储和操作的整数。UTC 代表协调世界时，这是世界调节时钟和时间的主要时间标准。

如果您需要根据当前日期获取月份中的某一天，可以使用 `getDate` 方法。以下是使用 `getDate` 方法的示例：

```js
const now = new Date();
const date = now.getDate();
console.log(date); // 15
```

在这个例子中，我们使用 `Date` 对象创建一个新的日期实例并将其分配给变量 `now`。然后我们在 `now` 对象上调用 `getDate` 方法来获取月份中的日期并将其分配给变量 `date`。最后，我们将 `date` 的值记录到控制台，这将输出当前月份中的日期。

`getDate` 将返回 `1` 到 `31` 之间的整数值，具体取决于月份中的日期。如果日期无效，它将返回 `NaN`（非数字）。

要获取月份，您可以像这样使用 `getMonth` 方法：

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
// 2
```

月份是从零开始的，所以 `January` 是 `0`，`February` 是 `1`，以此类推。在这个例子中，输出是 `2`，对应 `March`。如果月份无效，它将返回 `NaN`。

如果您需要获取完整年份，那么您可以像这样使用 `getFullYear` 方法：

```js
const now = new Date();
const year = now.getFullYear();
console.log(year);
// Output: 2024
```

在这个例子中，输出是 `2024`，这是当前年份。如果年份无效，它将返回 `NaN`。

`Date` 对象上还有许多其他方法，包括 `getHours`、`getMinutes`、`getSeconds` 等。我鼓励您通过 Mozilla 文档或其他资源自行探索更多内容。
