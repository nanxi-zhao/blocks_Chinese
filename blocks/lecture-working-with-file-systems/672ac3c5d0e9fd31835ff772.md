# --description--

要在 Windows 上使用资源管理器创建文件，您可以点击右上角的"新建"选项。这将显示您可以创建的不同类型文件的列表。

您也可以在相应的应用程序中创建文件。让我们创建一个文本文件来展示这个过程。

当您点击"新建"并选择文件类型时，您会在当前文件夹中看到一个带有默认名称的新文本文件。如果愿意，您可以重命名它。

创建文件夹的过程完全相同。点击右上角的"新建"按钮，然后选择"文件夹"选项，或者右键点击空白处，然后选择"新建"。接着选择"文件夹"选项。您会在当前位置看到一个新的空文件夹。

您可以通过选择文件或文件夹并点击顶部的"重命名"选项来重命名它们。当文件的当前名称被高亮显示时，您可以开始输入文件的新名称。按 Enter 键确认更改。

或者，您可以右键点击文件或文件夹并选择"重命名"选项，或者使用 F2 键盘快捷键。输入新名称并按 Enter 键确认更改。

您可以通过将文件或文件夹拖放到目标文件夹中来移动它们，前提是您已将目标文件夹作为标签页打开。

您也可以选择文件或文件夹并点击顶部的"剪切"选项。

然后，转到目标文件夹并通过点击顶部的"粘贴"按钮来粘贴它。如果您之前复制或剪切了文件，此按钮将被启用。

您也可以通过右键点击空白处然后点击粘贴图标来完成此操作。这将从原始位置移除文件或文件夹。

要删除文件或文件夹，请点击它，然后点击顶部的垃圾桶图标。如果您右键点击它，您还会看到一个"删除"选项。

很好。现在让我们看看如何在 macOS 上使用 Finder 完成这些操作。

Finder 是 macOS 上的默认文件管理器。

要在 macOS 上创建文件，您需要打开一个允许您创建所需类型文档的应用程序。目前，Finder 中没有内置的直接创建文件的方法。

例如，要创建一个简单的纯文本文件，您可以打开 TextEdit 并保存您的文件。对于任何类型的文件，这个过程都完全相同。

有很多方法可以打开 TextEdit。其中一种方法是转到 Launchpad。然后，搜索一个可以创建您感兴趣类型的文件的应用程序。在这种情况下，那就是"TextEdit"。点击图标打开应用程序。

进入应用程序后，您应该会在应用程序菜单中看到保存文件的选项。此菜单位于右上角，紧挨着苹果菜单图标。

在应用程序菜单选项之一中，您应该会看到保存文件的命令。这通常在文件菜单下。

在它旁边，您会看到保存文件的键盘快捷键。在 macOS 上通常是 `Command + S`。

当您第一次保存文件时，您应该会看到一个对话窗口，您可以在其中写入文件名并选择位置。

通常，文件扩展名会由应用程序自动添加，但您可以在保存文件之前更改它。

保存文件后，您可以通过点击 Dock 中的 Finder 图标打开 Finder。您应该会在保存文件的文件夹中看到您的新文件。

要创建新文件夹，您应该转到要创建新文件夹的位置，然后右键点击空白处。您会看到一个选项列表。第一个选项是"新建文件夹"。

如果您点击它，您可以为新文件夹分配一个名称。输入名称并按 Enter 键确认。

在 Finder 中有多种移动文件和文件夹的方法。如果您需要将它们移动到 Finder"收藏夹"部分列出的文件夹中，您可以直接拖放。

如果您需要将它们移动到不同的文件夹，您有两种选择。

您可以将目标文件夹作为标签页打开，然后将文件或文件夹拖放到标签页中。

或者您可以使用键盘快捷键。首先用 `Command + C` 复制文件，转到新文件夹，然后使用 `Command + Option + V` 将文件或文件夹粘贴到该位置。这也会从原始文件夹中移除它。

要删除文件或文件夹，您可以右键点击它并选择"移到废纸篓"。

您也可以将它们拖放到 Dock 上的废纸篓图标中。这是一样的。

当文件或文件夹在废纸篓中时，您可以右键点击它以立即删除，或者您可以清空废纸篓以永久移除所有已删除的文件和文件夹。

了解如何在 Windows 和 macOS 上创建、移动和删除文件及文件夹非常重要。通过掌握这些技能，您可以组织文件、快速定位它们，并通过删除不必要的文件来释放计算机上的存储空间。
