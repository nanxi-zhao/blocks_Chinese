# --interactive--

函数是可重复使用的代码片段，用于执行特定任务或计算值。可以把函数想象成一台机器，它接收一些输入，对其进行一些操作，然后产生输出。下面是声明函数的一个示例：

```js
function greet() {
  console.log("Hello, Jessica!");
}
```

在这个示例中，我们声明了一个名为[greet]的函数。在该函数内部，我们有一个[console.log]，它会记录消息[Hello, Jessica!]。如果我们尝试运行这段代码，控制台中不会显示该消息。这是因为我们需要调用这个函数。

函数调用或执行是指我们实际使用或执行函数的过程。要调用函数，你需要引用函数名，后面跟上一对括号：

:::interactive_editor

```js
function greet() {
  console.log("Hello, Jessica!");
}

greet(); // "Hello, Jessica!"
```

:::

现在消息[Hello, Jessica!]将被记录到控制台中。但是如果我们希望消息显示为[Hello, Nick!]或[Hello, Anna!]呢？我们不希望每次问候不同的用户时都编写一个新函数。相反，我们可以创建一个可重用的函数，它使用函数参数和实参。

参数是函数被调用时传递给函数的值的占位符。它们允许函数接受输入并与该输入一起工作。实参是在调用函数时传递给函数的实际值。下面是使用参数和实参的[greet]函数的更新版本：

:::interactive_editor

```js
function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice"); // Hello, Alice!
greet("Nick"); // Hello, Nick!
```

:::

[name]作为参数，而字符串[Alice]和[Nick]作为实参。现在我们有了一个可重用的函数，可以在我们的代码中使用不同的实参调用数十次。

当函数完成执行时，它将始终返回一个值。默认情况下，返回值将是[undefined]。这是一个示例：

:::interactive_editor

```js
function doSomething() {
  console.log("Doing something...");
}

let result = doSomething();
console.log(result); // undefined
```

:::

如果你需要函数返回特定的值，那么你需要使用[return]语句。这是一个使用[return]语句返回两个值之和的示例：

:::interactive_editor

```js
function calculateSum(num1, num2) {
  return num1 + num2;
}

console.log(calculateSum(3, 4)); // 7
```

:::

通常你会使用[return]语句，因为你可以在代码的其他地方使用函数输出的值。

到目前为止，我们一直在使用命名函数，但你也可以创建一个匿名函数。匿名函数是没有名称的函数，可以赋值给一个变量，如下所示：

:::interactive_editor

```js
const sum = function (num1, num2) {
  return num1 + num2;
};

console.log(sum(3, 4)); // 7
```

:::

在这个示例中，我们有一个名为[sum]的常量变量，并将其赋值为一个返回[num1]和[num2]之和的匿名函数。然后我们可以调用[sum]并传入数字[3]和[4]以获得结果[7]。

函数支持默认参数，允许你为参数设置默认值。如果函数在没有该参数的实参的情况下被调用，这些默认值将被使用。这是一个示例：

:::interactive_editor

```js
function greetings(name = "Guest") {
  console.log("Hello, " + name + "!");
}

greetings(); // Hello, Guest!
greetings("Anna"); // Hello, Anna!
```

:::

在这个示例中，如果没有为[name]提供实参，则默认为[Guest]。

总之，函数允许你编写可重用和有组织的代码。它们可以接受输入（参数），执行操作，并返回输出。
