# --interactive--

`map` 方法是 JavaScript 中一个强大且广泛使用的函数，它作用于数组。它旨在通过将给定函数应用于原始数组的每个元素来创建一个新数组。

此方法不修改原始数组，而是返回一个包含函数应用于每个元素结果的新数组。

以下是使用 `map` 方法处理数字数组的示例：

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

为了创建一个每个数字都加倍的新数组，我们使用 `map` 方法。`map` 方法接受一个回调函数，其中函数在数组的每个元素上调用。

在这种情况下，数组中的每个数字都将乘以 `2`。结果将是一个包含数字 `2,4,6,8,10` 的新数组。

回调函数最多可以接受三个参数。

第一个参数是正在处理的当前元素。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

第二个参数是正在处理的当前元素的索引。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

第三个参数是调用 `map` 的数组。

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

理解和有效使用 `map` 方法可以显著提高你在 JavaScript 中处理数组的能力。在未来的课程中，我们将更深入地探讨 `map` 的高级用法，并探索它如何成为构建动态和高效程序的强大工具。
