什么是 JSON，如何使用括号和点表示法访问值？

JSON 代表 JavaScript 对象表示法。它是一种轻量级的、基于文本的数据格式，通常用于在服务器和 Web 应用程序之间交换数据。

JSON 在 Web 开发中如此流行的原因之一是它既可被机器解析，又对人类可读。

由于 JSON 是语言无关的，你可以轻松地将 JSON 数据从 Java 应用程序发送到 Python 应用程序，或从 JavaScript 应用程序发送到 C# 应用程序。

JSON 支持许多数据类型，包括对象、数组、字符串、布尔值、null 和数字。

以下是 JSON 对象的示例：

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

如你所见，JSON 使用键值对来存储信息，每对之间用逗号分隔。每个键必须用双引号括起来，否则会出现错误。

要从 JSON 对象访问数据，你可以使用点表示法或括号表示法。在这个示例中，我们使用点表示法从 JSON 对象访问 `age`：

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

这个特定示例使用了所谓的 `import` 语句，它将 JSON 对象导入到此文件中，以便我们能够访问它。你将在未来的课程中了解更多关于 `import` 语句的知识。

你也可以使用括号表示法从 JSON 对象访问信息。以下是访问 `list of courses` 数组的示例：

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

在这里使用括号表示法特别有用，因为键包含多个用空格分隔的单词。如果我们尝试使用点表示法，会导致错误。

总之，JSON 是一种多功能的格式，可以存储许多数据类型，包括数组和嵌套对象。通过使用点表示法或括号表示法，你可以轻松访问 JSON 对象中存储的值。
