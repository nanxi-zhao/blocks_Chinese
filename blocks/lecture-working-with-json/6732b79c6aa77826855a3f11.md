# --interactive--

在 JavaScript 中有两种强大的方法来处理 JSON 数据：`JSON.parse()` 和 `JSON.stringify()`。这些方法通常用于在 JSON 字符串和 JavaScript 对象之间进行转换。

`JSON.stringify()` 用于将 JavaScript 对象转换为 JSON 字符串。当你想以一种可以在系统之间轻松共享或传输的格式存储或传输数据时，这非常有用。

以下是使用 `JSON.stringify()` 方法的方式：

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

`JSON.stringify()` 方法还接受一个称为替换器的可选参数，它可以是一个函数或数组。以下是使用数组作为可选替换器参数的示例：

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// 结果: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

在这个示例中，我们有一个包含四个属性的 `developerObj`。当我们使用 `JSON.stringify()` 方法时，可以为第二个参数传入一个数组，并指定我们想要字符串化的属性。结果将是一个只包含 `firstName` 和 `country` 属性的字符串化对象。

`JSON.stringify()` 方法的另一个可选参数是间隔参数。这允许你控制字符串化结果的间距：

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* 结果
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

大多数时候你不会为 `JSON.stringify()` 方法使用这些可选参数，但了解它们仍然很有帮助。

你在编程中会经常使用的另一个方法是 `JSON.parse()` 方法。`JSON.parse()` 将 JSON 字符串转换回 JavaScript 对象。当你从 Web 服务器或 `localStorage` 中检索 JSON 数据并且需要在应用程序中操作数据时，这非常有用。你将在未来的课程中了解更多关于 `localStorage` 的知识。

以下是使用 `JSON.parse()` 方法的示例：

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// 结果:
// { name: 'John', age: 30, isAdmin: true }
```

:::

这允许你将数据作为普通的 JavaScript 对象在程序中使用，使其更容易操作和使用。

在未来的模块中，我们将继续学习更多关于如何使用 JSON 以及 `JSON.parse()` 和 `JSON.stringify()` 方法的知识。
