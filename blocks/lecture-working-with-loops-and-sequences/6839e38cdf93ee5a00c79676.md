# --description--

在上一节里你已经了解了列表数据类型、元素访问和切片操作。本节我们继续学习列表的常用方法，比如 `append()`、`extend()`、`insert()`、`remove()`、`pop()`、`clear()`、`sort()`、`sorted()`、`reverse()` 和 `index()`。

`append()` 方法用于在列表末尾追加一个元素，例如把数字 `6` 添加到 `numbers` 列表：

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

如果把一个列表作为整体追加到另一个列表，会形成嵌套列表：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

```
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

如果希望把 `even_numbers` 中的每个元素逐一追加到 `numbers`，请使用 `extend()`：

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

要在特定索引插入元素，可以使用 `insert()`：

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)
print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

`remove()` 会删除第一个匹配的元素；若要按索引删除，使用 `pop()`：

```py
numbers = [1, 2, 3, 4, 5, 5]
numbers.remove(5)
print(numbers) # [1, 2, 3, 4, 5]

numbers = [1, 2, 3, 4, 5]
numbers.pop() # 删除并返回最后一个元素
```

`clear()` 用于清空列表：

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()
print(numbers) # []
```

`sort()` 会就地排序列表；`sorted()` 可用于任意可迭代对象并返回一个新的已排序列表：

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()
print(numbers) # [1, 2, 19, 35, 41, 67]

numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

`sort()` 和 `sorted()` 都接受可选参数 `key` 与 `reverse`，稍后我们学习元组时会详细讲解。

`reverse()` 会就地反转列表顺序，而 `index()` 可以查找元素首次出现的位置（未找到时抛出 `ValueError`）。

这组方法是使用列表时很实用的基础工具，掌握后会让你写出更简洁的代码。继续保持好奇，慢慢练习就会越来越顺手！

There are a few more methods for Python lists, but this initial list of methods is a good place to start.
