# --description--

元组（tuple）是 Python 中用于表示有序值序列的一种数据类型。元组可以包含不同类型的数据，例如：

```py
developer = ('Alice', 34, 'Rust Developer')
```

与列表不同，元组是不可变的（immutable），也就是说一旦创建就无法修改其中的元素。

如果尝试修改元组中的某项，会抛出 `TypeError`：

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'
"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

访问元组元素与列表类似，使用索引即可：

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

可以使用负索引从末尾开始访问，例如：

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

若索引超出范围，会抛出 `IndexError`。

也可以使用 `tuple()` 构造器把可迭代对象转换为元组：

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

同样可以使用 `in` 检查元素是否存在，并支持像列表那样的解包操作：

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

对元组也能使用切片操作来提取子序列，但不能删除或修改元素（尝试删除会报错）。

当你需要一个不会被修改的、固定长度的数据集合时，元组是很好的选择；如果需要增删改，则应使用列表。
