正如之前的模块所介绍的，循环用于重复执行一段代码。本节我们来学习 Python 中常见的循环类型以及一些搭配使用的技巧，掌握它们会让你更高效地处理序列数据。

首先是 `for` 循环，常用于遍历列表并对每个元素执行操作，例如：

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

输出为：

```md
Rust
Java
Python
C++
```

注意 `print(language)` 必须缩进在循环内部，否则会出现 `IndentationError`。

`for` 循环也可以遍历字符串等其他可迭代对象：

```py
for char in 'code':
    print(char)
```

并且可以嵌套 `for` 循环来处理多重序列：

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

另一种常用循环是 `while`，它会持续执行直到条件为 `False`，例如一个简单的猜数字游戏：

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

此外，`break` 用于立即终止循环，`continue` 用于跳过当前迭代并进入下一次循环。`for` 和 `while` 都可以配合 `else` 子句使用：当循环正常结束（未被 `break` 打断）时会执行 `else` 中的代码。

循环在 Python 中非常常见，下一些课程我们会学习如何在循环中使用 `enumerate()` 和 `range()` 等实用函数。继续练习，你会越来越熟练！
