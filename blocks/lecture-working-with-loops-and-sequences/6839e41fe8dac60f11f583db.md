# --description--

`range()` 函数用于生成整数序列，常用签名为 `range(start, stop, step)`。其中唯一必须提供的参数是 `stop`（结束值，非包含），`start` 默认是 `0`，`step` 用来指定步长。

例如：

```py
for num in range(3):
  print(num)
```

上面会打印 0、1、2（`stop` 是非包含的边界）。

如果想从 1 开始到 4，可以这样写：

```py
for num in range(1, 5):
  print(num)
```

要生成偶数序列可以使用步长参数：

```py
for num in range(2, 11, 2):
  print(num)
```

注意 `range()` 仅接受整数参数，传入浮点数会导致 `TypeError`；如果不传任何参数也会抛出 `TypeError`。

步长可为负数以生成递减序列，例如：

```py
for num in range(40, 0, -10):
  print(num)
```

还可以用 `list(range(...))` 把生成器转换为列表：

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

`range()` 是生成整数序列的便捷工具，掌握后会在很多场景中非常有用。
