上一节我们学习了元组的基本用法，本节介绍一些常用的元组方法和相关函数，帮助你更方便地查询与处理不可变序列。

首先是 `count()`，用于统计某个元素在元组中出现的次数：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

若元素不存在，`count()` 返回 `0`；传入错误的参数则会抛出 `TypeError`。

另一个常用的方法是 `index()`，用于查找某个元素首次出现的索引位置：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

若元素未找到，会抛出 `ValueError`。`index()` 也支持可选的 `start` 与 `stop` 参数，用于限制查找范围，比如：

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

对于任意可迭代对象（包括元组），`sorted()` 可以返回一个新的已排序列表：

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

`sorted()` 支持 `key` 与 `reverse` 参数，便于自定义排序逻辑或按相反顺序返回结果。

掌握这些方法后，你在处理元组时会更加得心应手。继续练习，会越来越熟练！
