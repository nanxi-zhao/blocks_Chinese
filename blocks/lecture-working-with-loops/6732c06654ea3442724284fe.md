# --interactive--

在之前的课程中，你学习了如何使用 `for` 循环、`for...in` 循环和 `for...of` 循环。在本课中，你将学习 `while` 循环和 `do...while` 循环。

`while` 循环将在条件为真时运行代码块。以下是 `while` 循环的基本语法：

```js
while (条件) {
  // 要执行的代码块
}
```

在执行代码块之前检查条件。如果条件为假，代码块将不会被执行。

当你不知道需要运行代码块多少次时，`while` 循环很有用。以下是使用 `while` 循环的示例：

:::interactive_editor

```js
let counter = 0;
while(counter < 5) {
  console.log(counter);
  counter++;
}
```

:::

在这个例子中，我们有一个名为 `counter` 的变量，初始化为 `0`。`while` 循环将继续运行，只要 `counter` 的值小于 `5`。在循环内部，我们将 `counter` 的值记录到控制台，然后将 `counter` 增加 `1`。

另一个类似于 `while` 循环的是 `do...while` 循环。这是基本语法：

```js
do {
  // 要执行的代码块
} while (条件);
```

`do...while` 循环和 `while` 循环的一个关键区别是，`do...while` 循环将在检查条件之前至少执行一次代码块。

如果条件为真，代码块将继续执行。如果条件为假，代码块将停止执行。

以下是使用 `do...while` 循环的示例：

:::interactive_editor

```js
let counter = 0;
do {
  console.log(counter);
  counter++;
} while (counter < 5);  
```

:::

在这个例子中，我们有一个名为 `counter` 的变量，初始化为 `0`。`do...while` 循环将 `counter` 的值记录到控制台，然后将 `counter` 增加 `1`。执行代码块后，它检查 `counter` 的值是否小于 `5`。如果是，循环将继续运行。如果不是，循环将停止。

在大多数情况下，你可能会比 `do...while` 循环更经常使用 `while` 循环。但是，了解这两种循环以及何时使用它们是很好的。

# --questions--

## --text--

`while` 循环和 `do...while` 循环的主要区别是什么？

## --answers--

`while` 循环至少执行一次代码块，而 `do...while` 循环则不会。
