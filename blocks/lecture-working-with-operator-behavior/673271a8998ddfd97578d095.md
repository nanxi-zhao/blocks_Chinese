# --interactive--

如果你正在处理数字并且需要将值增加或减少一，递增和递减运算符会让工作变得更简单。让我们用简单的方式分解一下。

递增和递减运算符分别用 `++` 和 `--` 表示。它们都允许你将变量的值调整 `1`。

你可以不用写 `x = x + 1` 或 `x = x - 1`，而只需使用 `x++` 来加 `1`，或使用 `x--` 来减 `1`。它更快、更简洁、更易读。

递增和递减运算符的工作方式有一个转折：它们有两种形式，前缀和后缀，区别在于值何时更新。对于递增运算符，前缀是 `++x`，后缀是 `x++`。

前缀（`++x`）先增加变量的值，然后返回新值。后缀（`x++`）先返回变量的当前值，然后增加它：

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6
```

:::

在上面的代码中，`++x` 意味着"先递增 `x`，然后使用它"。所以当你记录 `++x` 时，你立即得到递增后的值，即 `6`。

现在，让我们看看使用后缀的例子：

:::interactive_editor

```js
let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

在这个例子中，`y++` 意味着"先使用 `y`，然后递增它"。当你记录 `y++` 时，你得到 `5`，但 `y` 在那行代码之后变成 `6`。

递减运算符的作用与递增相同，只是它将值减少 `1`。同样，有两种形式：前缀（`--x`）先减少变量的值，然后返回新值。后缀（`x--`）先返回当前值，然后减少它：

:::interactive_editor

```js
let x = 5;
console.log(--x); // 4
console.log(x); // 4

let y = 5;
console.log(y--); // 5
console.log(y); // 4
```

:::

那么，你应该使用哪一种：前缀还是后缀？在许多情况下，使用哪一种并不重要。两者都能完成工作。但是，如果你立即在表达式中使用该值，区别就变得重要了。让我们看看这个例子：

:::interactive_editor

```js
let a = 5;
let b = ++a;
console.log(b); // 6 (a 在赋值前被递增)

let c = 5;
let d = c++;
console.log(d); // 5 (c 在赋值后被递增)
```

:::

所以，如果你需要立即更新的值，使用前缀。如果你想要当前值，而不关心稍后的递增，使用后缀。
