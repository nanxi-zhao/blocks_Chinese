# --interactive--

可选链运算符（`?.`）是 JavaScript 中的一个有用工具，让你可以安全地访问对象属性或调用方法，而不用担心它们是否存在。它就像处理可能缺少部分的对象时的安全网。

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30
};

console.log(person.name); // "Alice"
console.log(person.job); // undefined
```

:::

在这个例子中，`person.name` 存在，所以它记录 `Alice`。但 `person.job` 不存在，所以它给我们 `undefined`。

现在，假设我们想访问一个可能不存在的对象的属性：

```js
const person = {
  name: "Alice",
  age: 30
};

console.log(person.address.street); // 这将抛出错误！
```

这个例子将抛出 `Uncaught TypeError`。由于 `person.address` 是 `undefined`，我们无法访问 `street` 属性。这就是可选链运算符派上用场的地方。以下是使用可选链运算符的示例：

:::interactive_editor

```ts
const user = {
  name: "John",
  profile: {
    email: "john@example.com",
    address: {
      street: "123 Main St",
      city: "Somewhere"
    }
  }
};

console.log(user?.profile?.address?.street); // "123 Main St"
console.log(user?.profile?.phone?.number);   // undefined
```

:::

通过使用可选链运算符，我们告诉 JavaScript 只有在对象（或 `?.` 前的值）存在且不是 `null` 或 `undefined` 时才继续操作。

如果 `?.` 前的值是 `null` 或 `undefined`，JavaScript 返回 `undefined` 而不是尝试继续操作并抛出错误。

记住，可选链运算符在你不确定属性或方法是否存在时最有用。它有助于防止错误并使你的代码更加健壮。
