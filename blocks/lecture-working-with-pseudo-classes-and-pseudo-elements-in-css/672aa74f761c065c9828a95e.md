# --interactive--

伪类是特殊的CSS关键字，允许您根据元素的特定状态或位置来选择元素。元素的状态或位置可以包括：

- 当它处于活动状态时。
- 当鼠标悬停在它上面时。
- 当它是父元素的第一个子元素时。
- 当它是父元素的最后一个子元素时。
- 当链接已被访问时。
- 当它被禁用时。

要使用伪类，您需要在选择器后面添加一个冒号（[:]）和伪类的名称：

```css
selector:pseudo-class {
  /* CSS属性 */
}
```

让我们看看如何使用伪类来表示我们已经提到的每种状态和位置。

[:active]伪类允许您选择元素的活动状态，比如点击按钮：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>示例按钮</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

[:hover]伪类定义了元素的悬停状态。例如悬停在按钮或链接上：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">悬停在我上面！</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

[:first-child]伪类选择作为其父元素第一个子元素的元素。这里是针对[div]容器中第一个段落元素的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>第一个子元素</p>
  <p>第二个子元素</p>
  <p>第三个子元素</p>
  <p>最后一个子元素</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

该[div]中的第一个段落元素将获得浅珊瑚色背景和四周0.4rem的内边距。

[:last-child]伪类针对作为其父元素最后一个子元素的元素。这里是针对容器[div]元素中最后一个段落元素的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>第一个子元素</p>
  <p>第二个子元素</p>
  <p>第三个子元素</p>
  <p>最后一个子元素</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

[:visited]伪类允许您为用户已经访问过的链接设置样式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com">访问Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

[:disabled]伪类允许您为禁用模式下的交互元素设置样式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>禁用按钮</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

正如您所见，伪类允许您根据用户交互（如悬停或访问链接）来设置元素样式。这使您的网站感觉更具交互性。

除了已经提到的伪类之外，还有其他一些伪类，如：

- [:focus]
- [:first-of-type]
- [:last-of-type]
- [:nth-of-type]
- [:modal]
- [:enabled]
- [:checked]
- [:required]等。
