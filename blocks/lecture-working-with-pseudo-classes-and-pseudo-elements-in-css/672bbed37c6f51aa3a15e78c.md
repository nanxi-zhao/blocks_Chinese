# --interactive--

CSS最有趣的方面之一是伪元素的使用。在这个上下文中，"pseudo"意味着"不是真实的"，所以伪元素是虚拟或合成的元素，它们不直接匹配任何实际的HTML元素。它们允许您为元素的特定部分设置样式或在不添加额外HTML的情况下插入内容。

要应用伪元素，请使用双冒号（[::]）将其附加到原始元素的选择器上。请注意，选择器可以是任何类型，例如类或ID选择器。以下是伪元素的基本语法：

```css
selector::pseudo-element {
  property: value;
}
```

这种双冒号是区分伪元素和伪类的标志，伪类使用单冒号。

伪元素允许您为元素内容的特定部分设置样式或在元素之前或之后插入内容，但它们不能独立存在。伪元素所附加的元素称为其源元素。

让我们从[:before]和[:after]伪元素的示例开始。顾名思义，[:before]允许您在元素内容之前插入内容，而[:after]允许您在元素内容之后插入内容。

这是一个按钮元素的示例。在CSS中，我们将使用绝对定位和[:before]伪元素在按钮的[Learn More]文本之前添加一个星号。您将在未来的课程中了解更多关于绝对定位的内容。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">了解更多</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "★";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

[content]属性用于表示您希望在按钮文本之前添加的内容。在这个示例中，我们添加了一个星号。您会注意到您不仅可以插入内容，还可以为其设置样式。这是使用相同按钮的[:after]伪元素的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">了解更多</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '→';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

在[:after]伪元素中，[transition]属性用于在0.3秒内以缓动效果动画化变化，创建平滑渐进的转换而不是突然的转换。您将在未来的课程中了解更多关于[transition]属性的内容。

您还可以将伪类附加到使用[:before]和[:after]伪元素插入另一个内容中的内容。例如，内容的悬停状态。这是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">了解更多</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "→";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

通过悬停状态中的[transform: translateX(2px)]，内容在用户将鼠标悬停在按钮上时向右移动[2px]。[:after]本身中的transition属性确保该过程耗时[0.3s]。

这就是[transform]属性的作用——它允许您在特定方向上旋转、倾斜、缩放或平移元素。您将在未来的课程中了解更多相关内容。

在下一个示例中，我们将查看[:first-letter]伪元素。[:first-letter]伪元素定位元素内容的第一个字母，允许您为其设置样式。这是一个段落文本的示例。如果我们想要设置第一个字母的样式，可以像这样使用[:first-letter]伪元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>MasterPuti让您无需付费即可学习编程。</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

在最后一个示例中，我们将查看[:marker]伪元素，它允许您选择列表项的标记、项目符号或编号进行样式设置。[:marker]伪元素提供了一种通过自定义列表标记来匹配您的配色方案来增强网站品牌标识的方法。

这是一个无序列表和有序列表的示例。要更改列表项的标记颜色和大小，您可以像这样使用[:marker]伪元素：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>无序列表项目1</li>
  <li>无序列表项目2</li>
  <li>无序列表项目3</li>
  <li>无序列表项目4</li>
</ul>

<ol>
  <li>有序列表项目1</li>
  <li>有序列表项目2</li>
  <li>有序列表项目3</li>
  <li>有序列表项目4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

在本课程中，我们只涵盖了少数伪元素。但还有更多伪元素，如[:placeholder]、[:spelling-error]和[:selection]，我鼓励您自己探索。
