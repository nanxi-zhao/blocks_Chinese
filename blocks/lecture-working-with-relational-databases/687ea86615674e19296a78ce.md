# --description--

关系数据库有广泛的应用，包括Web开发、库存管理系统、医疗系统、电子商务应用程序等。

一些最常见的关系数据库是MySQL、PostgreSQL、SQLite和Microsoft SQL Server。让我们来了解一下它们中的每一个。

MySQL是一个开源的关系数据库管理系统。它在Web开发中非常受欢迎，因为它易于使用、可靠且高效地构建现实应用。它还有一个活跃的开发者社区来支持和维护它。

PostgreSQL官方描述为"世界上最先进的开源关系数据库"。它是一个免费的开源对象关系数据库系统，以其可靠性、数据完整性和可扩展性而闻名。例如，您可以定义自己的数据类型、自定义函数，并用不同的编程语言编写代码而无需重新编译数据库。

SQLite是一个免费的开源轻量级基于文件的SQL数据库引擎。它是世界上最广泛使用的数据库引擎之一。SQLite是自包含和无服务器的。它是一个零配置数据库，这意味着它不需要初始设置过程，这对快速入门非常有帮助。

这三个数据库都是免费和开源的。还有其他流行的专有选项，包括Microsoft SQL Server和Oracle数据库。

Microsoft SQL Server是由Microsoft开发并在各种应用程序中使用的流行关系数据库管理系统。

Oracle数据库是一个可扩展的关系数据库管理系统，通常用于需要高性能的企业应用程序。

很好。现在您对不同的选项有了更多了解，让我们看看如何安装PostgreSQL。

首先，您需要访问官方网站：<a href="https://www.postgresql.org/" target="_blank">https://www.postgresql.org/</a>。

到了那里后，点击"Download"按钮。

这将带您到一个页面，您可以在其中选择您的操作系统。如果您点击其中一个，将带您到该操作系统的说明。

我们将介绍Windows操作系统的安装过程，但该过程在操作系统之间是相似的。

从这个页面，您可以通过点击"Download the installer"链接来下载安装程序。

这将带您到一个页面，您可以选择要安装的PostgreSQL版本。如果您点击相应的图标，下载应该会开始。

然后，在您的下载文件夹或默认存储下载的文件夹中，您应该看到安装程序。

如果您双击它，将开始设置过程。点击Next。

选择PostgreSQL的安装目录。您会看到一个默认目录，但您可以自定义它。

然后，选择要安装的组件。有四个可能的组件：

- PostgreSQL Server，数据库引擎本身。
- pgAdmin 4，一个用户友好的数据库管理工具。
- Stack Builder，一个用于下载和安装其他PostgreSQL相关软件和扩展的工具
- 以及命令行工具，直接从终端与PostgreSQL服务器交互。这包括psql，一个用于执行SQL查询的交互式终端。

当您准备就绪时，点击Next。

然后，选择存储数据的目录。您会看到一个默认路径，但您可以自定义这个路径。当您准备就绪时，点击Next。

现在您需要为数据库超级用户输入密码，该用户对所有数据库和对象具有无限制访问权限。输入您的密码，重新输入，然后点击Next。

然后，选择服务器应监听的端口号。默认情况下，它是5432。然后，点击Next。

您需要为新的数据库集群选择一个区域设置。这对于数据库解释区域敏感的数据类型很重要。然后，点击Next。

最后，您会看到一个安装前摘要。花点时间查看这个摘要，然后点击Next。

您需要确认要开始安装过程。如果您点击Next，过程将开始。它应该只需要几分钟。

当过程完成后，根据您安装的组件，您可能需要选择是否启动Stack Builder来下载和安装其他工具、驱动程序和应用程序。

如果您决定启动Stack Builder，您会看到一个可以开始安装的屏幕。或者，您可以点击Cancel稍后再做。

恭喜！现在您已经在计算机上安装了PostgreSQL。该过程对于其他操作系统应该是相似的。之后，您可以开始使用数据库。

转到您的应用程序并搜索pgAdmin，这是一个开源的PostgreSQL管理和开发平台，如果您在组件中选择了它，应该已经安装了。点击pgAdmin 4打开它。

当应用程序启动时，您会看到一个初始加载屏幕。然后，您应该看到一个仪表板，您可以在其中启动新服务器、检查数据库、配置pgAdmin等。您还会找到有用的文档链接。

如果您点击左上角的"Servers"，您需要输入在安装过程中选择的密码。

输入密码后，您应该看到一个屏幕，您可以在其中访问数据库，包括它们的表格。

您还会找到另一个非常有用的工具psql，一个SQL查询的交互式终端。要在系统上搜索psql并点击它。

它将打开SQL Shell，您可以在其中开始运行命令。

这就是您如何在Windows上安装PostgreSQL。该过程对于macOS和Linux非常相似。

或者，如果您在macOS上安装它，可以使用Homebrew，一个包管理器。一旦您安装了Homebrew，运行此命令来安装PostgreSQL：

```bash
brew install postgresql@<version>
```

您可以通过在at（`@`）符号后写入数字来指定要安装的版本。
