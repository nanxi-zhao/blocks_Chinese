# --description--

让我们看看一些可以用来在SQL中插入和查看数据的基本命令。

首先，要查看数据，您需要插入数据。所以我们从插入开始。

本课程中我们将涵盖的示例将基于这个狗记录表格：

```sql
CREATE TABLE dogs(
  id SERIAL,
  name VARCHAR(100),
  age INTEGER
);
```

该表格将存储各种狗的名称和年龄。

如果我们假设使用PostgreSQL，我们可以对`id`使用`SERIAL`数据类型。由于`id`是使用`SERIAL`创建的，其值将是`INTEGER`，从`1`开始，并在插入新记录时自动递增。因此，在插入记录时您不需要为其传递值。

让我们从使用`INSERT INTO`语句向我们的`dogs`表格插入记录开始。有多种方法可以做到这一点。第一种是插入单行并指定将插入值的列。

您写入`INSERT INTO`，然后是表格名称（在这种情况下是`dogs`），然后是括号内的列，然后是`VALUES`，以及括号内的相应值。

这是一个示例，我们插入一条狗的记录，`name`为`'Gino'`，`age`为`3`，记住`id`将自动分配：

```sql
INSERT INTO dogs (name, age)
VALUES ('Gino', 3);
```

这通常是最安全的选择，因为您明确指定了列，所以值将按顺序分配给这些列。

请注意，`'Gino'`需要用单引号括起来，因为它是`VARCHAR`类型。

另一种选择是不指定列插入记录，如下所示：

```sql
INSERT INTO dogs
VALUES ('Gino', 3);
```

这也是有效的，但由于值是根据它们的顺序分配给列的，因此更容易出错。

您还可以通过用逗号分隔，在同一SQL命令中插入多条记录。

在这里，我们插入两条狗的记录，一条是`'Gino'`，另一条是`'Nora'`。

```sql
INSERT INTO dogs (name, age)
VALUES
  ('Gino', 3),
  ('Nora', 2);
```

就像您可以插入记录一样，您也可以从数据库中查询信息。在数据库的上下文中，查询是对数据的请求。

在SQL中，`SELECT`语句用于从一个或多个表格中查询数据。您可以自定义查询以获取所需的确切信息。

如果您需要查询`dogs`表格中的所有信息，您只需写入`SELECT`，然后是星号（`*`），然后是`FROM`和表格名称`dogs`：

```sql
SELECT *
FROM dogs;
```

在此命令中，星号是代表"所有列"的通配符。

您将获得表格中所有记录的所有数据：

```sql
 id | name | age 
----+------+-----
  1 | Gino |   3
  2 | Nora |   2
```

要查询特定列，您可以在命令中写入这些列的名称，紧跟在`SELECT`之后，用逗号分隔。在此示例中，我们查询`dogs`表格的`name`和`age`列：

```sql
SELECT name, age
FROM dogs;
```

结果将只包括`name`和`age`列：

```sql
 name | age 
------+-----
 Gino |   3
 Nora |   2
```

有时，您可能需要基于特定条件查询数据。例如，要获取所有年龄小于3岁的狗，您可以使用`WHERE`关键字和小于（`<`）比较运算符：

```sql
SELECT *
FROM dogs
WHERE age < 3;
```

结果将只包括年龄小于3岁的狗：

```sql
 id | name | age 
----+------+-----
  2 | Nora |   2
```

如果您只是想找到`'Gino'`的`age`，您可以使用等于（`=`）比较运算符：

```sql
SELECT age
FROM dogs
WHERE name = 'Gino';
```

这是结果：

```sql
 age 
-----
  3
```

这些是在SQL中插入和查看数据的常见和简单方法，但还有许多不同的选项可供选择，其中一些更高级。您将在接下来的课程中了解它们。
