# --interactive--

使用`calc()`函数，您可以直接在样式表中执行计算来动态确定属性值。这意味着您可以通过基于视口大小或其他元素计算尺寸来创建灵活和响应式的用户界面。

`calc()`是一个CSS函数。当您开始学习JavaScript时，您将了解更多关于函数的知识，但在本课中，您将学习理解`calc()`工作原理所需的基础知识。

函数是执行特定任务的代码块。CSS中已经定义了一些函数，因此您可以直接使用它们，并传递任何必要的值来自定义它们的任务执行方式。

在编程世界中，当我们运行函数执行的任务时，我们说我们在"调用"函数。我们传递给函数的值被称为参数。

正如您在下面的代码中看到的，要调用函数，您需要写函数名，后跟括号内的参数，参数之间用逗号分隔。函数名和左括号之间不应该有空格：

```css
function(argument1, argument2, argument3)
```

函数可能只需要一个值来知道该做什么。在这种情况下，它只需要一个参数。这就是`calc()`函数的情况。它只需要一个参数，因为它需要知道要计算什么。

为此，您传递一个称为表达式的参数。表达式是产生结果的值和运算符的组合。

这就是您可以调用`calc()`函数的方式。您写calc名称，后跟括号，在括号内写表达式：

```css
calc(expression)
```

表达式被求值以计算最终结果。"求值"意味着值和运算符在后台被转换为单个值。结果被分配给进行计算的CSS属性。

您可以对表示长度、角度、时间、百分比、数字和颜色的值执行计算。您还可以组合不同的单位，如像素、百分比和ems。

对于数字，表达式中的所有值（也称为操作数）都必须有相应的单位，如`px`、`em`和百分比（`%`）。根据运算符的不同，不同的操作数可能有不同的单位。

您可以在表达式中使用加法（`+`）、减法（`-`）、乘法（`*`）和除法（`/`）运算符。

如果存在多个操作数和运算符，`calc()`将遵循标准运算符优先级规则。如果需要，您还可以添加括号来建立操作顺序。

在下面的示例中，您可以看到一个包含文本`Hello, World!`的`div`。

使用CSS类型选择器选择`div`，您可以使用白色文本和深蓝色背景对其进行样式设置：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>Hello, World!</div>
```

```css
div {
  color: white;
  background-color: #1b1b32;
  width: calc(50% - 20px);
}
```

:::

这里的新内容是宽度是动态计算的。注意我们如何调用`calc()`函数并传递表达式作为参数。表达式有两个具有不同单位的操作数和一个运算符，即减法运算符。

百分比是相对单位。值（`50%`）将由父容器的宽度确定。然后，从该值中减去`20px`。这个表达式的结果将确定`div`的宽度。

`div`的宽度大约是其容器总宽度的一半，如果您调整父容器的大小，宽度将自动重新计算。

这就是使用`calc()`的关键优势。您可以根据应用程序或视口的不同方面动态确定CSS属性的值。

如果您需要在计算中使用CSS函数和变量，表达式也可以包含它们。您将在接下来的课程中了解更多关于CSS变量的知识。

很好。现在您已经了解了`calc()`函数的基础知识，让我们介绍一些最佳实践。

首先，您必须在加法（`+`）和减法（`-`）运算符周围添加空格。

例如，下面的表达式将不是有效表达式，因为减法运算符紧跟在第二个操作数之前。

```css
calc(100% -30px)
```

减法（`-`）运算符必须用空格包围，像这样。添加空格将创建一个有效的表达式。

```css
calc(100% - 30px)
```

这对于乘法和除法运算符不是必需的，但强烈推荐。

如果您需要执行计算并在其他计算中使用这些结果，还可以嵌套`calc()`函数调用。

另外，如果您在传递给`calc()`函数的表达式中使用零值表示长度，则必须包含单位。例如，这个表达式将无效：

```css
calc(100% - 0)
```

您需要添加单位，如px。

```css
calc(100% - 0px)
```

您还应该知道，目前如果您使用乘法或除法运算符，其中一个操作数必须是无单位的。对于除法运算符，特别是右操作数必须是无单位的。这将不是有效表达式，因为两个操作数都有单位（像素）。操作数5或50中的一个必须是无单位的：

```css
calc(5px * 50px)
```

您需要省略其中一个的单位。这两种替代方案都是有效的：

```css
calc(5 * 50px)
calc(5px * 50)
```

这是除法运算符的例子。这将不是有效表达式，因为它们都有单位：

```css
calc(50% / 5%)
```

当您有除法运算符时，应该从右操作数中移除单位：

```css
calc(50% / 5)
```

`calc()`函数对您作为Web开发人员非常有帮助。使用此函数，您可以动态设置属性值以创建灵活和响应式的设计。
