# --interactive--

在编程中，理解字符如何表示为数字是基础性的。这就是ASCII的用武之地。ASCII，全称为美国信息交换标准代码，是在计算机中表示文本的字符编码标准。它为每个字符分配一个数值，这些数值被机器普遍识别。

在本课程中，我们将探讨什么是ASCII，它是如何工作的，以及如何使用JavaScript方法如[charCodeAt()]和[fromCharCode()]与ASCII值进行交互。

ASCII是一个将字符（如字母、数字和符号）编码为数值的系统。每个字符都映射到一个特定的数字。

例如，大写字母[A]在ASCII中用数字[65]表示，而小写[a]用[97]表示。这种编码允许计算机存储和操作文本。

ASCII标准涵盖128个字符，包括：

- 大写和小写英文字母（A-Z，a-z）。
- 数字（0-9）。
- 常见的标点符号和符号（!，@，#等）。
- 控制字符（如换行符和制表符）。

在JavaScript中，您可以使用[charCodeAt()]方法轻松访问字符的ASCII代码。此方法在字符串上调用，并返回指定索引处字符的ASCII代码。

让我们看一个示例：

:::interactive_editor

```js
let letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

在这个示例中，[A]是字符串的第一个字符，调用[charCodeAt(0)]返回其ASCII值[65]。

您也可以将此方法与其他字符一起使用来查找它们的ASCII值：

:::interactive_editor

```js
let symbol = "!";
console.log(symbol.charCodeAt(0));  // 33
```

:::

在这里，感叹号[!]的ASCII代码作为[33]返回。

虽然[charCodeAt()]帮助您检索字符的ASCII值，但[fromCharCode()]方法允许您执行相反的操作：将ASCII代码转换为其对应的字符。

让我们看看它的实际效果：

:::interactive_editor

```js
let char = String.fromCharCode(65);
console.log(char);  //  A
```

:::

在这个示例中，[fromCharCode(65)]将ASCII值[65]转换回字符[A]。

另一个示例是将数字[97]转换为其对应的小写字母：

:::interactive_editor

```js
let char = String.fromCharCode(97);
console.log(char);  // a
```

:::

当您需要基于ASCII值操作或比较字符时，这些方法特别有用。

例如，您可以使用[charCodeAt()]通过比较其ASCII值来检查字符是大写、小写还是数字。

另一方面，[fromCharCode()]可用于从ASCII代码动态生成字符。
