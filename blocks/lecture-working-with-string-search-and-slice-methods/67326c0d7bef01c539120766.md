# --interactive--

在 JavaScript 中处理字符串时，有很多情况下你可能需要检查一个字符串是否包含特定的子字符串，子字符串是该字符串的较小部分。

例如，你可能想要在执行某些操作之前检查用户的输入是否包含特定的单词或字符。实现这一点的一种方法是使用 `includes()` 方法。

`includes()` 方法用于检查字符串是否包含特定的子字符串。如果在字符串中找到子字符串，则该方法返回 `true`，否则返回 `false`。

以下是基本语法：

```js
string.includes(searchValue);
```

对于语法，`searchValue` 是你想要在字符串中查找的子字符串。下面是一个示例：

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

在这个示例中，在字符串 `JavaScript is awesome!` 中找到了单词 `awesome`，所以 `includes()` 方法返回 `true`。

需要注意的是，`includes()` 方法是区分大小写的。这意味着需要精确匹配字符，包括大小写。

例如：

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

由于 `Awesome`（大写的 `A`）与 `awesome`（小写的 `a`）不匹配，结果是 `false`。

你还可以通过提供第二个参数来使用 `includes()` 方法检查从字符串中特定索引开始的子字符串：

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

在这里，从字符串的第 7 个位置开始搜索子字符串 `JavaScript`，确保跳过此位置之前的任何字符。

`includes()` 方法只返回 `true` 或 `false` 结果。它不提供子字符串在字符串中的位置信息或出现次数。如果你需要这种详细信息，其他方法（如 `indexOf()` 方法）可能更合适。
