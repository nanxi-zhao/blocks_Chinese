# --interactive--

在 JavaScript 中，模板字面量是一种强大而灵活的处理字符串的方式。与使用单引号（'）或双引号（"）的常规字符串不同，模板字面量使用反引号（`）定义。

它们允许更简单的字符串操作，包括直接在字符串内部嵌入变量，这一特性称为字符串插值。

模板字面量使创建跨越多行的字符串或在字符串中直接包含表达式（如变量甚至 JavaScript 代码）变得更加容易。

以下是模板字面量的示例：

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

注意使用反引号而不是单引号或双引号。`${name}` 语法是字符串插值的示例，其中变量 `name` 的值直接插入到字符串中。

字符串插值允许你在字符串内部嵌入变量和表达式。这比旧方法有了显著改进，旧方法需要使用 `+` 运算符连接字符串和变量。

以下是使用字符串连接和加号（+）运算符的示例：

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

以下是使用模板字面量和字符串插值的示例：

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

如你所见，使用模板字面量的字符串插值更加清晰易读，特别是当你处理多个变量时。

模板字面量的另一个强大特性是它们支持多行字符串。使用常规字符串时，你需要使用转义字符（\n）来创建新行。使用模板字面量时，你可以简单地将字符串写在多行上，并保留格式：

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

模板字面量的另一个特性是它们允许你直接在字符串中嵌入 JavaScript 表达式，如本示例所示：

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

当你需要在字符串中包含变量或表达式、格式化复杂字符串或处理多行文本时，模板字面量特别有用。与传统的字符串连接相比，它们更加简洁和易读。
