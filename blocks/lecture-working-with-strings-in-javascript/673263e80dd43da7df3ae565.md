# --interactive--

当使用 JavaScript 处理字符串时，有时你需要定位较大字符串中特定子字符串的位置。

子字符串是出现在较大字符串中的字符序列。例如，在字符串 `hello world` 中，`hello` 和 `world` 是子字符串。

要定位字符串内子字符串的位置，你可以使用 `indexOf()` 方法。JavaScript 中的 `indexOf()` 方法允许你在字符串中搜索子字符串。

如果找到子字符串，`indexOf()` 返回该子字符串第一次出现的索引（或位置）。如果未找到子字符串，`indexOf()` 返回 `-1`，这表示搜索不成功。

`indexOf()` 方法接受两个参数：第一个是你要在较大字符串中查找的子字符串，第二个是搜索的可选起始位置。如果你不提供起始位置，搜索将从字符串的开头开始。

在这种情况下，**参数**是你在调用函数或方法时提供给它的值，使该函数或方法能够使用你提供的特定信息执行任务。你将在未来的课程中了解更多关于参数的知识。

以下是使用 `indexOf()` 方法查找字符串 `awesome` 位置的示例：

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

在这个示例中，单词 `awesome` 在字符串 `JavaScript is awesome!` 中的索引 `14` 处开始，所以 `indexOf()` 方法返回 `14`。

现在，让我们看看当子字符串未找到时会发生什么：

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

由于单词 `fantastic` 没有出现在字符串中，该方法返回 `-1`。

你也可以通过向 `indexOf()` 提供第二个参数来指定在字符串中的何处开始搜索。以下是一个示例：

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

在这种情况下，对 `JavaScript` 的搜索从第 10 个字符之后开始，因此 `JavaScript` 的第二次出现是在索引 `27` 处找到的。

需要注意的是，`indexOf()` 方法是区分大小写的。

在这个示例中，以下内容将返回 `-1`，因为在字符串 `MasterPuti` 中找不到大写字母 `F`。

:::interactive_editor

```js
console.log("MasterPuti".indexOf("F")) // -1
```

:::

当你需要检查子字符串是否存在于字符串中并确定其位置以进行进一步操作时，使用 `indexOf()` 非常有用。
