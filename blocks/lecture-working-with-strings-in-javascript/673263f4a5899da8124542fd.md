# --interactive--

`prompt()` 方法是 JavaScript 与用户交互的重要组成部分。它是通过小弹出对话框获取用户输入的最简单方法之一。

你经常会看到它用于网页需要用户信息的情况，例如姓名或其他形式的文本输入。

那么，`prompt()` 方法到底是做什么的呢？它会打开一个对话框，要求用户提供一些输入，然后将用户输入的文本作为字符串返回。

`prompt()` 方法接受两个参数：第一个是将出现在对话框内的消息，通常提示用户输入信息。第二个是可选的默认值，它将初始填充输入字段。

```js
prompt(message, default);
```

以下是它如何工作的示例。

**注意**：此示例包含你尚未学习的代码。不用担心理解代码中的所有内容。这只是为了说明 `prompt()` 方法如何工作，并确保提示不会在页面加载时立即出现，因为这可能被视为侵入性行为。如果你启用了交互式编辑器，可以自己尝试一下。

:::interactive_editor

```html
<button id="prompt-btn">Show Prompt</button>
<p id="output"></p>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("prompt-btn");
const output = document.getElementById("output");
btn.addEventListener("click", () => {
  const userName = prompt("What is your name?", "Guest");
  output.textContent = "Hello, " + userName + "!";
});
```

:::

在这个示例中，当用户点击按钮时，`prompt()` 方法会显示一个对话框，其中包含消息 `What is your name?` 和一个初始包含值 `Guest` 的输入字段。

如果用户输入他们的姓名并按下"确定"，`userName` 变量将存储输入的值。如果用户按下"取消"，`userName` 变量将被设置为 `null`。`null` 表示用户没有提供任何输入。输出段落将显示使用提供的姓名或如果用户取消则显示 `Guest` 的问候消息。

请记住，`prompt()` 方法会暂停脚本的执行，直到用户与对话框交互。

这意味着你的其余 JavaScript 代码不会运行，直到用户提供输入并点击"确定"，或取消提示。

另一点需要考虑的是，虽然 `prompt()` 对于快速测试或小型应用程序很有用，但由于其破坏性性质和在不同浏览器中的不一致行为，在现代复杂的 Web 应用程序中通常会避免使用它。

通过理解 `prompt()` 方法，你获得了一种简单的方式与用户交互并通过浏览器直接检索信息，即使它在现代 Web 应用程序中可能不被广泛使用。
