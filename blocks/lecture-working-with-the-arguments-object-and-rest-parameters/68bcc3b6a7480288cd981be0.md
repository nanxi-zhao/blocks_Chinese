# --interactive--

正如你在之前的课程和研讨会中回忆的那样，你可以创建带有多个参数的函数，并用参数调用该函数。

这里是一个提醒：

```js
// 函数定义
function getSum(num1, num2) {
  return num1 + num2;
}

// 函数调用
getSum(3, 4); // 7
```

但是如果你有一个函数被调用时传入的参数比定义时接受的参数多会怎样呢？

:::interactive_editor

```js
// 函数定义
function getSum(num1, num2) {
  return num1 + num2;
}

// 用额外参数调用函数
console.log(getSum(3, 4, 5)); // 7
```

:::

在这种情况下，JavaScript 不会抛出错误。它会忽略额外的参数，只将数字 `3` 和 `4` 相加。接受可变数量参数的函数被称为 <dfn>可变参数函数</dfn>。

如果你正在处理可变参数函数，那么你可以利用 `arguments` 对象。这个类数组对象包含传递给函数的参数值。

这里是一个例子：

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// 结果:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

由于 `arguments` 对象是类数组的，你可以像这样访问特定索引处的参数：

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

你也可以像这样使用 `length` 属性来获取函数被调用时的参数数量：

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

尽管 `arguments` 对象看起来像一个真正的数组，但它没有内置的 `Array` 方法，如 `includes` 或 `push`。要使用这些方法，你需要首先使用 `slice`、`Array.from()` 或展开运算符等方法将 `arguments` 对象转换为真正的数组：

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

虽然可以使用 `arguments` 对象处理可变参数函数，但现代 JavaScript 应用程序通常会使用剩余参数语法。你将在未来的课程中了解更多相关内容。
