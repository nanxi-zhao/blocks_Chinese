# --description--

让我们来学习DOM节点及其在DOM树中的关系。

就像真正的树木有大枝和小枝以分层结构连接一样，DOM节点之间也存在直接和间接的关系。我们将使用这个示例来说明这些关系：

```html
<!DOCTYPE html>
<html>
  <head>
    <title>DOM Tree Example</title>
  </head>
  <body>
    <h1>Heading 1</h1>
    <p>Paragraph 1</p>
    <ul>
      <li>List item 1</li>
      <li>List item 2</li>
    </ul>
  </body>
</html>
```

让我们从顶部开始。DOM树的根是[html](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L3-L22)元素。它是HTML文档所有内容的顶级容器。所有其他节点都是这个根节点的后代。

然后，在根节点下方，我们在层次结构中找到其他节点。父节点是包含其他元素的元素。子节点是包含在另一个元素内的元素。

在这个示例中，[body](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L6-L21)元素是段落元素的父元素，而段落元素是[body](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L6-L21)元素的子元素。这些元素在DOM树中表示为具有相同关系的节点。

就像我们有父节点和子节点一样，我们也有兄弟节点。兄弟节点是共享同一父元素的元素。在我们的示例中，两个列表项元素共享相同的无序列表父元素，所以它们是兄弟节点。

同样，[h1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733689f9f31dd31e7d9c789.md#L21-L21)和段落元素也是兄弟节点，因为它们共享同一父元素，即[body](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L6-L21)元素。

我们还在层次结构的不同层级间有间接关系。后代节点是直接或间接包含在另一个元素内的元素。如果可以通过从祖先向下遍历DOM树到达某个元素，则该元素被视为另一个元素的后代。

在我们的示例中，列表项[li](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733689f9f31dd31e7d9c789.md#L24-L24)元素是[body](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L6-L21)元素的后代，因为它们间接包含在其中。

祖先节点是在DOM树层次结构中位于另一个元素上方的元素。在我们的示例中，[body](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L6-L21)元素是列表项[li](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733689f9f31dd31e7d9c789.md#L24-L24)元素的祖先。

理解这些关系对于使用JavaScript操作和导航DOM树至关重要。
