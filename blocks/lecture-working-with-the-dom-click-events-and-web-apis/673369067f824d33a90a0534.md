# --interactive--

当用户点击按钮或表单发生更改时，这被称为事件。在你的程序中，你需要有一种方法来监听这些事件并响应它们。

[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)方法用于监听事件。它接受两个参数：要监听的事件和事件发生时将被调用的函数。

以下是基本语法：

```js
element.addEventListener("event", listener);
```

元素是要监视事件的HTML元素，事件指定要监听的事件类型，例如["click"](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L27-L29)。

监听器是在事件发生时接收通知的对象。大多数时候，这将是你定义的处理事件的函数。

这是一个示例：

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

你也可以选择这样传递函数引用：

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

有时你可能想要创建一个单独的函数来处理事件。这可以使你的代码更易于阅读和维护。

监听器参数也可以是[null](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L44-L44)，或者它也可以是使用[EventListener](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L44-L44)接口的对象。[EventListener](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L44-L44)接口定义了一个名为[handleEvent()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L44-L44)的单一方法。每当监听的事件发生时，这个方法会自动调用。使用这个接口允许同一个对象在需要时处理多个事件。

大多数时候，你不需要使用这个接口，而是可以传递一个事件发生时将被调用的函数。

现在，让我们看一个示例，更好地理解这是如何工作的：

在这个示例中，我们有一个id为[btn](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L53-L53)的[button](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L53-L53)元素：

```html
<button id="btn">Show alert</button>
```

我们想要监听这个按钮上的["click"](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L27-L29)事件，并在按钮被点击时显示一个警告。

我们首先需要在JavaScript代码中访问那个[button](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L53-L53)元素。然后我们需要添加一个事件监听器来监听点击事件，并在按钮被点击时显示一个警告：

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

现在每次按钮被点击时，用户都会在屏幕上看到一个警告消息，如下所示：

```md
You clicked the button
```

你可以监听的另一种事件是["input"](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L103-L105)事件。当输入元素的值发生变化时，会触发这个事件。

这是一个示例：

```html
<input type="text" id="input" placeholder="Type something" />
```

我们想要监听这个输入元素上的输入事件，并在用户每次输入内容时将输入的值记录到控制台。

我们首先需要在JavaScript代码中访问那个[input](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L86-L86)元素。然后我们需要添加一个事件监听器来监听输入事件，并在用户每次输入内容时将输入的值记录到控制台：

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

还有许多其他事件你可以使用[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)方法来监听。一些常见事件包括[mouseover](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L110-L110)、[mouseout](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L110-L110)、[keydown](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L110-L110)、[keyup](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L110-L110)和[submit](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L110-L110)。

在未来的课程中，我们将涵盖更多事件以及如何使用[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)方法来监听它们。
