# --interactive--

在上一课中，你学习了如何使用[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)方法处理["click"](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L27-L29)和["input"](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L103-L105)事件。在本课中，你将学习如何使用[removeEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L155-L155)方法移除事件监听器。

[removeEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L155-L155)方法用于移除先前使用[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)方法添加到元素上的事件监听器。当你想要停止监听元素上的特定事件时，这很有用。

以下是[removeEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L155-L155)方法的基本语法：

```js
element.removeEventListener("event", listener);
```

就像[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)方法一样，[removeEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L155-L155)方法接受两个参数：要移除的事件和先前添加的监听器函数。

还有一个可选的第三个参数可以传递给[removeEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L155-L155)方法。这个参数可以是[options](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L24-L24)或[useCapture](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L26-L26)。

[options](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L24-L24)参数是一个对象，指定事件监听器的选项，例如事件监听器是否应该是被动的或只执行一次。

[useCapture](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L26-L26)参数是一个布尔值，指定事件是否应该在事件传播阶段被捕获。

大多数时候，你只需要向[removeEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L155-L155)方法传递事件和监听器参数。

如果我们想要在灰色和蓝色之间切换背景颜色，那么我们可以为此使用事件监听器。

我们首先需要访问[button](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L39-L39)元素以及[body](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L64-L64)元素：

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

然后我们需要创建负责在灰色和蓝色之间切换的函数：

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

我们使用布尔变量[isBgColorGrey](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L71-L71)来跟踪当前的背景颜色。如果背景颜色是灰色，我们就将其更改为蓝色，反之亦然。

然后我们需要向[button](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L39-L39)元素添加一个事件监听器，在按钮被点击时调用[toggleBgColor](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L73-L76)函数：

```js
btn.addEventListener("click", toggleBgColor);
```

我们可以更新HTML，包含一个段落元素，当鼠标悬停时将用于移除事件监听器：


要在段落被悬停时移除事件监听器，我们可以向段落元素添加一个事件监听器。我们使用[mouseover](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L154-L156)事件并传递一个从[button](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L39-L39)元素移除事件监听器的函数：

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

当段落被悬停时，按钮点击事件的事件监听器被移除，按钮被点击时背景颜色将不再改变。
