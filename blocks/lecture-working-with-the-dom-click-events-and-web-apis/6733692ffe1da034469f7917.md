# --interactive--

有时候你需要在JavaScript文件中操作HTML元素的样式。一个例子是当用户点击按钮时需要隐藏或显示菜单。

在本课中，你将学习如何使用[Element.style](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L33-L33)和[Element.classList](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L40-L40)属性操作样式。

[Element.style](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L33-L33)属性是一个只读属性，表示元素的内联样式。你可以使用这个属性来获取或设置元素的样式。

以下是带有一个ID为[para](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L19-L19)的段落元素示例：

```html
<p id="para">This is a paragraph</p>
```

你可以使用[style](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L33-L33)属性将段落元素的颜色更改为红色：

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

[style](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L33-L33)属性可以用于设置许多CSS属性，例如[color](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L33-L33)、[background-color](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L38-L38)、[font-size](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L38-L38)、[font-weight](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L38-L38)等。

操作样式的另一种方法是使用[Element.classList](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L40-L40)属性。[classList](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L40-L40)属性是一个只读属性，可以用于向元素添加、删除或切换类。让我们看几个示例。

在这个示例中，你将向一个ID为[para](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L19-L19)的段落元素添加一个名为[highlight](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L53-L55)的类。在CSS中，我们将设置[highlight](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L53-L55)类将段落元素的背景颜色更改为黄色。在JavaScript中，我们将使用[classList.add()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L60-L60)方法向段落元素添加[highlight](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L53-L55)类。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

你也可以通过将它们作为参数传递给[classList.add()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L60-L60)方法向元素添加多个类：

```js
classList.add("class1", "class2", "class3");
```

如果你需要从元素中删除一个类，你可以这样使用[classList.remove()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L74-L74)方法：

```js
classList.remove("highlight");
```

要切换元素上的类，你可以使用[classList.toggle()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L112-L112)方法。在这个示例中，我们有一个ID为[toggle-btn](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L83-L83)的[button](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L39-L39)元素和一个ID为[menu](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L85-L91)的[nav](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L85-L91)元素。在CSS中，我们为菜单添加了一些样式和一个名为[show](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L103-L105)的类，该类将[.menu](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L96-L101)的[display](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L97-L97)属性更改为[block](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L104-L104)。在JavaScript中，我们使用[getElementById()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-arrays-variables-and-naming-practices/6733b43e3b72d0180404c53d.md#L41-L47)方法访问菜单元素和[button](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L39-L39)元素。然后，我们向[button](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L39-L39)元素添加一个事件监听器，当按钮被点击时将在[.menu](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L96-L101)元素上切换[show](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L103-L105)类：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

当用户点击按钮时，[show](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L103-L105)类将被添加到[.menu](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L96-L101)元素上，菜单将被显示。如果用户再次点击按钮，[show](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L103-L105)类将从[.menu](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L96-L101)元素上删除，菜单将被隐藏。

通过使用[style](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L33-L33)、[add()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L60-L60)、[remove()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L74-L74)和[toggle()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733692ffe1da034469f7917.md#L112-L112)等方法和属性，你可以轻松地用JavaScript操作元素的样式，创建动态和交互式的网页。
