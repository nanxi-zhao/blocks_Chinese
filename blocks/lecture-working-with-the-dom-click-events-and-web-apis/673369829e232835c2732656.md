# --interactive--

对话框让你可以向用户显示重要信息或操作。使用HTML内置的[dialog](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L25-L27)元素，你可以轻松地在Web应用程序中创建这些对话框（包括模态和非模态对话框）。

模态对话框是一种对话框类型，它强制用户在访问应用程序或网页的其他部分之前与之交互。它有效地阻止与其他内容的交互，直到用户完成某个操作，例如关闭对话框或提交表单。

相比之下，非模态对话框允许用户在对话框打开时继续与页面或应用程序的其他部分交互。它不会阻止访问其他内容。

在本课中，你将学习如何使用[showModal()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L69-L69)和[close()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L121-L121)方法打开和关闭模态框。

当你想要确保用户专注于模态框的特定操作或消息时，你可以使用[showModal()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L69-L69)方法打开模态框对话框。这将为页面上的其他项目添加背景并禁用它们。这对于显示表单、确认和需要用户操作的关键信息的模态框来说是理想的。

以下是模态框对话框的HTML：

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

现在，你在页面上看不到任何东西，因为模态框在初始渲染时是关闭的。你可以使用[showModal()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L69-L69)方法自动打开模态框：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

浏览器中的结果将显示一个包含文本[This is a modal dialog.](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L38-L38)的模态框

最好将控制权交给用户。为了实现这一点，你可以向按钮添加点击事件监听器并使用[showModal()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L69-L69)方法：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

如果你需要显示一个对话框，同时仍然允许与对话框外的内容进行交互，那么你可以使用[show()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L92-L92)方法：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

要关闭模态框，你可以在[dialog](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L103-L106)元素内向模态框添加一个按钮并使用[close()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369829e232835c2732656.md#L121-L121)方法：

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

模态对话框和非模态对话框之间有什么区别？

## --answers--
