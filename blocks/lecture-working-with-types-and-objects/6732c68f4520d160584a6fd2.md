# --interactive--

`toString()`方法是JavaScript中的一项基本功能，用于将值转换为其字符串表示形式。这是一个可用于数字、布尔值、数组和对象的方法。`toString()`最常见的用途之一是将数字转换为其字符串表示形式。这是一个示例：

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

此方法接受一个可选的基数，该基数是`2`到`36`之间的数字。这个基数代表进制，如二进制的基数2或八进制的基数8。如果未指定基数，则默认为基数10，即十进制。以下是向`toString()`方法传递`2`作为参数的示例：

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

结果将是`1010`，这是十进制数`10`的二进制表示。

您还可以使用`toString()`方法将数组和对象转换为字符串。数组有自定义的`toString()`实现，将每个元素转换为字符串并用逗号连接。这是一个示例：

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

在此示例中，数组的所有元素都连接在一起形成字符串`1,2,3`。

当`toString()`方法与对象一起使用时，结果不会是对象属性的字符串化版本。

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

在此示例中，结果将是对象的默认字符串表示形式，即`[object Object]`。要获取`person`对象属性的字符串化版本，您需要使用`JSON.stringify()`，您将在未来的课程中了解更多相关内容。

总之，`toString()`方法用于将值转换为字符串。了解它如何与不同数据类型配合工作以及如何为自己的对象自定义它，可以大大提高您在JavaScript应用程序中操作和显示数据的能力。
