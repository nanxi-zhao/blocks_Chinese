# --interactive--

`Number()`构造函数用于创建数字对象。数字对象包含一些有用的属性和方法，如`isNaN`和`toFixed`方法。以下是使用`Number()`构造函数和`new`关键字的示例：

:::interactive_editor

```js
const myNum = new Number("34");
console.log(typeof myNum); // "object" 
```

:::

在此示例中，我们将一个字符串字面量传递给`Number()`构造函数，返回类型是`object`类型而不是`string`类型。

当`Number()`构造函数作为函数调用而不使用`new`关键字时，返回值将是原始数字类型。大多数时候，您将使用`Number()`构造函数将其他数据类型转换为数字数据类型。以下是将字符串转换为数字的示例：

:::interactive_editor

```js
const myNum = Number("100");
console.log(myNum); // 100

console.log(typeof myNum); // number
```

:::

当您从用户那里获取输入并且需要将该字符串输入转换为数字以便执行数学计算时，这很有帮助。

如果您尝试通过空字符串调用`Number()`构造函数，则结果将是数字`0`：

:::interactive_editor

```js
const num = Number("");
console.log(num); // 0
```

:::

这是因为JavaScript会尝试解析字符串，由于它不包含任何数字，结果将是零。

如果您尝试传递包含随机字符的字符串，则结果将是`NaN`或"Not a Number"。

:::interactive_editor

```js
const num = Number("random");
console.log(num); // NaN
```

:::

在处理布尔值时，`true`返回`1`，因为`true`被转换为一，`false`返回`0`，因为`false`被转换为零。

:::interactive_editor

```js
const boolTrue = Number(true);
const boolFalse = Number(false);

console.log(boolTrue); // 1
console.log(boolFalse); // 0
```

:::

如果您传递`null`，结果将是`0`，如果您传递`undefined`，结果将是`NaN`。

:::interactive_editor

```js
const undefinedNum = Number(undefined);
const nullNum = Number(null);

console.log(undefinedNum); // NaN
console.log(nullNum); // 0
```

:::

在处理数组时，有几点需要考虑。

空数组将返回`0`。包含单个数字的数组将返回该数字。包含多个数字的数组返回`NaN`。包含字符串的数组也将返回`NaN`。

:::interactive_editor

```js
const emptyArr = Number([]);
const arrOneNum = Number([7]);
const arrMultiNum = Number([7, 36, 12]);
const arrStr = Number(["str1"]);
const arrMultiStr = Number(["str1", "str2"]);

console.log(emptyArr); // 0
console.log(arrOneNum); // 7
console.log(arrMultiNum); // NaN
console.log(arrStr); // NaN
console.log(arrMultiStr); // NaN
```

:::

在处理对象时，结果总是`NaN`。

:::interactive_editor

```js
const obj1 = Number({});
const obj2 = Number({2: 2});
const obj3 = Number({key: "val"});
const obj4 = Number({key: true});

console.log(obj1); // NaN
console.log(obj2); // NaN
console.log(obj3); // NaN
console.log(obj4); // NaN
```

:::

总之，您将主要使用`Number()`构造函数进行类型转换，而不是创建数字或数字对象。
