## HTML基础

- **HTML的作用**：HTML代表网页的内容和结构。
- **HTML元素**：元素是HTML文档的构建块。它们代表标题、段落、链接、图像等。大多数HTML元素由开始标签（`<elementName>`）和结束标签（`</elementName>`）组成。

以下是基本语法：

```html
<elementName>内容放在这里</elementName>
```

- **空元素**：空元素不能有任何内容，只有开始标签。例如包括`img`和`meta`元素。

```html
<img>
<meta>
```

在某些代码库中常见在空元素内包含正斜杠`/`。这两种方式都是可以接受的：

```html
<img>
<img/>
```

- **属性**：属性是放置在HTML元素开始标签内的值。属性提供关于元素的附加信息或指定元素的行为方式。以下是属性的基本语法：

```html
<element attribute="value"></element>
```

布尔属性是在HTML标签中存在或不存在的属性。如果存在，则值为true，否则为false。布尔属性的例子包括`disabled`、`readonly`和`required`。

- **注释**：注释用于在代码中为自己和其他开发者留下笔记。以下是HTML中注释的语法：

```html
<!--这是HTML注释。-->
```

## 常见HTML元素

- **标题元素**：HTML中有六个标题元素。从`h1`到`h6`的标题元素用于表示其下方内容的重要性。数字越小，重要性越高，所以`h2`元素的重要性低于`h1`元素。

```html
<h1>最重要的标题元素</h1>
<h2>第二重要的标题元素</h2>
<h3>第三重要的标题元素</h3>
<h4>第四重要的标题元素</h4>
<h5>第五重要的标题元素</h5>
<h6>最不重要的标题元素</h6>
```

- **段落元素**：这用于网页上的段落。

```html
<p>这是一个段落元素。</p>
```

- **`img`元素**：`img`元素用于向网页添加图像。`src`属性用于指定该图像的位置。对于图像元素，最好包含另一个叫做`alt`属性的属性。以下是带有`src`和`alt`属性的`img`元素示例：

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg" alt="盘子上的一片千层面。">
```

- **`body`元素**：此元素用于表示HTML文档的内容。

```html
<body>
  <h1>猫咪照片应用</h1>
  <p>这是一个段落元素。</p>
</body>    
```

- **`section`元素**：此元素用于将内容划分为较小的部分。

```html
<section>
  <h2>关于我</h2>
  <p>你好，我是简·多伊，我是一名网页开发者。</p>
</section>
```

- **`div`元素**：这是一个通用的HTML元素，没有任何语义含义。它用作容纳其他HTML元素的通用容器。

```html
<div>
  <h1>我是一个标题</h1>
  <p>我是一个段落</p>
</div>
```

- **锚点（`a`）元素**：这些元素用于向网页应用链接。`href`属性用于指定当用户点击时链接应该跳转到哪里。

```html
<a href="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg">可爱的猫</a>
```

- **无序（`ul`）和有序（`ol`）列表元素**：要创建项目符号列表，你应该使用`ul`元素，在其中嵌套一个或多个`li`元素，如下所示：

```html
<ul>
  <li>猫薄荷</li>
  <li>激光笔</li>
  <li>千层面</li>
</ul>
```

要创建有序列表，你应该使用`ol`元素：

```html
<ol>
  <li>跳蚤治疗</li>
  <li>雷声</li>
  <li>其他猫</li>
</ol>
```

- **强调（`em`）元素**：这用于对文本的一部分进行强调。

```html
<p>猫<em>爱</em>千层面。</p>  
```

- **强重要性（`strong`）元素**：此元素用于对文本进行强烈强调，以表示紧迫感和严肃性。

```html
<p>
  <strong>重要：</strong>在继续之前，请确保佩戴安全护目镜。
</p>
```

- **`figure`和`figcaption`元素**：`figure`元素用于将图像和图表等内容分组。`figcaption`元素用于表示`figure`元素内内容的标题。

```html
<figure>
  <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="两只虎斑小猫一起睡在沙发上。">
  <figcaption>猫<strong>讨厌</strong>其他猫。</figcaption>  
</figure>
```

- **`main`元素**：此元素用于表示网页的主要内容。
- **`footer`元素**：此元素放置在HTML文档的底部，通常包含版权信息和其他重要页面链接。

```html
<footer>
  <p>
    无版权 - <a href="https://www.freecodecamp.org">freeCodeCamp.org</a>
  </p>
</footer>
```

## 标识符和分组

- **ID**：HTML元素的唯一标识符。ID名称在每个HTML文档中只能使用一次。

```html
<h1 id="title">电影评论页面</h1>
```

ID名称不能有空格。如果你的ID名称包含多个单词，那么可以在单词之间使用破折号，如下所示：

```html
<div id="red-box"></div>
```

- **类**：类用于对元素进行分组以进行样式化和行为控制。

```html
<div class="box"></div>
```

与ID不同，你可以在整个HTML文档中重复使用相同的类名。`class`值也可以有空格，如下所示：

```html
<div class="box red-box"></div>
<div class="box blue-box"></div>
```

## 特殊字符和链接

- **HTML实体**：HTML实体或字符引用是一组用于在HTML中表示保留字符的字符。例子包括与符号（`&amp;`）和小于符号（`&lt;`）。

```html
<p>这是一个 &lt;img /&gt; 元素</p>
```

- **`link`元素**：此元素用于链接到外部资源，如样式表和网站图标。以下是使用`link`元素链接到外部CSS文件的基本语法：

```html
<link rel="stylesheet" href="./styles.css" />
```

`rel`属性用于指定链接资源与HTML文档之间的关系。`href`属性用于指定外部资源URL的位置。

- **`script`元素**：此元素用于嵌入可执行代码。

```html
<body>
  <script>
    alert("欢迎来到freeCodeCamp");
  </script>
</body>
```

虽然你可以在`script`标签内编写所有JavaScript代码，但最佳实践是链接到外部JavaScript文件。以下是使用`script`元素链接到外部JavaScript文件的示例：

```html
<script src="path-to-javascript-file.js"></script>
```

这里使用`src`属性来指定外部JavaScript文件的位置。

## 样板和编码

- **HTML样板**：这是一个包含基本结构和每个HTML文档所需的重要元素的样板。

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>freeCodeCamp</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <!--标题、段落、图像等放在这里-->
  </body>
</html>
```

- **`DOCTYPE`**：这用于告诉浏览器你使用的HTML版本。
- **`html`元素**：这表示HTML文档的顶级元素或根元素。要指定文档的语言，你应该使用`lang`属性。
- **`head`元素**：`head`部分包含浏览器和搜索引擎所需的重要元数据。
- **`meta`元素**：这些元素表示你网站的元数据。这些元素有关于字符编码以及像Twitter这样的网站如何预览你的页面链接等详细信息。
- **`title`元素**：此元素用于设置显示在浏览器标签或窗口中的文本。
- **UTF-8字符编码**：UTF-8或UCS转换格式8是一种广泛用于网络的标准化字符编码。字符编码是计算机用来存储字符为数据的方法。`charset`属性在`meta`元素内用于将字符编码设置为UTF-8。

## SEO和社交分享

- **SEO**：搜索引擎优化是一种优化网页使其在搜索引擎中更加可见并排名更高的实践。
- **元（`description`）元素**：这用于为网页提供简短描述并影响SEO。

```html
<meta
  name="description"
  content="发现专家提示和技术，用于在小空间园艺、选择合适的植物以及维护繁荣花园方面的技巧。"
/>
```

- **开放图谱标签**：开放图谱协议使你能够控制你的网站内容在各种社交媒体平台上的显示方式，如Facebook、LinkedIn等。
通过设置这些开放图谱属性，你可以吸引用户点击并与你的内容互动。
你可以通过在HTML `head`部分内的一系列`meta`元素来设置这些属性。
- **`og:title`属性**：这用于设置社交媒体帖子显示的标题。

```html
<meta content="freeCodeCamp.org" property="og:title" />
```

- **`og:type`属性**：`type`属性用于表示在社交媒体上共享的内容类型。此类内容的例子包括文章、网站、视频或音乐。

```html
<meta property="og:type" content="website" />
``` 

- **`og:image`属性**：这用于设置社交媒体帖子显示的图像。

```html
<meta
  content="https://cdn.freecodecamp.org/platform/universal/fcc_meta_1920X1080-indigo.png"
  property="og:image"
/>
```

- **`og:url`属性**：这用于设置用户将点击的社交媒体帖子的URL。

```html
<meta property="og:url" content="https://www.freecodecamp.org" />
```

## 媒体元素和优化

- **替换元素**：替换元素是指其内容由外部资源而非CSS本身确定的元素。例如`iframe`元素。`iframe`代表内联框架。它是一个内联元素，用于直接在HTML页面中嵌入其他HTML内容。

```html
<iframe src="https://www.example.com" title="示例网站"></iframe>
```

你可以包含`allowfullscreen`属性，允许用户以全屏模式显示iframe。

```html
<iframe
  src="video-url"
  width="width-value"
  height="height-value"
  allowfullscreen
></iframe>
```

要在`iframe`中嵌入视频，你可以直接从YouTube和Vimeo等流行视频服务复制，或者通过`src`属性指向该视频的URL来自己定义。以下是嵌入来自YouTube的热门freeCodeCamp课程的示例：

```html
<h1>使用iframe元素嵌入的freeCodeCamp YouTube视频</h1>

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/PkZNo7MFNFg?si=-UBVIUNM3csdeiWF"
  title="YouTube视频播放器"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  referrerpolicy="strict-origin-when-cross-origin"
  allowfullscreen
></iframe>
```

还有一些其他替换元素，如`video`和`embed`。在特定情况下，一些元素会表现为替换元素。以下是带有`type`属性设置为`image`的`input`元素示例：

```html
<input type="image" alt="描述性文字放在这里" src="example-img-url">
```

- **优化媒体**：在网页上使用图像等媒体时，有三个工具需要考虑：大小、格式和压缩。压缩算法用于减小文件或数据的大小。
- **图像格式**：两种最常见的文件格式是PNG和JPG，但这些不再是提供图像的最理想格式。除非你需要支持旧版浏览器，否则你应该考虑使用更优化的格式，如WEBP或AVIF。
- **图像许可证**：处于公共领域的图像没有附加版权，可以自由使用而无需任何限制。特别在Creative Commons 0许可证下发布的图像被认为是公共领域。一些图像可能在宽松许可证下发布，如Creative Commons许可证或freeCodeCamp使用的BSD许可证。
- **SVG**：可缩放矢量图形基于路径和方程跟踪数据来绘制点、线和曲线。这意味着矢量图形（如SVG）可以缩放到任何大小而不会影响质量。

## 多媒体集成

- **`audio`和`video`元素**：`audio`和`video`元素允许你在HTML文档中添加声音和视频内容。`audio`元素支持流行的音频格式如mp3、wav和ogg。`video`元素支持mp4、ogg和webm格式。

```html
<audio src="CrystalizeThatInnerChild.mp3"></audio>
```

如果你想在页面上看到音频播放器，那么你可以添加带有`controls`属性的`audio`元素：

```html
<audio src="CrystalizeThatInnerChild.mp3" controls></audio>
```

`controls`属性使用户能够管理音频播放，包括调整音量以及暂停或恢复播放。`controls`属性是一个布尔属性，可以添加到元素中以启用内置播放控件。如果省略，则不会显示控件。

- **`loop`属性**：`loop`属性是一个布尔属性，使音频连续重播。

```html
<audio
  src="https://cdn.freecodecamp.org/curriculum/js-music-player/can't-stay-down.mp3"
  loop
  controls
></audio>
```

- **`muted`属性**：当在`audio`元素中出现时，`muted`布尔属性将以静音状态启动音频。

```html
<audio
  src="https://cdn.freecodecamp.org/curriculum/js-music-player/can't-stay-down.mp3"
  loop
  controls
  muted
></audio>
```

- **`source`元素**：在音频文件类型方面，不同浏览器支持的类型有所不同。为了适应这一点，你可以在`audio`元素内使用`source`元素，浏览器会选择它理解的第一个源。以下是为`audio`元素使用多个`source`元素的示例：

```html
<audio controls>
  <source src="audio.ogg" type="audio/ogg" />
  <source src="audio.wav" type="audio/wav" />
  <source src="audio.mp3" type="audio/mpeg" />
</audio>
```

我们目前学到的所有属性也都在`video`元素中受支持。以下是使用带有`loop`、`controls`和`muted`属性的`video`元素的示例：

```html
<video
  src="https://archive.org/download/BigBuckBunny_124/Content/big_buck_bunny_720p_surround.mp4"
  loop
  controls
  muted
></video>
```

- **`poster`属性**：如果你想在视频下载时显示图像，可以使用`poster`属性。此属性在`audio`元素中不可用，是`video`元素独有的。

```html
<video
  src="https://archive.org/download/BigBuckBunny_124/Content/big_buck_bunny_720p_surround.mp4"
  loop
  controls
  muted
  poster="https://peach.blender.org/wp-content/uploads/title_anouncement.jpg?x11217"
  width="620"
></video>
```

## 目标属性类型

- **`target`属性**：此属性告诉浏览器在哪里打开锚点元素的URL。此属性有四个重要的可能值：`_self`、`_blank`、`_parent`和`_top`。还有第五个值，称为`_unfencedTop`，目前用于实验性的`FencedFrame`API。你可能还没有理由使用这个值。
- **`_self`值**：这是`target`属性的默认值。这会在当前浏览上下文中打开链接。在大多数情况下，这将是当前标签或窗口。

```html
<a href="https://freecodecamp.org" target="_self">访问freeCodeCamp</a>
```

- **`_blank`值**：这会在新的浏览上下文中打开链接。通常，这将在新标签中打开。但有些用户可能会配置他们的浏览器在新窗口中打开。
```html
<a href="https://freecodecamp.org" target="_blank">访问freeCodeCamp</a>
```

- **`_parent`值**：这会在当前上下文的父级中打开链接。例如，如果你的网站有iframe，该iframe中的`_parent`值将在你的网站标签/窗口中打开，而不是在嵌入的框架中打开。

```html
<a href="https://freecodecamp.org" target="_parent">访问freeCodeCamp</a>
```

- **`_top`值**：这会在最顶层的浏览上下文中打开链接——想想"父级的父级"。这类似于`_parent`，但链接将始终在完整的浏览器标签/窗口中打开，即使对于嵌套的嵌入框架也是如此。

```html
<a href="https://freecodecamp.org" target="_top">访问freeCodeCamp</a>
```

## 绝对路径vs相对路径

- **路径定义**：路径是用于指定文件系统中文件或目录位置的字符串。在Web开发中，路径让开发者能够链接到图像、样式表、脚本和其他网页等资源。
- **路径语法**：有三个关键语法需要了解。首先是斜杠，根据你的操作系统可能是反斜杠（`\`）或正斜杠（`/`）。第二个是单点（`.`）。最后，我们有双点（`..`）。斜杠被称为"路径分隔符"。它用于表示文件夹或文件名之间文本的中断。单点指向当前目录，两个点指向上级目录。

```md
public/index.html
./favicon.ico
../src/index.css
```

- **绝对路径**：绝对路径是资源的完整链接。它从根目录开始，包括每个其他目录，最后是文件名和扩展名。"根目录"指的是层次结构中的顶级目录或文件夹。绝对路径还包括协议——可以是`http`、`https`和`file`，如果是网络上的资源还包含域名。以下是链接到freeCodeCamp标志的绝对路径示例：

```html
<a href="https://design-style-guide.freecodecamp.org/img/fcc_secondary_small.svg">
  查看fCC标志
</a>
```

- **相对路径**：相对路径指定相对于当前文件目录的文件位置。它不包括协议或域名，使其比同一网站内的内部链接更短且更灵活。以下是链接到同一文件夹中的`about.html`页面的示例：

```html
<p>
  在
  <a href="about.html">关于页面</a>了解更多
</p>
```

## 链接状态

- **`:link`**：这是默认状态。此状态表示用户尚未访问、点击或与之交互的链接。你可以认为此状态为页面上的所有链接提供了基础样式。其他状态在此基础上构建。
- **`:visited`**：当用户已经访问过链接的页面时应用此状态。默认情况下，这会使链接变为紫色——但你可以利用CSS为用户提供不同的视觉指示。
- **`:hover`**：当用户将光标悬停在链接上时应用此状态。此状态有助于为链接提供额外关注，以确保用户确实打算点击它。
- **`:focus`**：当我们将焦点放在链接上时应用此状态。
- **`:active`**：当用户激活链接时应用此状态。这通常意味着用鼠标主键点击链接，在大多数情况下是左键单击。
