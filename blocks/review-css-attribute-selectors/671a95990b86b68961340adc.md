## 使用不同的属性选择器和链接

- **定义**：`attribute` 选择器允许你根据 HTML 元素的属性（如 `href` 或 `title` 属性）来定位元素。

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

- **`title` 属性**：此属性提供有关元素的附加信息。以下是定位具有 `title` 属性的链接的方法：

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

- **组合选择器以匹配同时具有 `href` 和 `title` 属性的链接**：结合多个属性选择器。

```css
a[href][title] {
  color: green;
}
```

- **匹配空格分隔列表中的单个词**：定位具有特定类词的链接。

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

- **匹配以特定前缀开头的值**：定位以 https:// 开头的链接

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

- **匹配以特定后缀结尾的值**：定位以 .jpg 结尾的链接

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

- **匹配包含子字符串的值**：定位在值中任何位置包含 https 的链接。

```css
a[href*="https"] {
  color: teal;
}
```

- **总结**：这些模式使得根据属性和值一致地设置链接样式变得容易。

## 使用 `lang` 和 `data-lang` 属性定位元素

- **`lang` 属性**：此属性在 HTML 中用于指定元素内内容的语言。你可能希望根据编写语言的不同来区别设置元素样式，特别是在多语言网站上。

```css
p[lang="en"] {
  font-style: italic;
}
```

- **`data-lang` 属性**：像 `data-lang` 属性这样的自定义数据属性通常用于在元素中存储附加信息，例如指定文本特定部分使用的语言。以下是根据 `data-lang` 属性设置元素样式的方法：

```css
div[data-lang="fr"] {
  color: blue;
}
```

## 使用属性选择器、有序列表元素和 `type` 属性

- **`type` 属性**：在 HTML 中使用有序列表时，`type` 属性允许你指定使用的编号样式，如数字、字母或罗马数字。

```css
/*示例定位大写字母编号*/
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/*示例定位小写罗马数字*/
ol[type="i"] {
  color: green;
}
```
