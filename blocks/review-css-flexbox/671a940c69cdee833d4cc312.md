## CSS Flexbox和Flex模型简介

- **定义**：CSS flexbox是一个一维布局模型，允许你在容器内按行和列排列元素。
- **Flex模型**：该模型定义了flex项目在flex容器内如何排列。每个flex容器都有两个轴：主轴和交叉轴。

## [flex-direction](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性

- **定义**：该属性设置主轴的方向。[flex-direction](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)的默认值是[row](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)，它将所有flex项目放在同一行上，按照浏览器默认语言的方向（从左到右或从右到左）。
- **[flex-direction: row-reverse;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这会反转行中的项目。
- **[flex-direction: column;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这将使flex项目垂直对齐而不是水平对齐。
- **[flex-direction: column-reverse;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这会垂直反转flex项目的顺序。

## [flex-wrap](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性

- **定义**：该属性确定flex项目在flex容器内如何包装以适应可用空间。[flex-wrap](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)可以取三个可能的值：[nowrap](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[wrap](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)和[wrap-reverse](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)。
- **[flex-wrap: nowrap;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这是默认值。即使flex项目的宽度超过容器的宽度，也不会被包装到新行上。
- **[flex-wrap: wrap;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：当项目超出其容器的宽度时，该属性将包装这些项目。
- **[flex-wrap: wrap-reverse;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：该属性将以相反的顺序包装flex项目。
- **[flex-flow](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性**：该属性是[flex-direction](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)和[flex-wrap](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)的简写属性。

```css
flex-flow: column wrap-reverse;
```

## [justify-content](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性

- **定义**：该属性沿flex容器的主轴对齐子元素。
- **[justify-content: flex-start;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：在这种情况下，flex项目将对齐到主轴的起点。这可以是水平或垂直的。
- **[justify-content: flex-end;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：在这种情况下，flex项目对齐到主轴的末端，水平或垂直。
- **[justify-content: center;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这将使flex项目在主轴上居中。
- **[justify-content: space-between;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这将使元素在主轴上均匀分布。
- **[justify-content: space-around;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这将使flex项目在主轴内均匀分布，在第一个项目前和最后一个项目后添加空间。
- **[justify-content: space-evenly;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这将使项目在主轴上均匀分布。
