## CSS Grid基础

- **定义**：CSS Grid是一个二维布局系统，用于在网页中创建复杂的布局。网格将具有行和列，它们之间有间隙。要定义网格布局，你需要将[display](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性设置为[grid](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)。

```css
.container {
  display: grid;
}
```

- **[fr](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)（分数）单位**：该单位表示网格容器内空间的一部分。你可以使用[fr](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)单位创建灵活的网格。
- **在轨道间创建间隙**：有三种方法可以在CSS网格中创建轨道间的间隙。你可以使用[column-gap](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性在列之间创建间隙。你可以使用[row-gap](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性在行之间创建间隙。或者你可以使用[gap](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)简写属性在行和列之间创建间隙。
- **[grid-template-columns](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这用于设置网格轨道列的线条名称和大小。

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **[grid-template-rows](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这用于设置网格轨道行的线条名称和大小。
- **[grid-auto-flow](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这确定了自动放置的项目如何适应网格。

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **[grid-auto-columns](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这用于设置隐式创建的列的大小。

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **[place-items](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这用于在块和内联方向上对齐项目。
- **[align-items](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这用于设置网格容器中项目的对齐方式。
- **[repeat()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)函数**：该函数用于在轨道列表中重复部分。你可以使用[repeat()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)函数，而不是写[grid-template-columns: 1fr 1fr 1fr;](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)。

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **显式网格**：你可以使用[grid-template-columns](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)或[grid-template-rows](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性指定线条或轨道的数量。
- **隐式网格**：当项目放置在网格之外时，会自动为这些外部元素创建行和列。
- **[minmax()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)函数**：该函数用于设置轨道的最小和最大大小。

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **基于线的定位**：所有网格都有线条。要指定项目在线条上的起始位置，你可以使用[grid-column-start](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)和[grid-row-start](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性。要指定项目在线条上的结束位置，你可以使用[grid-column-end](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)和[grid-row-end](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性。你也可以选择使用[grid-column](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)或[grid-row](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)简写属性。

这是使用[grid-column](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性使元素跨越所有列的示例。

```css
.element {
  grid-column: 1 / -1;
}
```

- **[grid-template-areas](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：该属性用于为你要在网格上定位的项目提供名称。


```css
    "footer footer"; 
  gap: 20px; 
}
```

## 调试CSS

- **DevTools（开发者工具）**：DevTools允许你实时检查和修改CSS。样式面板显示应用于所选元素的所有CSS规则，包括继承的样式。你可以切换单个属性的开和关，编辑值，甚至直接在浏览器中添加新规则。这种即时反馈对于尝试不同样式而无需修改源代码非常有用。
- **CSS验证器**：验证器用于根据官方规范检查你的CSS，并报告任何错误或警告。你可以使用的一个流行验证器是W3C CSS验证器。
- **调试响应式设计**：DevTools有一个选项，允许你模拟你的网站在各种屏幕尺寸和设备上的外观。这可以帮助你识别断点问题或在不同视口尺寸上扩展效果不佳的样式。
