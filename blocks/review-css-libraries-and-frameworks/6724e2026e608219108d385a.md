## CSS框架

- **CSS框架**：CSS框架可以加快你的工作流程，在整个网站中创建统一的视觉风格，使你的设计在多个浏览器中看起来一致，并保持你的CSS代码更有组织性。
- **流行的CSS框架**：一些流行的CSS框架包括Tailwind CSS、Bootstrap、Materialize和Foundation。
- **潜在的缺点**：
  - 框架提供的CSS可能与你的自定义CSS冲突。
  - 你的网站可能看起来与其他使用相同框架的网站相似。
  - 大型框架可能导致性能问题。

## 两种类型的CSS框架

- **实用优先的CSS框架**：这些框架具有特定用途的小类，如设置边距、内边距或背景颜色。你可以根据需要将这些小类直接分配给HTML元素。Tailwind CSS被归类为实用优先的CSS框架。

这是使用Tailwind CSS样式化按钮的示例。

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

- **基于组件的CSS框架**：这些框架具有预构建的组件和预定义的样式，你可以添加到你的网站中。这些组件在CSS框架的官方文档中可用，你可以将它们复制粘贴到你的项目中。Bootstrap被归类为基于组件的CSS框架。

这是使用Bootstrap创建列表组的示例。与将小类应用到HTML元素不同，你将添加整个组件，包括HTML结构。

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

## CSS预处理器

- **CSS预处理器**：CSS预处理器是扩展标准CSS的工具。它将具有扩展语法的代码编译成原生CSS文件。它可以帮助为复杂项目编写更清晰、可重用、较少重复和可扩展的CSS。
- **功能**：CSS预处理器可以提供的一些功能包括变量、混合、嵌套和选择器继承。
- **流行的CSS预处理器**：一些流行的CSS预处理器包括Sass、Less和Stylus。
- **潜在的缺点**：
  - 将CSS规则编译成标准CSS可能导致开销。
  - 编译后的代码可能难以调试。

## Sass

- **Sass**：它是最流行的CSS预处理器之一。Sass代表"Syntactically Awesome Style Sheets"。
- **Sass支持的功能**：Sass支持变量、嵌套CSS规则、模块、混合、继承和用于基本数学运算的运算符等功能

## Sass支持的两种语法

- **SCSS语法**：SCSS（Sassy CSS）扩展了CSS的基本语法。它是Sass最广泛使用的语法。SCSS文件具有[.scss](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/blocks_english/review-css-libraries-and-frameworks/.scss)扩展名。

这是在SCSS中定义和使用变量的示例。

```scss
$primary-color: #3498eb;

header {
  background-color: $primary-color;
}
```

- **缩进语法**：缩进语法是Sass的原始语法，也称为"Sass语法"。

这是在缩进语法中定义和使用变量的示例。

```sass
$primary-color: #3498eb

header
  background-color: $primary-color
```

### 混合

- **混合**：混合允许你将多个CSS属性及其值组合在名称下，并在整个样式表中重用该CSS代码块。

这是在SCSS语法中定义混合的示例。在这种情况下，混合被称为[center-flex](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)。它有三个CSS属性，用于使用flexbox居中元素。

```scss
@mixin center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

这是使用你定义的混合的示例。

```scss
section {
  @include center-flex;
  height: 500px;
  background-color: #3289a8;
}
```
