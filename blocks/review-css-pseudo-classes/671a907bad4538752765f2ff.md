## 用户操作伪类

- **伪类定义**：这些是特殊的CSS关键字，允许你根据元素的特定状态或位置来选择元素。
- **用户操作伪类**：这些是特殊关键字，允许你根据用户交互来改变元素的外观，改善整体用户体验。
- **[:active](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)伪类**：此伪类让你选择元素的活动状态，比如点击按钮。
- **[:hover](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)伪类**：此伪类定义元素的悬停状态。
- **[:focus](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)伪类**：此伪类在元素获得焦点时应用样式，通常通过键盘导航或用户点击表单输入框。
- **[:focus-within](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)伪类**：此伪类用于在元素或其任何后代获得焦点时应用样式。

## 功能性伪类

- **功能性伪类**：功能性伪类允许你根据更复杂的条件或关系来选择元素。与基于状态定位元素的常规伪类（例如[:hover](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[:focus](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)）不同，功能性伪类接受参数。
- **[:is()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)伪类**：此伪类接受选择器列表（例如[ol](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[ul](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)），并选择与列表中某个选择器匹配的元素。

```html
<p class="example">This text will change color.</p>
<p>This text will not change color.</p>
<p>This text will not change color.</p>
<p class="this-works-too">This text will change color.</p>
```

```css
p:is(.example, .this-works-too) {
    color: red;
}
```

