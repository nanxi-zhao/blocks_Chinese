## 图概述

图是由边（连接）连接的节点（顶点）集合。每个节点可以连接到多个其他节点，形成网络。不同类型的图包括：

- 有向图：边有方向（从一个节点到另一个节点），通常用直线和箭头表示。
- 无向图：边没有方向，用简单线条表示。
- 顶点：每个节点都关联到一个标签或标识符。
- 有环图：包含环（从同一节点开始和结束的路径）。
- 无环图（DAG）：不包含环。
- 边标记图：每条边都有标记，通常绘制在相应边的旁边。
- 加权图：边有关联的权重（值），可用于执行算术运算。
- 不连通图：包含两个或多个没有边连接的节点。

图用于各种应用，如地图、网络、推荐系统、依赖关系解析。

## 图遍历

这涉及访问图中的所有节点。两种主要算法是：

- **广度优先搜索（BFS）**
  - 使用队列。
  - 逐层探索。
  - 在无权图中找到最短路径。

- **深度优先搜索（DFS）**
  - 使用栈（或递归）。
  - 在回溯之前完全探索一个分支。
  - 用于环检测和路径查找。

## 图表示

图可以用两种主要方式表示：

- **邻接表**
  - 每个节点都有其邻居的列表。
  - 对于稀疏图来说空间效率高。
  - 易于遍历邻居。

- **邻接矩阵**
  - 一个二维数组，行和列代表节点。
  - 对于大图来说空间密集。
  - 快速检查两个节点之间是否存在边。

## 树

树是一种特殊的图，它是无环且连通的。关键属性包括：

- 它们没有环或循环（从同一节点开始和结束的路径）。
- 它们必须是连通的（每个节点都可以从每个其他节点到达）。

### 常见的树类型

最常见的树类型是：

- 二叉树
  - 每个节点最多有两个子节点，一个左子节点和一个右子节点。

- 二叉搜索树（BST）
  - 一种二叉树，其中每个左子节点都小于其父节点，每个右子节点都大于其父节点。

## 字典树

也称为前缀树，用于存储字符串集合，其中每个节点代表一个字符。

共享前缀只存储一次，使它们在自动完成和拼写检查等任务中很高效。

搜索和插入操作的时间复杂度为 O(L)，其中 L 是字符串的长度。

## 优先队列

优先队列是一种抽象数据类型，其中每个元素都有优先级。

队列和栈只考虑插入顺序，而优先队列考虑元素的优先级。

标准队列遵循 FIFO（先进先出），栈遵循 LIFO（后进先出）。然而，在优先队列中，高优先级的元素在低优先级的元素之前被服务，无论它们的插入顺序如何。

## 堆

它是一种基于树的特殊数据结构，具有称为堆属性的特定属性。

堆属性确定父节点和子节点之间的关系。有两种类型的堆：

- 最大堆
  - 每个父节点的值大于或等于其子节点的值。
  - 最大元素在根部。

- 最小堆
  - 每个父节点的值小于或等于其子节点的值。
  - 最小元素在根部。

### Python `heap` 模块示例

```py
import heapq

# 创建空堆
my_heap = []

# 插入元素
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)

# 删除最小元素
print(heapq.heappop(my_heap))  # 3

# 推入 + 弹出一步完成
print(heapq.heappushpop(my_heap, 2)) # 2

# 将列表转换为堆
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### 使用优先级

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# 删除最小数字 = 最高优先级
print(heapq.heappop(my_heap))  # (1, "C")
```
