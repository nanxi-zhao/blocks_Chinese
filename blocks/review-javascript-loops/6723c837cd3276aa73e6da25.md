## 使用循环

- **`for`循环**：这种类型的循环用于重复执行代码块一定次数。此循环分为三个部分：初始化语句、条件和递增/递减语句。初始化语句在循环开始前执行。它通常用于初始化计数器变量。条件在循环的每次迭代前求值。迭代是循环的单次通过。如果条件为`true`，则执行循环内的代码块。如果条件为`false`，则循环停止并继续执行下一个代码块。递增/递减语句在循环的每次迭代后执行。它通常用于递增或递减计数器变量。

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

- **`for...of`循环**：这种类型的循环用于遍历可迭代对象的值。可迭代对象的例子包括数组和字符串。

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

- **`for...in`循环**：这种类型的循环最适合遍历对象的属性。此循环将遍历对象的所有可枚举属性，包括继承的属性和非数字属性。

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

- **`while`循环**：这种类型的循环将在条件为`true`时运行代码块。

```js
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

- **`do...while`循环**：这种类型的循环将在检查条件之前至少执行一次代码块。

```js
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

## `break`和`continue`语句

- **定义**：`break`语句用于提前退出循环，而`continue`语句用于跳过循环的当前迭代并移动到下一个迭代。

```js
// Example of break statement
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, and 4

// Example of continue statement 
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, 4, 6, 7, 8, and 9
```
