## 使用HTML、CSS和JavaScript

虽然HTML和CSS提供了网站结构，但JavaScript通过启用复杂功能（如处理用户输入、动画元素，甚至构建完整的Web应用程序）为网站带来交互性。

## JavaScript中的数据类型

数据类型帮助程序理解它正在处理的数据类型，无论是数字、文本还是其他类型。

- **数字**：数字表示整数和浮点值。整数的例子包括7、19和90。
- **浮点数**：浮点数是带小数点的数字。例子包括3.14、0.5和0.0001。
- **字符串**：字符串是用引号括起来的字符序列或文本。`"我喜欢编码"`和`'JavaScript很有趣'`是字符串的例子。
- **布尔值**：布尔值表示两种可能值之一：`true`或`false`。你可以使用布尔值来表示条件，例如`isLoggedIn = true`。
- **未定义和空值**：`undefined`值是已声明但未赋值的变量。`null`值是空值，或故意赋值为`null`的变量。
- **对象**：对象是键值对的集合。键是属性名，值是属性值。

在这里，`pet`对象有三个属性或键：`name`、`age`和`type`。值分别是`Fluffy`、`3`和`dog`。

```js
let pet = {
  name: "Fluffy",
  age: 3,
  type: "dog"
};
```

- **符号**：Symbol数据类型是唯一且不可变的值，可用作对象属性的标识符。

在下面的例子中，创建了两个具有相同描述的符号，但它们不相等。

```js
const crypticKey1= Symbol("saltNpepper");
const crypticKey2= Symbol("saltNpepper");
console.log(crypticKey1 === crypticKey2); // false
```

- **BigInt**：当数字对于`Number`数据类型太大时，你可以使用BigInt数据类型来表示任意长度的整数。

通过在数字末尾添加`n`，你可以创建一个BigInt。

```js
const veryBigNumber = 1234567890123456789012345678901234567890n;
```

## JavaScript中的变量

- 可以使用`let`关键字声明变量。

```js
let cityName;
```

- 要给变量赋值，可以使用赋值运算符`=`。

```js
cityName = "New York";
```

- 使用`let`声明的变量可以被重新赋值。

```js
cityName = "Los Angeles";
console.log(cityName); // Los Angeles
```

- 除了`let`，你还可以使用`const`来声明变量。但是，`const`变量不能被重新赋值。

```js
const cityName = "New York";
cityName = "Los Angeles"; // TypeError: Assignment to constant variable.
```

- 使用`const`声明的变量用于声明常量，这些常量在整个代码中不允许更改，例如`PI`或`MAX_SIZE`。

## 变量命名约定

- 变量名应该是描述性和有意义的。
- 变量名应该是驼峰命名法，如`cityName`、`isLoggedIn`和`veryBigNumber`。
- 变量名不应以数字开头。它们必须以字母、`_`或`$`开头。
- 变量名不应包含空格或特殊字符，除了`_`和`$`。
- 变量名不应是保留关键字。
- 变量名是区分大小写的。`age`和`Age`是不同的变量。

## JavaScript中的字符串和字符串不可变性

- 字符串是用引号括起来的字符序列。它们可以使用单引号和双引号创建。

```js
let correctWay = 'This is a string';
let alsoCorrect = "This is also a string";
```

- 字符串在JavaScript中是不可变的。这意味着一旦创建了字符串，你就不能更改字符串中的字符。但是，你仍然可以将字符串重新赋值为新值。

```js
let firstName = "John";
firstName = "Jane"; // 将字符串重新赋值为新值
```

## JavaScript中的字符串连接

- 连接是将多个字符串或将字符串与包含文本的变量组合在一起的过程。`+`运算符是连接字符串的最简单和最常用的方法之一。

```js
let studentName = "Asad";
let studentAge = 25;
let studentInfo = studentName + " is " + studentAge + " years old.";
console.log(studentInfo); // Asad is 25 years old.
```

- 如果你需要向现有字符串添加或追加内容，则可以使用`+=`运算符。当你想通过随着时间推移向字符串添加更多文本来构建字符串时，这很有帮助。

```js
let message = "Welcome to programming, ";
message += "Asad!";
console.log(message); // Welcome to programming, Asad!
```

- 连接字符串的另一种方法是使用`concat()`方法。此方法将两个或多个字符串连接在一起。

```js
let firstName = "John";
let lastName = "Doe";
let fullName = firstName.concat(" ", lastName);
console.log(fullName); // John Doe
```

## 使用`console.log()`记录消息

- `console.log()`方法用于将消息记录到控制台。它是调试和测试代码的有用工具。

```js
console.log("Hello, World!");
// Output: Hello, World!
```

## JavaScript中的分号

- 分号主要用于标记语句的结束。这有助于JavaScript引擎理解各个指令的分隔，这对正确执行至关重要。

```js
let message = "Hello, World!"; // 第一个语句在这里结束
let number = 42; // 第二个语句从这里开始
```

- 分号有助于防止代码执行中的歧义，并确保语句正确终止。

## JavaScript中的注释

- 任何被注释掉的代码行都会被JavaScript引擎忽略。注释用于解释代码、做笔记或临时禁用代码。
- 单行注释使用`//`创建。

```js
// 这是一个单行注释，会被JavaScript引擎忽略
```

- 多行注释使用`/*`开始注释，使用`*/`结束注释。

```js
/*
这是一个多行注释。
它可以跨越多行。
*/
```

## JavaScript作为动态类型语言

- JavaScript是一种动态类型语言，这意味着你在声明变量时不必指定数据类型。JavaScript引擎会根据赋给变量的值自动确定数据类型。

```js
let error = 404; // JavaScript将error视为数字
error = "Not Found"; // JavaScript现在将error视为字符串
```

- 其他非动态类型的语言，如Java，会导致错误：

```java
int error = 404; // 值必须始终是整数
error = "Not Found"; // 这在Java中会导致错误
```

## 使用`typeof`运算符

- `typeof`运算符用于检查变量的数据类型。它返回一个表示变量类型的字符串。

```js
let age = 25;
console.log(typeof age); // "number"

let isLoggedIn = true;
console.log(typeof isLoggedIn); // "boolean"
```

- 然而，在JavaScript中关于`null`有一个众所周知的怪癖。`typeof`运算符对`null`值返回`"object"`。

```js
let user = null;
console.log(typeof user); // "object"
```
