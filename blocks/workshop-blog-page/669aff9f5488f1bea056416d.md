# --description--

在本工作坊中，你将通过创建一个专门介绍猫 Whiskers 先生的博客页面来练习使用语义化 HTML。

开始项目，添加 `<!DOCTYPE html>`，添加一个 `html` 元素，其 `lang` 属性值为 `en`。

请记住，你在上一模块中学过如何创建类似的基本 HTML 模板。

```html
<!DOCTYPE html>
<html lang="en">
<!--所有其他元素都将放在这里-->
</html>
```
