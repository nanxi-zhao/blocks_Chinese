# --description--

要在字符串中匹配特定字符，可以使用正则表达式（regex）。

在 JavaScript 中，正则由一对斜杠包裹的模式表示，例如下面的例子会匹配字符串字面量 `"hello"`：

```js
const regex = /hello/;
```

请声明一个名为 `regex` 的变量，并按上例初始化。在后续步骤中你会修改该模式以匹配卡路里计数器所需的字符。
