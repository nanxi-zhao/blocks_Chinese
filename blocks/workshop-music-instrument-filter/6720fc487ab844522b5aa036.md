# --description--

我们给你一个包含多个乐器卡片的页面，接下来你要为这个静态页面添加筛选功能，可以根据乐器所属家族来过滤显示。

先从创建一个下拉菜单开始，给它添加 `class="select-container"`。做得很好，继续加油！

# --hints--

请确保页面中有一个 `select` 元素。

```js
assert.exists(document.querySelector("select"));
```

该 `select` 元素应具有 `select-container` 这个类名。

```js
assert.equal(document.querySelector("select").getAttribute("class"), "select-container");
```
