# --description--

由于你新增了另一个 `div` 元素，需要为它提供更好的无障碍支持，让屏幕阅读器能正确识别。

之前课程介绍过 `aria-live` 属性，用于创建“实时区域”（live region）。当实时区域的内容发生变化时，屏幕阅读器用户可以被自动通知，而不必手动聚焦或与其交互。

在 `#status` 元素上添加 `aria-live="polite"`。

`polite` 值会告诉屏幕阅读器在当前播报完成或用户停止输入后再播报更新，减少对用户的打断。

# --hints--

`#status` 元素应包含 `aria-live` 属性。

```js
assert.isTrue(document.getElementById("status").hasAttribute("aria-live"));
```

该属性的值应为 `"polite"`。

```js
assert.equal(document.getElementById("status").getAttribute("aria-live"), "polite");
```
