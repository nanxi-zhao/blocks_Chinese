# --description--

在你的 `.tabs` 元素内部，添加一个 `div` 元素，设置 `role` 属性为 `"tablist"`，以及设置 `aria-labelledby` 属性为 `"tabs-title"`。

这里需要使用 `aria-labelledby` 属性来将选项卡列表与可见标签关联起来，帮助屏幕阅读器等辅助技术向用户宣布选项卡列表的用途。

# --hints--

你应该在 `.tabs` 元素内部有一个 `div` 元素。

```js
assert.exists(document.querySelector(".tabs div"));
```

你的 `div` 元素应该有一个设置为 `"tablist"` 的 `role` 属性。

```js
assert.equal(document.querySelector(".tabs div").getAttribute("role"), "tablist");
```

你的 `div` 元素应该有一个设置为 `"tabs-title"` 的 `aria-labelledby` 属性。

```js
assert.equal(document.querySelector(".tabs div").getAttribute("aria-labelledby"), "tabs-title");
```
