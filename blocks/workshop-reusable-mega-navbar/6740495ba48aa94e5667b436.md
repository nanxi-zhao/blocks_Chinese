# --description--

在这个工作坊中，你将通过构建一个可重用的导航栏来练习使用React函数组件。

所有的CSS都已经为你提供好了，这样你就可以专注于创建导航栏。

首先定义一个 `Navbar` 函数组件。在下一步中，你将开始使用JSX构建组件。

确保在导出组件时使用命名导出，因为它将被导入到HTML文件中。

以下是命名导出的示例：

```js
export function multiply(a, b) {
  return a * b;
}
```

另外，React函数组件需要以大写字母开头，因为这是一种区分常规HTML元素的方式。
