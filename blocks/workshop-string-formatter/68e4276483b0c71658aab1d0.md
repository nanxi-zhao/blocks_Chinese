# --description--

有时候你需要创建一个不同大小写的新字符串。

要创建一个所有字母都转换为大写的新字符串，你可以使用 `toUpperCase()` 方法。

这里有一个例子：

```js
const sentence = "I love to code!";
sentence.toUpperCase(); // "I LOVE TO CODE!"
```

创建一个名为 `upperCaseInput` 的变量，并将使用 `toUpperCase()` 方法处理 `cleanedInput` 变量的结果赋值给它。
