# --description--

`fetch()` 方法返回一个 `Promise`，它是一个占位对象，当你的请求成功时会被兑现，当请求失败时会被拒绝。

如果 `Promise` 被兑现，它会解析为一个 `Response` 对象，你可以使用 `.then()` 方法来访问这个 `Response`。

下面是你如何将 `.then()` 链接到 `fetch()` 方法：

```js
fetch("sample-url-goes-here")
  .then((res) => res)
```

将 `.then()` 方法链接到你的 `fetch` 调用。在 `.then()` 方法内部，添加一个以 `res` 为参数的回调函数，然后将 `res` 输出到控制台，这样你就可以看到 `Response` 对象。打开浏览器控制台并展开 `Response` 对象来查看它包含的内容。

再次提醒，先不要在代码末尾加分号。
