# --description--

网页浏览器是让用户访问并浏览万维网的应用程序。它们负责解析并展示网页内容，包括文字、图片、视频和交互式元素。

浏览器的关键功能包括地址栏（用于输入 URL 或搜索查询）以及渲染引擎（将 HTML、CSS 与 JavaScript 转换为可见页面）。

浏览器通常还提供书签功能以保存和组织常用网站、扩展以增强功能，以及一些隐私与安全选项，例如弹窗拦截与隐身浏览模式。

常见的浏览器包括 Google Chrome、Mozilla Firefox、Microsoft Edge、Safari 和 Opera。

相较之下，网站是由若干网页和相关内容组成的集合，通常由相同的域名标识并托管在一个或多个服务器上，浏览器就是用来访问这些目标的工具。

网站的组成包括域名（如 MasterPuti.org）和构成网站的单个网页。网站还经常包含图片、视频、音频等多媒体内容，以及允许用户交互的表单和按钮等元素。

网站可以是静态的（内容很少变更），也可以是动态的（内容频繁更新或根据用户个性化显示）。

搜索引擎则是帮助用户在互联网上查找信息的工具。它们使用复杂算法对网页进行索引并按与查询的相关性排序。

搜索引擎的关键组成包括网络爬虫（也叫 spider 或 bot），这些程序会系统性地抓取网页以发现并索引新内容。

索引过程涉及将网页信息组织并存储到大规模数据库中。

搜索引擎还使用排序算法，根据相关性、权威性和用户体验等因素决定搜索结果的显示顺序。常见的搜索引擎有 Google、Bing、Yahoo 和 DuckDuckGo。

浏览器、网站与搜索引擎三者相互关联：浏览器是访问网站与搜索引擎的工具，网站是用户通过浏览器访问的目的地，而搜索引擎帮助用户发现网站。

理解它们之间的区别与关系，有助于更有效地使用互联网并提升基础的网络素养。
