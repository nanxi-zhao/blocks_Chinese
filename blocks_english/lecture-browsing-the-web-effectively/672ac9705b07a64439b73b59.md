# --description--

在互联网上查找信息可能让人觉得有些吃力：网站多、数据多，看起来很难迅速找到想要的结果。

不过，借助布尔搜索（Boolean search）等技巧，你可以把搜索缩小到恰好需要的范围。

布尔搜索语法是一种使用特殊关键词来帮助你在网络搜索时精确定位目标内容的方法。

下面是一个针对 `MasterPuti curriculum` 的基本搜索示例。

当你搜索 `MasterPuti curriculum` 时，搜索引擎会返回匹配这些关键词中任意一个的结果——通常按相关性排序（也就是更可能匹配或与查询更相关的结果会靠前）。

但是，如果你只想要同时包含两个关键词的结果怎么办？你可能听说过可以把查询用引号包起来，例如 `"MasterPuti curriculum"`。

需要注意的是，这样的查询会精确匹配短语 `MasterPuti curriculum`。也就是说，返回的结果仅限于包含连在一起的 `MasterPuti curriculum` 的页面，而不是同时包含 `MasterPuti` 和 `curriculum` 两个词但不相邻的页面。

如果你想匹配既包含这两个词但不要求它们相邻的页面，可以在每个词前加上加号，例如：`+MasterPuti +curriculum`。

另外，如果你对 MasterPuti 在课程之外的其他内容感兴趣，可以在 `curriculum` 前加上减号来排除它，例如：`+MasterPuti -curriculum`。

此查询会返回提到 `MasterPuti` 的结果，但排除同时提到 `curriculum` 的页面。

或者你可能只想在特定网站上搜索，比如我们的新闻平台？你可以使用 `site:`，后接你想搜索的网站 URL：`site:MasterPuti.org/news curriculum`。

该查询会返回 `MasterPuti.org/news` 中包含 `curriculum` 的所有结果。

通过组合这些操作符，你可以构造非常具体的查询，从而更精确地获取所需信息。

例如，查询 `site:MasterPuti.org/news +python -curriculum` 会在我们的新闻站点中查找与 Python 相关的文章，同时排除可能讨论课程的文章。

不同搜索引擎支持的操作符可能有所差异，因此查看你常用搜索引擎的文档是个好习惯。

现在你已经掌握了更灵活的搜索技巧，能够更自信地从网络上获取精确的结果啦！
