# --description--

在 JavaScript 中，`throw` 语句用于抛出自定义异常。异常表示程序遇到意外事件并中断了正常执行流程。

作为程序员，需要妥善处理这些异常，避免程序在运行时意外崩溃。`throw` 的基本语法如下：

```js
throw expression;
```

其中 `expression` 是表示异常的对象或值，例如内置的 `Error`、`TypeError` 或 `RangeError`。下面示例使用 `throw` 抛出一个 `TypeError`：

```js
function validateNumber(input) {
  if (typeof input !== "number") {
    throw new TypeError("Expected a number, but received " + typeof input);
  }
  return input * 2;
}
```

在这个例子中，如果 `input` 不是 `number`，就会抛出带自定义信息的 `TypeError`，否则函数返回 `input * 2`。

如果想抛出更通用的错误信息，可以使用 `Error` 构造函数，例如：

```js
function divide(numerator, denominator) {
  if (denominator === 0) {
    throw new Error("Cannot divide by zero");
  }
  return numerator / denominator;
}
```

上例在分母为 `0` 时抛出错误信息 `Cannot divide by zero`。

在下一课中，我们会讲如何在 `try`/`catch` 块中抛出并优雅处理这些错误。
