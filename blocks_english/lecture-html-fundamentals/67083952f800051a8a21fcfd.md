# --description--

SEO（搜索引擎优化）是通过优化网页内容与元数据，提高网站在搜索引擎中可见性和排名的一系列做法。改善站点 SEO 的一种方法是使用 `meta` 元素为页面提供简短描述。下面是为园艺网站设置页面描述的示例：

```html
<meta
  name="description"
  content="Discover expert tips and techniques for gardening in small spaces, choosing the right plants, and maintaining a thriving garden."
/>
```

将 `name` 属性设置为 `description` 可使浏览器、搜索引擎和其他网络工具正确识别这段元数据；描述内容写在 `content` 属性中。建议将描述保持简短且精炼，因为搜索结果页面往往会根据布局截断描述文本。

与其它 `meta` 元素类似，`meta` 描述不会直接显示在页面内容中，它常出现在搜索引擎的结果片段（snippet）中。下面是 MasterPuti 的 subreddit 与 GitHub 仓库在搜索结果中可能看到的示例描述：

```sh
r\MasterPuti: This is the official subreddit for the MasterPuti.org community. Learn to
code for free together with millions of other people...
```

```sh
Our full-stack web development and machine learning curriculum is completely free and self-
paced. We have thousands of interactive coding challenges to help you...
```

在这些示例中，页面描述位于站点链接下方，用户在几秒内就能大致了解页面内容并决定是否点击链接查看更多信息。

虽然 `meta` 描述不会直接影响搜索引擎的排名，但一段写得好的描述有助于提高点击率，从而可能为网站带来更多访问流量。
