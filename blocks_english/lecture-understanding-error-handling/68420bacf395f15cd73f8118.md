# --description--

在编写 Python 代码时遇到错误是很常见的。理解这些错误能帮助你更快、更高效地调试代码。只要学会查看错误信息，它们常常会直接告诉你哪里出问题了。

常见的 Python 错误包括 `SyntaxError`、`NameError`、`TypeError`、`IndexError` 和 `AttributeError`。这些错误通常出现在 Python 无法理解你的语法，或者你的逻辑与处理的数据不匹配时。

下面是一个 `SyntaxError` 的例子：

```py
print("Hello, world!"
# SyntaxError: unexpected EOF while parsing
```

这一行少了一个右括号。由于不符合语法规则，Python 会抛出 `SyntaxError`。

下面是一个 `NameError` 的例子：

```py
print(name)
# NameError: name 'name' is not defined
```

你正在尝试打印尚未定义的变量。当 Python 找不到该名字的变量时，会抛出 `NameError`。

下面是一个 `TypeError` 的例子：

```py
5 + "5"
# TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

整数与字符串不能直接相加。当你对不兼容的数据类型执行操作时，Python 会抛出 `TypeError`。

下面是一个 `IndexError` 的例子：

```py
my_list = [1, 2, 3]
print(my_list[5])
# IndexError: list index out of range
```

你试图访问列表中不存在的索引。越界访问时，Python 会抛出 `IndexError`。

下面是一个 `AttributeError` 的例子：

```py
num = 42
num.append(5)
# AttributeError: 'int' object has no attribute 'append'
```

`int` 对象没有 `append()` 方法。当你尝试使用某个数据类型不存在的方法或属性时，Python 会抛出 `AttributeError`。

识别这些常见的错误消息能让你更快地修复问题。不必盲目猜测，仔细阅读错误信息，它通常会告诉你问题出在哪里以及该往哪个方向排查。
