# --description--

在 Python 中，异常处理是编写健壮且容错程序的核心部分。它能让你以结构化的方式预测、捕获并应对错误。

异常处理是捕获并管理程序执行过程中发生错误的过程，从而避免代码意外崩溃。

Python 提供了 `try`、`except`、`else` 和 `finally` 等语句块来优雅地处理错误。下面是一个基本示例：

```py
try:
    x = 10 / 0
except ZeroDivisionError:
    print("不能除以零！")
```

- `try`：包含可能发生错误的代码块。

- `except`：当 `try` 中抛出指定类型的错误时执行的代码块。

- 在这个例子中，除以零会抛出 `ZeroDivisionError`，随后被捕获并处理。

下面是同时展示 `else` 和 `finally` 的示例：

```py
try:
    x = 10 / 2
except ZeroDivisionError:
    print("不能除以零！")
else:
    print('除法成功:', x)
finally:
    print('这个代码块总是会执行。')
```

- `else`：当 `try` 块未抛出异常时执行。

- `finally`：无论是否发生异常都会执行，适合用于关闭文件或释放资源等清理工作。

你也可以使用多个 `except` 块来捕获不同的异常：

```py
try:
    number = int('abc')
    result = 10 / number
except ValueError:
    print('这不是一个有效的数字。')
except ZeroDivisionError:
    print("不能除以零。")
```

通过使用不同的 `except` 子句，你可以让错误处理更有针对性、更有帮助。

你还可以使用异常对象（通常用 `as` 关键字别名），例如下面用 `e` 作为异常对象的别名：

```py
try:
    x = 1 / 0
except ZeroDivisionError as e:
    print(f'发生错误: {e}')
```

使用 `e` 可以访问实际的错误信息或对象，便于记录日志或调试。

你也可以在一个 `except` 中通过元组指定多个异常来一并捕获：

```py
try:
    number = int(input('输入一个数字: '))
    result = 10 / number
except (ValueError, ZeroDivisionError) as e:
    print(f'发生错误: {e}')
```

通过异常处理，你的程序可以更优雅地从错误中恢复。使用 `try`、`except`、`else` 和 `finally`，你可以预见潜在问题并构建更可靠的应用程序。
