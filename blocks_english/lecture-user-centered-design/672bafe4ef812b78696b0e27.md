# --interactive--

**注意**：一些交互式示例可能使用了你尚未学习的CSS。不用担心理解所有代码。示例的目标是向你展示这些设计概念的预览，以便你更好地理解工作原理。

深色模式是Web应用程序中的一项特殊功能，你可以将默认的浅色配色方案更改为深色配色方案。这有助于减少眼部疲劳并在低光环境下提高可读性。在设计深色模式功能时，了解最佳实践很重要，以确保你的深色模式功能有效且用户友好。

点击下面示例中的[Toggle Dark Mode]按钮，看看深色模式是如何工作的。

第一个考虑因素是避免在深色模式中使用饱和色彩。饱和色彩是明亮且强烈的颜色。例如，深灰色背景上的亮洋红色按钮可能过于强烈并导致眼部疲劳。相反，你应该在深色模式中使用去饱和色彩。去饱和色彩是不太强烈的颜色，饱和度较低，在深色模式下看起来更舒适。

深色模式的另一个考虑因素是使用纯黑色背景配白色文字。虽然这种高对比度可能有效，但对眼睛来说可能过于刺眼。相反，考虑使用深灰色背景配浅灰色文字来获得更柔和的对比度。文字在深色模式下会更容易阅读，也更舒适。

:::interactive_editor

  <div class="dark-mode-examples">
    <div class="box harsh-dark">

另一个考虑因素是在深色模式中使用网站的品牌标识。品牌标识是一组代表品牌的视觉元素，如标志、颜色和字体。在实施深色模式时，你应该考虑深色模式功能如何与你的品牌颜色和风格保持一致。让品牌图标和按钮保持全饱和度是可以的，而周围的元素则去饱和。

总的来说，在设计方面，你总是希望关注用户体验和对比度水平。深色模式也不例外，通过遵循这些最佳实践，你可以创建一个有效且用户友好的深色模式功能。
