# --interactive--

**注意**：一些交互式示例可能使用了你尚未学习的CSS和JavaScript。不用担心理解所有代码。示例的目标是向你展示这些设计概念的预览，以便你更好地理解工作原理。

无限滚动是一种设计模式，当用户向下滚动页面时加载更多内容。通常，这在Twitter等社交媒体网站上使用。例如，如果你已登录并想查看更多推文，可以向下滚动，更多推文将加载。这是无限滚动的一个示例。

window.addEventListener('scroll', () => {

  const container = document.querySelector('.infinite-scroll');

    container.appendChild(post);
  }
}

无限滚动也用作分页的替代方案。分页是一种将内容分成页面的设计模式。当需要显示大量内容时经常使用。分页的一个示例是在Google上搜索某些内容时，你会在多个页面上看到搜索结果。使用分页时，你必须点击按钮才能转到下一页。使用无限滚动时，你只需继续向下滚动，更多内容就会加载。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="pagination">
  <div class="post">帖子1</div>
  <div class="post">帖子2</div>
  <div class="post">帖子3</div>
  <!-- 更多帖子将在下一页 -->
</div>
<div class="pagination-controls">
  <button class="prev" disabled>上一页</button>
  <button class="next">下一页</button>
</div>
<script src="index.js"></script>
```

```css
.pagination {
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.post {
  padding: 12px;
  border: 1px solid #ccc;
  border-radius: 4px;
}

.pagination-controls {
  margin-top: 12px;
}

.pagination-controls button {
  padding: 8px 16px;
  margin-right: 8px;
}
```

```js
let currentPage = 1;
const postsPerPage = 3;
const totalPosts = 9; 
const totalPages = Math.ceil(totalPosts / postsPerPage);
const container = document.querySelector('.pagination');
const prevButton = document.querySelector('.prev');
const nextButton = document.querySelector('.next'); 

function renderPosts() {
  container.innerHTML = '';
  const start = (currentPage - 1) * postsPerPage;
  const end = start + postsPerPage;
  for (let i = start; i < end && i < totalPosts; i++) {
    const post = document.createElement('div');
    post.className = 'post';
    post.textContent = `帖子 ${i + 1}`;
    container.appendChild(post);
  }
  prevButton.disabled = currentPage === 1;
  nextButton.disabled = currentPage === totalPages;
}

prevButton.addEventListener('click', () => {
  if (currentPage > 1) {
    currentPage--;
    renderPosts();
  }
});

nextButton.addEventListener('click', () => {
  if (currentPage < totalPages) {
    currentPage++;
    renderPosts();
  }
});

renderPosts();
```

当你将无限滚动融入设计时，有几个最佳实践需要牢记。第一个考虑因素是提供一个"加载更多"按钮，当用户点击时加载下一组结果。这是让用户控制何时想查看更多内容的好方法。

另一个考虑因素是添加一个"返回"按钮。这为用户提供了返回上一页的能力，而无需一直滚动回到顶部。这创造了更好的用户体验，并让他们对自己的浏览体验有更多控制权。

有时你会看到带有"返回顶部"按钮的设计，将用户引导回结果页面的顶部。另一个考虑因素是提供加载指示器。用户应该清楚地知道更多内容正在加载；否则，他们可能会认为页面已损坏。

:::interactive_editor

```js
const container = document.querySelector('.infinite-scroll');
const loadingIndicator = document.querySelector('.loading-indicator');
const backToTopBtn = document.getElementById('back-to-top');

window.addEventListener('scroll', () => {
  if (window.scrollY > 400) {
    backToTopBtn.style.display = 'block';
  } else {
    backToTopBtn.style.display = 'none';
  }

  if (window.innerHeight + window.scrollY >= document.body.offsetHeight) {
    loadMorePosts();
  }
});

backToTopBtn.addEventListener('click', () => {
  window.scrollTo({ top: 0, behavior: 'smooth' });
});

function loadMorePosts() {
  loadingIndicator.style.display = 'block';
  setTimeout(() => {
    for (let i = 0; i < 3; i++) {
      const post = document.createElement('div');
      post.className = 'post';
      post.textContent = `帖子 ${container.children.length + 1}`;
      container.appendChild(post);
    }
    loadingIndicator.style.display = 'none';
  }, 1000);
}
```

:::

最后一个考虑因素是让用户可以访问页脚。如果页脚包含重要信息，那么应该始终让用户可以访问。

总之，无限滚动是在网站上显示内容的好方法。但是，在设计无限滚动时，你应该记住最佳实践，以便提供最佳的用户体验。
