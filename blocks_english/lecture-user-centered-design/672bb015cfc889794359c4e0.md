# --interactive--

**注意**：一些交互式示例可能使用了你尚未学习的CSS和JavaScript。不用担心理解所有代码。示例的目标是向你展示这些设计概念的预览，以便你更好地理解工作原理。

什么是模态框？它是网站可能在内容顶部显示给你的弹出窗口类型。HTML有一个[dialog]元素，你可以用来创建模态框。

模态框后面的内容通常会变暗。这有助于用户在视觉上专注于你希望他们交互的区域——在这种情况下是模态框。

始终让用户能够点击模态框外部来关闭它是很好的做法。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button id="open-modal">打开模态框</button>
<dialog>
  <h2>订阅我们的新闻通讯！</h2>
  <p>获取最新更新和优惠。</p>
  <button>订阅</button>
  <button>关闭</button>
</dialog>

<script src="index.js"></script>
```

```css
dialog {
  border: none;
  border-radius: 8px;
  padding: 20px;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

dialog::backdrop {
  background: rgba(0, 0, 0, 0.5);
}
```

```js
const dialog = document.querySelector('dialog');
const closeButton = dialog.querySelector('button:last-of-type');
const openModalButton = document.getElementById('open-modal');

closeButton.addEventListener('click', () => {
  dialog.close();
});

openModalButton.addEventListener('click', () => {
  dialog.showModal();
});

// 点击模态框外部关闭它
dialog.addEventListener('click', (event) => {
  const rect = dialog.getBoundingClientRect();
  const isInDialog = (
    event.clientX >= rect.left &&
    event.clientX <= rect.right &&
    event.clientY >= rect.top &&
    event.clientY <= rect.bottom
  );
  if (!isInDialog) {
    dialog.close();
  }
}); 

```

:::

你经常会看到模态框上有非常突出的按钮。这些被称为CTA，或行动号召。你希望这些按钮易于识别，因为用模态框打断用户流程的目的是促使他们采取特定行动。

模态框还应该有一个关闭按钮。虽然你可能真的很希望用户点击你的CTA，但重要的是要给他们一个退出模态框并恢复之前所做事情的选项。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button id="open-modal">打开模态框</button>
<dialog>
  <h2>订阅我们的新闻通讯！</h2>
  <p>获取最新更新和优惠。</p>
  <button class="cta">订阅</button>
  <button class="close">关闭</button>
</dialog>
<script src="index.js"></script>
```

```css
.cta {
  background-color: #007BFF;
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 4px;
  cursor: pointer;
}

.close {
  background-color: transparent;
  color: #007BFF;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
}
```

```js
const dialog = document.querySelector('dialog');
const closeButton = dialog.querySelector('.close');
const openModalButton = document.getElementById('open-modal');

closeButton.addEventListener('click', () => {
  dialog.close();
});

openModalButton.addEventListener('click', () => {
  dialog.showModal();
});

// 点击模态框外部关闭它
dialog.addEventListener('click', (event) => {
  const rect = dialog.getBoundingClientRect();
  const isInDialog = (
    event.clientX >= rect.left &&
    event.clientX <= rect.right &&
    event.clientY >= rect.top &&
    event.clientY <= rect.bottom
  );
  if (!isInDialog) {
    dialog.close();
  }
});

```

:::

当然，模态框存在可访问性问题，比如正确管理元素的焦点。但是，如果你将这些通用实践作为起点，你将有一个坚实的基础可以在此基础上构建。
