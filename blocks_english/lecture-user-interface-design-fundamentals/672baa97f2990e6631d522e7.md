# --interactive--

**注意**: 一些交互式示例可能使用了你尚未学习的 CSS。不用担心理解所有代码。示例的目的是向你展示这些设计概念的预览，以便你更好地理解它们的工作原理。

首先，你可能需要熟悉对比度的概念。对比度是指两种颜色之间的差异，或者说区分它们的难易程度。

对比度较高的颜色在视觉上会更加明显，而对比度较低的颜色在视觉上会更加相似。例如，黑色和白色具有很高的对比度比。而浅蓝色和浅紫色的对比度就较低。

:::interactive_editor

```html
<style>
  .example {
    padding: 20px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .high-contrast {
    background-color: black;
    color: white;
  }

  .low-contrast {
    background-color: #add8e6; 
    color: #dda0dd; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 5px;
  }
</style>

<div class="example high-contrast">
  <div class="label">高对比度</div>
  这段文本具有高对比度（黑色背景和白色文本）。
</div>

<div class="example low-contrast">
  <div class="label">低对比度</div>
  这段文本具有低对比度（浅蓝色背景和浅紫色文本）。
</div>
```

:::

当然，在这些示例中，由于布局的原因，区别可能更加明显。但是将这些颜色应用到文本上会怎样呢？你可以在白色背景上使用高对比度的黑色文本，而在蓝色背景上使用低对比度的紫色文本。

:::interactive_editor

```html
<style>
  .text-example {
    padding: 10px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .text-high-contrast {
    background-color: white;
    color: black;
  }

  .text-low-contrast {
    background-color: #add8e6; 
    color: #dda0dd; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 5px;
    display: block;
  }
</style>

<div class="text-example text-high-contrast">
  <span class="label">高对比度文本</span>
  这是高对比度文本：白色背景上的黑色文本。
</div>

<div class="text-example text-low-contrast">
  <span class="label">低对比度文本</span>
  这是低对比度文本：浅蓝色背景上的紫色文本。
</div>
```

:::

但是你如何确定什么样的对比度才"足够好"呢？你不能仅仅基于文本在你看来的样子来判断，因为每个用户都会有不同的体验。幸运的是，Web 内容可访问性指南（WCAG）为此提供了标准。

对比度比为 4.5:1 的文本被认为是 AA 标准，这是你为大多数用户实现可访问性应遵循的最低标准。对比度比为 7:1 的文本被认为是 AAA 标准，确保了最佳的可访问性水平。

:::interactive_editor

```html
<style>
  .contrast-example {
    padding: 15px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .aa-contrast {
    background-color: #ffffff; 
    color: #4b4b4b; 
  }

  .aaa-contrast {
    background-color: #ffffff; 
    color: #1a1a1a; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }
</style>

<div class="contrast-example aa-contrast">
  <span class="label">AA 标准（对比度比率 ~4.5:1）</span>
  此文本符合对比度的最低可访问性标准。
</div>

<div class="contrast-example aaa-contrast">
  <span class="label">AAA 标准（对比度比率 ~7:1）</span>
  此文本符合最高级别的可访问性对比度标准。
</div>
```

:::

有许多网站可以检查两种颜色之间的对比度比，但大多数浏览器都允许你直接在网站的开发者工具中进行检查。

让我们再看几个对比度比的示例，以便更好地理解这个概念。

下面是一个白色背景上黑色文本的示例，其对比度比为 21:1。

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }
</style>

<div class="contrast-21">
  <span class="label">对比度比率 21:1</span>
  这是白色背景上的黑色文本，具有最高的对比度比率 21:1。
</div>
```

:::

这个示例完全符合 AAA 标准。现在，让我们看看蓝色背景上的紫色文本。

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .purple-on-blue {
    background-color: #0000cc;
    color: #800080; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">对比度比率 21:1</span>
  这个示例在白色背景上的黑色文本完全符合 AAA 标准。
</div>

<div class="purple-on-blue">
  <span class="label">蓝色上的紫色（低对比度）</span>
  这不符合 AA 标准。
</div>
```

:::

这个示例的对比度比为 1.48:1，甚至不符合 AA 标准。

你如何解决这个问题呢？有三个方面会影响对比度比。

第一个是色调，它代表一般的颜色类型，如红色、蓝色、橙色。让我们转换到一个与蓝色距离更远的色调。在这种情况下，让我们使用红色。

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .red-on-blue {
    background-color: #0000cc; 
    color: #ff0000; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">对比度比率 21:1</span>
  这个示例在白色背景上的黑色文本完全符合 AAA 标准。
</div>

<div class="red-on-blue">
  <span class="label">蓝色上的红色（更高对比度的色调变化）</span>
  这不符合可访问性标准。
</div>
```

:::

不幸的是，这种改变只将对比度比提高到 1.49:1，这意味着你需要改变饱和度，或者说颜色的"量"。让我们增加文本中红色的量。这使我们更接近目标，对比度比为 3.54:1。

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .low-sat-red-on-blue {
    background-color: #0000cc; 
    color: #b23333; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .high-sat-red-on-blue {
    background-color: #0000cc; 
    color: #ff4d4d;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">对比度比率 21:1</span>
  这个示例在白色背景上的黑色文本完全符合 AAA 标准。
</div>

<div class="low-sat-red-on-blue">
  <span class="label">蓝色上的低饱和度红色（对比度 ~1.49:1）</span>
  这种红色饱和度较低，导致对比度较差。
</div>

<div class="high-sat-red-on-blue">
  <span class="label">蓝色上的高饱和度红色（对比度 ~3.54:1）</span>
  增加红色的饱和度可以改善对比度，但仍低于 AA 标准。
</div>
```

:::

然而，通过改变最后一个值（亮度），你可以进一步接近目标。亮度代表颜色中白色的存在量。

让我们减少红色的亮度。现在在浅蓝色背景上有了更深的红色，这使对比度比达到了 10.34:1。

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .blue-bg {
    background-color: #add8e6; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .low-sat-red-on-blue {
    background-color: #0000cc; 
    color: #b23333;
    margin-bottom: 20px;
  }

  .high-sat-red-on-blue {
    background-color: #0000cc; 
    color: #ff4d4d; 
    margin-bottom: 20px;
  }

  .dark-red-on-light-blue {
    background-color: #add8e6; 
    color: #8b0000;
  }
</style>

<div class="contrast-21">
  <span class="label">对比度比率 21:1</span>
  这个示例在白色背景上的黑色文本完全符合 AAA 标准。
</div>

<div class="low-sat-red-on-blue blue-bg">
  <span class="label">蓝色上的低饱和度红色（对比度 ~1.49:1）</span>
  这种红色饱和度较低，导致对比度较差。
</div>

<div class="high-sat-red-on-blue blue-bg">
  <span class="label">蓝色上的高饱和度红色（对比度 ~3.54:1）</span>
  增加红色的饱和度可以改善对比度，但仍低于 AA 标准。
</div>

<div class="dark-red-on-light-blue">
  <span class="label">浅蓝色上的深红色（对比度 ~10.34:1）</span>
  减少红色的亮度可以显著提高对比度比率。
</div>
```

:::

你可以继续调整颜色，以在你想要的视觉效果和可访问的对比度比之间找到平衡。但重要的是要记住，可访问性应该始终优先考虑。
