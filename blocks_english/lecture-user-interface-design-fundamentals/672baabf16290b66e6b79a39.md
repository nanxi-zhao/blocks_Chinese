# --interactive--

**注意**: 一些交互式示例可能使用了你尚未学习的 CSS。不用担心理解所有代码。示例的目的是向你展示这些设计概念的预览，以便你更好地理解它们的工作原理。

在设计网页时，创建连贯且视觉上吸引人的设计很重要。实现这一目标的方法之一是使用对齐。

对齐是排列文本和图像的方式，以在元素之间创建视觉连接。

它有助于在页面上创造秩序感和组织感，使用户更容易导航和理解内容。

有几种类型的对齐可以使用，但基本的有：

- 左对齐
- 居中对齐
- 右对齐
- 两端对齐
- 垂直对齐

左对齐、右对齐和居中对齐都是水平对齐的子类型，而垂直对齐用于沿垂直轴对齐元素。

让我们更仔细地看看每种对齐类型以及如何在设计中使用它们。

左对齐通常与文本一起使用，其中每个元素都与左边界对齐。将网页上的所有标题和段落与左边界对齐，使用户更容易阅读和跟随内容。

:::interactive_editor

```html
<style>
  body {
    font-family: Arial, sans-serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 700px;
    margin: 0 auto;
  }

  h1, h2, h3, p {
    text-align: left;
    margin: 10px 0;
  }

  p {
    line-height: 1.5;
  }
</style>

<h1>为什么左对齐很重要</h1>
<p>左对齐是在网页上呈现文本的最常见和有效的方式之一。</p>

<h2>一致性</h2>
<p>将所有标题和段落与左边界对齐，创造出干净一致的阅读流程。</p>

<h3>易于跟随</h3>
<p>用户可以轻松扫描和跟随内容，而不会感到困惑或错位。</p>
```

:::

左对齐的对立面是右对齐，其中每个元素都与右边界对齐。这通常用于网站上显示额外内容，如促销横幅或广告。

:::interactive_editor

```html
<style>
  body {
    font-family: Arial, sans-serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 700px;
    margin: 0 auto;
  }

  h1, h2, h3, p {
    text-align: right;
    margin: 10px 0;
  }

  p {
    font-size: 16px;
    line-height: 1.5;
  }
</style>

<h1>网页设计中的右对齐</h1>
<p>右对齐通常用于在网站上显示额外或促销内容。</p>

<h2>次要内容</h2>
<p>将文本与右边界对齐可以在视觉上将其与主要内容分开。</p>

<h3>促销用途</h3>
<p>这种对齐通常用于横幅、广告或侧边栏消息。</p>
```

:::

例如，与右边界对齐的广告使其从页面的其余内容中脱颖而出，但不会分散用户对主要内容的注意力。

居中对齐是指元素在页面上居中。这通常用于标题、标志和其他你想要吸引注意力的重要元素。

:::interactive_editor

```html
<style>
  body {
    font-family: Arial, sans-serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 700px;
    margin: 0 auto;
  }

  h1, h2, h3, p {
    text-align: center;
    margin: 10px 0;
  }

  p {
    font-size: 16px;
    line-height: 1.5;
    max-width: 600px;
    margin-left: auto;
    margin-right: auto;
  }
</style>

<h1>居中对齐以产生影响</h1>
<p>居中对齐非常适合吸引注意力到标题和重要元素上。</p>

<h2>引人注目的布局</h2>
<p>通过居中关键内容，使其脱颖而出并创造出平衡的视觉效果。</p>

<h3>常见用途</h3>
<p>这种技术通常用于标志、主页标题和行动号召消息。</p>
```

:::

两端对齐是指文本与左右边界都对齐。这通常用于描述性段落或文章，创造出干净专业的外观。

:::interactive_editor

```html
<style>
  body {
    font-family: Georgia, serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 700px;
    margin: 0 auto;
  }

  h1 {
    text-align: center;
    margin-bottom: 20px;
  }

  p {
    text-align: justify;
    line-height: 1.7;
    margin-bottom: 20px;
  }
</style>

<h1>两端对齐以创建干净的文本块</h1>

<p>
  两端对齐确保每行文本在左右边界之间均匀延伸。
  这创造出平滑的块状外观，看起来非常精致和专业。
  它通常用于报纸、杂志和正式文件中，以获得统一的外观。
</p>

<p>
  然而，在使用两端对齐文本时应注意避免不均匀的间距或"河流"状空白，
  特别是在窄列或屏幕上。适当的连字符和响应式设计有助于保持可读性。
</p>
```

:::

最后一种对齐类型是垂直对齐，用于沿垂直轴对齐元素。

垂直对齐可以用于例如网站上的联系表单。将所有表单输入（如姓名、电子邮件和消息字段）沿垂直轴对齐，使用户更容易填写表单。

:::interactive_editor

```html
<style>
  body {
    font-family: Arial, sans-serif;
    padding: 20px;
    background-color: #fff;
    color: #222;
    max-width: 400px;
    margin: 0 auto;
  }

  form {
    display: flex;
    flex-direction: column;
    gap: 15px;
  }

  label {
    font-weight: bold;
    margin-bottom: 5px;
  }

  input, textarea {
    padding: 8px;
    border: 1px solid #ccc;
    border-radius: 4px;
    width: 100%;
    box-sizing: border-box;
  }

  button {
    padding: 10px;
    background-color: #007bff;
    border: none;
    border-radius: 4px;
    color: white;
    cursor: pointer;
  }

  button:hover {
    background-color: #0056b3;
  }
</style>

<form>
  <label for="name">姓名</label>
  <input type="text" id="name" name="name" placeholder="例如：简·多伊">

  <label for="email">电子邮件</label>
  <input type="email" id="email" name="email" placeholder="janedoe@example.com">

  <label for="message">消息</label>
  <textarea id="message" name="message" rows="4" placeholder="在这里写下你的消息"></textarea>

  <button type="submit">提交</button>
</form>
```

:::

通过使用不同类型的对齐，你可以在页面上创造秩序感和组织感，使用户更容易导航和理解内容。
