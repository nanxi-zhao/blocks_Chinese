# --interactive--

外边距和内边距是CSS中创建结构良好、可读性强且视觉上吸引人的网页的重要属性。

外边距控制元素外部的空间，帮助将其与其他元素分离并定义布局结构，而内边距控制元素内部的空间，提高内容的可读性和美观性。

为了更好地理解[margin]和[padding]之间的区别，让我们看一些示例。

下面是页面上三个段落元素的HTML示例：

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

要为每个段落元素的顶部应用间距，你可以这样使用[margin-top]属性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

在此示例中，我们仅为每个段落元素的顶部应用[20px]的外边距。

我们还在四边上设置了实心黑框，以便你能更好地看到外边距。

四种不同的[margin]属性是[margin-top]、[margin-right]、[margin-bottom]和[margin-left]。

下面是使用所有四个属性的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

在此示例中，我们为段落元素的四边分配了间距值。还添加了实心黑框，以便你能更好地看到外边距。

使用[margin]属性的另一种方法是使用简写符号。你可以一次指定一个、两个、三个或四个值。让我们一起探讨这些选项。

在[margin]简写上使用单一值时，该确切值将应用于目标元素的四边。

下面是在[margin]简写上使用单一值的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

此代码示例将为段落元素的四边平均应用[10px]的[margin]。

使用两个值时，第一个值应用于[top]和[bottom]，而第二个值应用于元素的[left]和[right]边。

下面是为[margin]简写使用两个值的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

这将段落元素的[top]和[bottom]外边距设置为[10px]，[left]和[right]外边距设置为[20px]。

如果提供三个值，第一个值应用于[top]外边距，第二个值应用于[left]和[right]外边距，第三个值应用于[bottom]外边距。

下面是一个更好地理解的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

这将[top]的外边距设置为[10px]，[left]和[right]设置为[20px]，[bottom]设置为[30px]。

使用四个值时，这为你提供了更多控制，因为你可以独立指定目标元素每边的外边距值。

第一个值针对[top]，第二个值针对[right]，第三个值针对[bottom]，第四个值针对[left]。

下面是使用四个值的margin简写的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

这将[top]的外边距设置为[10px]，[right]设置为[20px]，[bottom]设置为[30px]，[left]设置为[40px]。

[padding]属性用于在元素内部应用空间，即内容与其边框之间。

像[margin]属性一样，四个[padding]属性是[padding-top]、[padding-right]、[padding-bottom]和[padding-left]。

下面是为段落元素设置[padding]的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

这将[top]的内边距设置为[10px]，[right]设置为[20px]，[bottom]设置为[30px]，[left]设置为[40px]。

如你所见，[padding]应用于边框内的内容，而[margin]应用于边框外侧。

就像[margin]属性一样，你也可以选择使用[padding]属性的简写。

你也可以在[padding]简写属性上指定一个、两个、三个或四个值。

下面是为前面的段落元素使用[padding]简写的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

在此示例中，使用简写代码将段落元素的[top]内边距设置为[10px]，[right]设置为[20px]，[bottom]设置为[30px]，[left]设置为[40px]。
