# --interactive--

在之前的课程中，你首次接触了`length`属性，该属性返回数组中元素的数量。下面是一个关于它如何工作的快速回顾：

:::interactive_editor

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

:::

数组中可能存在空槽。空槽被定义为其中没有任何内容的槽。这与值为`undefined`的数组不同。这类数组被称为稀疏数组。这是一个示例：

:::interactive_editor

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

:::

在这种情况下，尽管我们只有两个定义的元素，`1`和`4`。长度是`4`，因为最高索引（`3`）加`1` gives us a length of `4`。

现在让我们讨论如何创建一个固定长度的空数组。在JavaScript中有几种方法可以做到这一点，但一种常见的方法是使用带有数字参数的`Array()`构造函数。`Array()`构造函数可以与`new`关键字一起使用来创建一个新数组。这是一个示例：

:::interactive_editor

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [undefined, undefined, undefined, undefined, undefined]
```

:::

在这个示例中，我们使用`Array(5)`创建了一个新数组。这创建了一个长度为`5`的稀疏数组，其中所有槽都是空的。

另一种创建固定长度空数组的方法是使用带有长度参数的`Array.from()`方法。该方法创建一个指定长度的数组，所有元素都初始化为`undefined`：

:::interactive_editor

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5

// [undefined, undefined, undefined, undefined, undefined]
console.log(fixedLengthArray);
```

:::

如果你想创建一个特定长度的数组并用默认值填充它，你可以使用`Array.fill()`方法：

:::interactive_editor

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

:::

这创建了一个长度为三的数组，并用值`0`填充所有元素。

了解如何获取数组的长度和创建固定长度的数组对于许多编程任务都很重要，特别是当你需要为特定算法或数据结构初始化数组时。
