# --interactive--

CSS中的属性选择器是一个强大的工具，允许你根据HTML元素的属性来定位它们。这意味着你可以为具有特定属性或某些属性值的元素应用样式。

当你想要动态地为元素设置样式，或者仅靠类名无法提供足够的特异性时，它特别有用。

例如，你可以使用属性选择器来定位所有具有`href`属性的链接。当你想要为页面上的所有链接应用统一样式时，这特别方便。

下面是使用属性选择器来定位具有`href`属性的链接的方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

这个CSS规则将为任何包含`href`属性的链接应用蓝色和下划线，确保所有可点击的链接具有一致的样式。

但属性选择器可以更加具体。假设你只想定位具有`title`属性的链接。`title`属性通常提供有关链接的附加信息，你可能希望以不同的方式为这些链接设置样式，以表明它们有额外的信息可用。下面是实现方法：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

此规则为任何具有`title`属性的链接应用粗体文本并移除下划线。这是在视觉上将这些链接与页面上其他链接区分开来的好方法。

你也可以组合属性选择器以获得更精确的效果。例如，如果你想为同时具有`href`和`title`属性的链接设置样式，可以这样做：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

在这种情况下，只有同时包含`href`和`title`属性的链接才会被设置为绿色文本。这种控制级别正是使属性选择器在CSS中如此强大的原因。

Another example for an attribute selector is to match a single value within a space-separated list of values in an attribute.

Here is an example of an anchor element with multiple classes:

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

To target this specific anchor element, you can use the following selector:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

这里使用`[attr~=value]`语法来定位class属性包含单词`"primary"`的所有锚元素。

如果需要定位属性值以特定值为前缀的元素，则可以使用`[attr^=value]`语法。这是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

在此示例中，`a[href^="https://"]`选择器将定位`href`属性值以`"https://"`开头的所有锚元素。

要定位属性值以特定值结尾的元素，你可以使用`[attr$=value]`语法。这是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

在此示例中，`a[href$=".com"]`选择器将定位`href`属性值以`.com`结尾的所有锚元素。

使用属性选择器不仅可以增强网页的样式，还可以通过使链接等交互元素根据其属性更易于区分来提高可访问性。
