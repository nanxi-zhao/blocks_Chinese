# --interactive--

在 HTML 中使用有序列表时，[type] 属性允许您指定使用的编号样式，例如数字、字母或罗马数字。

[type] 属性可以设置在 [ol]（有序列表）元素上以控制编号格式：

- [1] 用于数字列表（例如 1, 2, 3, ...），
- [A] 用于大写英文字母列表（例如 A, B, C, ...），
- [a] 用于小写英文字母列表（例如 a, b, c, ...），
- [I] 用于大写罗马数字（例如 I, II, III, ...），
- [i] 用于小写罗马数字（例如 i, ii, iii, ...）。

以下是如何使用属性选择器定位具有特定 [type] 属性的有序列表：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>项目 1</li>
  <li>项目 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

在这个示例中，CSS 规则将对使用大写英文字母编号的任何有序列表应用紫色文本和粗体字重，这是由 [type="A"] 指定的。

同样，您可以通过选择具有 [type="i"] 的列表来定位使用小写罗马数字的有序列表：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>项目 1</li>
  <li>项目 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

此规则将把使用小写罗马数字的任何有序列表的文本颜色更改为绿色。

基于 [type] 属性设置有序列表样式的能力让您能够更好地控制文档中列表的外观，从而根据内容结构进行自定义。
