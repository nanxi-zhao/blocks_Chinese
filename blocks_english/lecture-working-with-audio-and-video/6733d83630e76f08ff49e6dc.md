# --description--

`HTMLMediaElement` API 是控制页面上音频和视频元素行为的强大工具。它扩展了基本的 `HTMLElement` 接口，因此你可以访问基本属性以及这些有用的方法。

你已经在之前的关于 `Audio` 元素的课程中探索了一些方法，如 `play()` 和 `pause()`。这些相同的方法在视频元素上也可用。

还有一些其他有用的方法，如 `addTextTrack()` 方法。此方法允许你指定与媒体元素关联的文本轨道，这对于为视频添加字幕特别有帮助。

或者是 `fastSeek()` 方法，它允许你将播放位置移动到媒体内的特定时间。

`HTMLMediaElement` API 还提供了对标准元素没有的新事件的访问。

当媒体开始和停止播放时，会触发 `play` 和 `pause` 事件。当到达媒体末尾时，会触发 `ended` 事件。

当由于数据缓冲而自动暂停播放时，会触发 `waiting` 事件。

而 `canplay` 和 `canplaythrough` 事件在媒体可以部分播放或完全播放时触发。

`HTMLMediaElement` 为你提供了对交互式媒体的大量控制，我们只是触及了表面。我鼓励你探索和试验 API，找到将其与你的应用程序集成的最佳方式。
