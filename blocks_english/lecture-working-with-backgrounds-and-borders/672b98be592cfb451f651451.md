# --interactive--

CSS 中的渐变是两种或多种颜色之间的平滑过渡，可以应用到元素的背景上，让页眉、卡片或页面看起来更有层次感，而无需使用图片。

CSS 主要有两种渐变：线性渐变（linear）和径向渐变（radial）。

线性渐变沿直线过渡颜色，你可以定义方向和参与过渡的颜色。

基本语法如下：

```css
background: linear-gradient(direction, color-stop1, color-stop2, ...);
```

在这个语法中，`direction` 表示渐变的方向（可以是角度如 `45deg`，也可以是关键字如 `to right`、`to bottom`，或指定某个边/角），`color-stop` 则指定颜色以及可选的位置点。

下面用一个简单的例子来帮助理解线性渐变的工作方式：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient{
  background: linear-gradient(to right, red, yellow);
  height: 40vh;
}
```

:::


上面的 CSS 会创建一个从左侧的 `red` 到右侧的 `yellow` 的线性渐变。示例中元素高度为视口高度的 `40vh`，这样更容易观察渐变效果（关于 `vh` 单位，我们会在后面课程中详细介绍）。

`to right` 表示渐变从左向右水平进行。

另一个常见的渐变类型是径向渐变（radial gradient）。

径向渐变从某个起点（通常是中心）向外呈圆形或椭圆形地辐射颜色过渡。

基本语法如下：

```css
background: radial-gradient(shape size at position, color-stop1, color-stop2, ...)
```

其中 `shape` 指定渐变的形状，可以是 `circle` 或 `ellipse`；`size` 决定渐变结束形状的大小，可能取 `closest-side`、`closest-corner`、`farthest-side`、`farthest-corner`、`contain` 或 `cover`；`position` 指定渐变中心的位置，可用关键字（如 `center`、`top left`、`bottom right`）或精确值（如 `50% 50%`、`10px 20px`）表示。

最后，颜色停点（color stops）是一系列颜色，指定了渐变经过的颜色及其（可选的）位置百分比或长度。

下面是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient{
  background: radial-gradient(circle closest-side at center, red, yellow 50%, green);
  height: 60vh;
}
```

:::


上面的 CSS 会创建一个以元素中心为起点的圆形径向渐变：中心为 `red`，在半径的 `50%` 处过渡到 `yellow`，最后以 `green` 结束。

`closest-side` 会让渐变的结束形状适配元素的最近一侧。示例中元素高度为视口高度的 `60vh`，以便更明显地展示效果。

掌握 CSS 渐变可以大大提升你的设计表现力，通过纯代码就能做出漂亮的背景效果，而无需依赖图片。

线性渐变适合平滑的颜色过渡，径向渐变适合圆形或聚焦效果，二者结合使用可以让页面更有创意和层次感。继续试试不同的配色和参数，你会越做越自信！
