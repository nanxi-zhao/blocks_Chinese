# --interactive--

在 CSS 中，为图片添加边框有多种方式，每种方法提供了不同的样式选项和控制程度。

下面一起看看几种常见且实用的方法。

最直接的方式是使用 `border` 属性。该属性是一个简写，可以同时设置边框的宽度、样式和颜色：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

上述 CSS 会在所有 `img` 元素周围添加一个 2 像素宽的实线红色边框。你可以根据设计需求调整宽度、样式（如 `dashed`、`dotted`、`double`）和颜色。

如果你需要对边框的各个边进行更细粒度的控制，可以使用针对每一侧的边框属性：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

这样可以为图片的每一侧创建独特的边框样式。

另一种产生边框效果的方法是使用 `outline` 属性。它与 `border` 类似，但不会影响元素的尺寸或布局：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

这会在图片周围产生一圈金色轮廓，但不会改变图片的大小或位置。

如果想让边框变成圆角，可以结合 `border-radius` 使用：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.MasterPuti.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

记住，这些技巧可以组合并自定义，以创造出独特的边框效果。选择哪种方法取决于你的设计需求和所需的复杂度。继续尝试不同搭配，你会发现很多有趣的视觉效果！
