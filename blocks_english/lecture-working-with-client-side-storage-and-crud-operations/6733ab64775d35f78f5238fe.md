# --description--

CRUD是一个首字母缩略词，代表创建（Create）、读取（Read）、更新（Update）和删除（Delete）。这四个操作是持久化存储的基本操作。

持久化存储是指以一种即使在断电或设备重启后数据仍然可用的方式保存数据。

理解CRUD基本操作的工作原理对Web开发至关重要，因为它构成了使用数据库和构建用户可以添加、查看、修改和删除数据的应用程序的基础。

现在，让我们更仔细地看看CRUD的每个部分：

- 创建（Create）是指创建新数据的过程。例如，在Web应用程序中，这可能是用户向博客添加新文章。

- 读取（Read）是从数据库中检索数据的操作。例如，当你访问博客文章或在网站上查看个人资料时，你正在执行读取操作来获取和显示存储在数据库中的数据。

- 更新（Update）涉及修改数据库中的现有数据。例如，编辑博客文章或更新个人资料信息。

- 删除（Delete）是从数据库中移除数据的操作。例如，当你删除博客文章或账户时，你正在执行删除操作。

CRUD在使用数据库、用户界面和RESTful API时都会用到。RESTful API是一组用于构建Web服务的约定，它允许客户端通过标准HTTP方法执行CRUD操作来与数据库或后端系统交互。

HTTP代表超文本传输协议，它是Web上数据通信的基础。有HTTP方法定义了可以通过Web对资源执行的操作。常见的方法有[GET](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L140-L140)、[POST](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L132-L132)、[PUT](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L144-L144)、[PATCH](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L38-L38)、[DELETE](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L152-L152)。

你将在未来的课程中学习更多关于RESTful API和HTTP的知识，但这里快速介绍一下CRUD如何映射到不同的HTTP方法：

- [POST](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L132-L132)用于创建新资源。

- [GET](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L140-L140)用于检索或读取数据。

- [PUT](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L144-L144)用于通过完全替换来更新资源。

- [PATCH](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L38-L38)用于部分更新资源。

- [DELETE](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ab64775d35f78f5238fe.md#L152-L152)用于移除资源。

以下是一个如何在JavaScript中使用简单数组表示CRUD操作的示例：

```js
let items = [];

// Create
function createItem(item) {
  items.push(item);
}

// Read
function readItems() {
  return items;
}

// Update
function updateItem(index, newItem) {
  items[index] = newItem;
}

// Delete
function deleteItem(index) {
  items.splice(index, 1);
}

// Example Usage
createItem('Book');
console.log(readItems()); // ['Book']
updateItem(0, 'Magazine');
console.log(readItems()); // ['Magazine']
deleteItem(0);
console.log(readItems()); // []
```

在这个示例中：

- 我们通过将项目推入数组来创建项目。

- 我们通过返回数组来读取项目。

- 我们通过修改给定索引处的数组元素来更新项目。

- 我们通过使用[splice()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/6732b284901f8c1539e20bcb.md#L26-L26)从数组中删除项目。

这是CRUD操作在概念层面上如何工作的基本表示。
