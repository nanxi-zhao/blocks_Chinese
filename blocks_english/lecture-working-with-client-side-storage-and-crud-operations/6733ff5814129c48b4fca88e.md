# --description--

在上一课中，我们学习了使用[localStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff4a9319c8486750886c.md#L27-L27)的相关知识，并简单介绍了[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)。

回想一下，[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)是指当用户关闭运行Web应用程序的标签页或窗口时，数据就会被清除。它适用于数据只需要在单个会话期间持久化的场景，比如在导航过程中保持表单数据或在结账过程中存储临时状态信息。

与[localStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff4a9319c8486750886c.md#L27-L27)类似，[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)使用键值对来存储和检索数据。与[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)一起使用的方法也与[localStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff4a9319c8486750886c.md#L27-L27)相同，唯一的真正区别是数据存储的时间长短。

以下是使用不同方法的几个示例：

- [sessionStorage.setItem()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L18-L18)：在[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)中存储键值对。

```js
sessionStorage.setItem('cart', '3 items');
```

- [sessionStorage.getItem()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L24-L24)：从[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)中检索给定键的值。

```js
let cart = sessionStorage.getItem('cart');
console.log(cart); // Outputs: '3 items'
```

- [sessionStorage.removeItem()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L31-L31)：使用键从[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)中删除特定项目。

```js
sessionStorage.removeItem('cart');
```

- [sessionStorage.clear()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L37-L37)：清除存储在[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)中的所有数据。

```js
sessionStorage.clear();
```

让我们看一个示例，我们在[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)中存储数据，这些数据只在浏览器标签页或窗口打开期间持续存在：

```js
// Store data in sessionStorage
sessionStorage.setItem('currentUser', 'JohnDoe');

// Retrieve the stored data
const user = sessionStorage.getItem('currentUser');
console.log(user); // 'JohnDoe'

// Remove a specific key from sessionStorage
sessionStorage.removeItem('currentUser');

// Clear all sessionStorage data
sessionStorage.clear();
```

在这个示例中，我们：

1. 在[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)中存储当前用户名（[JohnDoe](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L47-L47)）。

2. 检索并显示它。

3. 删除与键[currentUser](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L50-L50)关联的项目。

4. 清除所有[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)数据。

与[localStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff4a9319c8486750886c.md#L27-L27)的关键区别是，一旦用户关闭标签页，所有存储的会话数据都将丢失。

[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)在以下场景中特别有用：

- 存储多页面表单过程中的临时数据，如表单条目。

- 存储不需要在会话之间持久化的临时选择或偏好。

- 在单页应用程序中维护状态，这些状态在标签页关闭后不需要被记住。

[sessionStorage](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-client-side-storage-and-crud-operations/6733ff5814129c48b4fca88e.md#L12-L12)确保一旦用户离开页面，会话数据就会被清除，这对于不想在当前会话之外保留信息的场景非常有用。
