# --description--

你可能已经熟悉了一些基本快捷键，这些快捷键是从你的操作系统继承的。像 `Ctrl + S` 保存、`Ctrl + C` 复制和 `Ctrl + V` 粘贴在 VS Code 中都能工作。

但有一些特定于应用程序的快捷键仍然可以提升你的生产力。值得注意的是，这些快捷键中的一些可能因操作系统而异。

例如，`Shift + Alt + F` 将在当前打开的文件上运行你配置的格式化程序（例如，对于 JavaScript 项目，可以是 prettier）。

或者 `Ctrl + Shift + F`（Windows）或 `Cmd + Shift + F`（Mac）来搜索工作区中所有文件的文本内容。然后 `Ctrl + Shift + H`（Windows）或 `Cmd + Shift + H`（Mac）如果你想运行查找和替换。

如果你需要删除一行，`Ctrl + Shift + K`（Windows）或 `Cmd + Shift + K`（Mac）将删除它。

需要更多的代码空间吗？`Ctrl + B`（Windows）或 `Cmd + B`（Mac）将隐藏侧边栏 - 侧边栏包含文件列表和扩展菜单。

或者也许你只是看不清你的代码？`Ctrl + 加号`（Windows）或 `Cmd + 加号`（Mac）将增加编辑器的缩放比例，`Ctrl + 减号`（Windows）或 `Cmd + 减号`（Mac）将减小它。

最后，如果你忘记了这些快捷键中的任何一个，你总是可以使用 `Ctrl + Shift + P`（Windows）或 `Cmd + Shift + P`（Mac），它会为你打开命令面板，你可以选择任何你需要的命令。

有了这些知识，再加上一些练习，你就快成为 VS Code 的高级用户了。
