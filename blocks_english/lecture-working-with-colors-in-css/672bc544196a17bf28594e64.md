# --interactive--

在之前的课程中，你学习了如何开始使用线性和径向渐变。在本课中，我们将深入探讨如何使用 CSS 属性来创建更具视觉吸引力和复杂的设计。

CSS 中的渐变允许你在两个或多个指定颜色之间创建平滑过渡。它们通常用于在不需要图像文件的情况下为网页设计添加深度、纹理和视觉趣味。

CSS 支持两种主要类型的渐变：线性渐变和径向渐变。

线性渐变沿着直线创建颜色的渐进混合。你可以控制这条线的方向和使用的颜色。以下是线性渐变的基本示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

这创建了一个从左侧红色到右侧蓝色的渐变。`to right` 参数指定了渐变的方向。你可以使用其他方向，如 `to left`、`to top`、`to bottom`，甚至指定角度如 `45deg`。

你也可以在线性渐变中使用多种颜色停止点：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

这创建了一个从红色到黄色再到蓝色的渐变，在元素中均匀分布。

另一方面，径向渐变创建从中心点向外辐射的圆形或椭圆形渐变。这是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

这创建了一个从中心红色到边缘蓝色的圆形渐变。你也可以指定径向渐变的形状和大小：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

这创建了一个从元素左上角开始的椭圆形渐变。线性和径向渐变都允许你在精确位置指定颜色停止点。例如：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

这创建了一个红色占前 `20%`，然后在 `40%` 处过渡到黄色，最后在 `80%` 处过渡到蓝色的渐变。

渐变可以在任何使用图像的 CSS 地方使用。这使得它们在创建背景、按钮或与其他 CSS 属性结合创建文本效果时非常灵活。

记住，虽然渐变可以增强网站的视觉吸引力，但确保足够的对比度以保证可访问性是很重要的，特别是在文本背景使用渐变时。

## --text--

CSS 中线性渐变和径向渐变的区别是什么？

## --answers--

线性渐变只使用两种颜色，而径向渐变使用多种颜色。
