# --interactive--

在JavaScript中，[indexOf()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L22-L22)方法对于查找数组中特定元素的第一个索引非常有用。如果找不到该元素，则返回[-1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L37-L37)。以下是基本语法：

```js
array.indexOf(element, fromIndex)
```

[element](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L13-L13)表示你要在数组中搜索的值，[fromIndex](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L13-L13)参数是搜索应该开始的位置。[fromIndex](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L13-L13)参数是可选的。如果未提供[fromIndex](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L13-L13)，搜索从数组的开头开始。让我们看一个示例：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

:::

在这个示例中，我们有一个包含各种水果名称的数组[fruits](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L21-L21)。我们使用[indexOf()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L22-L22)方法查找字符串[banana](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L22-L22)在[fruits](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L21-L21)数组中的索引。由于[banana](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L22-L22)出现在索引[1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L23-L23)处，该方法返回[1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L23-L23)，存储在[index](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L22-L22)变量中并记录到控制台。

如果你搜索的元素在数组中找不到，[indexOf()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L36-L36)返回[-1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L37-L37)。例如：

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");
console.log(index); // -1
```

:::

在这里，我们使用[indexOf()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L36-L36)在fruits数组中搜索字符串[grape](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L36-L36)。由于[grape](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L36-L36)不在数组中，该方法返回[-1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L37-L37)，存储在[index](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L36-L36)变量中并记录到控制台。

如果你想在特定索引号之后开始查找项目，那么你可以传递第二个参数，就像这个示例中一样：

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);
console.log(index); // 4
```

:::

在这个示例中，搜索不是从数组的开始位置开始，而是从索引号[3](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L50-L50)开始，即[yellow](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L49-L49)，并得到输出[4](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-common-array-methods/67329f64e0ef5c5b7388158d.md#L51-L51)。
