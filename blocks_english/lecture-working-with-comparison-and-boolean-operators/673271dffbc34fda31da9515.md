# --interactive--

比较运算符允许您比较两个值并返回[true](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271dffbc34fda31da9515.md#L17-L17)或[false](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271dffbc34fda31da9515.md#L17-L17)结果。然后您可以使用结果来做决策或控制程序的流程。您在[if](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271dffbc34fda31da9515.md#L10-L10)语句、循环和许多其他需要根据某些条件做决策的情况下使用比较。让我们深入了解最常见的比较运算符，看看它们是如何工作的。

大于运算符，由右尖括号（[>](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271dffbc34fda31da9515.md#L20-L20)）表示，检查左侧的值是否大于右侧的值：

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

大于或等于运算符，由右尖括号和等号（[>=](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271dffbc34fda31da9515.md#L35-L35)）表示，检查左侧的值是否大于或等于右侧的值：

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

小于运算符，由左尖括号（[<](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271dffbc34fda31da9515.md#L50-L50)）表示，与[>](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271dffbc34fda31da9515.md#L20-L20)类似，但方向相反。它检查左侧的值是否小于右侧的值：

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

小于或等于运算符，由左尖括号和等号（[<=](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-comparison-and-boolean-operators/673271dffbc34fda31da9515.md#L65-L65)）表示，检查左侧的值是否小于或等于右侧的值：

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
