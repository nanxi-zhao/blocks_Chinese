# --interactive--

二元逻辑运算符帮助您评估两个表达式并根据它们的真实性返回结果。让我们看看三种最常见的二元逻辑运算符：逻辑AND、逻辑OR和空值合并运算符。

逻辑AND运算符由双_ampersand（[&&](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L17-L17)）表示。它检查两个操作数是否为真并返回结果。如果两个操作数都为真，它返回第二个值，即右侧的值：

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

在上面的示例中，文本[hello](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L17-L17)被记录到控制台，因为两个操作数都是[true](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L17-L17)。如果任一操作数为假，它返回假值：

:::interactive_editor

```js
const result = 0 && 3;

console.log(result); // 0
```

:::

由于[0](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L29-L29)是假值，数字[0](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L29-L29)被记录到控制台。如果两个操作数都为假，它返回第一个假值：

:::interactive_editor

```js
const result = false && 0;

console.log(result); // false
```

:::

由于[false](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L41-L41)是假值，那么[false](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L41-L41)被记录到控制台。逻辑AND运算符在您想要检查多个条件并确保所有条件都为真才继续执行时非常有用。这是一个示例：

:::interactive_editor

```js
if (2 < 3 && 3 < 4) {
 console.log('The if block runs'); 
} else {
 console.log('The else block runs');
} 
```

:::

在条件中，由于[2](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L53-L53)小于[3](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L53-L53)并且[3](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L53-L53)小于[4](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L53-L53)，那么句子"The if block runs"将被记录到控制台。

逻辑OR运算符检查至少有一个操作数为真。如果第一个操作数为真，它返回该值：

:::interactive_editor

```js
const result = 'This is truthy' || false;

console.log(result); // This is truthy
```

:::

如果第一个操作数为假但第二个为真，第二个值将被记录到控制台：

:::interactive_editor

```js
const result = 0 || 'This is truthy';

console.log(result); // This is truthy
```

:::

在[if/else](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L95-L99)语句中使用逻辑OR运算符是很常见的，像这样：

:::interactive_editor

```js
let userInput;

if (userInput || 'Guest') {
 console.log('A user is present');
} else {
 console.log('No user detected');
}
```

:::

由于我们没有给[userInput](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L93-L93)变量赋值，它目前是[undefined](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L93-L93)。[if](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L95-L99)语句中的条件检查[userInput](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L93-L93)变量或字符串[Guest](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L95-L95)是否为真。由于字符串[Guest](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L95-L95)在这种布尔上下文中为真，字符串"A user is present"将被记录到控制台。

空值合并运算符比逻辑OR和逻辑AND更复杂。由双问号（[??](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L111-L111)）表示，它在您只想在第一个值为[null](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L125-L125)或[undefined](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L130-L130)时返回值的情况下很有用。这是使用空值合并运算符的示例：

:::interactive_editor

```ts
const result = null ?? 'default';

console.log(result); // default
```

:::

由于[null](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L111-L111)是空值，字符串[default](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L111-L111)将被记录到控制台。空值合并运算符在[null](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L125-L125)或[undefined](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L130-L130)是唯一应该触发回退或默认值的情况下非常有用。这是处理用户偏好设置的示例：

:::interactive_editor

```ts
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

在上面的示例中，我们有一个名为[userSettings](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L124-L128)的对象，包含[theme](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L125-L125)、[volume](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L126-L126)和[notifications](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L127-L127)属性。我们使用点符号访问[theme](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L125-L125)，如[userSettings.theme](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L130-L130)。您将在未来的课程中了解更多关于如何使用对象的知识。由于用户的[theme](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L125-L125)目前设置为[null](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L125-L125)，那么字符串[light](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/6732720e95f6a0db526a2e4d.md#L130-L130)将被记录到控制台。
