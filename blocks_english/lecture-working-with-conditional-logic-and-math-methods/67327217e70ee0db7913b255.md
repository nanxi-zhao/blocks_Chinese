# --interactive--

在深入学习JavaScript时，您会很快发现执行数学运算是一个常见任务。虽然基本算术运算符可以处理简单计算，但JavaScript提供了一个内置的[Math](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)对象来处理更复杂的数学挑战。

这个方便的工具提供了各种方法，使执行高级计算和操作数字变得更加容易。让我们探索这些方法，看看它们如何简化您的编程体验。

[Math.random()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)方法生成一个介于[0](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)（包含）和[1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)（不包含）之间的随机浮点数。这意味着可能的输出可以是[0](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)，但它永远不会真正达到[1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)。这是使用[Math.random()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)方法的示例：

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// 0到1之间的任何数字 – 0包含而1不包含
```

:::

[Math.min()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L32-L32)和[Math.max()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L35-L35)都接受一组数字并分别返回最小值和最大值。这是使用这两种方法的示例：

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

第一个[console.log()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L33-L33)将记录数字[1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L32-L32)，因为[1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L32-L32)是那组数字中最小的。第二个[console.log()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L36-L36)将记录数字[9](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L35-L35)，因为[9](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L35-L35)是那组数字中最大的。

如果您想将数字四舍五入到最接近的整数，可以使用[Math.ceil()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L48-L48)和[Math.floor()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L58-L58)方法。这是使用[Math.ceil()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L48-L48)的示例：

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

[Math.ceil()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L48-L48)将[4.3](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L48-L48)向上舍入到最接近的整数，在这种情况下是[5](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L48-L48)。现在，让我们看看向下舍入数字：

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

[Math.floor()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L58-L58)将[4.7](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L58-L58)向下舍入到最接近的整数，在这种情况下是[4](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L58-L58)。[Math.round()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L68-L70)是[Math.ceil()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L48-L48)和[Math.floor()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L58-L58)的混合体。它将数字四舍五入到最接近的整数，考虑小数点：

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

所以，如果小数点小于[5](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L35-L35)，数字向下舍入。如果小数点是[5](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L35-L35)或更大，数字向上舍入。[Math.floor()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L58-L58)和[Math.random()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)的一个实际应用是在两个整数之间生成随机数。这是其语法：

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min)) + min;
console.log(randomNum);
```

:::

在[20](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L93-L93)和[1](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L93-L93)之间生成随机数看起来像这样：

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

另一个有用的[Math](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)方法是[Math.trunc()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L104-L104)方法。[Math.trunc()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L104-L104)移除数字的小数部分，只返回整数部分，不进行四舍五入：

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

如果您需要获取数字的平方根或立方根，可以分别使用[Math.sqrt()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L115-L115)和[Math.cbrt()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L116-L116)方法：

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

第一个日志语句将记录[9](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L35-L35)，因为[81](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L115-L115)的平方根是[9](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L35-L35)，而第二个日志语句将记录[3](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L116-L116)，因为[27](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L116-L116)的立方根是[3](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L116-L116)。如果您需要获取数字的绝对值，可以使用[Math.abs()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L126-L126)方法：

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

[Math.abs()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L126-L126)返回数字的绝对值，将负数转换为正数。我们要看的最后一个方法是[Math.pow()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L137-L137)方法：

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

[Math.pow()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L137-L137)接受两个数字并将第一个数字提升到第二个数字的幂。还有许多属于[Math](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-conditional-logic-and-math-methods/67327217e70ee0db7913b255.md#L19-L19)对象的方法，您可以自己探索。然而，这些只是在JavaScript代码库中更常用的几种方法。
