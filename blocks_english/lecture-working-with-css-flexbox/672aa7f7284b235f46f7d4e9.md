# --interactive--

CSS flexbox是一种一维布局模型，允许您在容器内按行和列排列元素。您还可以控制它们的顺序和方向。Web开发人员使用它来创建响应式网站和Web应用程序，这些应用程序能够适应不同的屏幕尺寸和方向。我们将flexbox称为一维布局模型，因为它专注于沿单个轴排列元素。轴可以是水平的或垂直的。

在开始使用flexbox之前，您应该了解两个关键概念：flex容器和flex项目。

flex容器是具有flex布局的HTML元素。您可以在flex容器内以各种方式排列和对齐元素。要使HTML元素成为flex容器，您需要在其CSS样式中添加[display: flex](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L77-L77)。

flex项目是flex容器的直接子元素。这些元素可以根据其属性在flex容器内排列和对齐。它们还可以收缩或扩展以适应可用空间。

这是一个包含[main](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L21-L25)容器和三个子[div](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L22-L24)元素的示例：

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

如果您只在CSS样式表中设置这些[div](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L22-L24)元素的[width](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L43-L43)、[height](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L44-L44)和[background-color](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L48-L48)，每个子元素都将放置在自己的行上，因为容器默认情况下不是flex。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

但是如果您向[main](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L21-L25)容器添加[display: flex](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L77-L77)，[div](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L22-L24)元素将重新排列以适应同一行，必要时它们会收缩：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

默认情况下，flex容器将是块级元素，因此容器本身将相对于其他元素和容器位于自己的行上。

现在您对flex容器和flex项目有了更多了解，您还应该了解flex属性。这些属性决定flex项目如何在flex容器内排列、调整大小和分布。一些最常用的属性包括[flex-direction](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L102-L102)、[justify-content](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L102-L102)、[align-items](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L102-L102)和[flex-wrap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L102-L102)。

很好。现在让我们谈谈flex模型。该模型定义了flex项目如何在flex容器内排列。每个flex容器都有两个轴：

- 主轴。
- 交叉轴。

这些轴的方向决定了不同属性如何影响flex项目的布局和分布。默认情况下，flex容器的主轴是水平的，交叉轴是垂直的。flex项目沿主轴方向排列。交叉轴垂直于主轴。

[flex-direction](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L102-L102)属性设置主轴的方向。[flex-direction](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L102-L102)的默认值是[row](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L114-L114)，它将所有flex项目放在同一行上，按照浏览器默认语言的方向（从左到右或从右到左）：

```css
flex-direction: row; /* Default */
```

要反转行中的项目，您可以使用[flex-direction: row-reverse](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L117-L117)：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

这将反转flex项目的顺序。如果您想垂直对齐flex项目，您只需要在flex容器中将[flex-direction](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L102-L102)设置为[column](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L156-L156)。这将改变主轴的方向：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

现在[div](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L22-L24)元素将垂直对齐，因为主轴将是垂直的，交叉轴将是水平的。

您还可以使用[flex-direction: column-reverse](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672aa7f7284b235f46f7d4e9.md#L197-L197)垂直反转flex项目的顺序：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS flexbox是一个强大的布局模型，它提供了一种灵活高效的方式来排列容器内的元素。通过理解flex容器、flex项目和各种flex属性的关键概念，您可以创建动态和响应式的网站，这些网站能够适应不同的屏幕尺寸和方向。
