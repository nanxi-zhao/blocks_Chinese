# --interactive--

Flex属性是您可以应用到flex容器上的属性，用于确定子元素的分布。我们将介绍一些最常用的属性：[flex-wrap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L10-L10)、[justify-content](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L10-L10)和[align-items](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L10-L10)。

让我们从[flex-wrap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L10-L10)开始。这个属性决定了flex项目如何在flex容器内换行以适应可用空间。[flex-wrap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L10-L10)可以取三个可能的值：[nowrap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L12-L12)、[wrap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L12-L12)和[wrap-reverse](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L12-L12)。[nowrap](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L12-L12)是默认值——即使flex项目的宽度超过了容器的宽度，它们也不会换行到新行。

在下面的代码中，我们有三个[div](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L21-L23)元素。让我们关注[width](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L29-L29)。用红色边框包围的[main](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L20-L24)容器的[width](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L29-L29)为[200px](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L29-L29)，而它的三个子[div](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L21-L23)元素组合起来的[width](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L35-L35)为[240px](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L35-L35)（每个[80px](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L35-L35)）：

很好。现在您知道如何沿主轴分布flex项目了。但您可能还想沿交叉轴分布它们。请记住，交叉轴垂直于主轴。您可以使用[align-items](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L10-L10)属性来实现这一点。要沿交叉轴居中项目，您只需要向flex容器添加[align-items: center](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-flexbox/672bd658c0e190f674a5e057.md#L395-L395)：

:::interactive_editor

```css
#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```
