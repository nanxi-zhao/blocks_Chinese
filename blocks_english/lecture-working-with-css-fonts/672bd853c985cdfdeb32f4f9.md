# --interactive--

CSS默认不会对文本应用任何阴影。这是一个没有任何阴影的段落示例。但如果您确实需要添加阴影，可以使用[text-shadow](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L29-L29)属性。

```css
text-shadow: /* Values */
```

在CSS中，您可以通过[X](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L16-L16)偏移、[Y](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L16-L16)偏移、模糊半径和颜色的组合来描述阴影。首先，您需要指定[X](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L16-L16)和[Y](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L16-L16)偏移，它们分别代表阴影相对于文本的水平和垂直距离。这些值是必需的。

这是如何在阴影上应用[X](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L16-L16)和[Y](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L16-L16)偏移的示例。我们应用[text-shadow](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L29-L29)属性，[x-offset](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L18-L18)为[3px](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L29-L29)，[y-offset](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L18-L18)为[2px](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L29-L29)：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px;
}
```

:::

在浏览器中，文本和阴影看起来相似，但我们也可以通过在偏移之前或之后指定值来自定义阴影的颜色。让我们设置阴影颜色。这里我们将使用十六进制颜色，但您可以使用任何有效的颜色格式。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

我们在偏移之后写入颜色，但如果您愿意，您也可以在偏移之前写入颜色。这是等效的：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: #00ffc3 3px 2px;
}
```

:::

在浏览器中，阴影将具有特定颜色，因此我们可以很容易地区分它与文本。现在我们可以区分阴影和文本，了解正值和负值如何影响阴影偏移也很重要。[X](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L16-L16)偏移和[Y](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L16-L16)偏移的正值将分别向右和向下移动阴影，而负值将向左和向上移动阴影。这是一个使用负值的示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: -3px -2px #00ffc3;
}
```

:::

现在阴影将相对于文本向左和向上移动。很好。但阴影看起来不太好看，因为它看起来完全像原始文本但放在其后面。为了让它看起来更好，我们需要添加第三个值，即模糊半径。这个值是可选的，但使阴影看起来更平滑和更微妙。模糊半径的默认值为零。值越高，模糊越大，这意味着阴影变得更浅。

在这里，我们将模糊半径设置为[3px](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L99-L99)，我们回到偏移的正值：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

现在它开始看起来像阴影了。它更模糊和微妙，因此我们可以专注于主要文本，而阴影在背景中添加了一些样式。从这里开始，我们可以稍微调整值，直到找到适合我们需求的偏移、颜色和模糊半径的组合。

了解文本可以有多个阴影也是有帮助的。您只需要在同一个[text-shadow](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L29-L29)属性中用逗号分隔写入它们。它们将按层应用，从前到后，第一个阴影在顶部。

这是一个应用了三层阴影的段落示例。注意所有这些阴影都在同一个[text-shadow](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L29-L29)属性中定义，并用逗号分隔：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 
    3px 2px 3px #00ffc3, 
    -3px -2px 3px #0077ff, 
    5px 4px 3px #dee7e5;
}
```

:::

[text-shadow](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-fonts/672bd853c985cdfdeb32f4f9.md#L29-L29)是一个强大的CSS属性，可以让您的文本脱颖而出。它还可以增强Web应用程序的整体设计。通过选择偏移、模糊和颜色，您可以创建各种阴影效果。
