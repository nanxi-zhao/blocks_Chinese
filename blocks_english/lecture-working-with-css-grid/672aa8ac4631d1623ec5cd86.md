# --interactive--

CSS Grid是一个强大的布局系统，允许Web开发人员轻松创建复杂且响应式的网页布局。

想象一下您正在房间里摆放家具——CSS Grid就像在地板上有一个看不见的网格，帮助您将所有东西精确地放置在您想要的位置。

当我们构建网站时，我们经常需要在页面上排列不同的元素。

在CSS Grid之前，这有时很棘手，特别是对于复杂的布局。CSS Grid通过将您的网页划分为行和列来简化这个过程，创建一个网格状结构。

让我们想象您正在处理一个包含多个嵌套项目的容器[div](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/672aa8ac4631d1623ec5cd86.md#L23-L30)，像这样：

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

如果您想以网格格式样式化这些元素，可以将[display](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/672aa8ac4631d1623ec5cd86.md#L53-L53)设置为[grid](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/672aa8ac4631d1623ec5cd86.md#L53-L53)并应用列，像这样：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

在这段代码中，我们告诉浏览器创建一个三列等宽的网格，这就是[1fr 1fr 1fr](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/672aa8ac4631d1623ec5cd86.md#L54-L54)的含义，我们在每个网格项目之间添加了20像素的间距。

现在，您可能想知道："那Flexbox呢？它不也用于布局吗？"

您说得对！Flexbox是另一个CSS布局模型，它也很有用。但有一些关键区别。

Flexbox是一维的，而Grid是二维的。这意味着Flexbox非常适合在单行或单列中排列东西，而Grid在创建具有行和列的布局方面表现出色。

Flexbox是以内容为先的，意味着它根据内容调整布局。而Grid是以布局为先的，允许您创建布局然后将项目放入其中。Grid为您提供更精确的放置控制。您可以确切地告诉项目占据哪一行和哪一列。

这是用于比较的Flexbox示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

这创建了一个flex容器，其中的项目在主轴上均匀分布。

Grid和Flexbox都有各自的优势，通常，最佳布局是两者的结合。您可以使用Grid进行整体页面布局，然后使用Flexbox对齐每个网格区域内的项目。

总之，CSS Grid是一个强大的工具，允许进行精确的二维布局。虽然一开始可能看起来复杂，但通过练习，它将成为创建响应式和复杂Web布局的宝贵工具。
