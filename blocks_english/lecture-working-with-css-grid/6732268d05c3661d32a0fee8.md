# --interactive--

在之前的课程中，我们一直在使用[grid-template-columns](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732268d05c3661d32a0fee8.md#L27-L27)属性，并将其值设置为几个分数单位。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

虽然以下代码完全有效，但有一种更简单的方法来重复轨道列表的部分或全部。

[repeat()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732268d05c3661d32a0fee8.md#L61-L61)函数用于重复列或行的轨道部分或全部。此函数接受重复次数和您希望重复的轨道。

这是使用repeat函数的早期示例的修订版本：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

在浏览器中样式不会改变，但这是为列编写重复值的更简洁方式。

[repeat()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732268d05c3661d32a0fee8.md#L61-L61)函数将接受您可以用于行或列的任何有效模式。

这是使用[repeat()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732268d05c3661d32a0fee8.md#L61-L61)函数的示例，将第一列和第三列设置为20像素，第二列和第四列设置为一个分数单位。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

有时，您可能选择写出每个单独的值而不是使用[repeat()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-css-grid/6732268d05c3661d32a0fee8.md#L61-L61)函数。但在某些时候，当您想要为轨道列表重复特定模式时，这个函数非常有用。
