# --description--

文件管理应用程序是一种在计算机上轻松存储、组织和编辑文件的方式。无论您使用的是 Mac 还是 Windows 计算机，都有内置的应用程序可以用来组织您的文件。

如果您使用的是 Windows 计算机，默认的文件管理器是文件资源管理器。这用于浏览、搜索和管理文件和文件夹，以及执行复制、移动和删除等文件操作。

要找到文件资源管理器，您可以转到开始菜单或按键盘上的 Windows 徽标键。

要固定文件夹，您可以右键单击鼠标并选择"固定"。要移动文件或文件夹，您首先需要选择它，然后选择"剪切"，然后导航到新位置进行粘贴。

如果您使用的是 Mac 计算机，默认的文件管理器是 Finder。这提供了对本地和远程文件的访问，支持文件预览，并提供了各种组织功能，如标签和智能文件夹。

要在 Mac 上访问 Finder，请转到底部的 Dock 并单击 Finder 图标。

您会注意到左侧有一个侧边栏，您最喜欢的文件夹位于其中。

组织文件的一种方法是使用标签。要为文件或文件夹添加标签，您可以在 Finder 中选择该项目，然后右键单击鼠标。您将在下拉菜单中看到"标签..."选项，您可以应用自定义彩色标签。

要搜索已标记的项目，您可以滚动到侧边栏的底部并单击相应的彩色标签，或者您可以在搜索字段中按颜色名称搜索标签。

组织文件的另一种方法是使用智能文件夹。智能文件夹是根据您设置的标准（如文件类型、日期或关键字）自动组织的文件集合。

无论您是在 Windows 上使用文件资源管理器还是在 Mac 上使用 Finder，这些应用程序都提供了灵活性来管理您的文件，具有标签、搜索和智能文件夹等功能。
