# --description--

让我们学习如何在 Windows 和 macOS 上搜索文件和文件夹。

我们从 Windows 开始。您可以从任务栏或文件资源管理器进行搜索。

要从任务栏搜索，请输入您要查找的文件或文件夹的名称。您也可以搜索关键词。您会看到来自您的 PC 和 OneDrive 的结果。

如果您只想查看文档，可以点击"文档"标签。您还有应用程序和网页结果的标签。

另一种选择是使用搜索文件资源管理器。您会在资源管理器窗口的右上角找到它，那里您可以看到"搜索文档"。

您有三个选项来定义搜索范围：

- 您可以从主页搜索，快速查找来自您的 PC 和云端的文件。

- 您可以从文件夹搜索，查找存储在其中的文件。

- 您可以从这台电脑搜索，进行缓慢但深入的搜索。

这就是您在 Windows 上搜索文件和文件夹的方法。现在让我们看看如何在 macOS 上完成此操作。

首先，您可以使用 Spotlight，这是一个帮助您在 Mac 上查找文件的工具。要打开 Spotlight，请转到右上角的菜单栏并找到放大镜图标。如果您在菜单栏上没有看到这个图标，您可以在控制中心设置中添加它。

转到您的设置，按"控制中心"筛选，点击此选项，然后向下滚动直到找到"Spotlight"选项。您也可以使用键盘快捷键打开 Spotlight：`Command + Spacebar`。

当您打开 Spotlight 时，您会看到 Spotlight 搜索，您可以通过输入文件或文件夹的名称来搜索任何文件或文件夹。当您开始输入时，您会看到按类别分组的结果，包括建议、文件夹、演示文稿、照片、文档和相关搜索。

您也可以通过点击"显示更多"来展开特定类别。

另一种搜索您最近打开的文件的方法是通过点击 Dock 中的 Finder 图标打开 Finder。然后，在 Finder 侧边栏中转到"最近"。在那里，您会找到所有您最近查看过的文件。

这就是您在 macOS 上搜索文件和文件夹的方法。通过这些搜索工具，您可以优化工作流程并快速找到所需的文件和文件夹。
