# --interactive--

`reduce` 方法是 JavaScript 中的一个函数，允许你处理数组并将其压缩为单个值。这个单个值可以是数字、字符串、对象，甚至是另一个数组。

它被称为 `reduce`，因为它将数组减少为单个输出。虽然一开始可能看起来很复杂，但理解 `reduce` 可以在许多情况下大大简化你的代码。

本质上，`reduce` 通过按顺序将函数应用于数组中的每个元素来工作，将每次计算的结果传递给下一次。这个函数通常被称为 reducer 函数。

reducer 函数接受两个主要参数：累加器和当前值。累加器是存储操作运行结果的地方，当前值是正在处理的数组元素。

让我们看一个示例来说明 `reduce` 是如何工作的：

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce(
  (accumulator, currentValue) => accumulator + currentValue,
  0
);

console.log(sum); // 15
```

:::

在这个示例中，我们使用 `reduce` 来获取数组中所有数字的总和。

reducer 函数接受累加器（从 `reduce` 的第二个参数指定的 `0` 开始）并将每个数字加到它上面。

每次加法的结果成为下一次迭代的新累加器。

`reduce` 方法还可以接受初始值作为其第二个参数。这是累加器开始的值。在上面的示例中，我们将其设置为 `0`。

如果你不提供初始值，`reduce` 将使用数组的第一个元素作为初始累加器，并从第二个元素开始处理。

`reduce` 的伟大之处之一是它的灵活性。因为你定义了 reducer 函数，所以你可以完全控制如何处理数组以及想要产生什么样的结果。这使得 `reduce` 极其强大，但也可能使其在一开始难以理解。

通过练习，你将掌握使用 `reduce` 方法。
