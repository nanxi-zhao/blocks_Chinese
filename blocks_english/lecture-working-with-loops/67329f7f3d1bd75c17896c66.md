# --interactive--

编程中的循环用于重复执行代码块多次。

循环的一个例子是当你设计一个需要打印出项目列表的程序时。你可以使用循环来打印列表中的每个项目。

另一个例子是当你设计游戏并想让角色在屏幕上移动时。你可以使用循环每次循环运行时将角色移动一定像素。

在 JavaScript 中，有几种类型的循环可以使用。在本课中，我们将介绍 `for` 循环。以下是 `for` 循环的基本语法：

```js
for (初始化; 条件; 增量或减量) {
  // 要执行的代码块
}
```

初始化语句在循环开始前执行。它通常用于初始化计数器变量。计数器变量是用于跟踪循环运行次数的变量。

条件语句在循环的每次迭代前进行评估。迭代是循环的单次遍历。

如果条件为真，循环内的代码块将被执行。如果条件为假，循环停止，你继续执行下一个代码块。

循环的最后一部分是增量/减量语句。该语句在循环的每次迭代后执行。它通常用于增加或减少计数器变量。

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

在上面示例的第一部分中，我们初始化计数器变量 `i` 为 `0`。在 `for` 循环中使用 `i` 作为计数器变量是常见惯例。

下一部分是检查条件。在这种情况下，条件是检查 `i` 是否小于 `5`。由于 `i` 是 `0`，条件为真，循环内的代码块被执行。

循环内的代码块是将 `i` 的值记录到控制台。`i` 的值是 `0`，所以控制台将显示 `0` 的值。

然后执行增量语句。在这种情况下，我们将 `i` 增加 `1`。所以 `i` 现在是 `1`。

然后我们再次检查条件，即检查 `i` 是否小于 `5`。由于 `i` 现在是 `1`，条件仍然为真，循环内的代码块再次执行。

我们继续重复这个过程，直到条件为假。在这种情况下，当 `i` 是 `5` 时，条件为假，循环停止。

当你使用循环时，应该小心不要创建总是为真的条件。如果你这样做，循环将永远运行，你的程序将崩溃。这被称为无限循环。

可以创建嵌套的 `for` 循环。嵌套循环是将一个循环放在另一个循环内部。我们稍后会看到你可能想要这样做的例子。

当你需要重复执行代码块一定次数时，循环在编程中是有益的。尽管使用 `for` 循环一开始可能有些棘手，但通过练习你会掌握它。
