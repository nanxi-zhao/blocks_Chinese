# --interactive--

JavaScript 提供了对数字执行基本算术运算的工具，如加法、减法、乘法和除法。JavaScript 还包括用于更复杂算术运算的运算符，如取余和指数运算。

所有这些工具都称为算术运算符。让我们详细看看这些运算符，如何使用它们以及如何组合它们。

加法运算符是加号（`+`）。加法运算符允许你找到两个或多个数字的总和。在加法运算中，数字的顺序并不重要：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

减法运算符是减号（`-`）。它允许你找到两个数字之间的差值。当你想从另一个数字中减去一个数字时，使用减号，通常是从一个较大的数字中减去一个较小的数字：

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

如果较小的数字在前面，你会得到一个负数结果：

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

你也可以将数字赋值给变量，然后使用变量名进行减法运算：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

在 JavaScript 中，乘法运算符用星号（`*`）表示，用于找到两个或多个数字的乘积。你相乘的数字的顺序并不重要：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

在 JavaScript 中，除法运算符是斜杠（`/`），这与传统数学中使用的除法符号（`÷`）不同。你使用除法运算符执行除法运算。在这种情况下，你相除的数字的顺序很重要：

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

需要注意的是，如果你试图除以零，JavaScript 将返回 `Infinity`：

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

确保避免这类计算，这样你就不会在代码中得到意外的结果。

取余运算符用百分号（`%`）表示，返回除法的余数。数学中的余数是在执行除法后剩余的值：

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

指数运算符用双星号（`**`）表示，将一个数字提升到另一个数字的幂：

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

可以在单个运算或表达式中混合使用运算符：

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

当你在单个表达式中混合不同的运算符时，JavaScript 引擎遵循一个称为运算符优先级的系统来确定运算的顺序。运算符优先级决定了表达式中运算的执行顺序。你将在未来的课程中了解更多关于运算符优先级的知识。
