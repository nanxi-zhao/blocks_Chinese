# --interactive--

在设计页面时，您将使用不同的属性，如宽度、高度、内边距、外边距等。当您定义这些属性时，需要指定要使用的长度单位。

您可以使用两种类型的单位来定义这些属性：相对单位和绝对单位。在本课程中，我们将只关注绝对单位。

绝对长度单位是固定长度，不相对于任何其他东西。相对意味着长度相对于其他东西，如屏幕大小或父元素的大小。我们将在接下来的几节课中更多地讨论相对单位。

最常见的绝对单位是像素（`px`）。像素是CSS中的固定大小单位，提供对尺寸的精确控制。这意味着1px始终等于1/96英寸。

需要注意的是，虽然为了CSS布局的目的，1px被标准化为1/96英寸，但像素的实际物理大小可能因显示设备而异。

让我们看一个使用像素设置盒子宽度和高度的例子：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
}
```

:::

在这个例子中，我们有一个红色盒子，宽度和高度设置为`100px`。无论屏幕大小如何变化，这个盒子的尺寸都将保持不变。

那么，什么时候应该使用像素这样的绝对单位呢？这取决于具体情况。但通常在需要精确控制元素尺寸、间距和布局时会使用像素。有时您可能会在边距、内边距和边框中使用像素。

下面是使用像素设置两个盒子边距的例子：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="box"></div>
<div class="box"></div>
```

```css
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 10px;
}
```

:::

请记住，外边距是盒子外部的空间。所以，在这个例子中，盒子的四周都会有`10px`的外边距。

其他类型的绝对单位包括以下几种：

- `in`（英寸）单位，等于96px
- `cm`（厘米）单位，等于25.2/64英寸
- `mm`（毫米）单位，等于1/10厘米
- `q`（四分之一毫米）单位，等于1/40厘米
- `pc`（派卡）单位，等于1/6英寸
- `pt`（点）单位，等于1/72英寸

这些单位大多数用于打印而不是屏幕显示。

虽然您将使用的最常见单位是像素，但了解其他绝对单位的存在是很重要的。
