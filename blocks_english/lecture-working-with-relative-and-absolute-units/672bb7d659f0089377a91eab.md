# --interactive--

CSS中的百分比是相对单位，允许您将大小、尺寸和其他属性定义为父元素的比例。当您使用百分比值时，您实际上是在说："使这个元素占其容器的X%。"

例如，如果您在元素上设置`width: 50%`，它将占据其父容器宽度的一半。这使得百分比在创建适应不同屏幕尺寸的响应式设计时非常有用。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

百分比非常适合创建适应各种屏幕尺寸的流体布局。例如，将容器设置为`width: 80%`可确保它占据其父元素宽度的80%，无论设备如何。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

使用百分比处理灵活的图像是另一种常见做法。通过在图像上应用`max-width: 100%`，您可以允许它们在较小的屏幕上缩小，同时保持宽高比。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

虽然不太常见，但百分比也可以用于字体大小以创建可扩展的排版。例如，`font-size: 120%`会使文本比其父元素的字体大小大20%。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

百分比在垂直居中时特别方便。以下是您如何使用百分比和`transform`属性将元素垂直居中的示例。

此示例将元素定位在其容器顶部50%的位置，然后使用`transform`将其向上移动自身高度的一半，从而有效地实现垂直居中。

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

您将在未来的课程中更详细地学习绝对定位和变换属性的工作原理。

请记住，百分比总是相对于某些东西的。对于像`width`这样的水平属性，它们相对于父元素的宽度。对于像`height`这样的垂直属性，它们通常相对于父元素的高度（如果已指定）。

但是，在嵌套具有基于百分比尺寸的元素时要小心，因为这可能导致意外结果。还要记住，如果父元素没有定义高度，基于百分比的高度可能会很棘手。

总之，CSS中的百分比是创建灵活、响应式设计的强大工具。当您希望元素按比例缩放到其容器或需要适应不同屏幕尺寸的布局时，请使用它们。
