# --interactive--

在使用 JavaScript 时，你可能会遇到需要将字符串重复指定次数的情况。

无论是生成重复的模式还是简单地复制文本，[repeat()]方法都提供了一种简单有效的方式来实现这一目标。

JavaScript 中的 [repeat()]方法是一个内置函数，允许你将字符串重复指定的次数。以下是基本语法：

```js
string.repeat(count);
```

`string` 是你想要重复的字符串，`count` 是你希望字符串重复的次数。这是一个示例：

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

在这种情况下，字符串 `Hello!` 重复了三次，结果是 `Hello!Hello!Hello!`。

虽然 [repeat()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/underscore/modules/repeat.js#L7-L18) 方法很有用，但需要注意一些例外情况和限制。

`count` 参数必须是非负数。如果你传入一个负数，JavaScript 将抛出一个 `RangeError`。

```js
let word = "Test";
console.log(word.repeat(-1));  // 抛出 RangeError: Invalid count value
```

`count` 必须是一个有限的数字。如果你尝试将字符串重复无限次或使用 `Infinity` 作为计数，也会得到一个 `RangeError`。

在 JavaScript 中，`Infinity` 是一个特殊值，表示无限数量。它用于表示大于任何有限数字的数值。

```js
let word = "Test";
console.log(word.repeat(Infinity));  // 抛出 RangeError: Invalid count value
```

如果计数不是整数（比如像 `2.5` 这样的小数），[repeat()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/underscore/modules/repeat.js#L7-L18) 方法会将其向下舍入到最接近的整数。

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

如果你传入 `0` 作为计数，[repeat()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/underscore/modules/repeat.js#L7-L18) 方法将返回一个空字符串。

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

[repeat()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/underscore/modules/repeat.js#L7-L18) 方法可以简化涉及字符串复制的任务，使你的代码更加简洁易读。

无论是生成重复的文本模式还是用字符填充空间，[repeat()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/underscore/modules/repeat.js#L7-L18) 都可以帮你避免编写循环或更复杂的代码。
