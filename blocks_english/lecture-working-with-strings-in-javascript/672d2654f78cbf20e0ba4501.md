# --interactive--

在 JavaScript 中，字符串被视为字符序列，可以使用方括号表示法访问字符串中的每个字符。这允许你根据字符的位置（称为索引）从字符串中检索特定字符。

索引是字符串中字符的位置，它是从零开始的。这意味着字符串的第一个字符的索引是 `0`，第二个字符的索引是 `1`，以此类推。

例如，在字符串 `hello` 中，字符 `h` 的索引是 `0`，`e` 的索引是 `1`，`l` 的索引是 `2`，以此类推。

方括号表示法使用方括号（`[]`）和你想要访问的字符的索引。让我们看一个示例：

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[1]); // "e"
```

:::

在这个示例中，我们可以访问索引 `1` 处的字符，即 `e`。

要获取字符串的最后一个字符，可以使用字符串的长度减去一。字符串的 `length` 属性告诉你它包含多少个字符，所以要访问最后一个字符，你需要从长度中减去一：

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[greeting.length - 1]); // "o"
```

:::

在这种情况下，`hello` 的 `length` 是 `5`，最后一个字符（`o`）的索引是 `4`，即 `5 - 1`。

如果你想获取多个字符，可以这样使用方括号表示法：

:::interactive_editor

```js
let greeting = "hello";
let firstTwo = greeting[0] + greeting[1]; // "he"
console.log(firstTwo);
```

:::

在这个示例中，我们使用方括号表示法连接第一个和第二个字符，形成字符串 `he`。

当你需要访问字符串中的特定字符时，方括号表示法非常有用，例如从姓名中提取首字母或检查特定字母以进行验证。
