# --interactive--

当使用 JavaScript 处理字符串时，有时你需要包含 JavaScript 引擎可能误解的特殊字符。

两个常见任务包括在字符串中创建换行和转义某些字符（如引号）以确保它们正确显示。

在许多编程语言中，包括 JavaScript，你可以使用称为转义序列的特殊字符在字符串中创建换行。最常见的换行转义序列是 `\n`。

例如，如果你想将字符串分成多行，你会在想要新行开始的地方使用 `\n`：

:::interactive_editor

```js
let poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

`\n` 转义序列告诉 JavaScript 在该点插入换行符，这会导致字符串显示在多行上。

处理字符串时另一个重要概念是转义字符。有时，你需要在字符串中包含 JavaScript 通常用于其他目的的字符，例如引号。

如果你只是在字符串中使用引号而不转义它们，可能会导致错误，因为 JavaScript 会认为你试图结束字符串。

例如，这会导致错误：

```js
let statement = "She said, "Hello!"";
```

JavaScript 会感到困惑，因为它认为字符串在单词 `"said,"` 之后结束，但你希望 `"Hello!"` 周围的引号成为字符串的一部分。

为了解决这个问题，你可以在引号前放置反斜杠（`\`）来转义内部引号：

:::interactive_editor

```js
let statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

反斜杠告诉 JavaScript 将引号视为字面字符，所以它们在输出中正确显示。

你也可以转义其他特殊字符，例如反斜杠本身（`\\`），或用单引号包围的字符串中的单引号（`\'`）。

这是使用单引号的另一个示例：

:::interactive_editor

```js
let quote = 'It\'s a beautiful day!';
console.log(quote); // It's a beautiful day!
```

:::

通过使用 `\'` 转义单引号，JavaScript 知道将其作为字符串的一部分包含，而不是提前结束字符串。

当你格式化输出或处理字符串中的特殊字符时，转义和创建换行是必不可少的。这些技术帮助你防止错误并确保文本按预期显示。
