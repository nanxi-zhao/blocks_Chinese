# --interactive--

让我们来学习DOM以及为什么它对Web开发如此重要。DOM代表文档对象模型。它是一个编程接口，让我们能够与HTML文档进行交互。

通过DOM，你可以添加、修改或删除网页上的元素。你甚至可以通过让元素监听和响应事件来使你的网站具有交互性。

在DOM中，HTML文档被表示为一个节点树。每个节点代表HTML文档中的一个HTML元素：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

这是代表我们示例基本DOM结构的图表：

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

当然，这可以根据网页HTML标记的结构变得更加详细和复杂。

HTML文档是DOM层次结构中的根节点。它有一个子节点，即[html](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L3-L22)元素。这是HTML文档的根元素，因为所有其他节点都从它派生。

[html](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L3-L22)元素有两个子元素：[head](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L19-L21)和[body](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L6-L21)。

[head](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L19-L21)元素包含有关文档的元数据。它提供有关网页的基本信息。

[body](file:///F:/Code/blocks_Chinese/blocks_english/lab-blog-post-card/src/index.html#L6-L21)元素包含网页的可见内容。

你可以使用JavaScript访问这些元素，在代码中使用它们，甚至动态地修改它们。这就是DOM和Web API的强大之处。

要在JavaScript中访问这些元素，你可以使用[getElementById()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-arrays-variables-and-naming-practices/6733b43e3b72d0180404c53d.md#L41-L47)和[querySelector()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L99-L99)方法。这些方法是Web API，因为它们提供了使用JavaScript与DOM交互的标准化方式。

使用[getElementById()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-arrays-variables-and-naming-practices/6733b43e3b72d0180404c53d.md#L41-L47)，你可以获得一个代表具有指定[id](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-attribute-selectors/6733b1933b72d0180404c4ec.md#L51-L51)的HTML元素的对象。请记住，[id](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-attribute-selectors/6733b1933b72d0180404c4ec.md#L51-L51)s在每个HTML文档中必须是唯一的，所以这个方法只会返回一个[Element](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L62-L62)对象。这里你可以看到一个示例：

```js
const container = document.getElementById("container");
```

这行JavaScript代码获取[id](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-attribute-selectors/6733b1933b72d0180404c4ec.md#L51-L51)值为[container](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L70-L73)的元素，并将该对象赋值给一个JavaScript常量。你必须将[id](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-attribute-selectors/6733b1933b72d0180404c4ec.md#L51-L51)作为参数用引号括起来传递。如果你将这个对象记录到控制台，你会在输出中看到它：

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

[querySelector()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L99-L99)比[getElementById()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-arrays-variables-and-naming-practices/6733b43e3b72d0180404c53d.md#L41-L47)更广泛。使用[querySelector()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L99-L99)，你可以获得HTML文档中与作为参数传递的CSS选择器匹配的第一个元素。

在这个示例中，你将获得与[section](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L91-L94)选择器匹配的元素：

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

如果你想使用[querySelector()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L99-L99)通过类名选择元素，你需要在类名前加上点（`.`）。这里是一个示例：

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

你还有其他方法可以匹配多个元素，如[getElementsByClassName()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L186-L186)和[querySelectorAll()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/67336894ae148431a870694d.md#L194-L194)。你将在未来的课程中学到更多相关内容。

通过理解如何使用Web API高效地操作DOM，你可以创建强大且具有交互性的Web应用程序。
