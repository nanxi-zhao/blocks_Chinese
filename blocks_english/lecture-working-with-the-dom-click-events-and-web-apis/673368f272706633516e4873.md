# --interactive--

你已经习惯了在HTML和CSS项目中使用属性。但你知道吗，你可以在JavaScript文件中向HTML元素添加属性？

在本课中，你将学习如何使用[setAttribute()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)方法向HTML元素添加属性。

这是基本语法：

```js
setAttribute(attribute, value);
```

让我们看几个示例，更好地理解如何使用[setAttribute()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)方法。

在第一个示例中，我们在HTML中有一个[p](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L25-L25)元素：

```html
<p id="para">I am a paragraph</p>
```

要添加[class](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)属性，我们首先需要使用[getElementById()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-arrays-variables-and-naming-practices/6733b43e3b72d0180404c53d.md#L41-L47)方法访问那个[p](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L25-L25)元素。然后我们可以使用[setAttribute()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)方法添加[class](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)属性并将值设置为[my-class](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)。

我们使用[outerHTML](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L43-L43)来记录整个HTML元素包括其属性。

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

如果你在HTML元素上有现有属性，你可以使用[setAttribute()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)方法更新其值。

在这个示例中，我们有一个[class](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)属性设置为[my-class](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L67-L67)的[div](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L53-L53)元素：

```html
<div class="my-class"></div>
```

要将[class](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)属性值更新为example，我们可以再次使用[setAttribute()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)方法：

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

那么，什么时候使用[setAttribute()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L41-L41)方法的一些现实示例是什么呢？

如果你正在构建一个动态图像库，当用户点击缩略图时，你可能需要更新图像元素的[src](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L76-L76)属性。

另一个示例是，如果你在处理表单验证并且需要向[input](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L78-L78)元素添加某些属性如[required](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L78-L78)或[minlength](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368f272706633516e4873.md#L78-L78)。
