# --description--

[Event](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L10-L10)对象是一个在用户以某种方式与你的网页交互时触发的有效载荷。这些交互可以是任何事情，从点击按钮或聚焦输入框到摇晃他们的移动设备。

像所有JavaScript对象一样，[Event](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L10-L10)对象有许多可能有用的属性。可用的属性取决于触发这个有效载荷的事件。

所有[Event](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L10-L10)对象都有[type](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L14-L14)属性。这个属性揭示了触发有效载荷的事件类型，例如["keydown"](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L14-L14)或["click"](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L14-L14)。这些值将对应于你可能传递给[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L14-L14)的相同值，在那里你可以捕获和利用[Event](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L10-L10)对象。

[Event](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L10-L10)对象总是有[target](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L16-L16)属性。[target](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L16-L16)属性是触发事件的对象的引用。最常见的是，这将是某种[HTMLElement](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368ccf52205329b729378.md#L163-L163)对象，或[Document](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368e7bd043f331919514d.md#L92-L92)或[Window](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368e7bd043f331919514d.md#L54-L54)对象。但它也可能是更具体的东西，比如[AudioContext](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L16-L16)。

事件也有方法，这些方法是作为对象上的属性暴露的函数。一个常用的方法是[preventDefault()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L18-L18)，当调用时阻止事件的默认行为。

例如，如果你想自己处理表单提交，你可能会调用[preventDefault()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L18-L18)来阻止浏览器尝试将表单数据作为[POST](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L20-L20)请求提交。你将在未来的课程中学习更多关于[POST](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L20-L20)请求的内容。

你也可能会遇到[stopPropagation()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L22-L22)方法。这个方法阻止事件冒泡或传播到父元素。你将在未来的课程中学习更多关于这意味着什么的内容。

还有大量特定于[Event](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L10-L10)对象某些实现的属性。例如，[FetchEvent](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L24-L24)将有一个request属性来包含触发事件的请求。

如果你不确定有哪些属性可用，你可以记录相关的[Event](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673368fbe12a2b337645053d.md#L10-L10)对象，甚至可以查看文档。
