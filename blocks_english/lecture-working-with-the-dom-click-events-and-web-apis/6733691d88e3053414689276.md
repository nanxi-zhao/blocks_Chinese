# --interactive--

在之前的课程中，你学习了如何使用[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)方法处理事件。但在JavaScript中有另一种不推荐的方式处理事件。

内联事件处理器是HTML元素上的特殊属性，用于在事件发生时执行JavaScript代码。

以下是带有内联点击事件处理器的[button](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369101e5c4a33db0e8a02.md#L39-L39)元素示例：

:::interactive_editor

```html
<button onclick="alert('Hello World!')">Show alert</button>
```

:::

当用户点击按钮时，[alert](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733691d88e3053414689276.md#L19-L19)函数被调用，并显示一条包含[Hello World!](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733691d88e3053414689276.md#L19-L19)消息的警告对话框。

使用内联事件处理器的另一种方式是调用在HTML文档的[script](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733691d88e3053414689276.md#L33-L37)标签中定义的函数。

以下是定义名为[changeBgColor](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733691d88e3053414689276.md#L34-L36)的函数并从内联点击事件处理器调用它的示例：

:::interactive_editor

```html
<script>
  function changeBgColor() {
    document.body.style.backgroundColor = "lightblue";
  }
</script>

<button onclick="changeBgColor()">Change background color</button>
```

:::

当用户点击按钮时，[changeBgColor](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/6733691d88e3053414689276.md#L34-L36)函数被调用，页面的背景颜色更改为浅蓝色。虽然可以这样使用内联事件处理器，但这不被认为是最佳实践。

原因之一，内联事件处理器只能用于向元素附加一个事件监听器。如果你想向同一元素附加多个事件监听器，你需要使用[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)。另一个原因是内联事件处理器将HTML和JavaScript代码混合在一起，这会使你的代码更难阅读和维护。最好通过使用[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)向元素附加事件监听器来将HTML代码和JavaScript代码分开。

内联事件处理器在现代JavaScript中不推荐使用。因此，在JavaScript中处理事件时，最好坚持使用[addEventListener()](file:///F:/Code/blocks_Chinese/blocks_english/lecture-working-with-the-dom-click-events-and-web-apis/673369067f824d33a90a0534.md#L68-L70)方法。
