## 终端、Shell和命令行基础

- **命令行**：用户输入命令的文本界面。
- **终端**：提供访问命令行的应用程序。
- **终端模拟器**：为终端添加额外功能。
- **Shell**：解释输入到终端中的命令（例如，Bash）。
- **PowerShell / 命令提示符 / Microsoft终端**：在Windows上访问命令行的选项。
- **终端（macOS）**：macOS上的内置选项，还有第三方替代品如iTerm或Ghostty。
- **终端（Linux）**：选项因发行版而异，有许多第三方模拟器如kitty。
- **术语**：虽然"终端"、"shell"和"命令行"经常互换使用，但它们有特定的含义。


## 命令行快捷方式

- **上下箭头**：在历史记录中循环浏览上一个/下一个命令。
- **Tab键**：自动补全命令。
- **`Control+L`**（Linux/macOS）或输入[cls](file:///D:/Program%20Files/Git/usr/bin/cls.exe)（Windows）：清除终端屏幕。
- **`Control+C`**：中断正在运行的命令（在PowerShell中如果选择了文本也用于复制）。
- **`Control+Z`**（仅限Linux/macOS）：将任务挂起到后台；使用[fg](file:///D:/Program%20Files/Git/usr/bin/fg.exe)恢复它。
- **`!!`**：立即重新运行上次执行的命令。

## Bash基础

- **Bash**（Bourne Again Shell）：广泛使用的类Unix shell。  
  关键命令：

  - **[pwd](file:///D:/Program%20Files/Git/usr/bin/pwd.exe)**：显示当前目录。
  - **[cd](file:///D:/Program%20Files/Git/usr/bin/cd.exe)**：更改目录。
    - **`..`** 指向父目录（上一级）。
    - **`.`** 指向当前目录。
  - **[ls](file:///D:/Program%20Files/Git/usr/bin/ls.exe)**：列出文件和文件夹。
    - **`-a`**：显示所有文件，包括隐藏文件。
    - **`-l`**：显示文件的详细信息。
  - **[less](file:///D:/Program%20Files/Git/usr/bin/less.exe)**：一次一页地查看文件内容，包含导航选项，包括向后滚动和搜索。
  - **[more](file:///D:/Program%20Files/Git/usr/bin/more.exe)**：一次显示一屏文件内容，向后滚动功能有限，导航功能基本。
  - **[cat](file:///D:/Program%20Files/Git/usr/bin/cat.exe)**：一次性显示整个文件内容，无需滚动或导航，适用于较小的文件。
  - **[mkdir](file:///D:/Program%20Files/Git/usr/bin/mkdir.exe)**：创建新目录。
  - **[rmdir](file:///D:/Program%20Files/Git/usr/bin/rmdir.exe)**：删除空目录。
  - **[touch](file:///D:/Program%20Files/Git/usr/bin/touch.exe)**：创建新文件。
  - **[mv](file:///D:/Program%20Files/Git/usr/bin/mv.exe)**：移动或重命名文件。
    - 重命名：`mv oldname.txt newname.txt`
    - 移动：`mv filename.txt /path/to/target/`
  - **[cp](file:///D:/Program%20Files/Git/usr/bin/cp.exe)**：复制文件。
    - **`-r`**：递归复制目录及其内容。
  - **[rm](file:///D:/Program%20Files/Git/usr/bin/rm.exe)**：删除文件。
    - **`-r`**：递归删除目录及其内容。
  - **[echo](file:///D:/Program%20Files/Git/usr/bin/echo.exe)**：显示一行文本或变量的值。
    - 使用 `>` 覆盖文件中的现有内容。（例如，`echo "text" > file.txt`）
    - 使用 `>>` 将输出追加到文件中**而不覆盖现有内容**（例如，`echo "text" >> file.txt`）。
  - **[exit](file:///D:/Program%20Files/Git/usr/bin/exit.exe)**：退出终端会话。
  - **[clear](file:///D:/Program%20Files/Git/usr/bin/clear.exe)**：清除终端屏幕。
  - **[find](file:///D:/Program%20Files/Git/usr/bin/find.exe)**：搜索文件和目录。
    - **`-name`**：按名称模式搜索文件（例如，`find . -name "*.txt"`）。
  - 使用 **[man](file:///D:/Program%20Files/Git/usr/bin/man.exe)** 后跟命令（例如，`man ls`）访问详细的手册/帮助页面。

## 命令选项和标志

- **选项**或**标志**：修改命令的行为，通常以连字符为前缀：
  - **长格式（两个连字符）**：
    - 示例：`--help`, `--version`
    - 值使用等号附加，例如，`--width=50`。
  - **短格式（一个连字符）**：
    - 示例：`-a`, `-l`
    - 值使用空格传递，例如，`-w 50`。
    - 多个短选项可以链接在一起，例如，`ls -alh`。
- **`--help`**：你总是可以使用带有此标志的命令来了解任何命令的可用选项。
