## 色彩理论

- **色彩理论定义**：这是研究颜色如何相互作用以及它们如何影响我们感知的学科。它涵盖了色彩关系、色彩和谐以及色彩的心理影响。
- **原色**：这些颜色即黄色、蓝色和红色，是所有其他颜色的基本色调。
- **间色**：这些颜色是通过混合等量的两种原色产生的。绿色、橙色和紫色是间色的例子。
- **复色**：这些颜色是通过将原色与相邻的间色结合产生的。黄绿色、蓝绿色和蓝紫色是复色的例子。
- **暖色**：这些颜色包括红色、橙色和黄色，能唤起舒适、温暖和舒适的感觉。
- **冷色**：这些颜色包括蓝色、绿色和紫色，能唤起平静、宁静和专业的感觉。
- **色轮**：色轮是一个圆形图，显示颜色如何相互关联。它是设计师的重要工具，因为它帮助他们选择色彩组合。
- **类似色搭配**：这些色彩搭配创造出连贯和舒缓的体验。它们具有在色轮上相邻的类似颜色。
- **互补色搭配**：这些色彩搭配创造出高对比度和视觉冲击力。它们的颜色在色轮上相对位于两端。
- **三色搭配**：这种色彩搭配具有鲜艳的颜色。它们由大致等距的颜色组成。如果将它们连接起来，会在色轮上形成一个等边三角形。
- **单色搭配**：对于这种色彩搭配，所有颜色都是通过调整同一基础颜色的亮度、暗度和饱和度得出的。这能唤起统一与和谐的感觉，同时仍能创造对比。

## 在CSS中使用颜色的不同方法

- **命名颜色**：这些颜色是浏览器识别的预定义颜色名称。示例包括[blue](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[darkred](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[lightgreen](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)。
- **[rgb()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)函数**：RGB代表红色、绿色和蓝色——光的三原色。这三种颜色以不同强度组合，创造出广泛的色彩。[rgb()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)函数允许你使用RGB色彩模型定义颜色。

```css
p {
  color: rgb(255, 0, 0);
}
```

- **[rgba()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)函数**：此函数添加第四个值alpha，用于控制颜色的透明度。如果未提供，alpha值默认为[1](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)。

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

- **[hsl()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)函数**：HSL代表色相、饱和度和亮度——定义颜色的三个关键组成部分。

```css
p {
  color: hsl(120, 100%, 50%);
}
```

- **[hsla()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)函数**：此函数添加第四个值alpha，用于控制颜色的不透明度。

```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

- **十六进制**：十六进制代码（简称hex代码）是一个六字符字符串，用于表示RGB色彩模型中的颜色。"hex"指的是十六进制数系统，使用数字0到9和字母A到F。

```css
h1 {
  color: #FF5733; /* 一种红橙色 */
}

p {
  background-color: #4CAF50; /* 一种绿色 */
}
```

## 线性渐变和径向渐变

- **线性渐变**：这些渐变沿着直线创建颜色之间的渐进混合。你可以使用[to top](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[to right](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[to bottom right](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)等关键字或[45deg](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)等角度来控制这条线的方向。你可以使用任何有效的CSS颜色和任意多的颜色停靠点。

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

- **径向渐变**：这些渐变创建从中心点向外辐射的圆形或椭圆形渐变。

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
