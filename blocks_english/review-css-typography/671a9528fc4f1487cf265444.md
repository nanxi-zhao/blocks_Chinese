## 排版简介

- **定义**：排版是选择合适的字体和格式的艺术，使文本在视觉上吸引人且易于阅读。"字体"指的是单个字符的设计和排列方式。
- **字体系列定义**：字体系列是一组字符、数字和符号的整体设计和风格。它就像字体家族的蓝图。
- **字体定义**：字体是具有特定特征的字体系列的具体变体，如大小、粗细、样式和宽度。
- **衬线字体系列**：这种字体系列具有古典风格，在字符末端有小线条。衬线字体系列通常用于印刷材料，如书籍。
- **无衬线字体系列**：这种字体系列具有更现代的外观，在字符末端没有小线条。无衬线字体系列通常用于数字设计，因为它们在屏幕上易于阅读。一些示例包括Helvetica、Arial和Roboto。
- **字体粗细**：这是字符的粗细程度，包括细体、常规、粗体和黑体。
- **字体样式**：这是字符的倾斜和方向，如斜体和倾斜体。
- **字体宽度**：这是字符占据的水平空间，如紧缩体和扩展体。
- **基线**：这是大多数字符所在的假想线。
- **大写字母高度**：这是大写字母的高度，从基线测量到顶部。
- **小写字母x高度**：这是小写字母的平均高度，不包括上升部和下降部。
- **上升部**：这些是小写字母中延伸到x高度以上的部分，如字母'h'、'b'、'd'和'f'的顶部。
- **下降部**：这些是小写字母中延伸到基线以下的部分，如'y'、'g'、'p'和'q'的尾部。
- **字距调整**：这是如何调整特定字符对之间的间距以提高其可读性和美观性。例如，减少字母A和V之间的间距。
- **字间距**：这是如何调整文本块中所有字符之间的间距。这本质上是字符之间的距离。它会影响文本的密集程度和开放程度。
- **行距**：这是文本行之间的垂直空间。它从一行的基线测量到下一行的基线。
- **排版最佳实践**：你应该选择清晰简单的字体，使你的设计易于理解。这对于网站的主要文本尤其重要。如果字体易于阅读，用户更可能与你的内容互动。你应该最多使用两到三种字体来创建视觉一致性。使用太多字体会使文本更难阅读，并削弱你品牌的识别度。这也可能通过增加网站的加载时间来影响用户体验。你可以使用字体大小为标题、副标题、段落和其他元素创建视觉层次结构。例如，网页上的主标题应该有较大的字体，其次是较小字体大小的副标题。这将为层次结构中的每个元素提供特定的字体大小，帮助用户通过视觉方式浏览结构。

## CSS [font-family](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性

- **定义**：字体家族是一组共享共同设计的字体。属于同一字体家族的所有字体都基于相同的核心字体，但它们在样式、粗细和宽度方面也有变化。你可以按优先级顺序指定多个字体家族，从高到低，用逗号分隔。这些替代字体将作为后备选项。你应该始终在字体家族列表的末尾包含一个通用字体家族。

```css
font-family: Arial, Lato;
```

- **常见字体家族**：以下是一些在CSS中常用的字体家族：serif、sans-serif、monospace、cursive、fantasy

## 网络安全字体

- **定义**：这些字体是极有可能安装在计算机或设备上的字体子集。当浏览器必须渲染字体时，它会尝试在用户的系统上找到字体文件。但如果找不到字体，它通常会回退到默认的系统字体。

- **网络安全字体**：
  - 无衬线字体通常用于Web开发，因为它们在字符末端没有小的"脚"或线条，所以在屏幕上易于阅读。一些网络安全的无衬线字体示例包括：Arial、Verdana和Trebuchet MS。
  - 相比之下，衬线字体在字符末端有小的"脚"，所以它们通常用于传统印刷。网络安全的衬线字体包括：Times New Roman和Georgia。

## At规则和[@font-face](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11) At规则

- **定义**：At规则是向浏览器提供指令的语句。你可以使用它们来定义样式表的各个方面，如媒体查询、关键帧、字体面等。
- **[@font-face](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这允许你通过指定字体文件、格式和其他重要属性（如粗细和样式）来定义自定义字体。为了使[@font-face](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11) at规则有效，你还需要指定[src](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性。此属性包含对字体资源的引用。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
       url("path/to/font.woff"),
       url("path/to/font.otf");
}
```

- **字体格式**：对于每个字体资源，你也可以指定格式。这是可选的。这是给浏览器的字体格式提示。如果省略格式，资源将被下载，并在下载后检测格式。如果格式无效，资源将不会被下载。可能的字体格式包括[collection](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[embedded-opentype](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[opentype](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[svg](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[truetype](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)、[woff](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)和[woff2](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype"),
       url("path/to/font.woff") format("woff");
}
```

- **[woff](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)和[woff2](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：[woff](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)代表"Web Open Font Format"。这是Mozilla与Type Supply、LettError和其他组织合作开发的广泛使用的字体格式。[woff](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)和[woff2](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)的区别在于用于压缩数据的算法。
- **OpenType**：这是由Microsoft和Adobe开发的可缩放计算机字体格式，允许用户访问字体中的附加功能。它在主要操作系统中广泛使用。
- **[tech()](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)**：这用于指定字体资源的技术。这是可选的。

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype") tech(color-COLRv1),
       url("path/to/font.woff") format("woff");
}
```

## 使用外部字体

- **定义**：外部字体是不直接包含在你的项目文件中的字体文件。它们通常托管在单独的服务器上。要将这些外部字体包含在你的项目中，你可以在CSS中使用[link](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)元素或[@import](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)语句。
- **Google Fonts**：这是Google提供的一项服务，提供字体集合，其中许多字体是专门为Web开发设计的。

这是使用[link](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)元素的示例：

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap" rel="stylesheet">
```

```css
.roboto-thin {
  font-family: "Roboto", sans-serif;
  font-weight: 100;
  font-style: normal;
}
```

这是使用[@import](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)语句的示例：

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

- **Font Squirrel**：这是另一个流行的资源，你可以用于向项目添加自定义外部字体。

## [text-shadow](file:///E:/1.work/byy/%E5%9C%A8%E7%BA%BF%E8%AF%BE%E7%A8%8B/blocks/node_modules/@types/node/stream/consumers.d.ts#L11-L11)属性

- **定义**：此属性用于对文本应用阴影。你需要指定X和Y偏移，它们分别表示阴影相对于文本的水平和垂直距离。这些值是必需的。X偏移和Y偏移的正值将分别向右和向下移动阴影，而负值将向左和向上移动阴影。

```css
p {
  text-shadow: 3px 2px;
}
```

- **阴影颜色**：你也可以通过在偏移之前或之后指定此值来自定义阴影的颜色。如果未指定颜色，浏览器将自动确定颜色，所以最好显式设置它。

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

- **模糊半径**：模糊半径是可选的，但会使阴影看起来更平滑和更微妙。半径模糊的默认值为零。值越高，模糊越大，这意味着阴影变得更轻。

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

- **应用多个文本阴影**：文本可以有多个阴影。阴影将按层应用，从前到后，第一个阴影在最上面。

```css
p {
  text-shadow: 
    3px 2px 3px #00ffc3, 
    -3px -2px 3px #0077ff, 
    5px 4px 3px #dee7e5;
}
```
