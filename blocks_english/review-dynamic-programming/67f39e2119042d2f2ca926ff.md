## 动态规划简介

- **定义**：动态规划是一种算法技术，通过将复杂问题分解为更简单的子问题并存储结果来避免重复计算，从而解决复杂问题。
- **重叠子问题**：在解决较大问题时，相同的较小问题会多次出现。我们存储这些子问题的解决方案，而不是重复计算它们。
- **最优子结构**：问题的最优解包含其子问题的最优解。这意味着我们可以通过组合较小部分的最佳解决方案来构建最佳解决方案。

## 动态规划解决方案

- **记忆化（自顶向下方法）**：记忆化存储昂贵函数调用的结果，并在相同输入再次出现时返回缓存的结果。

```py
def climb_stairs_memo(n, memo={}):
    """使用记忆化的动态规划"""
    # 检查我们是否已经计算过这个值
    if n in memo:
        return memo[n]  # 返回缓存的结果 - O(1) 查找！

    # 基本情况
    if n <= 2:
        return n

    # 计算一次并存储在 memo 中供将来使用
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **制表法（自底向上方法）**：制表法从底层开始构建解决方案，用子问题的解决方案填充表格。

```py
def climb_stairs_tabulation(n):
    """使用制表法的动态规划"""
    if n <= 2:
        return n

    # 创建数组来存储从 0 到 n 的所有步骤的结果
    dp = [0] * (n + 1)
    dp[1] = 1  # 到达第 1 步有 1 种方式
    dp[2] = 2  # 到达第 2 步有 2 种方式

    # 迭代地构建解决方案
    for i in range(3, n + 1):
        # 到达第 i 步的方式 = 到达 (i-1) 的方式 + 到达 (i-2) 的方式
        dp[i] = dp[i-1] + dp[i-2]

    return dp[n]
```

## 使用动态规划的实际应用

- **路线优化**：GPS 系统使用动态规划算法找到位置之间的最短路径。
- **文本处理**：拼写检查器和自动完成功能经常依赖动态规划来计算单词之间的编辑距离。
- **金融建模**：投资策略和投资组合优化经常使用动态规划技术。
- **资源分配**：背包问题及其变体出现在调度、预算和资源管理中。

## 何时使用动态规划

你应该在以下情况下考虑使用动态规划：

- 问题可以分解为重叠的子问题。
- 问题表现出最优子结构。
- 天真的递归解决方案会涉及重复计算。
- 你需要优化时间复杂度而以空间复杂度为代价。
