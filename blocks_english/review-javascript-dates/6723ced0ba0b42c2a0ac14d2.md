## [date()]对象和常用方法

- **定义**：[date()]对象用于在JavaScript中创建、操作和格式化日期和时间。在下面的例子中，使用[new]关键字创建[Date]对象的新实例，然后将[Date]对象赋值给变量[now]。如果你将[now]的值记录到控制台，你会看到基于运行代码的计算机系统时钟的当前日期和时间。

```js
const now = new Date();
```

- **[Date.now()]方法**：此方法用于获取当前日期和时间。[Date.now()]返回自1970年1月1日00:00:00 UTC以来的毫秒数。这被称为Unix纪元时间。Unix纪元时间是计算机系统中表示日期和时间的常见方式，因为它是一个易于存储和操作的整数。UTC代表协调世界时，这是世界调节时钟和时间的主要时间标准。
- **[getDate()]方法**：此方法用于获取基于当前日期的月份中的某一天。[getDate()]将返回1到31之间的整数值，具体取决于月份中的哪一天。如果日期无效，它将返回[NaN]（非数字）。

```js
const now = new Date();
const date = now.getDate();
console.log(date); // 15
```

- **[getMonth()]方法**：此方法用于获取月份。月份是从零开始的，所以一月是0，二月是1，以此类推。在这个例子中，输出是2，对应三月。如果月份无效，它将返回[NaN]。

```js
const now = new Date();
const month = now.getMonth();
console.log(month); // 2
```

- **[getFullYear()]方法**：此方法用于获取完整的年份。如果年份无效，它将返回[NaN]。

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); // 2024
```

## 格式化日期的不同方法

- **[toISOString()]方法**：此方法用于以扩展的[ISO]（ISO 8601）格式格式化日期。ISO 8601是表示日期和时间的国际标准。格式是[YYYY-MM-DDTHH:mm:ss.sssZ]。

```js
const date = new Date();
console.log(date.toISOString());
```

- **[toLocaleDateString()]方法**：此方法用于根据用户的区域设置格式化日期。

```js
const date = new Date();
console.log(date.toLocaleDateString());  // 11/23/2024
```

[toLocaleDateString()]方法接受两个可选参数：locales和options。

locales参数是表示要使用的区域设置的字符串。例如，你可以传入["en-US"]表示英语（美国）或["fr-FR"]表示法语（法国）。如果你不传入locales参数，则使用默认区域设置。第二个可选参数是options参数。这个参数是一个对象，允许你指定日期字符串的格式。

```js
const date = new Date();
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options)); // Saturday, November 23, 2024
```
