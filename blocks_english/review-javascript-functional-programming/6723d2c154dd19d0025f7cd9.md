## 纯函数与非纯函数

- 纯函数是指对于相同的输入总是产生相同的输出，并且没有任何副作用的函数。它的输出仅依赖于输入，不会修改任何外部状态。
- 非纯函数具有副作用，这些副作用是对函数外部可观察到的程序状态的改变。

## 函数式编程

- 函数式编程是一种软件开发方法，强调使用函数来解决问题，专注于要做什么而不是如何做。
- 函数式编程鼓励使用有助于避免副作用的技术，例如使用不可变数据结构和高阶函数。
- 正确使用时，函数式编程原则会让代码更清晰、更易于维护

## 柯里化

- 柯里化是一种函数式编程技术，它将一个多参数的函数转换为一系列单参数的函数。

下面是一个普通函数与柯里化函数的例子：

```js
// 普通函数

function average(a, b, c) {
  return (a + b + c) / 3;
}

// 柯里化函数

function curriedAverage(a) {
  return function(b) {
    return function(c) {
      return (a + b + c) / 3;
    };
  };
}

// 柯里化函数的使用

const avg = curriedAverage(2)(3)(4);
```

- 当处理接受许多参数的函数时，柯里化可能特别强大。
- 柯里化让你的代码更加灵活且易于重用。
- 你可以使用箭头函数更简洁地创建柯里化函数：

```js
const curriedAverage = a => b => c => (a + b + c) / 3;
```

- 虽然柯里化可以带来更灵活和可重用的代码，但如果过度使用也会让代码更难阅读。
