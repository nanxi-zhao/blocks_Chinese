## JavaScript中的集合

- `Set`是管理数据集合的内置选项。
- 集合确保其中的每个值只出现一次，使其在消除数组中的重复项或处理不同的值集合时非常有用。
- 你可以使用`Set()`构造函数创建一个`Set`：

```js
const set = new Set([1, 2, 3, 4, 5]);
console.log(set); // Set { 1, 2, 3, 4, 5 }
```

- 可以使用以下方法操作集合：

  - `add()`：向`Set`添加新元素。
  - `delete()`：从`Set`中删除元素。
  - `has()`：检查`Set`中是否存在元素。
  - `clear()`：从`Set`中删除所有元素。
  - `keys()`和`values()`：都返回包含`Set`值的`SetIterator`。它们是相同的，因为`keys()`是`values()`的别名。
  - `forEach()`：用于遍历`Set`的值。

## JavaScript中的弱集合

- `WeakSet`是对象的集合，允许你存储弱持有的对象。

## 集合与弱集合

- 与集合不同，`WeakSet`不支持数字或字符串等原始类型。
- `WeakSet`只存储对象，并且对这些对象的引用是"弱"的，这意味着如果对象在代码的其他地方没有被使用，它会自动被删除以释放内存。

## JavaScript中的映射

- `Map`是保存键值对的内置对象，就像对象一样。
- 映射与标准JavaScript对象的不同之处在于它们能够允许任何类型的键，包括对象和函数。
- 当涉及到频繁添加和删除键值对时，`Map`比标准对象提供更好的性能。
- 你可以使用`Map()`构造函数创建一个`Map`：

```js
const map = new Map([
  ['flower', 'rose'],
  ['fruit', 'apple'],
  ['vegetable', 'carrot']
]);
console.log(map); // Map(3) { 'flower' => 'rose', 'fruit' => 'apple', 'vegetable' => 'carrot' }
```

- 可以使用以下方法操作映射：

  - `set()`：向`Map`添加新的键值对。
  - `get()`：从`Map`中检索键的值。
  - `delete()`：从`Map`中删除键值对。
  - `has()`：检查`Map`中是否存在键。
  - `clear()`：从`Map`中删除所有键值对。

注意，映射和集合都有`size`属性，返回其中唯一元素的数量。

## JavaScript中的弱映射

- `WeakMap`是键值对的集合，就像`Map`一样，但对键的引用是弱的。键必须是对象，值可以是你喜欢的任何东西。

## 映射与弱映射

- 弱映射与弱集合类似，它们只存储对象，并且对这些对象的引用是"弱"的。
