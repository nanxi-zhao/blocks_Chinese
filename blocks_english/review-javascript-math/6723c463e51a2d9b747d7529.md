## 使用数字数据类型

- **定义**：JavaScript的`Number`类型包括整数、浮点数、`Infinity`和`NaN`。浮点数是带小数点的数字。正`Infinity`是大于任何其他数字的数字，而`-Infinity`是小于任何其他数字的数字。`NaN`（`Not a Number`）表示无效的数值，如字符串`"Jessica"`。

## 常见算术运算

- **加法运算符**：此运算符（`+`）用于计算两个或多个数字的和。
- **减法运算符**：此运算符（`-`）用于计算两个数字的差。
- **乘法运算符**：此运算符（`*`）用于计算两个或多个数字的积。
- **除法运算符**：此运算符（`/`）用于计算两个数字的商
- **除零运算**：如果你试图除以零，JavaScript将返回`Infinity`。
- **余数运算符**：此运算符（`%`）返回除法的余数。
- **指数运算符**：此运算符（`**`）将一个数字提升到另一个数字的幂。

## 数字和字符串的计算

- **解释**：当你使用`+`运算符与数字和字符串一起使用时，JavaScript会将数字强制转换为字符串并将两个值连接起来。当你使用`-`、`*`或`/`运算符与字符串和数字一起使用时，JavaScript会将字符串强制转换为数字，结果将是数字。对于`null`和`undefined`，JavaScript在数学运算中将`null`视为0，将undefined视为`NaN`。

```js
const result = 5 + '10';

console.log(result); // 510
console.log(typeof result); // string

const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' * 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number

const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number
```

## 运算符优先级

- **定义**：运算符优先级确定表达式中运算的求值顺序。优先级较高的运算符在优先级较低的运算符之前求值。括号内的值将首先求值，乘法/除法的优先级高于加法/减法。如果运算符具有相同的优先级，则JavaScript将使用结合性。

```js
const result = (2 + 3) * 4;

console.log(result); // 20

const result2 = 10 - 2 + 3;

console.log(result2); // 11

const result3 = 2 ** 3 ** 2;

console.log(result3); // 512
```

- **定义**：结合性告诉我们当存在多个相同类型的运算符时，表达式的求值方向。它定义了表达式是从左到右（`左结合`）还是从右到左（`右结合`）求值。例如，指数运算符也是从右到左结合的：

```js
const result4 = 5 ** 4 ** 1; // 625

console.log(result4);
```

## 递增和递减运算符

- **递增运算符**：此运算符用于将值增加一。前缀表示法`++num`首先增加变量的值，然后返回新值。后缀表示法`num++`首先返回变量的当前值，然后增加它。

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6


let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

- **递减运算符**：此运算符用于将值减少一。前缀表示法和后缀表示法的工作方式与递增运算符相同。

```js
let num = 5;

console.log(--num); // 4
console.log(num--); // 4
console.log(num); // 3
```

## 复合赋值运算符

- **加法赋值（`+=`）运算符**：此运算符对值执行加法并将结果赋给变量。
- **减法赋值（`-=`）运算符**：此运算符对值执行减法并将结果赋给变量。
- **乘法赋值（`*=`）运算符**：此运算符对值执行乘法并将结果赋给变量。
- **除法赋值（`/=`）运算符**：此运算符对值执行除法并将结果赋给变量。
- **余数赋值（`%=`）运算符**：此运算符将变量除以指定数字并将余数赋给变量。
- **指数赋值（`**=`）运算符**：此运算符将变量提升到指定数字的幂并将结果重新赋给变量。

## 布尔值和相等性

- **布尔值定义**：布尔值是只能有两个值的数据类型：`true`或`false`。
- **相等性（`==`）运算符**：此运算符在检查值是否相等之前使用类型强制转换。
```js
console.log(5 == '5'); // true
```

- **严格相等性（`===`）运算符**：此运算符不执行类型强制转换，并检查类型和值是否相等。
```js
console.log(5 === '5'); // false
```

- **不等性（`!=`）运算符**：此运算符在检查值是否不相等之前使用类型强制转换。
- **严格不等性（`!==`）运算符**：此运算符不执行类型强制转换，并检查类型和值是否不相等。

## 比较运算符

- **大于（`>`）运算符**：此运算符检查左侧的值是否大于右侧的值。
- **大于等于（`>=`）运算符**：此运算符检查左侧的值是否大于或等于右侧的值。
- **小于（`<`）运算符**：此运算符检查左侧的值是否小于右侧的值。
- **小于等于（`<=`）运算符**：此运算符检查左侧的值是否小于或等于右侧的值。

## 一元运算符

- **一元加法运算符**：此运算符将其操作数转换为数字。如果操作数已经是数字，则保持不变。
```js
const str = '42';
const num = +str;

console.log(num); // 42
console.log(typeof num); // number
```

- **一元负号（`-`）运算符**：此运算符对其操作数取反。
```js
const num = 4;
console.log(-num); // -4
```

- **逻辑非（`!`）运算符**：此运算符翻转其操作数的布尔值。因此，如果操作数为`true`，则变为`false`，如果为`false`，则变为`true`。

## 位运算符

- **按位与（`&`）运算符**：此运算符在两个操作数的对应位都为1的每个位位置返回1。
- **按位与赋值（`&=`）运算符**：此运算符对指定数字执行`按位与`运算并将结果重新赋给变量。
- **按位或（`|`）运算符**：此运算符在任一或两个操作数的对应位为1的每个位位置返回1。
- **按位或赋值（`|=`）运算符**：此运算符对指定数字执行`按位或`运算并将结果重新赋给变量。
- **按位异或（`^`）运算符**：此运算符在两个操作数中只有一个（但不是两个）的对应位为1的每个位位置返回1。
- **按位非（`~`）运算符**：此运算符反转数字的二进制表示。
- **左移（`<<`）运算符**：此运算符将所有位向左移动指定的位数。
- **右移（`>>`）运算符**：此运算符将所有位向右移动。

## 条件语句、真值、假值和三元运算符

- **`if/else if/else`**：`if`语句接受一个条件，如果该条件为`真值`则运行代码块。如果条件为`false`，则移动到`else if`块。如果这些条件都不为`true`，则将执行`else`子句。`真值`是在布尔上下文中（如`if`语句）求值为`true`的任何值。`假值`是在布尔上下文中求值为`false`的值。
```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got an B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

- **三元运算符**：此运算符通常用作编写`if else`语句的较短方式。
```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`); // It's a sunny day!
```

## 二元逻辑运算符

- **逻辑与（`&&`）运算符**：此运算符检查两个操作数是否都为`true`。如果都为`true`，则返回第二个值。如果任一操作数为`假值`，则返回`假值`。如果两个操作数都为`假值`，则返回第一个`假值`。
```js
const result = true && 'hello';

console.log(result); // hello
```

- **逻辑或（`||`）运算符**：此运算符检查至少有一个操作数为真值。
- **空值合并（`??`）运算符**：此运算符仅在第一个值为`null`或`undefined`时返回值。
```js
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

## `Math`对象

- **`Math.random()`方法**：此方法生成0（包含）到1（不包含）之间的随机浮点数。这意味着可能的输出可以是0，但永远不会达到1。
- **`Math.max()`方法**：此方法接受一组数字并返回最大值。
- **`Math.min()`方法**：此方法接受一组数字并返回最小值。
- **`Math.ceil()`方法**：此方法将值向上舍入到最近的整数。
- **`Math.floor()`方法**：此方法将值向下舍入到最近的整数。
- **`Math.round()`方法**：此方法将值舍入到最近的整数。
```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

- **`Math.trunc()`方法**：此方法删除数字的小数部分，仅返回整数部分，不进行舍入。
- **`Math.sqrt()`方法**：此方法将返回数字的平方根。
- **`Math.cbrt()`方法**：此方法将返回数字的立方根。
- **`Math.abs()`方法**：此方法将返回数字的绝对值。
- **`Math.pow()`方法**：此方法接受两个数字并将第一个数字提升到第二个数字的幂。

## 常见数字方法

- **`isNaN()`**：`NaN`代表"Not-a-Number"。它是一个特殊值，表示不可表示或未定义的数值结果。`isNaN()`函数属性用于确定值是否为`NaN`。`Number.isNaN()`提供了更可靠的方法来检查`NaN`值，特别是在全局`isNaN()`函数可能导致意外结果的情况下。
```js
console.log(isNaN(NaN));       // true
console.log(isNaN(undefined)); // true
console.log(isNaN({}));        // true

console.log(isNaN(true));      // false
console.log(isNaN(null));      // false
console.log(isNaN(37));        // false


console.log(Number.isNaN(NaN));        // true
console.log(Number.isNaN(Number.NaN)); // true
console.log(Number.isNaN(0 / 0));      // true

console.log(Number.isNaN("NaN"));      // false
console.log(Number.isNaN(undefined));  // false
```

- **`parseFloat()`方法**：此方法解析字符串参数并返回浮点数。它设计用于从字符串开头提取数字，即使字符串后面包含非数字字符。
- **`parseInt()`方法**：此方法解析字符串参数并返回整数。`parseInt()`在遇到第一个非数字字符时停止解析。对于浮点数，它仅返回整数部分。如果在字符串开头找不到有效的整数，则返回`NaN`。
- **`toFixed()`方法**：此方法在数字上调用，并接受一个可选参数，即小数点后出现的位数。它返回具有指定小数位数的数字的字符串表示。
