## 对象基础

- **定义**：对象是由属性组成的数据结构。属性由键和值组成。要从对象中访问数据，可以使用点表示法或括号表示法。

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person["name"]); // Alice
```

要设置现有对象的属性，可以将点表示法或括号表示法与赋值运算符一起使用。

```js
const person = {
  name: "Alice",
  age: 30
};

person.job = "Engineer"
person["hobby"] = "Knitting"
console.log(person);  // {name: 'Alice', age: 30, job: 'Engineer', hobby: 'Knitting'}
```

## 从对象中删除属性

- **`delete`运算符**：此运算符用于从对象中删除属性。

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

## 检查对象是否有属性

- **`hasOwnProperty()`方法**：此方法返回一个布尔值，指示对象是否将指定属性作为自己的属性。

```js
const person = {
  name: "Alice",
  age: 30
};

console.log(person.hasOwnProperty("name")); // true
console.log(person.hasOwnProperty("job")); // false
```

- **`in`运算符**：如果属性存在于对象中，此运算符将返回`true`。

```js
const person = {
  name: "Bob",
  age: 25
};

console.log("name" in person);  // true
```

## 访问嵌套对象中的属性

- **访问数据**：访问嵌套对象中的属性涉及使用点表示法或括号表示法，就像访问简单对象中的属性一样。但是，你需要链接这些访问器以深入嵌套结构。

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

## 原始数据类型和非原始数据类型

- **原始数据类型**：这些数据类型包括数字、字符串、布尔值、`null`、`undefined`和符号。这些类型被称为"原始"类型，因为它们代表单个值而不是对象。原始值是不可变的，这意味着一旦创建，它们的值就不能改变。
- **非原始数据类型**：在JavaScript中，这些是对象，包括常规对象、数组和函数。与原始类型不同，非原始类型可以保存多个值作为属性或元素。

## 对象方法

- **定义**：对象方法是与对象关联的函数。它们被定义为对象的属性，并且可以访问和操作对象的数据。方法中的`this`关键字指向对象本身，从而可以访问其属性。

```js
const person = {
  name: "Bob",
  age: 30,
  sayHello: function() {
    return "Hello, my name is " + this.name;
  }
};

console.log(person.sayHello()); // "Hello, my name is Bob"
```

## 对象构造函数

- **定义**：在JavaScript中，构造函数是一种特殊类型的函数，用于创建和初始化对象。它通过`new`关键字调用，并可以初始化新创建对象的属性和方法。`Object()`构造函数创建一个新的空对象。

```js
new Object()
```

## 使用可选链运算符（`?.`）

- **定义**：此运算符让你可以安全地访问对象属性或调用方法，而无需担心它们是否存在。

```js
const user = {
  name: "John",
  profile: {
    email: "john@example.com",
    address: {
      street: "123 Main St",
      city: "Somewhere"
    }
  }
};

console.log(user.profile?.address?.street); // "123 Main St"
console.log(user.profile?.phone?.number);   // undefined
```

## 对象解构

- **定义**：对象解构允许你以更简洁和可读的方式从对象中提取值并将其分配给变量。

```js
const person = { name: "Alice", age: 30, city: "New York" };

const { name, age } = person;

console.log(name); // Alice
console.log(age);  // 30
```

## 使用JSON

- **定义**：JSON代表JavaScript对象表示法。它是一种轻量级的、基于文本的数据格式，通常用于在服务器和Web应用程序之间交换数据。

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

- **`JSON.stringify()`**：此方法用于将JavaScript对象转换为JSON字符串。当你想要以易于在系统之间共享或传输的格式存储或传输数据时，这很有用。

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString); // '{"name":"John","age":30,"isAdmin":true}'
```

- **`JSON.parse()`**：此方法将JSON字符串转换回JavaScript对象。当你从Web服务器或localStorage检索JSON数据并且需要在应用程序中操作数据时，这很有用。

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);

// result: { name: 'John', age: 30, isAdmin: true }
console.log(userObject);
```
