## 字符串基础

- **定义**：字符串是用单引号、双引号或反引号括起来的字符序列。字符串是原始数据类型，它们是不可变的。不可变性意味着一旦创建了字符串，就无法更改。
- **从字符串中访问字符**：要从字符串中访问字符，你可以使用方括号表示法并传入索引号。索引是字符串中字符的位置，它是从零开始的。

```js
const developer = "Jessica";
developer[0] // J
```

- **`\n`（换行符）**：你可以通过使用`\n`换行符在字符串中创建换行。

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

- **转义字符串**：你可以通过在引号前放置反斜杠（`\`）来转义字符串中的字符。

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

## 模板字面量（模板字符串）和字符串插值

- **定义**：模板字面量是用反引号（`）定义的。它们允许更简单的字符串操作，包括直接在字符串中嵌入变量，这一特性称为字符串插值。

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; // "Hello, Jessica!"
```

## ASCII、`charCodeAt()`方法和`fromCharCode()`方法

- **ASCII**：ASCII，全称为美国信息交换标准代码，是计算机中用于表示文本的字符编码标准。它为每个字符分配一个数值，这是机器普遍认可的。
- **`charCodeAt()`方法**：此方法在字符串上调用，返回指定索引处字符的ASCII码。

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

- **`fromCharCode()`方法**：此方法将ASCII码转换为其对应的字符。

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

## 其他常用字符串方法

- **`indexOf`方法**：此方法用于在字符串中搜索子字符串。如果找到子字符串，`indexOf`返回该子字符串第一次出现的索引（或位置）。如果未找到子字符串，`indexOf`返回-1，表示搜索失败。

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```

- **`includes()`方法**：此方法用于检查字符串是否包含特定子字符串。如果在字符串中找到子字符串，则该方法返回true。否则，它返回false。

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

- **`slice()`方法**：此方法提取字符串的一部分并返回新字符串，而不修改原始字符串。它接受两个参数：起始索引和可选的结束索引。

```js
const text = "MasterPuti";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

- **`toUpperCase()`方法**：此方法将所有字符转换为大写字母，并返回所有大写字符的新字符串。

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

- **`toLowerCase()`方法**：此方法将字符串中的所有字符转换为小写。

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

- **`replace()`方法**：此方法允许你在字符串中查找指定值（如单词或字符）并将其替换为另一个值。该方法返回带有替换内容的新字符串，并保持原始字符串不变，因为JavaScript字符串是不可变的。

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

- **`repeat()`方法**：此方法用于将字符串重复指定次数。

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

- **`trim()`方法**：此方法用于从字符串的开头和结尾删除空格。

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

- **`trimStart()`方法**：此方法从字符串的开头（或"开始"）删除空格。

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

- **`trimEnd()`方法**：此方法从字符串的末尾删除空格。

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

- **`prompt()`方法**：`window`的此方法用于通过对话框形式从用户那里获取信息。此方法接受两个参数。第一个参数是将出现在对话框中的消息，通常提示用户输入信息。第二个参数是可选的默认值，它将初始填充输入字段。

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
