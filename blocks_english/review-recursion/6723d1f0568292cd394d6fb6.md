# --description--

- 递归是一种编程概念，它允许你重复调用函数直到达到基础情况。

以下是计算数字阶乘的递归函数示例：

```js
function findFactorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * findFactorial(n - 1);
}
```

在上面的示例中，`findFactorial`函数被递归调用直到`n`达到`0`。当`n`为`0`时，达到基础情况，函数返回`1`。然后函数返回`n`和对`findFactorial(n - 1)`的递归调用结果的乘积。

- 递归允许你处理具有未知深度的事物，例如深度嵌套的对象/数组，或文件树。
- 调用栈用于跟踪递归函数中的函数调用。每次调用函数时，它都会被添加到调用栈中。当达到基础情况时，函数调用会从栈中移除。
- 你应该仔细定义基础情况，因为无限调用它可能导致代码崩溃。这是因为递归会不断堆积越来越多的函数调用，直到系统内存耗尽。
- 递归在解决数学问题如阶乘和斐波那契数列、遍历树和图、生成排列和组合等方面都有其用途。
