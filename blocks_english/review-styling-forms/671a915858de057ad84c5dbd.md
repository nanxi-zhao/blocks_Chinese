## 表单输入样式的最佳实践

- **输入样式**: 与所有文本元素一样，你需要确保应用于文本输入的样式是可访问的。这意味着字体需要适当大小，颜色需要与背景有足够的对比度。输入元素也是可聚焦的。在编辑样式时，你应该注意在元素获得焦点时保留明显的指示器，比如粗边框。

## 为输入使用 `appearance: none`

- **`appearance: none`**: 浏览器会对许多元素应用默认样式。`appearance: none` CSS 属性让你完全控制样式，但有一些注意事项。在为输入元素构建自定义样式时，你需要确保焦点和错误指示器仍然存在。

## 样式化 `datetime-local` 和 `color` 属性的常见问题

- **常见问题**: 这些特殊类型的输入依赖复杂的伪元素来创建日期和颜色选择器等组件。这给样式化这些输入带来了重大挑战。一个挑战是默认样式完全依赖于浏览器，所以你编写的让选择器看起来符合你意图的 CSS 在另一个浏览器上可能完全不同。
