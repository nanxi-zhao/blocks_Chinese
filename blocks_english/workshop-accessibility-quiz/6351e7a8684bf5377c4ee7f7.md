# --description--

即使 `label` 中已经嵌套了对应的 `input`，显式通过 `for` 与 `id` 绑定仍是更佳实践，可提高可访问性与可维护性。

现在请给四个单选题的 `label` 都添加 `for` 属性，与各自的单选 `input` 的 `id` 完全匹配。继续，语义关联更清晰！
