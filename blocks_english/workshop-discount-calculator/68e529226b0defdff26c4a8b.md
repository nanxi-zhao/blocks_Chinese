# --description--

现在你将开始构建折扣系统接口。Python 提供了一种定义抽象类的方法，这些抽象类可以作为其他类的蓝图。这是通过 `abc` 模块的 `ABC` 类（抽象基类）来完成的。

在你的代码顶部，添加以下导入语句：

```py
from abc import ABC, abstractmethod
```

这将导入 `ABC` 类和 `abstractmethod` 装饰器，你将使用它们来定义必须由继承自你的抽象类的任何类实现的方法。
