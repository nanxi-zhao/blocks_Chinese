# --description--

现在是时候为 `button` 元素应用一些样式了。

首先为你的 `button` 元素添加一个 class 属性。接下来，应用正确的工具类来设置红色文本颜色，深浅度设置为 `700`。

然后，在你的 `button` 开始标签中，使用正确的工具类将 `font-size` 设置为 `2xl`，它代表 `1.5rem` 或（`24px`）。

# --hints--

你的 `button` 元素应该有一个 class 属性。

```js
const btnEl = document.querySelector("button");
assert.isTrue(btnEl?.hasAttribute("class"));
```

你的 `button` 开始标签应该有 `text-red-700` 工具类。

```js
const btnEl = document.querySelector("button");
assert.isTrue(btnEl?.classList.contains("text-red-700"));
```

你的 `button` 开始标签应该有 `text-2xl` 工具类。

```js
const btnEl = document.querySelector("button");
assert.isTrue(btnEl.classList.contains("text-2xl"));
```
