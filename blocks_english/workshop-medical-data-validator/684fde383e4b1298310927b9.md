# --description--

在正则表达式中，量词用于指定某个字符可以重复的次数。例如 `+` 表示匹配前一个字符一次或多次：

```py
import re

book = "Fahrenheit 451"
print(re.search('\\d', book))
# <re.Match object; span=(11, 12), match='4'>

print(re.search('\\d+', book))
# <re.Match object; span=(11, 14), match='451'>
```

因此请在你的正则模式后追加 `+` 量词，以匹配一个或多个数字。
